　　在自下方的深渊中飞出之时，叶尘便已经暗暗运转逆生百转神功，以他如今的修为和肉身强度，瞬间便运转到了第七重，几乎可以说毫不费力。
<br/>　　陡然拔升了数十倍的真元，全部灌入灭世魔刀之内！
<br/>　　之前他一时大意，险些着了对方的道，如今自然不能留手了。
<br/>　　啨~~~
<br/>　　灭世魔刀似乎也感应到主人的变化，顿时发出一道欢快的鸣声。
<br/>　　轰！！
<br/>　　这一刻，
<br/>　　一道可怕到极点的毁灭气息，瞬间几乎将整个无名星球覆盖，已经躲到数千里之外，三大家族的那些人，感受到那股令他们心悸的气息，一个个顿时全都吓得浑身不停的颤抖，一些修为弱的，甚至早已经跪倒在了地上，差点要晕厥过去了。
<br/>　　至于那首当其冲的紫袍老者，更是两眼瞪得老大，眼睁睁的看着九天之上，那道淡灰色的刀芒，犹如天堑般，朝自己劈落下来！
<br/>　　“不！！！”
<br/>　　紫袍老者顿时大吼了一声，来不及多想，当即挺起手中的金色巨枪，迎了上去。
<br/>　　可惜，
<br/>　　咔嚓！
<br/>　　几乎连一秒钟都没有撑到，金色巨枪瞬间破碎，化成了一堆粉末！
<br/>　　“噗！！”
<br/>　　法宝被摧毁，紫袍老者顿时狂喷了一大口鲜血，整个人狠狠朝下方坠落。
<br/>　　而那灰色刀芒，则已经如影随形般，继续朝他劈落下来。
<br/>　　“千重盾！”
<br/>　　紫袍老者含血大吼了一声，再次将盾形法宝祭了出去。
<br/>　　然而，
<br/>　　“咔嚓！”
<br/>　　依旧连一秒钟都没有撑住，便再次破碎裂开，化成了一缕粉末。
<br/>　　“不！！！”
<br/>　　紫袍老者见状，顿时再次发出了一声低吼，当即一拍胳膊上的乾坤镯，
<br/>　　嗖！嗖！嗖！嗖！
<br/>　　好家伙，竟然一口气祭出了七八件法宝，甚至最后连一些低级的灵器级别的法宝，也全都甩了出来。
<br/>　　可惜，这些法宝加在一起，却依旧没能撑得住一秒，在那灰色的刀芒之下，犹如豆腐渣一般，尽数破碎开来。
<br/>　　“不要！不要杀我！求求你……”
<br/>　　眼看着，将自己的全部法宝都丢了出去，依旧未能挡住这一击，紫袍老者顿时彻底绝望，当即也不顾的脸面，开始大声求饶。
<br/>　　可惜，叶尘根本理都没理，
<br/>　　灰色刀芒直接从紫袍老者的头顶穿过，瞬间贯穿他全身上下。
<br/>　　“彭！”
<br/>　　片刻之后，
<br/>　　紫袍老者犹如一尊石雕般，瞬间化成了粉碎，连同神魂和肉身，彻底消失在这世间。
<br/>　　……
<br/>　　足足过去了许久，
<br/>　　远处那三大家族的众人，方才从之前的震撼中，缓缓回过神来，
<br/>　　“无极宗的那位特使大人，竟然就这么死了？”
<br/>　　“这小……这位姓叶的前辈，到底是何方神圣？先是羞辱了擎苍宗的韩月公子，如今又杀了无极宗的这么多强者，他难道就一点不怕两大仙门的报复吗？”
<br/>　　“以他的年纪和修为，能够做出如此壮举，应该足以在整个修真界的天骄榜中，跻身前十，不！至少也是前五的存在！此等绝世天骄，又岂会没有自己的背景？”
<br/>　　“不错！此人背后的势力，恐怕至少也是八大仙门级别的存在！”
<br/>　　……
<br/>　　在众人的议论声中，叶尘踏空而行，转眼之间，便已经来到冥沧荨的跟前，
<br/>　　“傻丫头，怎么也不知道躲开？没事吧？”
<br/>　　原来，冥沧荨之前因为担心叶尘的安危，所以一时间竟忘记了躲避，被之前两人的战斗余波所殃及，受了不轻的伤势。
<br/>　　叶尘一边说着，自己伸手握住冥沧荨的小手，动用万古长青珠，调动一丝造化之力，渡入到了冥沧荨的体内。
<br/>　　“我，我没事……”
<br/>　　冥沧荨感受到叶尘手掌上的温度，顿时俏脸羞得通红，一颗芳心忍不住一阵乱颤，下意识的想要将玉手，从叶尘手掌中抽出，可是却又无比喜欢那种被温暖包裹的感觉，一时间内心无比纠结起来。
<br/>　　可惜这种感觉并没有持续太久，叶尘将造化之气渡入冥沧荨体内，替她修复了伤势之后，便将手掌收回。
<br/>　　他并没有想到，这么一个细微的举动，竟然让冥沧荨对他的误会又加深了……
<br/>　　“走吧！我送你回家！”
<br/>　　叶尘丝毫没有察觉到，冥沧荨看向他的眼神，已经开始发生变化，随口又说了一句。
<br/>　　按照他之前的推测，冥沧荨的哥哥冥沧山回家之后，肯定会恶人先告状，若是让冥沧荨自己一个人回去，说不定是要吃亏的，所以他才提出跟她一起回去。
<br/>　　而冥沧荨，自然不知道叶尘的想法，此时的她，还沉溺在之前的甜蜜之中，听到叶尘这话，先是微微一愣，继而芳心更加狂颤起来，
<br/>　　“他要送我回家？这是什么意思？难道是想要向父亲提亲不成？这也……太直接了吧？我是该答应他呢，还是……”
<br/>　　叶尘要是知道冥沧荨此时的想法，估计非得吐血不可。
<br/>　　“还愣着干嘛？前面带路啊！”
<br/>　　叶尘见冥沧荨愣在那里，小脸红扑扑的低着头，不知道在想些什么，不由得开口催促了一句。
<br/>　　“哦哦，好！”
<br/>　　冥沧荨当即应了一声，连忙走到叶尘前面，开始带路，眼角眉梢的笑意，已经完全无法掩盖。
<br/>　　……
<br/>　　南斗星，玄冥城，
<br/>　　这里是南斗星域的核心地带，同时也是南斗星域四大家族之一，冥家的根基所在。
<br/>　　南斗星域，有雷、宋、冥、田四大家族，分别盘踞南斗星的一方，四家呈鼎足之势，已经超过了万年，彼此虽然互相争斗厮杀，却谁也没能将会灭掉，最近千年以来，一直相安无事，还算比较和平。
<br/>　　“叶大哥，这里就是我家，玄冥城，我先带你进去四处逛逛吧！”
<br/>　　冥沧荨指着前方的城门，俏脸上洋溢着一抹兴奋之色，还隐隐透着一抹娇羞。
<br/>　　然而，
<br/>　　等两人来到城门口，
<br/>　　“快看！是大小姐！”
<br/>　　哗啦啦！
<br/>　　玄冥城门口的那些金甲侍卫，顿时冲了出来，竟然将冥沧荨和叶尘两人包围了起来。
<br/>　　冥沧荨见状，顿时又惊又怒，
<br/>　　“你们想干什么？想造反吗？”
<br/>　　为首的那名金甲侍卫，拱手道：
<br/>　　“大小姐，实在抱歉！家主有令，任何人一旦见到大小姐，务必就地拿下！”

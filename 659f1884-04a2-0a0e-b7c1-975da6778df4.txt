
王令歌跟王惊蛰约的地方就在白云山上，一处在林间里的幽静场所，四层高的红色砖瓦建筑，周围是几栋独栋的别墅，正院大门紧闭，门卫室里站着两个虎视眈眈的黑西装，门旁也没有挂任何牌子，让人一看完全都不知道这是啥地方。<br/>　　<br/>　　王惊蛰和林汶骐是打车过来的，王令歌告诉他俩到了之后就先在门口等他一下，他到了后在一起进去。<br/>　　<br/>　　“啪”两人在门口站着点了根烟，林汶骐瞅着紧闭的大门，跟王惊蛰说道：“这地方一看，就让我想到了两个词，纸醉金迷和闲人免进，此处啊肯定是现在很流行的那种私人制的会员场所，你自己单独过来可能都进不去，喏，看见门口那两条黑背了么，你在往前两步，他们肯定就出来咬你了”<br/>　　<br/>　　王惊蛰笑道：“你嘴可真损，人家那是工作”<br/>　　<br/>　　林汶骐撇嘴说道：“工作是工作，但这帮人啊我都不是说，就是狗眼看人低，你开一五十万左右的车过来，人家都不带爱搭理你的，他们的眼界起步都是百万以上的”<br/>　　<br/>　　这时大门右边的林间公路上，开过来一辆大街上随处可见的黑色迈腾，车还没到门口呢，门卫里的两个黑西装见状，就立马全都走了出来，同时按了下手里的遥控器，大门朝着两边开启了。<br/>　　<br/>　　迈腾开到门口，车窗放下后，王令歌探出脑袋笑道：“上车吧，等半天了啊？”<br/>　　<br/>　　“没，也是刚到”<br/>　　<br/>　　林汶骐和王惊蛰坐进车里，两个门卫在旁边两脚并拢立正，点头说道：“您好，王先生”<br/>　　<br/>　　王令歌笑着回应了一声，顺着大门开了进去，王惊蛰斜了着眼睛说道：“他们工作态度还是可以的么”<br/>　　<br/>　　林汶骐说道：“没看出来啊，他们都认识令歌，你换一个人试试”<br/>　　<br/>　　王令歌问道：“你俩说什么呢？”<br/>　　<br/>　　“老林说，这地方要是开一辆五十万的车，门口的两人都不一定正眼看，从你这我看也不是这么回事啊”<br/>　　<br/>　　王令歌哈哈一笑，拍着方向盘说道：“他说的没错，这里的人都认识我，你要再换一辆车过来，他们确实就不是这态度了”<br/>　　<br/>　　林汶骐呲牙笑了，说道：“你这王家大公子也够低调的了哈”<br/>　　<br/>　　王令歌摇头说道：“车对于我来说就是一代步的，品质和价格我都不在乎，我开车就两点，空调好用能打着火就行了，其他都无所谓”<br/>　　<br/>　　迈腾开到那栋四层楼前的时候，停车场里停了有七八辆车，粗略扫了几眼，确实都是奔驰s那一级别的，其中还有几辆顶级的幻影和欧陆，这辆迈腾一停到这些车旁边，就显得极其眨眼了。<br/>　　<br/>　　门口一串银色西装的中年看见迈腾过来后，一路快步的赶了过来，在王令歌停好车熄了火的时候，正好就把车门给拉开了，时间拿捏的相当准确了。<br/>　　<br/>　　“令歌，房间都安排好了，还是梅兰间，你点的几个菜也准备完了”<br/>　　<br/>　　“谢了，许叔”王令歌说话的时候伸出手，这经理模样的人从身上拿出两张卡递了过来，他接到手里后随手就给了王惊蛰和林汶骐说道：“以后我要是没在羊城，你俩想过来消遣，带卡过来就行了”<br/>　　<br/>　　王惊蛰低头扫了一眼，也没推辞，就跟林汶骐都接了过来：“我这一看，你在这里的力度，相当可以了啊”<br/>　　<br/>　　王令歌眨了眨眼睛，低调的说道：“自家买卖，你随便点就行了”<br/>　　<br/>　　王惊蛰“哦”了一声，说道：“我说的么，这里的布置还挺有内涵的，我就估计跟你们家应该关系匪浅”<br/>　　<br/>　　“我爷爷亲自布下的，以前打算留给自己当养老山庄用的，后来觉得一个人住在这里空空荡荡的和外界来往也不太方便，就转做他用了”王令歌解释了一句，然后跟那经理说道：“许叔，他俩是我兄弟，以后过来的时候你看着点哈”<br/>　　<br/>　　许叔顿时望了他俩一眼，点头说道：“记住了，令歌”<br/>　　<br/>　　王惊蛰和林汶骐可能都不知道，王令歌往这里带人是很有说道的，平常他几乎很少主动带什么人来这里应酬，带人过来后顶多就说一声这是我朋友，让他主动介绍成兄弟的，这么多年也就向征和曹清道有这待遇了，所以许叔一听他刚才的话，就对王惊蛰和林汶骐的定位心里有数了。<br/>　　<br/>　　几人往楼里走，穿过大厅来到电梯门前，等了会后电梯下来从里面走出两个长得挺漂亮的女人，手挽着手说笑着往面走，王惊蛰扭着脑袋看着她们的背影，直到人走没了，他才说道：“有点眼熟呢看着？”<br/>　　<br/>　　林汶骐说道：“好像电视里演过娘娘的那俩？”<br/>　　<br/>　　王令歌拍了下王惊蛰肩膀，轻声说道：“你要是嫌咱们三吃饭太单调，她们人还没有走远，可以上来一起吃”<br/>　　<br/>　　王惊蛰惊愕的说道：“这也行”<br/>　　<br/>　　王令歌矜持的说道：“吃完了，喝好了，你要是不想走的话，后面也有房间”<br/>　　<br/>　　王惊蛰瞬间一呆，顿时就悟了，连连说道：“不用，不用，家里有，你问老林吧”<br/>　　<br/>　　林汶骐摆手说道：“别，我也不好这一口”<br/>　　<br/>　　片刻后，三人上了楼上一间挺雅致的包房里，进来的时候几个小菜和两瓶酒已经摆好了。<br/>　　<br/>　　“上次大排档吃的太匆忙了，这次都闲着呢，尽兴，必须尽兴啊”王令歌让他俩坐下，搓了搓手说道：“今个不喝趴下两，谁也不能走，喝多了直接夜宿在这，想要什么活动都能准备”<br/>　　<br/>　　“不是，我看你这好像还是带着事来的呢，拳脚上咱俩没较量上，酒场上还得要分个高低啊？”<br/>　　<br/>　　“主要是我看咱们岁数都差不多，既然都是兄弟，那不得分出个大小王来么，拳脚无眼容易擦枪走火，那就在酒上找找吧”<br/>　　<br/>　　“我爹好喝酒，小的时候就没少培养我，差点都把我泡在酒缸里养了，我酒量应该还行”王惊蛰低调的蔑视着。<br/>　　<br/>　　王令歌哎呀了一声，说道：“咱俩经历都差不多啊，我年纪小那时候，跟他说话没两句我就醉了，他天天喷出的口气都是酒精的，我也是被熏陶着成长起来的”<br/>　　<br/>　　王惊蛰流着汗的说道：“啥也别说了，爹都不靠谱，咱俩能不缺胳膊不缺腿的活到现在，也真挺不容易的了”<br/>　　<br/>　　“哈哈，来吧，来吧，先喝两杯开个胃……”<br/>　　<br/>　　一场酣畅淋漓的酒局，三人喝了四个多小时，喝到最后眼神都有点迷离了，清醒度降到了六成以下，说话的时候舌头都有点打结了。<br/>　　<br/>　　“唉，那个什么惊蛰啊，我过几天得要去大澳一趟，你有事没？没事，跟我逛逛去啊”王令歌胳膊搭在王惊蛰的肩膀上，喷着浓郁的酒气说道。<br/>　　<br/>　　“去那干嘛啊？”<br/>　　<br/>　　“刘运辉死了，我需要出面过去那边交代一下，刘运辉家里在大澳还是可以的，我们王府不至于怕他什么，不过为了王家的声名呢，有些善后的工作还是要做一下的，口碑，明白不？”<br/>　　<br/>　　王惊蛰斜了着眼睛说道：“你让我去，怎么的，还想让我负荆请罪嘛？都是兄弟，你说话好使，我肯定不能让你为难”<br/>　　<br/>　　“你给我滚蛋，你这是打我脸呢，你知道么？”王令歌瞪着眼珠子，拍着自己的脸蛋说道：“我是带你去那边体验下大澳的风情，那里的赌场还是不错的，还有就是……唉，你不是有个姐么，叫出来一起过去玩呗”<br/>　　<br/>　　“唰”王惊蛰往后仰了下身子，呵呵的笑了：“醉翁之意不在酒系不系，我拿你当哥们，你却想泡我姐？”<br/>　　<br/>　　王令歌尴尬的说道：“你别扯了，我都没见过王冬至，泡啥啊，就是出去走走，看看，无关情爱”<br/>　　<br/>　　“哦，王冬至”王惊蛰意味深长的说道：“我什么时候和你说我姐叫王冬至了，没有吧？”<br/>　　<br/>　　(=)

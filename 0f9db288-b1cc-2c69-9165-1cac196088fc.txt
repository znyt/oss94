　　“叮！第二次任务开始发布，修炼任务。”
<br/>　　“修炼任务：活着走出北斗荒古禁地，拜入一方修仙门派，踏入修仙之途！”
<br/>　　“任务要求：三个月内，成功开辟苦海，踏入修炼之路！”
<br/>　　“任务奖励：成功，三千价值点，失败，扣除一千价值点。”
<br/>　　“任务提醒：半小时后，降临荒古禁地。”
<br/>　　就在众人满心欢喜的耍弄着此次兑换到的各种技能时，那冷漠的声音再次响起，将众人从欢喜中挣脱出来。
<br/>　　因为一听到任务，他们便会想起几个小时之前发生的事，此刻，众人脸上再次爬满忧虑，既有对自己的生命忧虑，又有对未来的担忧。荒古禁地，一听名字便知道不是什么好地方。
<br/>　　半个小时候，随着“轰隆”一道声响，铜棺再次到达目的地。
<br/>　　铜棺这一次并没有落地，而是直接光芒一闪，垂下一道光幕，众人一阵眩晕，便从光幕之中落了下来。之后虚空一阵波动，铜棺便隐匿于虚空之中，消失不见了。
<br/>　　其实，它并没有消失，依旧在荒古禁地上方，暂时镇压这荒之力，只不过，普通人发现不了而已。
<br/>　　“光明！”
<br/>　　“我终于看到了光明！”
<br/>　　“是那么熟悉的光明世界！”
<br/>　　“这里是哪里？难道就是荒古禁地吗？”
<br/>　　众人心中带着紧张而又激动的心情，打量着周边的一切。
<br/>　　清新的空气扑面而来，甚至还带着泥土的清香与花草的芬芳，自然的气息洋溢在周围的一切，再也没有之前荧惑古星上的那种昏暗与满目血色的苍凉。
<br/>　　远处是连绵起伏的修理山峰，佳木葱茏。山顶近处是奇形怪状的岩石与苍劲的古木，更有水桶粗细的老藤如虬龙般盘绕，更有如茵的绿草与芬芳的野花，充满活力与生机。
<br/>　　“这里难道就是北斗的荒古禁地？怎么一点也不像是禁地？”有人小心问道。
<br/>　　“荒古禁……”叶梵三人来到一块石碑前，这石碑只有半截，而且还缠绕着藤蔓，有人工打磨过的迹象。可惜，只能看出三个字。
<br/>　　“这里果然是荒古禁地！”夜邪感慨道。
<br/>　　同时，他心里也极为清楚，此刻怕是早有几双眼睛在深渊下盯着自己等人了。荒古禁地平时许多大人物都不敢靠近，有血肉的生物根本不能长存，连古之圣贤都难以承受此地的荒力侵蚀，几乎没人可以在此长存。
<br/>　　自古至今，也不知道埋葬了多少绝代高手，连鼎盛的圣地来此，也照样被灭了，成了劫灰。
<br/>　　他知道，自己等人不能在此地长留，如若不然，便会被荒力带走生命，如不是此时有铜棺的镇压，恐怕他们早已化为飞灰了。
<br/>　　“此地恐怕不是什么净土，你们注意没有，附近似乎没有什么鸟兽，太安静了！”有细心之人感觉到了不对劲的地方。
<br/>　　“嗯？”此话一出，几人顿时心中一震，太安静了，又是并不是一种好的兆头。
<br/>　　这里有花，有草，有藤，有树，表面看起来生机勃勃，可却仔细一看却十分异常，周围竟然没有任何动物。
<br/>　　偌大的山体，竟然听不见任何鸟叫兽吼，看不到任何蚂蚁活动的痕迹，近乎死寂，这就十分可怕了。
<br/>　　“谁说没有动物了，你们看天上那是什么？”叶梵在这时抬头看天，发现了什么，手指遥遥指向天际。
<br/>　　远处，似乎有一只雄鹰在盘旋，非常特别，因为它浑身竟然是金灿灿的，纵然相隔很远，依然可以看见那种绚丽，似黄金铸成，通体金光闪烁。
<br/>　　这时，金色雄鹰向这片区域飞了一段距离，越发的清晰了，而后他突然俯冲向一片山区，片刻后又冲天而起，此刻它利爪上抓着一只大象，向着远处飞去。
<br/>　　“不是吧，我……我没有看错吧？”看到这一幕后，一向大大咧咧的庞博当场结巴起来，其他人更不用说了，几人也都近乎石化。
<br/>　　“那恐怕是金翅大鹏鸟吧。”夜邪开口说道。
<br/>　　“好像是。”叶梵愣愣地点了点头，说道。
<br/>　　“我们到底来到一个怎样的世界？”
<br/>　　“大家都别愣着了，此处很可能会有危险，我们赶紧在四周找找出路，找到后之后赶紧走！”夜邪此刻开口说道。
<br/>　　“不错，此地不宜久留！”此话一出，其他人纷纷赞同，大家都觉得此地有些诡异，不是久留之地。
<br/>　　之后，夜邪带着庞博与叶梵，有意无意地向着山顶走去。
<br/>　　来到几十米外的山顶，夜邪三人便见到几株水桶般粗细的老藤环绕着一块空地，那里有一个一米见方的泉池，汩汩而流，似甘露神泉一般。
<br/>　　在泉池边上还仗着十几株半米多高的小树，叶片宽大，青翠欲滴，形似人的手掌，好像有几个多臂的小人站在那里。每株小树定断都挂着一个红彤彤的果实，形似樱桃，但足有鸡卵那么大。
<br/>　　“好东西！”远远就能够闻道浓郁的果香，三人赶紧走到上前，也不管有没有毒，直接抓起来便吃。
<br/>　　三人每人吃了两个，还剩下三个，留给叶梵的同学。
<br/>　　“你最好是再吃一个，那样对你有好处。”夜邪劝慰道，他自是知道，叶梵是荒古圣体，需要的资源是别人的十倍，百倍，而荒古禁地的不死药，便是助他开启修行之路的良药，若是没有了它们，真不知道叶梵要到何时才能开辟苦海。
<br/>　　叶梵看了看夜邪，见他认真的模样，也便听了他的话，再吃了一个果子。
<br/>　　“你是不是知道这是什么？”叶梵问道。
<br/>　　“不错，这是不死药之一，其内蕴含着神秘的力量，而你的体质特殊，需要它来助你，才可节省许多时间，踏上修行之路。”夜邪也没有隐瞒，直接说道。
<br/>　　“那不是说，这是好东西了！”庞博顿时双目放光。
<br/>　　“对，这是天底下最好的东西之一，你们以后千万别到处乱说，否则会惹祸的。”夜邪叮嘱道。
<br/>　　“那是，谁有好东西会乱说啊，闷声发大财才是王道。”庞博拍拍胸口说道。
<br/>　　“这泉水也是好东西！”夜邪闻了闻汩汩而流的泉水，充满芬芳的香气，连忙俯身下去饮了起来。
<br/>　　叶梵与庞博见此，也毫不客气的喝了起来。
<br/>　　“这神泉也极为不简单，蕴含极大的生命力，关键时刻能救命，赶紧多收些。”
<br/>　　直到三人喝不动了，夜邪才开口说道。说着，他在储物戒指中开辟出一方空间，将这生命泉水受到里面。这东西有强大的生命力，能活死人，肉白骨，这种好东西遇到了没理由放弃。
<br/>　　“啊？我们手中只有这个啊，那不是亏了？”两人见此，拿出矿泉水瓶子，装了满满的两瓶，有些不满道。
<br/>　　“走，别太贪心！”一装完，夜邪便拉着两人向外走去，再不走，恐怕就麻烦了。
<br/>　　至于其他的不死神药，夜邪虽然眼红，但却也知道那不是自己该得到的，自己能得到两颗果实，已经在占了叶梵的便宜了。跟着主角走，才有好处可以拿。
<br/>　　几人会合后，叶梵如原著般将两颗果实给了柳依依与张子陵，之后十人便在夜邪的带领下向着山外走去。
<br/>　　有着夜邪的带路，也没有原著中的恩怨，省去了很多麻烦，节省了不少时间。很快，他们便再次看到一截石碑，上面依旧刻着“荒古禁地”四个大字。
<br/>　　几人顿时知道，他们走出了荒古禁地。
<br/>　　这时，众人看到远处高峰上，有一片规模浩大的建筑物，那是一片绵延的殿宇，像是天上的宫阙。
<br/>　　十人依旧不停，向着宫殿的方向追去，一路上遇到很多奇怪的东西，也有一些猛兽，渐渐地，众人感觉到自己的身体在发热，好似有烈火在燃烧，片刻后，就有人晕了过去。
<br/>　　有了第一个，自然也就有第二个，第三个……直至最后一个，最后，十人全部晕倒。荒之力，或者神果之力，在改造着他们的身体。

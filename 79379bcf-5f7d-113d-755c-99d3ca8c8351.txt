　　妙心直勾勾地盯着那些尸骨，有些激动地说道：“果然他们都是死在这里，那说明炼尸缸也就在这附近了……”
<br/>　　说完朝那边斜坡走了过去，想要下去检查那几副白骨。叶少阳一把拉住她。
<br/>　　“这里一共有八条路，我怀疑这也是阵法的一部分，不是随便走的。我们研究一下。”
<br/>　　妙心怔了一下，定睛看去，这八道斜坡就像发散的太阳光，从坑底的中心位置延伸上来，看上去风平浪静的，但是听叶少阳这么一说，妙心也警觉起来，朝深坑的底部看去，最中间的位置，被那两道龙气挡住，看不清中间有什么。
<br/>　　“来都来到这里了，我们下去看看中间有什么，然后再做打算，大家小心一点就是了。”卢晓清说完，拿出一沓符纸，画成隐气符，分给大家一人一张。
<br/>　　拿着卢晓清画的隐气符，叶少阳心中有些打鼓，按说以卢晓清的法力，所画的隐气符能将效用发挥到最大，面对一般邪物都会有效，但是面对眼前的局面，叶少阳还是没有底气。
<br/>　　风平浪静，仿佛没有任何危险，但越是这样反常，叶少阳心中就越是不安，台阶上那几具白骨就是最好的证据。这里绝对不是一个平静的地方。不过也如卢晓清所说，既然来都来了，总不能不下去。
<br/>　　贴上隐气符之后，大伙选择了有尸骨的那条斜坡，走了下去。
<br/>　　一口气走到白骨边上，任何异常都没有发生。妙心蹲下检查那几具白骨，通过各自包里一些残存的东西，确认这几人之中的一个，就是自己的先祖，其余都是他的弟子。
<br/>　　“总算是完成一件事了。”妙心长出了一口气，让大伙在这等着，自己陆续把几具尸体都搬到了最上方的墓道里放着，等着一会再把他们都带出去。
<br/>　　做完这件事，大伙继续往下走，一路下到深坑的底部，这时候总算看到了摆在深坑最中心位置的那个东西，妙心脱口就叫了出来：炼尸缸！
<br/>　　在见到炼尸缸之前，在叶少阳本能地想象中，炼尸缸应该就是一口家里腌咸菜的那种缸，最多大一点，但是真正见到这东西时，叶少阳才知道自己完全想错了：
<br/>　　炼尸缸非常大，至少有两三个人那么高，外形倒是跟普通家用的缸没什么区别，都是口小肚子大，外层是暗红色的，上面刻着一条蟠龙，缸口一圈，刻着一些道家的图腾和符文，想来是加持炼尸缸用的。
<br/>　　周围没有风，但是有一股逼人的寒气，从炼尸缸的方向袭来，被叶少阳等人深切地感觉到，不自觉地都往后退了两步。
<br/>　　“这就是炼尸缸……”叶少阳喃喃说道，法术界传闻中最著名的人造邪器，总算是见着了。
<br/>　　“这炼尸缸，是给皇帝用的，所以上面有蟠龙的纹饰，”妙心说道，“错不了了，就是它。”
<br/>　　大伙静静围观着炼尸缸，过了好一会，梧桐问妙心：“我们现在怎么办，毁了它吗？”
<br/>　　“这……听我爷爷说，炼尸缸是可怕的邪器，几乎是毁不了的，只能镇压，我想封印应该是在缸口，我要检查一下封印，如果有什么问题，争取修复……”
<br/>　　“不是说炼尸缸极难对付吗，怎么一点动静都没有？”卢晓清皱眉问道。
<br/>　　“应该是被封印住了。”妙心绕着炼尸缸走了一圈，没有看到任何封印，只能看到炼尸缸的顶部有一团很粗的红绳，将整个缸口环绕起来。
<br/>　　“封印，应该就是在这里了。”妙心指着缸口的红绳说道，“想要查看封印的情况，只有上去看看了。”
<br/>　　大伙一听她说要上去，都是吃了一惊。
<br/>　　“会有危险吗？”梧桐问。
<br/>　　“当然……我也不知道上面是什么情况，既然来了，总要试试看。”
<br/>　　妙心深吸了一口气，冲梧桐笑道：“我要是下不来，以后你就是法术界第一美人了。”
<br/>　　梧桐弯嘴一笑，“现在也是我。”
<br/>　　“美得你！”
<br/>　　妙心说完，来到炼尸缸的前面，突然就为难了：炼尸缸的外壳是琉璃的，十分光滑，而且中间鼓两头扁，又很高，想爬上去完全没有着力点。
<br/>　　大伙四下寻找能够用来垫脚的东西，可是这个深坑的底部除了这个炼尸缸什么都没有，一时间都有些不知道怎么办。
<br/>　　“我有办法上去。”叶少阳突然开口说道，把勾魂索解下来，一个助跑，就地跃起，将勾魂索朝炼尸缸上面甩过去，顶端的钩子挂在了缸沿上，用力扯了下，很牢固，对妙心使了个眼色。
<br/>　　“多谢。”妙心从叶少阳手中结果勾魂索的另一头，蹬着缸沿，一步步挪了上去。大伙都把法器拿在手中，警惕地望着妙心的行动，随时准备应付可能发生的意外。
<br/>　　妙心顺着勾魂索一直爬到炼尸缸的顶端，低头朝里面观察了一会，一只手抓着勾魂索，腾出一只手，往背包里翻找起来，动作很僵硬，过了一会儿，实在没办法，只好俯下身子，挨个观察大伙。
<br/>　　“你看什么，里面有什么？”梧桐十分好奇地问。
<br/>　　妙心不理她，目光继续在大伙身上转来转去，最终定在叶少阳身上，犹豫了一番，说道：“你上来，抱着我。”
<br/>　　“什么！”叶少阳怀疑自己听错了。
<br/>　　“我需要用两只手作法，你上来抱着我，我好作法。”这也是妙心实在没办法的情况下，想到的主意。
<br/>　　“这……”叶少阳顿时有点为难，看了一眼梧桐，道：“她不行么？”
<br/>　　“我是姑娘，力气不够，你上去吧。”梧桐说道，“这是办正事，你不要太拘泥小节。”
<br/>　　好吧……叶少无奈地挠了挠头，抓着勾魂索，紧贴着妙心爬了上去，与她平行，伸头朝炼尸缸里看去，头顶上空的五道灵光穿过了氤氲，将炼尸缸里的东西大致照亮。
<br/>　　有一个人，躺在炼尸缸的最底部，身上穿着黄澄澄的甲胄，应该是黄金打造的，黄金甲将他全身都包裹起来，连眼睛都没露出来。

　　陆千刃心中还以为，玄北圣者是因为寒月漓来的，毕竟寒月漓的师尊，可是冰漓长老，也许是因为冰漓长老的原因，玄北圣者才会出手的。
<br/>　　只要是放了寒月漓便是，毕竟寒月漓在北邙学院中有着无数老古董看重，杀了她，也是风险极大。
<br/>　　不过楚风眠，只是孤家寡人一个，得到了无数奇遇，就算是杀了，学院也不可能怪罪于他。
<br/>　　而且楚风眠的手中，还有着祖龙至尊剑，这件龙族至宝。
<br/>　　祖龙至尊剑，虽然是在龙族血脉的手中，才能发挥最为全部的力量，但是就算没有龙族血脉的加持，这祖龙至尊剑，也依然是天级灵器的巅峰。
<br/>　　楚风眠足以打碎金罗殿，这祖龙至尊剑的锋利，要发挥了九成的威力。
<br/>　　得到祖龙至尊剑，陆千刃的力量，也会在强大数成。
<br/>　　“两个人，我要一起带走。”
<br/>　　听着陆千刃说完，天空之上的玄北圣者，也缓缓开口道。
<br/>　　这语气，虽然平静，但是其中言语之中的意思，却是让所有人都能听得明白，这句话，没有丝毫一点的商量余地。
<br/>　　“什么？”
<br/>　　陆千刃听到这句话，脸色一下子暴怒起来。
<br/>　　他都已经让步了，没想到玄北圣者，居然是丝毫不给面子。
<br/>　　他们陆家，在北邙学院可所谓的一手遮天，无法无天的存在，在北邙学院中还没有什么人，居然敢不给他们陆家的面子。
<br/>　　“玄北，你真的是要插手我们陆家的事？跟我们陆家为敌？”
<br/>　　陆千刃看着玄北圣者，冷哼一声的说道。
<br/>　　“我知道你离开学院，在外得到奇遇有所领悟，但是跟我们陆家为敌，你还要自己掂量掂量这点！”
<br/>　　陆千刃的话，已经是绝对的威胁，抬出来陆家，来威胁玄北圣者。
<br/>　　只是玄北圣者的表情，在听到这陆千刃的话，脸色都没有任何的变化，反而是双眼之中，都露出了几分嘲笑。
<br/>　　“陆家的事？”
<br/>　　玄北圣者平静的开口道。
<br/>　　“这件事，是陆家的人，也同样是我玄北的事，你陆家的人，敢来镇压我的弟子，是不是没有将我这个师尊，放在眼中啊！”
<br/>　　这一番话，让在场猛然安静下来。
<br/>　　无数人的眼珠子都是在猛转，似乎是在再三确认这句话。
<br/>　　弟子？
<br/>　　师尊？
<br/>　　楚风眠，是玄北圣者的弟子？
<br/>　　无数围观众人心中，都是一下子惊异万分。
<br/>　　玄北圣者，在北邙学院的老祖之中，都是一尊极为神秘的存在，一些知晓玄北圣者的人，都知道，玄北圣者向来都是一个人，独来独往。
<br/>　　还从未听说过，玄北圣者，居然还有这样一个弟子。
<br/>　　楚风眠，不过是一个北邙学院的新人，什么时候居然拜师了玄北圣者？
<br/>　　“弟子？谁是你的弟子？”
<br/>　　陆千刃听到玄北圣者的话，同样一愣，似乎也没有想到玄北圣者居然是说出这种话。
<br/>　　陆千刃心中想过一千个，一万个玄北圣者出手的理由，肯定是看重了楚风眠的资质，为了卖一个人情，甚至是为了夺取祖龙至尊剑，这种理由都想过。
<br/>　　却是想不到，玄北圣者居然是为了救自己的弟子而来。
<br/>　　“楚风眠，是我的弟子。”
<br/>　　玄北圣者，冷冷的开口道。
<br/>　　“怎么？这件事，不单单是你们陆家的事，也是我玄北的事！”
<br/>　　“你们陆家的人，围攻我的弟子，被我的弟子击败也就罢了，陆千刃，你一个踏上了生死台阶的人物，北邙学院的老祖，居然都对一个小辈下手，未免有些太过了吧！”
<br/>　　玄北圣者的一句话，一下子震在了陆千刃的心中，令陆千刃哑口无言。
<br/>　　这里，本来就是生死台，在生死台上，一切弟子之间的交手，都不允许有其他长老参与的。
<br/>　　陆雀出手，本来就是违背的规矩，如今更是被楚风眠击败。
<br/>　　就算是楚风眠杀了他们两个人，也都是在规矩之中。
<br/>　　而陆千刃，身为北邙学院的老祖，不顾身份的对付楚风眠，已经是犯了规矩。
<br/>　　之前没有人为楚风眠出头，他便是可以肆无忌惮，其他人也不可能冒着得罪陆家的风险，来帮楚风眠。
<br/>　　可如今玄北圣者出现，这一切都不同了，有着玄北圣者出头，陆千刃所做的，的确已经是违背的学院的规则。
<br/>　　陆千刃的目光闪烁，似乎是要考虑这什么。
<br/>　　要杀楚风眠，现在本来是最好的机会，但是如今玄北圣者突然出来，玄北圣者的实力，要远在陆千刃之上。
<br/>　　现在陆千刃就算是强行想要轰杀楚风眠，都绝无可能。
<br/>　　“玄北，你真的要跟我们陆家为敌？”
<br/>　　陆千刃还是不死心，一双眼睛，看向玄北圣者开口道。
<br/>　　“你说他是你的弟子，有何证据。”
<br/>　　“证据？我玄北从不说假话。”
<br/>　　玄北圣者冷哼一声道，言语中已经有着几分冷笑道。
<br/>　　“陆千刃，再不走，你也别想全身而退！”
<br/>　　从玄北圣者的身上，一股灵力便是猛然爆发出来，如今玄北圣者身上的气息，也是在节节攀升。
<br/>　　这一股气息，化为威压，猛然向着陆千刃压迫过去。

<br/>　　这二人，虽然都是踏上了生死台阶的强者，但是交锋瞬间，便是分出了胜负，其中的差距极大。
<br/>　　“你！”
<br/>　　陆千刃连连后退，脸色也是惊恐万分。
<br/>　　这力量，远远超越他的想象。

<br/>　　陆千刃看着楚风眠，冷声说道。
<br/>　　“小子，这一次算你运气好，有人为你出头，我就饶过你这一次，将我陆家的人交出来吧！”
<br/>　　陆雀跟陆画二人，刚刚就被楚风眠镇压，落在了楚风眠的手中。
<br/>　　听到陆千刃的话，楚风眠脸色一变，突然冷笑一声道。
<br/>　　“放人？这两个废物，在生死台上被我镇压，是杀是剐也是由我来决定。”

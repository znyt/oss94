　　不多时，伏卓便带着苏莫，来到了一座塔楼之前。
<br/>　　此塔楼位于千渊殿的中央部位，高达数百丈，共有十三层，壮阔雄伟。
<br/>　　塔楼有不知名的岩石构成，迷茫着淡淡的黄色霞光，显得极为不凡。
<br/>　　“弟子伏卓，有要事求见殿主！”
<br/>　　来到紧闭的塔门之前，伏卓躬身行礼，面色恭敬无比。
<br/>　　殿主曾经吩咐过，要是有从飞升殿到来的人族，愿意留在大夏，要领来见他。
<br/>　　所以，伏卓才将苏莫带来面见殿主。
<br/>　　“进来吧！”
<br/>　　淡淡的声音响起，塔楼的大门，无声打开了。
<br/>　　“上官昊，我们进去见殿主。”伏卓深深的吸口气，招呼苏莫一声。
<br/>　　对于面见殿主，伏卓有些忐忑，毕竟，他只是千渊殿一个一品真神的弟子，而殿主可是九品真神，还是千渊城第一强者。
<br/>　　苏莫点了点头，跟着伏卓走进了塔楼之中。
<br/>　　在此之前，他已经做好的准备。
<br/>　　首先，他将不败战气元神，和其他一些剑魂，都隐藏在了吞噬元神之中，只留下了一个万化神剑元神。
<br/>　　其次，他将体内的混沌之力，分解了一半，变成了五行玄力，这也导致他的气息有所改变。
<br/>　　变得与混沌族有异，就算他有混沌不灭体，但是，体内充满了五种属性的玄力，也不会让别人认为他是混沌族。
<br/>　　为了留下来，苏莫不得不如此。
<br/>　　若是这殿主知晓他是混沌族，还不知道会有什么变故。
<br/>　　毕竟，他如今，可不适合去混沌神朝。
<br/>　　帝一魂肯定有后手，说不定在混沌神朝，已经布下了天罗地网。
<br/>　　塔楼第一层，是一个宽敞的大厅，比一个小型的演武场还要宽广，干净整洁，古朴大气。
<br/>　　不过，客厅之中，此时却是空无一人。
<br/>　　嗡~~
<br/>　　就在此刻，空间一阵波动，一具身影瞬间出现在了大厅主座之上。
<br/>　　此人，是一位面色刚毅的中年男子，一身锦袍，面容肃穆，威严无比。
<br/>　　“拜见殿主！”见到中年人现身，伏卓立刻躬身行礼，恭敬无比。
<br/>　　苏莫亦是抱拳行了一礼，同时他心中暗暗震惊，此人虽然没有多强的气息，但是整个人就如同无尽的深渊，非常的可怕。
<br/>　　“伏卓，何事？”锦袍中年人淡淡的问道，虽然是在问伏卓，但是他的目光，却是一直盯在苏莫的身上。
<br/>　　千渊殿的人并不多，中年人自然认识伏卓，也知晓伏卓管理飞升殿。
<br/>　　那么，眼前的青年，来路便不言而喻了。
<br/>　　“殿主，这位是上官昊，今日才从下界飞升而来！”伏卓恭敬的说道。
<br/>　　“嗯！”
<br/>　　锦袍中年微微颌首，打量了苏莫一番，诧异的问道：“你虚神境三变，是如何来到神域的？”
<br/>　　“禀殿主，我是和一位半神境的前辈，一起破开界障，进入神域的，只不过，并没有飞升到同一个地方！”苏莫沉声回答道。
<br/>　　“你以为本殿主无知吗？”
<br/>　　锦袍中年人闻言，脸色一板，冷漠的道：“界障玄妙无比，没有足够的实力，就算有人带你，你也不可能到达神域。”
<br/>　　神域的界障，只允许有足够实力的人通过，任何人都不可能带弱者进入，不然，就算是藏在下界最强大的空间宝器之中，也会被灭杀。
<br/>　　“禀殿主，我的实力也不弱，虽然是虚神境三变，但是，实力堪比下界的半神，所以才能来到神域。”苏莫郑重的道，他知道此人不好糊弄，心中也提高了警惕。
<br/>　　殿主闻言，微微颌首，道：“无数岁月以来，只要是能来到神域的下界武者，无不是天才，你虚神境三变能来到神域，想必是天才中的天才，告诉我，你有何不凡之处？”
<br/>　　“殿主，我在下界的确是天才，拥有万化神剑元神。”苏莫沉声道，他必须表现出一些天赋，才能让解释此事。
<br/>　　言罢，苏莫头顶混沌之光闪耀，万化神剑元神升腾而起。
<br/>　　至于他的吞噬元神，则是蛰伏在识海深处，对方能不能感知到，那就不得而知了。
<br/>　　“不错，的确是万化神剑元神！”
<br/>　　殿主见此，眸光微亮，不过，却是疑惑的问道：“你这万化神剑元神，居然是混沌之力融炼而成，你修炼了混沌族的武学？”
<br/>　　“殿主，我修炼五行功法，并没有修炼混沌族的武学，不过，当初机缘巧合，得到了很多混沌源石，便用来融练了混沌了元神。”苏莫早有说辞，不急不慌的说道。
<br/>　　“原来如此！”殿主微微点头，五行之力和混沌之力有联系，这个小子用混沌之力融炼了元神，倒也不会排斥。
<br/>　　“不过，你就算有混沌万化神剑元神，也不可能进入神域，我观你气血如龙，血脉之力极强，你拥有什么血脉？”殿主又问道。
<br/>　　苏莫心中郁闷，没想到此人如此追根究底，不过，他也不能不答。
<br/>　　“殿主，我血脉之力的确很强，但是，我自己并不清楚是什么血脉？”苏莫沉声道。
<br/>　　言罢，苏莫便稍稍催动了血脉之力，霎时之间，身上光芒四射。
<br/>　　他身上的光芒，很不纯粹，像金色、又像白色，还有琉璃般的玉色。
<br/>　　苏莫的血脉之力，乃是很多种血脉大融合后的结果，他还就不相信了，这个殿主能看出他的血脉是何种？
<br/>　　“这血脉……的确很强？”
<br/>　　殿主紧紧的盯着苏莫，良久也看不出这是什么血脉之力，最后只能说——很强。
<br/>　　“不错！你的天赋确实很高，难怪能从下界来到神域。”殿主面上露出了微笑，下界什么情况，他并不清楚。
<br/>　　不过，眼前这个上官昊，天赋很高倒是事实。
<br/>　　千渊城虽然人口百万，但千渊殿弟子不过只有几千人，个个都是精挑细选的。
<br/>　　而眼前的上官昊，天赋在这几千弟子之中，也能算是能排进前十了。
<br/>　　最主要的是，他一眼便能看出，这上官昊不足百岁，最多七八十岁的年纪。
<br/>　　如此年轻，就有虚神境三变的修为，天赋非常优秀，百岁成就真神，希望不小。
<br/>　　随即，殿主抬起手掌，掌中有源力涌动，凝聚成了一块巴掌大小的令牌。
<br/>　　这令牌呈灰色，一面雕刻着‘夏’字，一面雕刻着‘千渊’二字。
<br/>　　“这是我们大夏神王殿千渊分殿的令牌，你且收下，从今日起，你就是千渊分殿的弟子！”殿主手掌一抛，令牌便向苏莫飞了过去。
<br/>　　“多谢殿主！”苏莫接过令牌，郑重的道谢。
<br/>　　“嗯！”
<br/>　　殿主微微颌首，随即向伏卓吩咐道：“伏卓，你带上官昊去‘天九殿’，那里便是上官昊以后的住处，你再告诉他千渊殿的规矩。”
<br/>　　“天九殿！”伏卓闻言，顿时心中一惊，他没想到殿主如此重视这个上官昊。
<br/>　　千渊殿有数千弟子，三名长老，和一位殿主，住所可是泾渭分明的。
<br/>　　殿主和三位长老，居住在这玄渊塔之中，下面的弟子，则是分别居住在阁楼、院落和宫殿之中。
<br/>　　一般的普通弟子，只能居住阁楼，天赋杰出，且修为强大的弟子，则居住在独立的院落之内。
<br/>　　只有殿主看重的天才，才有资格居住在宫殿之中。
<br/>　　而千渊府内，只有九座宫殿，前八座都已经分配给了殿中最优秀的八名弟子，唯有第九座‘天九殿’，一直没有归属。
<br/>　　而如此，殿主居然将天九殿，分配给了这个上官昊。
<br/>　　伏卓心中震惊不已，这可不是一座简单的宫殿，更是代表了上官昊在殿主心目中的分量。
<br/>　　而且，天九殿中，有超级强大的辅助阵法，对任何人而言，都是顶级的修炼宝地。
<br/>　　“是，弟子明白！”伏卓心念电转，没有多想，立刻点头应允。
<br/>　　既然殿主吩咐了，他自然不敢有怨言。
<br/>　　只不过，千渊殿之中，有很多天才人物，一直都窥视天九殿，这些人怕是欲哭无泪了。
<br/>　　若是知晓天九殿被上官昊拥有，这些人估计不会甘心，毕竟上官昊修为太低。
<br/>　　“嗯，去吧！”殿主挥了挥手，身上光芒微闪，便消失不见。
<br/>　　“好快！”苏莫心中暗暗震惊，他看出来了，殿主并不是用了什么空间奥义，就是速度快，快到了极致。
<br/>　　“上官兄，恭喜恭喜！”
<br/>　　伏卓的脸上，露出了笑容，向苏莫抱拳道喜，在他看来，苏莫得到了殿主的重视，住进了天九殿，日后前途无量了。
<br/>　　“呵呵，我们日后就是同门了，还请伏师兄多多照顾！”苏莫却是不清楚天九殿的含义，抱拳回礼。
<br/>　　“哪里哪里？上官师弟以后多多提携才是！”
<br/>　　伏卓面带笑意，手一摆，道：“走，我带师弟去天九殿。”
<br/>　　苏莫心中疑惑，这个伏卓怎么这么客气了，不过，他也没有深究，便点了点头。
<br/>　　随即，苏莫便跟着伏卓，离开了塔楼，向千渊府深处走去。
<br/>　　千渊府很大，人却不是很多，显得颇为安静。
<br/>　　“算是有了落脚之地！”一边行走，苏莫一边暗想。
<br/>　　既然在此安顿了下来，那么，接下来就是安静的修炼了，争取二十年之内，成为真神。
<br/>　　“神王？是什么境界？”苏莫心中暗想，这个待会要询问伏卓一番。
<br/>　　大夏神王殿，想必最强者乃是‘神王’，不知道他自己据此境界，还有多远！

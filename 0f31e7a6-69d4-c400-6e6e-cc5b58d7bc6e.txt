　　“死吧！”
<br/>　　雪魔带头冲过来，手中长刀砍向四宝。
<br/>　　这一击，他用尽了全力，四宝几乎昏厥，毫无反抗之力。林三生当然不可能丢下他，情急中将天地规尺迎了上去。
<br/>　　承受了长刀一击，天地规尺直接脱手而飞，强大的气息波动震得手臂发麻，一口血差点吐出来。
<br/>　　雪魔倒转刀锋，横着朝四宝脖子上抹去，林三生右手麻木，情急中伸出左手，朝刀锋抓去。
<br/>　　这一刀力道之大，顶着他的手掌横推出去，林三生一只手几乎被斩断，猩红色的鬼血从伤口涌出，喷在刀锋上，腾起一股白汽。
<br/>　　长刀顶着他的手，抹向四宝的脖子，在这一秒钟，一向成竹在胸的林三生感受到了深深的绝望，眼看着长刀落下，突然锵的一声，在距离四宝脖子几厘米远的地方突然被挡住了。
<br/>　　一支判官笔，格挡在长刀之下。
<br/>　　林三生猛然转头，是徐文长，手持判官笔，淡淡地望着雪魔。
<br/>　　“看你们打的这么热闹，我竟有些耐不住了，看来修行还是不够。”
<br/>　　雪魔和林三生一样呆滞了片刻，一起朝他身后的高台上望去。
<br/>　　一时间，只见四座灵石上的符文一起泛起了蓝绿色的光，汇聚成四道光束，互相照过去，对角折射，四象生八卦，成了八八六十四道，如同织成了一张大网，覆盖在井口上方，整个平面之中，唯一没有被网覆盖的就是中间部分，为青铜鼎所占据。
<br/>　　青绿色的光落在上面，大鼎上的符文也凸显而出，如同浮雕。
<br/>　　“差点忘了，还差了这个。”
<br/>　　徐文长从衣袖里摸出了一个东西，朝高台上丢了过去，是一张绢纸，上面密密麻麻地写满了字，飞到巨鼎上方，落下去，燃烧起来，字迹飞起，绕着巨鼎旋转，凝聚在一起，形成了一股黑洞，一瞬间，海岛剧烈地摇晃起来。
<br/>　　仿佛岩浆喷发一般，一股水柱从巨鼎下的海眼中喷射出来，撞击着巨鼎的底部，鼎中烈火沸腾，从来都是火在下面烘烤容器，热水沸腾，眼前却是反过来了。
<br/>　　“本末倒置，水火相济，乾坤易位，天翻地覆，阴阳化生……”徐文长念着咒语，操控着那张在巨鼎里燃烧的绢纸，烈火越旺，海眼中喷射出来的水也越多，总算达到阴阳调和的地步，四块灵石射出的蓝光瞬间散开，一圈又一圈的光华，朝四周岛屿蔓延开来。
<br/>　　海眼被激活了。
<br/>　　“该死！”
<br/>　　雪魔看见这一幕，知道行动失败了，对着手下众人找了招手，迅速后退。
<br/>　　他心里清楚，海眼一旦激活，便是接上了天地间阴阳二气的力量，凭他们的力量，根本没办法破坏——甚至想要靠近海眼都做不到，再说徐文长已经腾出了手，雪魔自忖奈何不了他，干脆走了。
<br/>　　离开海岛，雪魔看见了位于海上的那个巨大的包围圈，眼前一亮，飞奔过去。
<br/>　　就算任务失败，如果能将捉鬼联盟一网打尽，也是大功一件，眼下，他们的大军已经层层叠叠地将捉鬼联盟围住，一个都跑不掉。
<br/>　　鬼兵围着捉鬼联盟，捉鬼联盟却围着穷奇，狂攻不止。
<br/>　　如果他们一拥而上，势必造成可怕的杀伤，但血池罗刹冷静地观察局面，觉得包围圈还不够严密，强攻的话固然能杀死几个，但其余的很有可能突围成功。
<br/>　　她的胃口很大，她想要擒杀捉鬼联盟所有人，或者俘虏他们，或者……也有可能劝降他们？
<br/>　　于是她下令不要进攻，反而向后退，在中间留下一块足够大的空间，免得被他们突围，一边传令下去，让更手下进入阵型，严阵以待，只要人足够多，那捉鬼联盟就一个都逃不掉了。
<br/>　　“叶少阳，你回头看一看，你们已经插翅难逃，而今你有两条路，要么全体战死，要不然就归顺随于我，我保证你们安全，我知道你不愿低头，但我也知道，你最重视手下的性命，你一心求死没人拦你，但你忍心让你这么多门人跟你一起去死吗？”
<br/>　　雪魔也来到她身边，默不作声地望着叶少阳。
<br/>　　对于血池罗刹的警告，叶少阳似乎没有听见，骑在小九的背上，手持七星龙泉剑，不断对穷奇发动猛攻。
<br/>　　穷奇是上古四大凶兽之一，实力强到能与证道强者金刚三藏一决高低，叶少阳不是他对手，三界之内，除了酆都大帝等极少数几个超一流强者，几乎没人能奈何得了它。
<br/>　　但面对捉鬼联盟一干人默契的围攻之下，也只能勉强防守，局面大约相持。
<br/>　　“这如何是好！”
<br/>　　徐文长又回到高台上，远远地俯瞰着远方的战局，不免心惊肉跳，而今捉鬼联盟几乎陷入绝境，于公，他们如今是阴司编外的最强的一股力量，于私，他们是自己找来帮忙的，能激活海眼，全靠他们牵制了太阴山大军，如今大功告成，他们万一全军覆没……徐文长过不了自己这一关。
<br/>　　“如何是好？”
<br/>　　他看到林三生也抱着四宝来到身边，一把抓住了他，“你诡计多端，如今这局面，如何是好？”
<br/>　　林三生目光灼灼地望着战场，仿佛没听到徐文长的话，片刻，他转头看了徐文长一眼，飞快问道：“岛上安全了吗？”
<br/>　　“海眼已活，化生阴阳，可看做是长盛不衰的法阵，纵然千军万马也攻不进来，唯有我有开启法阵的咒语。”“那就好。”
<br/>　　林三生将四宝丢到巨鼎边上，让他待在核心法阵之中，自己飞身下了高台，来到岸边。
<br/>　　慕寒、慈心师太带着一干法师，一脸惶惑地站在岸边，之前的战斗中，他们也死了不少人，连青城派掌教王道乾也战死了。冲和子在站在边上，带着他残余的一些弟子，也是一脸焦急地望着战场。
<br/>　　见到林三生过来，他主动询问现在怎么办。
<br/>　　“人都在这了是吧？”林三生问。
<br/>　　“其余都战死了，就剩他们了。我师兄进入海底，半天不见，想来是被某个同量级的强者缠住了，加之奈何？”

　　以叶尘如今的修为，自然不需要御剑飞行。
<br/>　　只不过，他刚刚击败烈日宗众人，牵动了体内的伤势，急需运功调息，这才动用紫琼剑，他好方便疗伤。
<br/>　　足足用了三四天的时间，叶尘终于来到了空间通道的尽头，而他体内的伤势，也好了七八成。
<br/>　　前方出现一道亮光，叶尘穿身而过。
<br/>　　哗啦！
<br/>　　下一刻，
<br/>　　叶尘已经出现在一片冰雪荒原之中，而且重力暴涨，以他如今的修为，竟然都微微有些不适。
<br/>　　“想来，这里应该就是太玄星了！”
<br/>　　流光派的传送阵，不但比较低级，而且还是单程的传送阵，只是随即传送到太玄星便算了事，对于普通的修真者来说，着实风险不小，也难怪平时封锁，不轻易开启。
<br/>　　叶尘先取出吞天鼎，神念在鼎内空间中扫了一下，顿时眉头紧皱，脸色有些难看，
<br/>　　如今他进入修真界，也已经有好几日的时间了，曦月的残魂也变得越发虚弱，而七叶回仙草的效果，也已经越来越弱。
<br/>　　没办法，地球上的灵花灵草，等级普遍不高，已经不足以阻止曦月的残魂消散。
<br/>　　“不行！如今看来，我已经等不到千万更高级的星域，必须立即搜寻一些，可以凝聚神魂的天材地宝才行！”
<br/>　　打定主意之后，叶尘当即将自己的神念，全部释放，开始在周围搜寻起来。
<br/>　　天玄星，是天玄星域的主星，叶尘前世只来过一次。
<br/>　　而且整个天玄星，地域广阔，足足是地球的数万倍，因而他对这里的环境并不熟悉，如今又是在荒郊野外，必须要先找到一个人问问路才行。
<br/>　　“找到了！百里之外，似乎有人！”
<br/>　　神念扫描到人的气息之后，叶尘没有任何犹豫，直接身体一晃，便朝那个方向疾驰而去。
<br/>　　天玄星上的重力，着实增加了不少，叶尘一开始还有些不太适应，不过飞奔了一段路程之后，体内真元便已经可以运转自如。
<br/>　　片刻之后，叶尘便已经来到百里之外，只见远处一片丛林之中，有七名修真者，六男一女，正在和一群白狼厮杀。
<br/>　　这些白狼，个个高达十余米，不但体型庞大，而且还能口喷冰锥！
<br/>　　而那群修真者，实力最高的也才只有筑基境后期，而且人数远远不及狼群，在群狼的围攻之下，自然节节败退，
<br/>　　“张师兄，我们的援兵，怎么还没有到？再这么耗下去，我们今天恐怕都要死在这里啦！”
<br/>　　“张师兄，实在不行，你带着小师妹先走，我们顶着！”
<br/>　　“是呀！走一个算一个，总比全都死在这里的好！”
<br/>　　……
<br/>　　那名实力最高的青年男子，应该就是众人口中的张师兄，一咬牙，冲着身后唯一的那名女子，道：
<br/>　　“小师妹，我们来掩护你，你快逃命去吧！”
<br/>　　那女子虽然被众人保护在中央，但早就吓得腿都软了，不过倒也挺讲义气，
<br/>　　“不行！要走一起走，我不能丢下你们不管！”
<br/>　　“唉！”
<br/>　　那位张师兄还想再劝，就在这个时候，
<br/>　　唰！
<br/>　　一道紫色金芒在空中划过，
<br/>　　“嗷嗷嗷嗷！！！”
<br/>　　伴随着一道道惨叫之声，围攻七人的那些雪狼群，顿时全都身首异处，连周围白色的地面，都被狼血染的通红！
<br/>　　“是金丹境高手！”
<br/>　　众人先是一愣，继而纷纷大喜。
<br/>　　不过，等他们看清楚那出手之人，顿时又再次愣住，
<br/>　　只见，他们眼前，出现了一个约莫二十岁不到的少年，之前那道剑芒，显然应该就是此人所发。
<br/>　　而这出手之人，自然正是叶尘。
<br/>　　经历过流光派这件事，叶尘已经无心去救一些陌生人，不过初到天玄星，总得先问一下路。
<br/>　　“我等是北寒城弟子，晚辈张明德，多谢前辈救命之恩！”
<br/>　　那位张师兄率先回过神来，快步来到叶尘跟前，恭恭敬敬的行了一礼道。
<br/>　　叶尘摆了摆手，直接开门见山道：
<br/>　　“别忙着谢我，我救人可是有条件的！”
<br/>　　七人听到这话，顿时纷纷脸色一变，不知道叶尘意欲何为，一时间都暗生警惕。
<br/>　　张明德深吸了一口气，道：
<br/>　　“不知道是什么条件？还请前辈明示！”
<br/>　　叶尘淡淡道：
<br/>　　“我初到天玄星，对这里的情况不怎么熟悉，我要你们带我去天玄城！”
<br/>　　呼！
<br/>　　叶尘这话一出口，七人顿时纷纷松了一口长气。
<br/>　　他们原本还以为，叶尘会提出什么过分的条件，没想到只是让他们带路。
<br/>　　而且天玄城，乃是天玄星第一大城，整个天玄星无人不知。
<br/>　　然而，
<br/>　　还没等众人高兴起来，叶尘接着又道：
<br/>　　“我赶时间，所以最好在一天之内抵达！”
<br/>　　“什么？”
<br/>　　听到这话，众人顿时全都两眼一瞪，
<br/>　　“一天之内？这怎么可能！从我们北寒城到天玄城，足足有近百万里，而且中间还隔着十万大山，就算是乘坐最快的流云飞舟，没有十天半月，也不可能抵达啊！”
<br/>　　说话的，是那名美丽少女，一脸的愤愤之色。
<br/>　　“十天半月？”
<br/>　　叶尘听到这话，顿时眉头一皱，冷然道：
<br/>　　“不行！最多三天，我必须到天玄城！如果你们办不到，那就把命还给我吧！”
<br/>　　说完这话，叶尘猛然将身上的气势，释放了出来。
<br/>　　以曦月如今的状态，已经支撑不了太久，说不得，叶尘也只好用强硬手段，逼一下这些人了。
<br/>　　轰！
<br/>　　感受到叶尘身上释放出来的强大气势，众人顿时如遭雷劈，个个面如土色，
<br/>　　“你这人怎么能这样？你这是强人所难……”
<br/>　　那美丽少女还想和叶尘理论，却被张明德一把拉住，冲她使了一个眼色之后，转而望向叶尘，道：
<br/>　　“前辈，想要一天之内抵达天玄城，也不是没有可能，我北寒城内，有一头四阶妖兽，名为九翅紫电鹫，可日行三十余万里！”
<br/>　　“前辈不如先随我等进城，待我们向城主大人请示之后，便可将九翅紫电鹫借于前辈，三日之内，定能抵达天玄城，前辈意下如何？”
<br/>　　张明德说完这话的同时，眼眸中闪过一抹寒意，一闪而逝。

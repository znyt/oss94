　　这二人交手的灵力碰撞，已经是足以撼动整个擂台。
<br/>　　这武斗大会擂台上的禁制，让一些生死境巅峰的武者，可都难以撼动。
<br/>　　楚风眠跟夜天君二人的实力，几乎都已经是要超出生死境的范畴了。
<br/>　　更加令众人惊叹的是，楚风眠跟夜天君的境界，居然还都是神力境极限的地步，他们二人，谁都没有度过生死大劫。
<br/>　　没有度过生死大劫，便是达到了如今这一步的实力，将来的前途，不可限量。
<br/>　　这一战，不管是谁胜，胜者，在未来，都会成为北域的绝对大人物。
<br/>　　在许多人思考之际。
<br/>　　擂台之上。
<br/>　　这二人的短暂交锋，已经结束，两人的身躯，都因为这爆发出来的力量，各自后退了数步。
<br/>　　这短暂的交手，二人的实力，不相上下。
<br/>　　夜天君后退数步，稳定住身形，脸色却是露出几分讥讽道。
<br/>　　“很好，没想到你居然足以接下我三成的力量，看来我还是有些小看了你。”
<br/>　　三成实力？
<br/>　　听到夜天君的这句话，周围安静的看台上，顿时引起了轩然大波。
<br/>　　刚刚夜天君的这一拳的力量，在众人看来，都是恐怖到极点的一拳了，楚风眠能够接下，已经是足以令人不可思议了。
<br/>　　但是听现在夜天君的意思，这还仅仅是夜天君的三成实力？
<br/>　　三成实力，便是如此恐怖，那夜天君的真正全力，该是到了什么程度？
<br/>　　在场武者中，不少人这才意识到，夜天君能够在北域第一大宗门北邙学院之中，拥有如此至高无上的地位，凭借的，可不是其他。
<br/>　　而是实力，在北邙学院那种尽是天才的地方，他都足以称王，他的实力，不言而喻。
<br/>　　夜天君，也许才是隐藏的最深的一个。
<br/>　　“若是这仅仅只是夜天君的三成实力的话，那楚风眠，怎么可能跟夜天君抗衡？”
<br/>　　“楚风眠的奇遇多，夜天君的奇遇也不少，更何况他还是先天神体，同境界本来便是无敌，天资也要超越任何人。”
<br/>　　“这楚风眠，要是在没有其他的手段，只怕是跟夜天君，无法抗衡。”
<br/>　　在众人的心中，这一战的天平，已经是向着夜天君倾斜过去。
<br/>　　“你的实力，不过如此，的确有着走到这一步的资格，不过碰到我，是你绝对的不幸，要是你在潜心修炼十年，真的有可能成长到跟我抗衡的地步，但是可惜，你没有这个机会了。”
<br/>　　夜天君看着楚风眠，大笑着开口道。
<br/>　　“小子，今日你可没有那么好运了，现在你就算是想逃都不可能，今日你，必死无疑！”
<br/>　　开口同时，夜天君猛然上前一步，只见他身上的气势，再次攀升起来，比起刚才，的确是要强大数倍的地步。
<br/>　　他刚刚所说的三成实力，并非虚假，如今这爆发出来的力量，的确是要比起刚刚，要强大的多。
<br/>　　这夜天君猛然上前，混乱之力，全部凝聚在他的拳头上，再一次一拳轰击出去。
<br/>　　这一拳，跟刚刚那一拳的威力，截然不同。
<br/>　　这一拳的打出，其中的灵力，在擂台上，猛然迸发出去，开始向着周围扩散，几乎是一瞬之间，这些灵力，都是化为了一道灵力长河，一道由混乱之力凝聚出来的长河。
<br/>　　这一道混乱长河，向着擂台上的所有角落，扩散而去，令这擂台上，没有任何躲藏的死角。
<br/>　　夜天君的这一拳打出，其中的灵力，都足以凝聚出这样一条混乱长河，一旦是有武者，陷入其中，便是无时无刻都会受到混乱之力的冲击。
<br/>　　整个擂台，都被封锁，根本没有任何可以躲闪的可能。
<br/>　　这是要灵力雄厚到什么地步，才足以做到？
<br/>　　在这擂台上，这一招，根本无路可躲，除非是楚风眠离开擂台，才足以躲开。
<br/>　　但是一旦离开擂台，便是代表着认输。
<br/>　　以楚风眠的性格，又怎么可能？
<br/>　　“混乱长河！这是夜天君自创的强大秘术，不知道有多少强者，死在了这一招的手中。”
<br/>　　不少武者，都是认出了夜天君的这一招。
<br/>　　在外，夜天君动用这一招，都不知道是有多少强者，死在了其中。
<br/>　　在这擂台上，更是没有任何可以退的空间，这一招的力量，比起在外面，更是还要恐怖无数倍。
<br/>　　“动用这一招，看来夜天君是丝毫不打算给楚风眠机会了。”
<br/>　　“夜天君如此认真，楚风眠根本没有任何一丝一毫的机会。”
<br/>　　众人看向楚风眠，现在的楚风眠，仿佛已经是被逼入到了绝境之中。
<br/>　　除非是楚风眠，真的有抗衡这混乱长河的力量，不然的话，这一招之下，楚风眠，必败。
<br/>　　“剑！”
<br/>　　面对这混乱长河的轰杀过来，楚风眠的脸色依然未变，只是口中淡然吐出了一个字。
<br/>　　就在他的手心中，祖龙至尊剑，猛然出窍，出窍一刻，便是剑锋向着那混乱长河，猛然斩了过去。
<br/>　　“嘶！”
<br/>　　这一道剑锋出现的一刻，有着开天辟地之势。
<br/>　　剑光猛然扫过，直接斩向了那混乱长河，只见这混乱长河，在楚风眠的剑光之下，都被硬生生的撕裂开了一个口子。
<br/>　　在这夜天君全力凝聚而出的混乱长河面前，楚风眠的剑锋，居然都是足以撕裂出一个口子。

<br/>　　楚风眠站在原地，一动未动，就是动用剑锋，居然就是将这夜天君的全力，如此轻而易举的给抵挡住了。
<br/>　　“怎么可能？混乱长河，都被此人的剑锋撕裂了。！”
<br/>　　“此人的剑术，莫非之前表现出的，还不是极限？”
<br/>　　“这种剑术，好古老，应该早已经失传了才对，此人怎么拥有的？”
<br/>　　看到楚风眠一剑破开混乱长河，不少众人都是直接站了起来。
<br/>　　在夜天君展现出全力的一刻，众人的心中，楚风眠已经是一个死人了。
<br/>　　没有任何人想过，楚风眠居然足以是跟夜天君抗衡。
<br/>　　楚风眠表现出来的实力，的确极为惊人，尤其是楚风眠的剑术，更是北域年轻一代的第一。
<br/>　　但是在绝对的力量面前，剑术也要被摧毁才对。
<br/>　　可楚风眠居然便是逆流而上，以剑锋，撕裂了混乱长河，这个结果，简直是所有人心中，相反的局面。

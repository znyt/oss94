        &lt;h3&gt;  第三卷第三百零五章  魔女彭慧&lt;/h3&gt;
<br/>
<br/>        江俞琪去找彭慧，彭慧在开会，江俞琪一直等着彭慧开完会，才到彭慧的办公室。
<br/>
<br/>        彭慧开会还是周边的几个项目，马上要进入招标流程了，方案的确定，标书的参数都需要有一个对比和界定。
<br/>
<br/>        彭慧在心里一直是认定江俞轩他们的方案不错，符合高新的发展，而且经过经济师和第三方造价公司的测算，他们的价格也是最合理的。
<br/>
<br/>        彭慧和江家姐弟也没有过多的交往，如果说要是走得近的还就是在杨东家吃饭的那一次，他们算是在一起聊了几句话，当时刘翼飞也在。
<br/>
<br/>        在这次会议上，彭慧力排众议，据理力争，为西城高新也为江家争得了一席之地。
<br/>
<br/>        主持人还是刘翼飞，不过这次的刘翼飞已经不再反对了。
<br/>
<br/>        彭慧说：“同事们，我对事不对人，在这些方案中，你们可以看出来那一个最适合高新的发展，凭良心说话，如果是你家里盖房子，你会选择那一个？有些人的私心很重，光为眼前的蝇头小利考虑，却不考虑大局，我不想作为多年以后被骂的对象。我们现在是为了一个城，而不是一个人，我们工作是为了大众，也不是为某个人服务的。当你们在权衡利弊思虑再三的时候，请你们想一想，你们到底是为谁服务。”
<br/>
<br/>        刘翼飞：“我们是讨论方案，你说这些干什么？”
<br/>
<br/>        “我只是希望大家站在公平合理的角度来考虑问题，请抛开私人偏见和个人利益，我不想在座的各位同事后期都背负着骂名，我更希望各位同事后期的工作一番顺利。”彭慧苦口婆心。
<br/>
<br/>        “彭总，我觉得你这些话都是废话，我们都是工作人员，眼睛都亮着呢，谁好谁坏还是能看出来的，谁有没有私心我们也能看出来，不要认为这个世界上就你一个人大公无私，品德高尚，在座的每一个人都会是公仆。刘总，您说对吗？”韩湘淡淡的说，她负责记录会议内容。
<br/>
<br/>        “闲话就少说了，大家再仔细看看你们手中的方案，这些方案还需要专家评审，我们个人是说了不算。”刘翼飞敲打着桌面，什么时候能战线统一，他也就不左右为难了。
<br/>
<br/>        “我主要是怕有些人会干扰专家的意见和左右专家的评审。”彭慧冷冰冰毫不留情的声音又响起来了。
<br/>
<br/>        “彭慧，现在是开会时间，请不要夹带任何私人感情在里面。”刘翼飞脸红了，在座的谁有权利干扰专家的意见？这彭慧胆子是越来越大了了。
<br/>
<br/>        “我只是实事求是，大家想一想看一看，我们隔壁的单位，那一个标不是被左右了的，排除异己，疯狂收纳，我就怕我们这也会变成那样。”彭慧盯着刘翼飞。
<br/>
<br/>        “如果你不相信我，我退出项目组。”刘翼飞简直是气急了，他这么几年也算是兢兢业业了，没有功劳也有苦力。
<br/>
<br/>        自从彭慧到他们单位之后，他就老是觉得有一双眼睛在他背后盯着他，让他提心吊胆，可是刘翼飞又不能把彭慧怎么样，彭慧职务虽然没有他高，可是人家姓彭啊。
<br/>
<br/>        “不是不相信刘总，是怕你顶不住压力。”彭慧是打一棍子然后给一颗糖。
<br/>
<br/>        “唉，还是你们理解我，开会，开会，大家先看方案，然后集体投票表决。胜出的两家交给专家组评审，不准有人干扰专家组的评审结果，不准打招、请客，周一进行标书文件审核。”刘翼飞发话了。
<br/>
<br/>        会议室只听到呼吸声和翻书的声音，这些方案在之前已经有商家来给他们讲过ppt的，在所有的ppt的讲解中，江家是最出彩的，包括设计、构造、后期的发展规划都符合现代化发展。但是当时他们投票的时候却被否决了，大家敢怒不敢言，只有彭慧一个人三番五次的对抗刘翼飞，刘翼飞的决定每次都被彭慧破坏掉，所以这个项目拖了很久的时间，一直到西岭的项目被提上日程，但是西岭又遭到了彭慧的反对，于是刘翼飞决定将两个项目同时进行，而彭慧被大家私下里称为：“魔女”
<br/>
<br/>        刘翼飞想分散彭慧的注意力，于是将高新的项目交给彭慧来处理，彭慧才和杨东等人来往密切，
<br/>
<br/>        正如江俞轩所说，高新是一个面子工作，不但里子面子都要好，更要成为西城的一道风景线，连同西城新区将会是南郊新区乃至整个西城的标志，古风与现代接合，新与旧的完美转换。
<br/>
<br/>        刘翼飞不是傻子，好与坏、对与错他还是能分得清的，不过就是他需要顶住压力，在见了陈子昂和江俞轩之后，他的思想明显的发生了倾斜，陆宇霆周围聚集了一帮年轻有为的思想开放的年轻人，这些年轻人将会是新时代的领军人物，他们的思想活跃，行动敏捷，能跟上社会发展的步伐，知道年轻一代的思想，契合社会发展的轨迹，不管怎么说，都是陆家人，不过是一新一旧而已。
<br/>
<br/>        俩个小时之后，大家都抬起了头，这两个小时，每个人都在内心煎熬着，做着思想斗争，江家以六比四微弱的优势赢得了大家的投票，而最后的一票是刘翼飞投的。
<br/>
<br/>        当刘翼飞投完最后的一票时，彭慧笑了，有些人傻眼了。
<br/>
<br/>        江俞琪一直等到彭慧开完会才见到彭慧，但是彭慧对江俞琪不熟悉，可以说不认识，从江俞轩到西城之后，江俞琪就是幕后人。
<br/>
<br/>        “你找我有什么事情吗？”彭慧淡淡的问江俞琪。
<br/>
<br/>        “我是江氏集团的江俞琪。”江俞琪拿出自己的名片递给彭慧。
<br/>
<br/>        “江俞轩是你弟弟？”彭慧看了一眼名片，将名片放到了办公桌上。
<br/>
<br/>        “是的，原本他今天要来拜访您的，只是他今天陪着杨总去体检了。”江俞琪解释着。
<br/>
<br/>        “谁来都一样，你说有什么事情？”彭慧还是不亲不疏的语气。
<br/>
<br/>        “您是高新项目的负责人，我想看看高新项目最近有没有进展。想探听一下领导们的意思。”江俞琪也不藏着掖着，明白人说明白话，她就是来看高新项目的进展情况的。
<br/>
<br/>        “项目已经通过会议，马上进入评审阶段，后期会进入招标流程，这个项目拖得时间有些久了，会加快速度的，你们后期关注招标情况就好了。”彭慧公事公办，她心里对江家的方案很钟情，但她不会给他们开特殊通道的，后期他们也是会经过竞争的。
<br/>
<br/>        她彭慧是为大众服务的，不是为某个人服务的，你的好我会看到，我会给予你支持，但是我不会和你有太多的纠葛，我们只限于工作上的交流。
<br/>
<br/>        江俞琪听彭慧这样说，也就打消了想再深层次沟通的想法，也就不再多问，便起身告辞，临出门的时候，彭慧说：“江俞轩陪杨总体检，结果什么时候出来？”
<br/>
<br/>        “应该会很快，俞轩是在西医找人体检的。”
<br/>
<br/>        “好的，知道了，回去替我向他们问个好。”彭慧看着江俞琪微微一笑。

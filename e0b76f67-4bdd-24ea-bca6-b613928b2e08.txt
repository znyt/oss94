　　“比…比武台，没了…”
<br/>　　周遭观赛武者，瞪大了眼睛。
<br/>　　“我没记错的话，东离剑宫的比武台，可是极东铁石所成，武道大能也顶多能划出数分痕迹。”
<br/>　　“这叶流，一掌便轰成了齑粉？”
<br/>　　这场盛事，可是关系到了古帝之墓的名额之争，中域各大势力齐聚，天骄云集。
<br/>　　东离剑宫作为主持者，自然也不会有半分怠慢。
<br/>　　这比武台，直接便用了极东铁石所铸。
<br/>　　极东铁石，本身价值不大。
<br/>　　但这种奇特的石头，只有在极东之地方才存在。
<br/>　　铁石，存于极东之地，如同最普通的石头，但却在极东之地的荒风之下，吹袭千万年而未损分毫。
<br/>　　极东铁石，并无半分修炼用途。
<br/>　　但它的特点，便是硬，硬到了极点。
<br/>　　这种铁石，甚至无法用来炼制武器，因为它根本就无法承载元力。
<br/>　　这次，倒是被东离剑宫用作了建造比武台。
<br/>　　另外，东离剑宫，作为当世三大剑宫之一。
<br/>　　宫内，剑诀凌厉异常。
<br/>　　剑宫内的弟子，也个个剑道水准惊人。
<br/>　　平日里，剑宫弟子、执事，乃至长老，也皆是拿这些极东铁石来练手，以及磨砺剑气。
<br/>　　有一传言，便是东离剑宫之所以无数年前便在此开宗立派，也与极东铁石有关。
<br/>　　而现在，坚硬无比的极东铁石，竟是在叶流一掌之下便悉数化作齑粉。
<br/>　　也难怪在场各大观赛者，个个面露惊色。
<br/>　　“叶圣府的最强掌法，叶圣掌，果然厉害。”冉琦面露惊色，也点了点头。
<br/>　　一旁，萧逸淡漠一笑，不语。
<br/>　　叶圣掌，大半年前与叶流一战时，萧逸就见识过了，自是知晓其中厉害。
<br/>　　“胜了。”顾怜星脸色一喜。
<br/>　　“这叶流果然有些本事。”公孙火舞同样面露惊色。
<br/>　　之前，她与岳天一战时。
<br/>　　她的鞭影飞扬，虽也将比武台的地面打裂些许。
<br/>　　可，裂痕，顶多在数分左右。
<br/>　　可想而知极东铁石之坚硬。
<br/>　　而整个比武台的范围，可是囊括千米，厚度则在数米以上。
<br/>　　这二者差距，可想而知。
<br/>　　“还未胜。”冉琦摇了摇头，语气凝重，“北影栋，没你们想的那么简单。”
<br/>　　萧逸点了点头，“北隐栋，丝毫未损。”
<br/>　　“什么？”一旁，顾怜星、公孙火舞二人，大惊失色。
<br/>　　比武台上。
<br/>　　叶流负手而立，漂浮于齑粉之上。
<br/>　　偌大个比武台，一掌化作齑粉，那无数齑粉，足足堆了数米高。
<br/>　　嘭…
<br/>　　忽然，一声爆响。
<br/>　　一股元力，顷刻爆发，直将比武台范围搅得风云大作。
<br/>　　漫天粉末，吹扬而起。
<br/>　　极东铁石的粉末，在狂风吹袭下，漫天飞扬。
<br/>　　“哼。”观赛席上，数个武者出手，试图挡下这些粉末。
<br/>　　然而，其结果却是未能损粉末分毫，反倒被吹了一身狼狈。
<br/>　　唯独观赛席首席之上，傲东楼数道剑气打出，将吹来的粉末悉数绞成了齑粉。
<br/>　　以及萧逸这边，一股澎湃紫炎凝聚，粉末还未能吹袭到身边，已被悉数烧成虚无。
<br/>　　除此之外，全场观赛者，尽皆被极东铁石粉末吹了满身。
<br/>　　“噗…呸…”
<br/>　　不少武者，连忙擦去脸上或发间粉末。
<br/>　　“这什么鬼极东铁石，就算被轰出了齑粉，竟还是如此坚硬，该死。”
<br/>　　此时，比武台上，偌大个比武台已消失，粉末也已消散。
<br/>　　除了叶流负手而立外，对面，北隐栋的身影，同样无损。
<br/>　　“有些本事。”叶流微微皱眉。
<br/>　　北隐栋脸色冰冷，“叶圣府叶流，不过如此罢了。”
<br/>　　“只是，比武台被你毁了，还如何战？”
<br/>　　二人的目光，同时看向剑宫裁判。
<br/>　　剑宫裁判，则看向观赛首席上的傲东楼。
<br/>　　傲东楼点点头，指尖一出，一道剑气凌空而落。
<br/>　　剑气，精准地半空分散，随后四处飞舞。
<br/>　　待得剑气消散。
<br/>　　原地，已出现了一个剑气划出的圆圈。
<br/>　　圆圈范围，与之前的比武台大小，相差无几。
<br/>　　“地面，有我东离剑宫的阵法，你们打不破，可放心一战。”
<br/>　　“而老夫划下的剑气范围，则是你们的比拼范围。”
<br/>　　“战斗继续。”
<br/>　　傲东楼说罢，缓缓坐下。
<br/>　　比武台，不，现今只能算是个比武圈。
<br/>　　比武圈内，二人的战斗，再次打响。
<br/>　　观赛席上，萧逸看着二人的战斗，微微皱眉思索。
<br/>　　这二人的实力，在伯仲之间，短时间内难分胜负。
<br/>　　“北隐宗。”萧逸在思索的，是这个宗门。
<br/>　　北隐宗，这个宗门，他当然知晓，乃是中域霸主势力，声名和势力，更甚中域十八府。
<br/>　　那么，北隐宫呢？
<br/>　　萧逸还记得，当初初来中域，须踏过十万大山，百万险地范围。
<br/>　　那时，遇到的那群家伙，可是自称北隐宫。
<br/>　　那邪魅年轻人，手段颇多，还有他手下那群被他称为影大、影二、影三等等的诡异武者。
<br/>　　那时，这群家伙的实力修为倒不算太强。
<br/>　　可他们的手段，却异常惊人。
<br/>　　以萧逸的本事，几乎可以做到同境界无敌，乃至跨几个级别都轻松碾压。
<br/>　　可对上那群家伙，却让他费尽了功夫，手段尽出，才一一解决。
<br/>　　还有最后遇到的那自称‘北隐无敌’的男子。
<br/>　　哪怕是萧逸现今眼力，也不敢轻视那男子。
<br/>　　所谓士别三日，当刮目相看。
<br/>　　可对于绝世妖孽来说，可是一段时间不见，就是一飞冲天。
<br/>　　算算时间，距离他初来中域至今，也有好些年了。
<br/>　　那叫‘北隐无敌’的男子，此刻的实力，怕早已不知成长到什么地步。
<br/>　　萧逸不敢确定，但却敢肯定，此人的天赋，绝对是他所见过的年轻妖孽中，最强的几人之一。
<br/>　　能与他相媲美的，他暂时能想到的，也就莫悠一人。
<br/>　　“北隐宗，北隐宫。”萧逸皱着眉。
<br/>　　他不知道那‘北隐无敌’，和北隐宗是否有干系。
<br/>　　但他很肯定，就面前这个北隐宗少宗主北隐栋，与当年他遇到的‘北隐无敌’相比，简直天差地别。
<br/>　　当年临别时，北隐无敌那随意回头，眼眸中的霸绝，还有那如若漠视苍生的睥睨之意，几乎是他生平所见。
<br/>　　恰在此时，周遭观赛席上，惊呼连连。
<br/>　　“是叶圣六绝，终于出现了，名震中域的叶圣六绝。”
<br/>　　惊呼声，打断了萧逸的思索。
<br/>　　“叶圣六绝？”萧逸自语一声，“胜负，也差不多该分出了。”

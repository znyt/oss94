        最近，苏漫雪觉得自己对诺筱颖太敏感了。还有那个傅家大少爷，他到底要什么时候才肯跟她正式领证结婚？
<br/>
<br/>        苏漫雪好害怕，害怕哪天，她现在所拥有的一切奢侈生活全部都化作泡沫消失了。
<br/>
<br/>        中午，诺筱颖端着餐盘在餐厅吃饭的时候，看到了昨天早上给她送早餐的赵斌。
<br/>
<br/>        她主动走过去，在他对面的座位上坐下，还将赵斌吓了一跳。
<br/>
<br/>        “你？”赵斌一时间没想起诺筱颖是谁。
<br/>
<br/>        诺筱颖微微一笑：“谢谢你昨天早上给我送的早餐，只是，我不明白，你为什么要送早餐给我？”
<br/>
<br/>        “你是诺筱颖？”赵斌这才缓过神来。
<br/>
<br/>        诺筱颖微笑着点了点头。
<br/>
<br/>        昨天走得太匆忙，他都没有好好地看清楚傅总在意的这个女孩到底长什么样子。
<br/>
<br/>        不过，现在，他总算是看清楚了。
<br/>
<br/>        虽然不是太惊艳的那种美女，却也是一股甜美的小清新。
<br/>
<br/>        “昨天的早餐多少钱，我数给你……”
<br/>
<br/>        “你认识傅昼景吧？”赵斌慌忙打断了诺筱颖的话。
<br/>
<br/>        诺筱颖眨了眨眼睛，讷讷地点了点头。
<br/>
<br/>        “他让我送的。”赵斌咧嘴一笑，本来他是想问她，是怎么认识他老板的。
<br/>
<br/>        但是，傅总事先就有过嘱咐，不许他在诺筱颖前面透漏了傅总的身份。
<br/>
<br/>        “你是他朋友？”诺筱颖下意识地问。
<br/>
<br/>        赵斌愣了一下，才木讷地点头，其实，他是他下属，说白了，就一小跟班。
<br/>
<br/>        “那你也是学建筑设计的喽？”诺筱颖接着说。
<br/>
<br/>        赵斌顿时一句话也说不上来了。
<br/>
<br/>        “对不起，我话太多了。”诺筱颖见赵斌只是盯着自己无言以对，连忙道歉，然后乖乖吃饭。
<br/>
<br/>        仅仅只是因为在同一张桌子上吃饭，也能闹出点传闻来。
<br/>
<br/>        听说，设计部新来的那个大专实习生，跟总经理助理赵斌好上了。
<br/>
<br/>        昨天，总经理助理赵斌亲自给诺筱颖送早餐，第二天中午，他俩又在同一个桌子上吃饭了。
<br/>
<br/>        姚明娜听到这个消息，不由地喜上眉梢，原来是她误会傅昼景了。
<br/>
<br/>        诺筱颖和赵斌才是一对啊！
<br/>
<br/>        昼景人那么好，对下属的女朋友稍微关照一下，也是应该的。
<br/>
<br/>        午休时间过后，姚明娜便拿这事当回事，特意跑到傅昼景的办公室里去说了一通。
<br/>
<br/>        “小赵啊！你有女朋友了，怎么也不跟我说一声？”
<br/>
<br/>        姚明娜无故突然跑来寒暄，还真有些令赵斌招架不住。
<br/>
<br/>        “姚副部长！您过来是找傅总的吗？”赵斌陪着笑容问。
<br/>
<br/>        姚明娜却白了赵斌一眼，对他的话充耳不闻，径直朝总经理办公室走去。
<br/>
<br/>        此时，傅昼景正在看文件。
<br/>
<br/>        姚明娜走到门口，很有礼貌地叩了三下门。
<br/>
<br/>        “进来。”傅昼景头也不抬地说。
<br/>
<br/>        姚明娜只身进来，还未走到傅昼景办公桌前，便开始说事了：“公司里不是禁止员工之间谈恋爱吗？”
<br/>
<br/>        “嗯。”傅昼景回答得漫不经心。
<br/>
<br/>        姚明娜走到傅昼景的办公桌前坐下，一边翘着二郎腿，一边双手抱臂，明明是来打小报告的，却还一副理直气壮的样子。
<br/>
<br/>        “你的助理跟设计部那个新来的大专实习生诺筱颖正在谈恋爱，你打算怎么处理这事情？”
<br/>
<br/>        “这种事情，不归我管。”傅昼景漠不关心地应了声。
<br/>
<br/>        看到傅昼景的反应，姚明娜不由地洋洋得意。
<br/>
<br/>        早知道诺筱颖其实是赵斌的女朋友，她还跟她较个什么劲啊？
<br/>
<br/>        “那你先忙，我走啦！”
<br/>
<br/>        姚明娜勾唇，昂头挺胸地转身离去。
<br/>
<br/>        傅昼景这才微微抬眸，看着她离去的背影，无奈地摇了摇头。
<br/>
<br/>        诺筱颖跟赵斌谈恋爱？可能吗？想想都觉得不可能！明娜竟然还跑到他这里来没事说事。
<br/>
<br/>        不过，这样的流言蜚语，对诺筱颖来说，未必不是一件好事。
<br/>
<br/>        至少，解除了姚明娜对她的针对，同时也令同事们不再对她呼来喝去。
<br/>
<br/>        总经理助理虽然在公司的职位层次不高，但是地位还是相当高的。虽然只是个助理，但是总归是“皇帝”身边的“大红人”。
<br/>
<br/>        诺筱颖拿了出差条后，都不知道自己已经“被”成了总经理助理的“女朋友”。
<br/>
<br/>        到了花园别墅区，诺筱颖在门卫那里做了登记，然后找到物业出示工作证，接着被物业的一个女管理员带到了苏漫雪的别墅里。
<br/>
<br/>        这个小区的别墅，是滨江城里的高档别墅了。处于环城的位置，交通也便利。
<br/>
<br/>        才一个多月不见，曾经的闺蜜就能买得起这么大的别墅里，诺筱颖总归是有点点羡慕之情。
<br/>
<br/>        她拿着量房仪，一边量尺寸，一边在画本上记录。一个人，一会儿蹲下，一会儿踮脚，一会儿上楼，又一会儿下楼。
<br/>
<br/>        不知不觉，一上午就这么过去了。
<br/>
<br/>        房子的尺寸，诺筱颖掌握后，便一个人坐在楼梯上，开始看客户资料。
<br/>
<br/>        必须有酒吧台和舞池，还要有健身房和家庭影院，此外，房子的隔音效果要做得非常好。
<br/>
<br/>        这样的设计要求，不像是个家，反倒像是一个小型的娱乐会所。
<br/>
<br/>        而苏漫雪给她的感觉，也压根就不像是一个快要结婚的持家女人，反倒是像一个只会贪图享乐拿钱挥霍的女人。
<br/>
<br/>        诺筱颖不禁在想，苏漫雪的未婚夫会是什么样的一个男人？
<br/>
<br/>        不过，能纵容漫雪做这些她自己喜欢做的事情，她的未婚夫一定很宠她！
<br/>
<br/>        只要她过得好就好，反正绝不绝交，也已经无所谓了。
<br/>
<br/>        诺筱颖欣慰地笑了笑，然后按照苏漫雪在资料上提出的要求，开始构思这里的室内设计。
<br/>
<br/>        一份三明治、一瓶水、一支画笔、一个画本，时间分分秒秒流逝，诺筱颖就这样一个人待在这个毛坯房里画了一整天。
<br/>
<br/>        几张草图出来后，诺筱颖满意地露出了一个灿烂的笑容，再看看窗外，乌云密布，像是要下雨了。
<br/>
<br/>        她赶紧收拾了一下东西，关上了别墅大门，想赶在下雨前回家，结果刚离开花园别墅小区，谁知大雨倾盆而下。
<br/>
<br/>        诺筱颖将包包举到头顶，在大雨中小跑，公交车站台在马路对面，她还得穿过马路才行。

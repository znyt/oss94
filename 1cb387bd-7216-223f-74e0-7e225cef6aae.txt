        滨江城的夜生活和临海城一样丰富多彩，就连大型商场都要到凌晨一点才关门。
<br/>
<br/>        诺筱颖来到了滨江城市中心一家最大的百货商场大门前，只见前坪花坛里的一块大石头上标注着“傅氏集团旗下聚福百货”。
<br/>
<br/>        看样子，傅氏集团是霸占了滨江城的整个经济枢纽。
<br/>
<br/>        诺筱颖来到了一楼的电子产品区域，左看右看，也没能挑中一台价格实惠又性能好的笔记本电脑。
<br/>
<br/>        这里最便宜的笔记本电脑，都要五千块钱，着实令诺筱颖感到揪心。
<br/>
<br/>        此时，在这人来人往的商场里，进来了一群便衣特警，他们迅速分散到每个角落里，然后悄悄地告知身旁的顾客，要淡定地离开商场。
<br/>
<br/>        正在给诺筱颖介绍笔记本电脑的女店员通过对讲机得知消息后，慌忙将拉住诺筱颖的手，轻声说道：“这位顾客，您好，我们商场由于特殊原因要提前打洋了。请您走那边的安全通道离开。”
<br/>
<br/>        被女店员莫名其妙地这么一说，诺筱颖下意识地抬眸朝左侧方的安全通道大门那看去。
<br/>
<br/>        只见陆陆续续有顾客，纷纷往那安全通道的大门赶。
<br/>
<br/>        诺筱颖又四下看了看，发现很多顾客似乎都接到了什么指令一样，纷纷往就近的安全通道大门走去。
<br/>
<br/>        “那能不能让我买了这台笔记本电脑再走？我今晚真的急需电脑工作！”诺筱颖看着女店员，恳求道。
<br/>
<br/>        女店员见诺筱颖一脸诚恳，似乎真的有重要的事情等着她去做一样，无奈之下，心一软，便答应了诺筱颖的要求。
<br/>
<br/>        “你确定，就要这台了吗？”
<br/>
<br/>        “嗯！”
<br/>
<br/>        “那你跟我来，我带你去买单！”女店员说着，从柜台下重新拿了一个未开封的笔记本电脑出来。
<br/>
<br/>        她急急忙忙地开了单，然后一边抱着笔记本电脑，一边领着诺筱颖朝收银台那走去。
<br/>
<br/>        “刷卡还是付现金？”
<br/>
<br/>        “刷卡！”
<br/>
<br/>        诺筱颖说着，掏出银行卡递给了女店员。
<br/>
<br/>        “小香，帮我刷最后一单！”女店员走到收银台前，跟收银台的女孩说的。
<br/>
<br/>        被唤作小香的女孩，看了一眼女店员，拒绝道：“我正准备关机了！”
<br/>
<br/>        “你这不是还没关机嘛！快点帮我刷这最后一单啦！”女店员接着说道，然后将诺筱颖的银行卡递了过去。
<br/>
<br/>        小香瘪瘪嘴，接过银行卡给诺筱颖刷单。
<br/>
<br/>        诺筱颖输入了自己的银行密码，在凭条上签了字。
<br/>
<br/>        女店员这才将手里的笔记本电脑递给诺筱颖。
<br/>
<br/>        小香立即关了收银台的电脑，拉着女店员的手，急匆匆地离开。
<br/>
<br/>        诺筱颖低着头，看着自己笔记本电脑，无意间撞到了一个人。
<br/>
<br/>        也就是这个时候，那个被她撞到的人，突然搂住她，并将某个东西绑在了她的腰间。
<br/>
<br/>        诺筱颖还未看清楚对方是谁的时候，耳畔忽然传来一个恶魔般的声音。
<br/>
<br/>        “这位小姐，恭喜您，成为我们选中的冒险使者！”
<br/>
<br/>        什么冒险使者？！
<br/>
<br/>        诺筱颖心惊，转过身去的时候，那个被她撞到的人瞬间不见了踪影。
<br/>
<br/>        耳畔，忽然传来一声尖叫。
<br/>
<br/>        “啊——有炸弹！”
<br/>
<br/>        诺筱颖怔了怔，只见面前一个吓得脸色惨白的女人，指着她的腰部大叫。
<br/>
<br/>        随即，那个女人落荒而逃。
<br/>
<br/>        也正是因为那个女人的尖叫，还未离开商场的顾客顿时一片惊慌，大家就像热锅上的蚂蚁，急得四处逃窜。
<br/>
<br/>        诺筱颖举起笔记本电脑，下意识低头看了看。
<br/>
<br/>        只见自己的腰间，紧紧地系着一根银色的铁链，而铁链上竟然还绑着一颗定时炸弹。
<br/>
<br/>        倒数的时间，仅仅只有十分钟！
<br/>
<br/>        诺筱颖一阵心悸地瞪大了眼睛，双腿一软，跌坐在地板上。
<br/>
<br/>        就在此时，人群中陆陆续续有几个男人纷纷向她跑来。
<br/>
<br/>        “这位小姐，请您不要惊慌，他是我们的拆弹专家！他一定会救您的！”
<br/>
<br/>        这句话，诺筱颖已经分不清是谁说的了。
<br/>
<br/>        耳畔一片嘈杂，大家四处逃窜，甚至还传来了小孩的大哭声。
<br/>
<br/>        她被几个男人围着，时间在此刻，只能以秒计数。
<br/>
<br/>        渐渐地，诺筱颖身边便只剩下一个男人了，而这个男人正头冒冷汗地替她拆着腰间的定时炸弹。
<br/>
<br/>        本来，他还有个助理，现在就连助理也撤了，最后的时间，仅仅只剩下了五分钟。
<br/>
<br/>        诺筱颖的脸色已经惨白到毫无血色，她的手心里满是冷汗。
<br/>
<br/>        “那个……警察先生，您、您也赶紧离开吧！”
<br/>
<br/>        这句话，诺筱颖鼓足了很大的勇气才说出来。
<br/>
<br/>        正在拆弹的这个男人，顿时感到束手无策。
<br/>
<br/>        “让我来！”
<br/>
<br/>        耳畔忽然响起一个极具穿透力的声音。
<br/>
<br/>        诺筱颖抬眸看去，只见一个身材颀长，穿着黑色塑身衣，戴着假面眼镜的男人朝她走了过来。
<br/>
<br/>        正在拆弹的男人，看到这个男人后，幽微地松了口气。
<br/>
<br/>        “太复杂了，很难。这是我刚刚记录的线路图，给你参考。”男人将任务交接给这个戴着假面眼镜的男人后，迅速加快步伐跑着撤退。
<br/>
<br/>        而此时的时间，只剩下最后三分钟了。
<br/>
<br/>        诺筱颖巍巍颤颤地盯着这个接手继续替她拆弹的男人，如此近的距离，她觉得他令自己有种似曾相识的感觉。
<br/>
<br/>        “你也走吧！”
<br/>
<br/>        她这回是死定了。
<br/>
<br/>        诺筱颖显得有些绝望。
<br/>
<br/>        男人一边专心致志地拆弹，一边淡淡地回答：“怎么，心疼我？怕我跟你一起死？”
<br/>
<br/>        他说完这句话后，诺筱颖猛然间又想起了一个人。
<br/>
<br/>        仍旧是那晚，被她救了的那个受伤的男人。
<br/>
<br/>        这令诺筱颖情不自禁地打量起面前的这个男人。
<br/>
<br/>        虽然，她看不到他的眼睛，但是他的脸部轮廓特别精致，除了眼睛，五官中的其它格外深邃立体。
<br/>
<br/>        “是你？”诺筱颖惊颤。
<br/>
<br/>        傅夜沉身心一怔，抬眸瞥了诺筱颖一眼，他还真没想到，她竟然认出了自己。
<br/>
<br/>        看到面前这个男人像似抬眸瞥了自己一眼，诺筱颖笃定了自己的想法后，又接着问：“你腹部上的伤，已经痊愈了吗？还有，你那个朋友，体内的蛇毒应该已经全部祛除了吧？”

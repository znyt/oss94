　　斗气化翼，永远都是修炼者心中的梦想，琥乾微微一笑，道：“这一路，我们将会亲自护送你们，启程！”
<br/>　　居高临下的俯视着下方的空地，挥手间，伴随着一阵尖哨，狮鹫兽双翅猛然一震，巨大的身躯便是离地而起。
<br/>　　而空地上，未能进入内院的学员也满怀期待，挥手欢送，同时也期待着，他们能够进入内院的一天。
<br/>　　坐在狮鹫兽上，慕清霄瞥了眼下方空地，站在最中央的古薰儿，金帝焚天炎，再一次擦肩而过。
<br/>　　伴随着狮鹫兽的双翼急速振动，空地上的人影越来越小，到最后，犹如蚂蚁一样。
<br/>　　而在高空俯视着整个迦南学院，几乎能将其所有位置尽收眼底，当然，除去某些秘境场所。
<br/>　　碧空如洗的蓝天上，十只巨大的狮鹫兽振动着双翼，朝着绵延如无止尽的学员后山掠去。
<br/>　　在狮鹫兽的四个方向，琥乾与三名外院长老，分别在四个方向，将十只狮鹫兽护卫在其中。
<br/>　　雄浑的斗气在他们全身浮现，狂风肆意，却不能撼动他们一丝一毫。
<br/>　　......
<br/>　　......
<br/>　　茫茫林海，其中遍布着各种各样的魔兽。
<br/>　　每隔一段时间，便有着充满凶悍气息的魔兽自林海中爆射而出，朝着上空的狮鹫兽队伍发动袭击。
<br/>　　偶尔，更有成群结队的飞行魔兽追掠而来。
<br/>　　只是，每当这些魔兽接近狮鹫兽百米处，感受到斗皇的恐怖气势后，都犹如惊弓之鸟，瞬间溃散开来。
<br/>　　当然，也并非全然如此，由于这茫茫无尽林海中，魔兽遍布的极其密集，其中自然也不乏一些实力恐怖的魔兽。
<br/>　　因此，在狮鹫队伍进入迦南学院后山半个小时后，便开始有实力强横的魔兽，不顾四名斗皇压迫性的气势，对着狮鹫队伍强袭而来。
<br/>　　而这也是需要四名斗皇护送的原因。
<br/>　　魔兽袭来，四名斗皇挥手间，雄浑的能量匹练如雷霆般划过天际...
<br/>　　旋即，伴随着一声声闷响，许多体积较大的魔兽，便是瞬间炸裂开来，内脏鲜血自上空倾洒而下。
<br/>　　在一段遥远的路程后，各种各样的凶戾魔兽前赴后继的狂涌而来，却依旧没有一头魔兽，能够突破四名斗皇的封锁。
<br/>　　坐在狮鹫兽上，其余学员都目瞪口呆的望着四名斗皇压倒性的力量，慕清霄则闭目养神，不为所动。
<br/>　　伴随着狮鹫兽前进，一路血雾漫天，琥乾与各长老，以强悍的实力，硬生生在天空中杀出一条大道。
<br/>　　终于，在半个多时辰后，狮鹫兽的速度就渐渐慢了下来。
<br/>　　“领域空间。”
<br/>　　低喃一声，慕清霄睁开双眼，视线投向前面碧空如洗的蓝天，没什么特别的，可在他看来，却充满了违和感。
<br/>　　无尽的林海，唯独脚下有一处深不见底的山涧。
<br/>　　“降落！”
<br/>　　凌厉的目光消失，琥乾四人纷纷收回斗皇的气势，笑眯眯的挥了挥手，十只狮鹫便展动双翼，向下方山涧的对面落下。
<br/>　　狮鹫兽携带着阴影与狂风，安稳的降落在地面，慕清霄微微一笑，道：“仙儿，我们下去吧。”
<br/>　　四十多名学员，闪身跃下狮鹫兽后，错错落落的分散在空地上，目光四周眺望，眼中满是茫然。
<br/>　　似乎是看出了学生们的困惑，琥乾淡淡一笑，道：“内院，可是迦南学院的重地，保护措施，自然不是外院可比的。”
<br/>　　说着，琥乾向前面空地走了十几步，脚步顿住，随手一挥，一道能量自其手中爆射而出，最后对着面前的空间射去。
<br/>　　旋即，诡异的一幕便是浮现了出来，只见能量在射过某一处空荡荡的空间时，却是带起一阵阵水波般的涟漪。
<br/>　　涟漪迅速波动着，最后一扇足有七八丈高的银色大门，凭空浮现！
<br/>　　望着诡异出现的银色大门，周围学生皆是一惊，显然，他们都没有想到，这看似极其普通的地方，竟然还隐藏着如此玄妙。
<br/>　　银色大门，在学员们的注视下，伴随着清脆的声响，缓缓开启，大门开启，其中场景也是一如既往的茂密森林，似乎与周围的景象并无区别。
<br/>　　“都跟上来。”
<br/>　　挥了挥手，琥乾率先向银色大门中走去，其后，数十名学员满脸好奇的跟了上去。
<br/>　　慕清霄与小医仙走在最后面，消失在银色大门中的学员们，果然证实他的想法，领域空间，而且还不小。
<br/>　　“走吧，里面估计就是内院的范围了。”
<br/>　　小医仙微微点头，玉手拉着他的衣袖，寸步不离的跟在他的身后，两人相继迈入银色大门中。
<br/>　　在两人迈入银色大门后，大门才缓缓紧闭而上，最后一圈银色能量涟漪扩散而出，最后完全消失。
<br/>　　与此同时，外界再度恢复以往的平静如常。
<br/>　　踏进大门后，倒与外界没什么不同，唯独在森林的入口处，不知何时，出现了两名白发苍苍的老者。
<br/>　　在他们的身后，还站着将近二十名年轻人。
<br/>　　慕清霄视线在他们身上扫过，发现在他们胸口左上方位置，都刻印着一枚类似塔状图纹的徽章。
<br/>　　“呵呵，琥老头，你依旧是这么准时，今年的新生如何？”此时，两名老者其中一名走上前来，笑眯眯的问道。
<br/>　　“与去年差不多，总体稍微要好上一点吧。”说着，琥乾的目光下意识的在慕清霄与小医仙身上掠过。
<br/>　　“真没想到，今年轮到你们两个老家伙值勤啊。”
<br/>　　“三年值勤一次，这也是没办法的事情，谁叫人手不够。”老者无奈的摇了摇头。
<br/>　　“也对，你们都过来。”
<br/>　　说着，琥乾转身对着慕清霄与小医仙，还有新生前五的男学员招了招手。
<br/>　　等几人来到身边，琥乾指着两位老者，道：“这两位是内院的长老，这位是苏长老，另外一位是庆长老，你们以后在内院，有事就可以找他们。”
<br/>　　“咦，这几位是新生前五，那这两位是？”
<br/>　　老者眼中闪过一丝困惑，相比新生前五，慕清霄与小医仙，无疑更有吸引力，无论是外表或者气质。
<br/>　　仅仅是站在那里，注定就是焦点。

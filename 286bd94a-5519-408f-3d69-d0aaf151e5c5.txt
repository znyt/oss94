　　在雷云皓和炎澜天形成防御的刹那，恐怖的三色剑气和十条犀利的灵蛇，便狠狠的击在了两人的防御之上。
<br/>　　三色剑气的威力，着实恐怖，几乎是瞬间就破开了雷球，斩进了雷球之中。
<br/>　　而那本命灵剑阵组成的十条灵蛇，威力同样恐怖，直接洞穿了汹涌的黑焰，向炎澜天的身体袭杀而去。
<br/>　　天空之上，发出两招攻击的苏莫身形一闪，立刻向一旁闪避。
<br/>　　因为在他现身出手的刹那，灵枫和卓剑豪便再次向他展开了攻击。
<br/>　　啊！啊！
<br/>　　就在苏莫闪避之间，两声惨叫声响起，三色剑气和灵剑击中了雷云皓和炎澜天。
<br/>　　轰！轰！轰隆隆！！
<br/>　　恐怖的劲力炸开，雷球崩溃，直接炸裂开来，雷云皓的身体直接被斩飞了出去，一路洒下大片鲜血。
<br/>　　而炎澜天的火焰防御，同样被灵蛇剑阵绞碎，但其身体并没有被击飞，而是被百柄灵剑架在了身上，彻底被制住了。
<br/>　　轰！
<br/>　　雷云皓的身体并没有飞出战台，而是砸在了战台上，发出一声猛烈的爆响。
<br/>　　所有人均是震撼不已，犀利！真是无比的犀利！
<br/>　　这才刚刚开战，炎澜天和雷云皓便被苏莫强势击败，这是多么恐怖的实力啊！
<br/>　　“炎澜天，你败了！”
<br/>　　苏莫退到远处站定，便意念一动，立刻收回了百柄本命灵剑。
<br/>　　对方已经被制住，按照会武的规则，已经败了，就要退出战斗。
<br/>　　灵枫和卓剑豪，一击被苏莫避了过去之后，也没有再继续攻击，而是脸色难看的望着被击败的炎澜天和雷云皓。
<br/>　　这才刚刚开始，便有两人已败，让他们的心沉到了谷底。
<br/>　　炎澜天闻听苏莫之言，脸色一阵青一阵红，被瞬间击败，让他感到无比的憋屈，更是丢进了脸面。
<br/>　　可是，当着无数人的面，他却不得不承认败了。
<br/>　　没有说一句话，炎澜天咬了咬牙，果断转身飞回了看台。
<br/>　　“可恶！”
<br/>　　就在此时，远处倒在战台上的雷云皓翻身而起，赤红的眼眸盯着苏莫，满脸的愤怒之色。
<br/>　　此刻，雷云皓非常的凄惨，身前一道巨大的剑痕从肩膀延伸到了腹部，骨头和肠子都露了出来，血流如柱。
<br/>　　雷云皓彻底怒了，被苏莫一击斩成重伤，让得他彻底暴怒了。
<br/>　　他并没有败，只是受伤了而已，并没有失去再战的资格。
<br/>　　嗡嗡~~
<br/>　　下一刻，雷云皓身形一颤，霎时之间化为了一只庞大的妖禽。
<br/>　　这只妖禽正是雷云皓的本体，雷云鸟！
<br/>　　雷云鸟体长千丈，双翼一展更为巨大，通体闪烁着刺目的雷光。
<br/>　　啾！
<br/>　　一声鸣叫，雷云鸟的身形冲天而起，庞大的身形碾碎空间，向苏莫飞冲而去。
<br/>　　苏莫目光淡淡，望着急速而来的雷云鸟，心中暗暗的摇了摇头，刚才一剑，他本可以轻易击败对方。
<br/>　　但是担心自己的一剑，会直接将对方斩杀，他才稍稍控制了力道，没有施展出最强的攻击，但是却没能将对方击出战台。
<br/>　　不过，这也无所谓了，不过是多出一招而已。
<br/>　　他的实力，足以碾压任何武帝境之内的武者。
<br/>　　轰隆！
<br/>　　只见庞大的雷云鸟冲天而起之后，尖利的口器张开，口中顿时吐出一颗庞大的雷球。
<br/>　　这雷球足有宫殿般大小，其内传出滔天的雷霆之力，震动苍穹，撼天动地，威势一时无两。
<br/>　　在雷云皓出手的同时，灵枫和卓剑豪同时出手了，灵枫再次击出一拳，磅礴的拳劲杀向苏莫，威力同样恐怖。
<br/>　　而卓剑豪意念一动，近两百柄灵剑顿时分散而开，以最快的速度笼罩苏莫的四周。
<br/>　　所有灵剑气机相连，形成了一片剑气领域，将苏莫牢牢的困在了其中。
<br/>　　卓剑豪心中清楚，苏莫有隐藏身形之法，不困住苏莫，他们是在太被动了。
<br/>　　所以，他第一时间动用剑阵，封住了苏莫的闪避空间。
<br/>　　这样的话，苏莫就算隐藏身形，还在剑气领域之中，而其就算逃了出去，他也能第一时间知晓。
<br/>　　轰隆隆！！
<br/>　　巨大的雷球和犀利的拳茫，呈夹击之势，裹挟毁天灭地之威，瞬间便临近了苏莫。
<br/>　　苏莫目光淡淡，处于围攻之中，依旧面不改色。
<br/>　　眼见攻击袭来，他微微摇了摇头，还是尽早结束战斗吧！
<br/>　　手中长剑一挥，苏莫瞬间连出两剑，分别斩向雷球和拳茫。
<br/>　　这两剑，威力更为巨大，恐怖的剑威如火山沸腾，如瀚海汹涌，配合上气息摄人的三色剑气，简直恐怖到了极致。
<br/>　　瞬息之间，两道剑气便斩中了雷球和拳茫，如同利刃切开豆腐一般，瞬间便斩碎了雷球和拳茫。
<br/>　　而后，三色剑气威势不减，以肉眼难辨的速度，向雷云鸟和灵枫袭杀而去，瞬间即至。
<br/>　　灵枫拥有瞬移的能力，躲避和逃跑的速度，绝对是最顶尖的存在，其身形一闪，便出现了百里之外，完美的避开了三色剑气。
<br/>　　不过，雷云皓的本体雷云鸟就没有这个本事了，体型本就庞大的雷云鸟，根本避不开苏莫的剑气，电光火石之间，只能再次从口中吐出一只雷球，企图挡住苏莫的剑气。
<br/>　　但是，雷球的威力虽然强大，但完全无法抗衡剑气，瞬间便被撕开。
<br/>　　毫无意外，三色剑气狠狠的斩在了雷云鸟的身上。
<br/>　　啾！
<br/>　　雷云鸟口中发出一声哀鸣，被直接斩飞了出去，一路洒下无尽的鲜血。
<br/>　　这一次，雷云鸟不再有好运，直接飞出了战台，甚至飞越了看台，跌落到了看台后方的虚空中。

<br/>　　强势！
<br/>　　实在是太强势了！
<br/>　　炎澜天、雷云皓等所谓的妖孽，根本不是苏莫的一合之敌啊！

<br/>　　完全的碾压，实力好像不在一个层次！
<br/>　　灵枫和卓剑豪，更是脸色狂变，苏莫的强大，出乎了他们的预料，让他们都有些心惊胆颤。
<br/>　　不过，心惊归心惊，他们不可能认输，哪怕是败了也要败的堂堂正正。
<br/>　　“杀！”
<br/>　　眼见雷云皓和灵枫的攻击无功，卓剑豪施展了剑阵的攻击，已经包围了苏莫的近两百柄灵剑，刹那间形成了一条巨大的剑龙。
<br/>　　剑龙之身长百丈有余，呈盘旋之状将苏莫围困，而后龙口一张，直接向苏莫一口咬去。

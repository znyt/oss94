夏夕绾迅速抬眸，一张俊美的容颜在她的视线里放大了，陆子羡来了！
<br/>
<br/>        他怎么来了？
<br/>
<br/>        陆子羡垂着俊美的眼睑看了她一眼，然后清寒的目光就落在了装着曼陀罗花毒液的小瓶子上，“那是什么？”
<br/>
<br/>        夏夕绾纤长的羽捷一颤，迅速伸手将小瓶子藏在了自己的口袋里，“没…没什么，就是双双带给我的小玩意儿。”
<br/>
<br/>        女孩很不会撒谎，就算现在竭力强装着自然和镇定，但陆子羡还是蹙了一下俊美的眉心，“我看着怎么像毒？”
<br/>
<br/>        “…”
<br/>
<br/>        夏夕绾想起眼前这个陆子羡可是天之骄子，帝都最年轻风华的院士，想瞒他太难了，他应该已经看出了什么。
<br/>
<br/>        夏夕绾想拍自己的脑袋，她怎么这么不小心被他给看到了，但是他总是这么神出鬼没的，走路一点声音都没有。
<br/>
<br/>        “子羡，这个是我的私事，你不用管，我再怎么说也是你的嫂嫂，所以我请求你一件事，你千万不要将这件事告诉你哥哥。”
<br/>
<br/>        看着女孩眸里的请求，陆子羡没有说话。
<br/>
<br/>        这时夏夕绾突然意识到自己的手腕还被他抓在手心里，她迅速动了一下，想要抽回自己的手。
<br/>
<br/>        但是没抽回，因为陆子羡抓着没放。
<br/>
<br/>        关于陆寒霆生气的事情，夏夕绾已经深思过了，她感觉陆先生十分介意她跟他的姑姑还有弟弟来往，虽然她还不知道原因，就算陆先生和自己的姑姑还有这个亲弟弟都没有多少感情，也不至于这么抵触甚至戒备吧。
<br/>
<br/>        她总觉得陆先生像是有什么事情瞒着她。
<br/>
<br/>        不过奶奶说的对，也许她真的没有给足陆先生安全感，无论陆先生的理由是什么，在明知道陆先生不喜欢的前提下，她都应该先跟他姑姑还有这个弟弟保持距离。
<br/>
<br/>        可是夏夕绾迅速一滞，陆子羡拽住她干什么？
<br/>
<br/>        “你找我有事？”夏夕绾疑惑的看着他。
<br/>
<br/>        “没事。”陆子羡缓缓将她纤细的皓腕给松开了，然后拿了自己的外套，他漫不经心道，“听说夏妍妍今天早晨救了一个老夫人，还用了金针封穴？”
<br/>
<br/>        夏夕绾点头，“好像是的。”
<br/>
<br/>        陆子羡没有回头，而是淡淡道，“这个金针封穴我曾经看见一个女孩用过，那是在帝都城的大街上。”
<br/>
<br/>        夏夕绾的记忆迅速回到了两年前，那次她去帝都城，曾经在大街上救过一个重症病人，当时还有另外一个人，那个人有着一双极其漂亮的手，拿着手术刀，她和那个人当街给那个病人做了一场手术。
<br/>
<br/>        夏夕绾的眸子落在了陆子羡那双手上，他抓着自己的外套，蜷起的五指白皙修长，犹如艺术品。
<br/>
<br/>        夏夕绾瞳仁微微一缩，她想起来了，那个拿着手术刀的人就是…他！
<br/>
<br/>        是陆子羡！
<br/>
<br/>        “竟然是你？”夏夕绾露出了震惊的表情。
<br/>
<br/>        陆子羡回头看着她，点了一下头，轻声道，“恩，是我。”

　　来人是个中年人，乃是柳絮城的城主，洞玄六重修为。
<br/>　　北山郡一百五十城，每城都有城主。
<br/>　　只不过，由于很多城池内都会有大大小小的势力、家族等等，大大地削弱了城主的实权。
<br/>　　所以，很多城主存在的意义，都只是收税，缴纳到北山郡，并保证城内安定，而无权过问各大势力和家族的事宜。
<br/>　　除非是某些手段过人，本身修为又强的城主，才能做到真正掌控全城。
<br/>　　柳城主无疑就是其中之一。
<br/>　　“柳城主，您怎么来了？”身旁的武者问道。
<br/>　　“废话。”柳城主看了眼远处的两片巨大光幕，沉声道，“闹出如此大的动静，当我是瞎子聋子吗？”
<br/>　　“那两个对战的武者，可是无忧公子和猎妖师易霄。”柳城主问道。
<br/>　　“正是。”身旁的武者回答道，“这两人都用上了灵器，攻击都达到了洞玄境层次，怕是会波及到我们柳絮城。”
<br/>　　“无妨。”柳城主摆摆手，道，“战斗就快结束了。”
<br/>　　“哦？”众人面露疑惑。
<br/>　　柳城主说道，“那两人体内真气已经耗尽了，都在调动灵器的力量才能继续战斗罢了。”
<br/>　　“呵。”柳城主淡淡道，“灵器内的力量虽多，但还不是他们这个境界能够完全调用的，顶多调用个两三成罢了。”
<br/>　　“哦？是这样吗？”众武者脸色的疑惑之色更浓。
<br/>　　柳城主点点头，道，“实力归实力，修为归修为。要想完全驾驭灵器内的力量，起码需要洞玄境。”
<br/>　　“算了，你们没有灵器，跟你们说也不明白。”
<br/>　　......
<br/>　　另一边，萧逸与钟无忧战斗。
<br/>　　此时的萧逸，脸色变得无比难看，仿佛某种疼痛在折磨着他。
<br/>　　他发现，自己似乎理解错了什么东西。
<br/>　　灵器内的力量，在他使用了两三成后，竟开始不受他的控制了。
<br/>　　他想再从中抽调力量来释放紫炎，竟对身体产生了极大的压迫和负担。
<br/>　　每再从烈焰手套中调动一分力量，身体就会痛苦几分。
<br/>　　但也没有办法，紫炎不能消失，否则自己危矣。
<br/>　　半晌之后，钟无忧的脸色也忽然变了，像萧逸一样，仿佛在承受某种痛苦，但也同样在坚持释放暗影千仞。
<br/>　　.....
<br/>　　远处，柳城主和一众武者也发现了这个情况。
<br/>　　“柳城主，怎么看他们两人的脸色，好像很痛苦似的？”众武者疑惑道。
<br/>　　柳城主白了他们一眼，道，“明明无法再调动灵器内的力量了，却还在强行调动，只会对身体造成损害，当然痛苦。”
<br/>　　“有多痛苦？”一武者问道。
<br/>　　柳城主道，“那种感觉，就像某股力量在你体内不断冲撞，破坏。强行摧毁你的肉身、撕扯你的经脉，压迫你的内脏，你说有多痛苦。”
<br/>　　众武者当即脸色大变，他们没有灵器，也不知道那种感觉。但只是单纯想一想，已经觉得可怕。
<br/>　　十多分钟后，暗黑之刃与紫炎仍在剧烈交锋着。
<br/>　　只不过，明显可以看到，萧逸与钟无忧两人的情况都非常不妙。
<br/>　　脸色苍白，口吐鲜血，甚至皮肤上也不断溢出血丝，两人都狼狈至极。
<br/>　　“战斗就要结束了。”柳城主沉声道，“谁先承受不住灵器的反噬，调动不出力量，便败了。”
<br/>　　半晌，还是钟无忧的暗黑之幕率先消失。
<br/>　　全身无力地倒下，就连手上握着的千影剑也脱手而出。
<br/>　　“噗。”钟无忧猛地吐出一口腥血，脸色苍白得吓人。
<br/>　　“哼。”这时，萧逸身上仍旧缠绕着漫天紫炎火海，迅速朝钟无忧而去。
<br/>　　远远看去，百丈紫炎火海，仿佛要将钟无忧彻底吞噬。
<br/>　　钟无忧瞳孔一缩，却毫无办法。
<br/>　　萧逸大手一挥，紫炎飞跃，在钟无忧身边肆意飞舞。
<br/>　　萧逸只需一个念头，便可将钟无忧烧成灰烬。
<br/>　　“钟无忧，你败了。”萧逸冷冷地说道。
<br/>　　“我败了...”钟无忧身体一震，脸色变得更加苍白。
<br/>　　这时，萧逸将紫炎散去，转过身，准备离去。
<br/>　　“你不杀我？”钟无忧惊讶地看着萧逸。
<br/>　　“呵。”萧逸苍白的脸色上露出一丝淡然的笑意，“之前我与天残地缺四人战斗时，你藏身暗处，始终没有出手。”
<br/>　　“若那时你们联手，今天败的，不一定是你。”
<br/>　　钟无忧冷声道，“我说过，那四个废物不值得本公子去救。”
<br/>　　萧逸头也不回，淡淡道，“值得也好，不值得也罢，你那时没有出手，我这时便饶你一命。”
<br/>　　“值得也好，不值得也罢？”钟无忧一愣，想起了之前自己的话语。
<br/>　　当时他也曾说，你恢复也好，不恢复也罢，半个时辰后本公子再取你性命。
<br/>　　同样是霸道的话语，同样是出自天才之口。
<br/>　　只不过，他之前的霸道，被易霄轻易破去，不得不出手。
<br/>　　而这次，易霄的霸道话语，说不杀他便不杀他，他钟无忧却无可奈何。
<br/>　　这时，前方再次传来萧逸的冰冷话语。
<br/>　　“当然，我不介意你来找我麻烦，更不介意你动用暗影楼的力量。”
<br/>　　“只不过，下一次我会取你性命；你暗影楼的人，我也见一个杀一个。”
<br/>　　话音落下，萧逸化作一道火光，转瞬间往远方离去。
<br/>　　钟无忧瘫坐在原地，失神地呢喃着，“败了，我钟无忧竟然败了....”
<br/>　　他的目光，变得空洞。
<br/>　　他是北山郡最出色的天才，暗影楼最寄予重望的接班人，竟如此便败了，还是被人饶过性命才得以活命。
<br/>　　这无疑是一种耻辱，对他来说更是比死还难受。
<br/>　　他甚至能听得出刚才易霄话语中的杀意，易霄根本不在乎他的身份，真的会杀掉他。
<br/>　　“呵呵。”
<br/>　　半晌，他忽然笑了出来，眼神也重新恢复了光彩，直直地盯着远方，那里是易霄离去的方向。
<br/>　　“易霄，你还真是个有意思的人。败于你手中，败了便败了。别以为我会怕你的警告，下一次，还不一定是谁杀谁呢。”
<br/>　　说罢，他站了起来，拍了拍身上的灰尘，吞下几口疗伤丹药，再次恢复了翩翩公子的模样，扬长而去。
<br/>　　.......
<br/>　　远处，包括柳城主在内的一众武者，都在看着钟无忧离开，以及易霄远去的方向。
<br/>　　“无忧公子竟然败了，这个北山榜前十中年纪最小的超级天才，竟也会败于同龄人手中。”
<br/>　　“倒是易霄，我北山郡竟又有一位超级天才，倒是可喜可贺之事。”
<br/>　　柳城主满意地笑着。
<br/>　　一众武者也是面露兴奋之色，今日的战斗，对他们来说，只能用精彩去形容。
<br/>　　甚至于，还让他们有了一定的领悟。
<br/>　　毕竟刚才那个层次的战斗，已经算是洞玄境的战斗了。
<br/>　　“天才猎妖师易霄，果真名不虚传。虽然戴着面具，但观他的年纪，应该还只是个少年吧。”
<br/>　　“如此天赋，在整个北山郡都是最顶尖的。”
<br/>　　“啧啧，今日之后，易霄之名怕是会传遍整个北山郡，无人不知，无人不晓。”
<br/>　　.......
<br/>　　远处，萧逸跑了数十里后，才一个踉跄，脚步不稳停了下来。
<br/>　　“噗。”
<br/>　　一口腥血吐出，面如金纸。
<br/>　　其实他也只是强弩之末，坚持不了多久。
<br/>　　幸亏自己修炼了修罗战体，拥有着强悍的肉身，远超其他武者，刚才才硬是忍下了灵器的反噬和损坏肉体。
<br/>　　否则，刚才一战，败的必然是自己。
<br/>　　正常武者的肉体，虽然不弱，但也不强。平日里修炼，九成九天地灵气都会用于转化为真气，只有一小部分的灵气用于自动淬炼肉体。
<br/>　　也是因此，武者战斗时，才会经常需要真气护身，否则很容易就会受伤。
<br/>　　萧逸的肉体，远强于钟无忧。所以才能比钟无忧承受灵器反噬更久，胜了那场战斗。
<br/>(本章完)

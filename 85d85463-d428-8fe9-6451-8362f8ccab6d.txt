　　阁楼密室之中，苏莫的身上气息涌动。
<br/>　　资源充足，他修为的进展颇快，只需三五年左右，就能晋升到二品中级的地步。
<br/>　　时间不断的流逝，随着大量的源石被消耗，苏莫的修为稳步增长。
<br/>　　一晃眼，就过去了四年多，苏莫如愿以偿的达到了二级中品的修为。
<br/>　　不过，他暂时没有出关的意思，资源足够，他准备在修炼一段时间，然后便前去时空界域。
<br/>　　然而，他一直闭关，可把外界的许焱急坏了。
<br/>　　紫霄剑宫，一座差不多的阁楼之中，许焱来回的踱步。
<br/>　　他在绫裳剑宫安排了人，注意苏莫的动向，但是四年多了，苏莫却是从出门一步。
<br/>　　“居然还不出来？不会闭关几十年吧？”许焱脸色阴沉，这倒是也有可能。
<br/>　　毕竟，修为到了真神之境，一次闭关数十年，再正常不过了。
<br/>　　这让许焱有些郁闷，因为苏莫一日不出关，他就不能安心的修炼或者外出。
<br/>　　“哎！”
<br/>　　少倾，许焱叹了口气，那秦陨不出关，他只能继续等待了。
<br/>　　随即，许焱就地打坐，也开始静静的修炼。
<br/>　　时光荏苒，眨眼间又过去了半年。
<br/>　　这一日，门外传来了一阵脚步声，许焱立刻睁开了眼眸。
<br/>　　而后，阁楼的大门被推开，有三人急切的走了出来。
<br/>　　这三人是两男一女，两名男子是三品低级的修为，女子则是二品高级的修为。
<br/>　　两名男子一个略高，一个略矮，体格均是极为壮硕。
<br/>　　而那名女子，身穿红纱长裙，肌肤白嫩，吹弹可破，容貌颇为娇美，一双丹凤眼之中，蕴含着淡淡的媚意。
<br/>　　不仅如此，此女身材极为火爆，丰腴白皙、凸凹有致，勾起诱人的曲线。
<br/>　　“许焱师兄，那秦陨离开了绫裳剑宫！”个子略高的男子急忙开口说道。
<br/>　　“哦？”许焱闻言，眸中精光一闪，立刻站了起来，问道：“他在何处？”
<br/>　　“他下了神剑山，在风雪城中！”矮个青年说道。
<br/>　　“走，我们去找他！”许焱招呼一声，不由分说，便火速立刻了阁楼。
<br/>　　两名青年和娇美的红裙女子，急忙跟上，跟着许焱而去。
<br/>　　“苗妃师妹，等会按原计划行动，由你先出手，一举成功！”
<br/>　　许焱带着三人，快速离开了紫霄剑宫，向神剑山下而去。
<br/>　　一边行走，许焱一边吩咐那名红裙女子。
<br/>　　想要让苏莫严重触犯殿规，简单的也就是以下犯上、背叛神王殿、残害同门、奸.污女弟子这四种了。
<br/>　　而以下犯上、背叛神王殿，这两个殿规，他们不好施手，没有办法让苏莫触犯这两种殿规，那就只有残害同门和奸污女弟子这两种了。
<br/>　　所以，许焱的办法，是先让红裙女子苗妃勾引苏莫，然后他们一举将苏莫擒获。
<br/>　　这个办法很简单，主要就是看苗妃的魅力了。
<br/>　　若是一般的女子，就算再美丽，也不可能影响到一名真神境武者的心志。
<br/>　　但是，苗妃可不同，不仅修炼了强大的媚术，还拥有诱.惑男人的狐媚元神。
<br/>　　可以这么说，修为不到三品的男人，就算精神意志再强大，都很难抵挡的住苗妃的诱惑。
<br/>　　再者，就算苗妃失败了，他们还有第二个计划。
<br/>　　“许焱师兄，你放心吧，既然是宫主的吩咐，妃妃自然不会懈怠！”
<br/>　　苗妃闻言轻笑，殷红的双唇微微翘起，勾起了一丝诱人的弧度，继续道：“那秦陨不过是二品的修为，即便再妖孽，也要拜倒在我的石榴裙下！”
<br/>　　“如此最好！”许焱轻轻的点了点头，心中却是轻轻一挡，这个苗妃实在诱人，让他都有些吃不消。
<br/>　　很快，四人便进入了神剑山脚下的风雪城中，几人修为强大，神念一扫，便发现了苏莫。
<br/>　　“已经快出城了，苗妃你先行一步吧！”许焱立刻吩咐道。
<br/>　　“嗯！”苗妃点了点头，身形一展，如同一只红色的蝴蝶，向苏莫追了过去。
<br/>　　……
<br/>　　风雪城内，苏莫大步而行，很快便接近了城门。
<br/>　　此行离开神剑山，他自然是要前去时空界域，他并未只会绫裳。
<br/>　　因为，他到时空界域，也不会待很长的时间，过不了多久就会回来。
<br/>　　很快，苏莫走出了城门，身形一展，冲天而起，向时空界域的方向而去。
<br/>　　不过，他刚刚飞出数里，后方突然响起一道柔美的声音。
<br/>　　“这位师弟可是秦陨？”
<br/>　　声音柔美，充满了灵动，尽显女人娇气。
<br/>　　苏莫闻言，立刻回头一看，便见后方一名红裙女子，快速追来上来。
<br/>　　几乎是呼吸之间，女子便飞到了苏莫的身侧，和他并排而行。
<br/>　　“你是？”苏莫微微皱眉，诧异的打量着红衣女子，此女很美，比之绫裳都不差多少。
<br/>　　但是，若说是女人的那种诱.人韵味，那比绫裳强了十倍。
<br/>　　“我叫苗妃，前段时间听说绫裳剑宫新来了一位弟子，身份神秘，和绫裳宫主走的很近，就是你吧？”苗妃轻笑一声，娇声娇气的说道。
<br/>　　“是我，幸会！”苏莫点了点头，别人一脸热情，他倒也没有冷淡。
<br/>　　“秦陨师弟，你这是要去哪里？”苗妃转头看着苏莫，浅笑嫣然，眸中渐渐的出现了一丝粉色的光晕。
<br/>　　“出去历练一段时间！”苏莫沉声道，眸光扫过苗妃，他的心中不知为何，突然生出了一丝悸动。
<br/>　　不过，这丝悸动，丝毫影响不了他。
<br/>　　“师弟，师姐也是要外出历练，要不同行如何？”苗妃柔声说道。
<br/>　　“还是算了吧，我喜欢独行，去些危险的地方！”苏莫一边飞行，一边摇头拒绝了。
<br/>　　他的心踪，不会透漏给任何不熟悉之人，更不会和不熟悉之人同行。
<br/>　　苗妃闻言，眸中闪过一道异色，他如此绝色，这个秦陨居然毫不犹豫的拒绝了？
<br/>　　这让她不仅暗怒，这人也太清心寡欲了吧！
<br/>　　不过，她也不在意，因为修为低于他的男人，没有谁几个人能抵挡住她的诱.惑。
<br/>　　“秦陨，你觉得师姐我容貌如何？”苗妃凝视着苏莫，微笑着问道。
<br/>　　“这……！”
<br/>　　苏莫闻言，转头看向苗妃，但就在此刻，异变陡生。
<br/>　　只见苗妃的双眸深处，好像有妖狐在闪动。
<br/>　　一股粉红色的雾气，从苗妃的身上弥漫而出，瞬间就将苏莫包裹在了其内。
<br/>　　不仅如此，一股特殊的芳香，充斥在天地之间，瞬间便进入了苏莫的脑海之中。
<br/>　　这股芳香，让人沉醉、让人沉沦、让人心底的欲.望无限的膨胀、更能让人失去理智。

　　猎妖殿内，铁达与萧逸在任务处做好登记，在这个任务结束前，萧逸便是这个猎妖队的一员。
<br/>　　“来，易霄兄弟，我给你介绍一下。”铁达大叔和声道。
<br/>　　“这是林飞，后天六重修为。”
<br/>　　铁达大叔指着一个队员说道。
<br/>　　萧逸打量了林飞一眼，此人身材消瘦，但浑身上下散发着干练，一双眼睛更是炯炯有神，如鹰隼般尖锐。
<br/>　　林飞对萧逸微微点头，笑道，“我是冲云箭武魂，百米之内的风吹草动都瞒不过我的眼睛。”
<br/>　　“箭武魂？”萧逸微微吃惊。
<br/>　　箭武魂属于器武魂的一类，比较稀少。擅长远攻，发出的箭威力极强，比同境界的武者强了数倍都不止。
<br/>　　而且冲云箭是黄色武魂，威力十分不弱。
<br/>　　“这是黄源，后天六重修为，你可以叫他黄胖子。”
<br/>　　铁达大叔指向另一个队员。
<br/>　　黄源是个胖子，身材高大，但气势沉稳，想来本事不错。
<br/>　　黄源也对萧逸打了个招呼，道，“易霄兄弟，不用生分，给面子的话叫一声胖哥，不给面子，叫一声死胖子也行。”
<br/>　　“呵，胖兄。”萧逸对这个爽朗的队友颇有好感。
<br/>　　黄源继续道，“我是巨力猿武魂，有我在，一定可以保护好你这个孱弱的炼药师的。”
<br/>　　“有劳。”萧逸不置可否，笑笑。
<br/>　　“这是杜新，后天五重修为。”铁达大叔继续介绍道，“别小看杜新，他可是老油条了，在外行走的经验一点儿不比我差。”
<br/>　　“过奖了。”杜新不冷不淡地回答了一句，而后闭嘴不言。
<br/>　　铁达大叔见杜新态度不太好，连忙笑笑，对萧逸道，“易霄兄弟，他是刺猬猪武魂，性子急，脾气躁，你别跟他一般见识。”
<br/>　　“刺猬猪？”萧逸忍住脸不笑，点点头。
<br/>　　倒是一旁的少女已经捂着肚子大笑起来，“原来杜新你的武魂是刺猬猪阿，就是那种一战斗就全身飙刺的猪妖兽，哈哈，难怪你之前不告诉我你的武魂。”
<br/>　　“哼。”杜新脸色难看，不再说话。
<br/>　　“哦。”铁达大叔为了缓解气氛，道，“你们别笑，我还是金刚岩武魂呢，就一块石头，比杜新差多了。”
<br/>　　金刚岩是一种路边的坚硬石头，武魂则是橙阶，确实比刺猬猪低级。
<br/>　　铁达大叔能凭此武魂修炼到先天三重，着实不易。
<br/>　　“至于浅沫，她全名是苏浅沫，武魂你也知道了，是长生藤。今年才15岁，已经是后天境了，可是个天才呢。”
<br/>　　“当然嘛，跟易霄兄弟你比是比不上的，毕竟你年纪轻轻已经是二品炼药师了。”
<br/>　　铁达大叔赞叹着。
<br/>　　杜新不屑道，“切，浅沫才15岁就后天二重了，易霄怎么也有16、7岁吧，才后天一重。”
<br/>　　“长生藤？”萧逸没有将杜新的话听入耳中，而是暗暗疑惑。
<br/>　　以他的炼药师知识，竟然没听说过长生藤这种植物，也不知是何品阶。
<br/>　　不过，这支猎妖队是临时组成的，大家也都是萍水相逢，完成任务后就各散东西了，萧逸也不好意思去问。
<br/>　　“易霄，你的武魂呢？”铁达大叔问道。
<br/>　　“哦。”萧逸反应过来，淡淡地回答道，“控火兽。”
<br/>　　“控火兽？”铁达大叔愣了愣，道，“你说什么？”
<br/>　　“控火兽阿。”萧逸道。
<br/>　　“控火兽？”铁达瞪大了眼睛，“那种一脚就能踢飞的最垃圾的妖兽？被誉为全大陆最低等的赤色武魂？”
<br/>　　林飞和黄胖子都面露不可置信。
<br/>　　“哈哈哈哈。”杜新嘲讽道，“控火兽武魂，笑死我了。易霄兄弟，你这辈子的武道修炼，成就有限阿。”
<br/>　　萧逸翻了个白眼，懒得理他。
<br/>　　按他的了解，控火兽确实弱，但却是和他最匹配的武魂，而且现在已经晋升到绿色了。
<br/>　　当然，在其他武者眼中，武魂是一生都无法改变的。
<br/>　　萧逸自然也不会去解释。
<br/>　　与杜新的嘲讽不同，林飞和黄源都亲切地搭过了萧逸的肩膀。
<br/>　　“易霄兄弟放心，虽然你战力低下，但我们会保护好你的。”林飞和黄源同时拍着胸脯保证道。
<br/>　　“额，谢了。”萧逸苦笑一声。
<br/>　　虽然现在自己只有后天一重的修为，但战力可不弱，越阶战斗对他来说，轻而易举。
<br/>　　起码先天之下，休想伤他分毫。
<br/>　　再者，体内八条主脉要不了多久就会恢复的。
<br/>　　这时，一直大大咧咧的苏浅沫忽然面露悲沉之色，亲昵地拉过萧逸的手。
<br/>　　“易霄，我没想到你这么可怜，赤阶的控火兽武魂，你这辈子都难以突破到先天吧。”苏浅沫柔声说着。
<br/>　　“之前那么大声吼你，对不起阿。”苏浅沫面露愧疚之色。
<br/>　　而后，更是伸出纤纤素手，在萧逸胸膛上轻轻地拍了拍，道，“别难过，想哭就哭出来吧。”
<br/>　　“我没想哭。”萧逸无语。
<br/>　　“不用解释。”苏浅沫认真地说道，“放心吧，以后我会照顾好你的。”
<br/>　　“阿？”萧逸愣了愣。
<br/>　　苏浅沫注意到自己的语误，连忙道，“哦，不是，我的意思是在任务中会照顾好你的，不会让你受伤的。”
<br/>　　“呼...吓我一跳。”萧逸松了口气。
<br/>　　苏浅沫顿时气道，“看你那松了口气的样子，怎么，本小姐很差吗？你之前还直勾勾地看着人家呢。”
<br/>　　不得不说，苏浅沫长得美丽可爱，就算没有倾国倾城，但也是沉鱼落雁的。
<br/>　　而且，才15岁，身材已经凹凸有致，不难想象长大些必是个绝美女子。
<br/>　　生气之时，胸脯起起伏伏，看得一旁的杜新双眼发直。
<br/>　　“小姐，稳。”萧逸连忙道，“你再生气下去，杜新就该流一地口水，脱水而死了。”
<br/>　　“嗯？”苏浅沫嘟起嘴，看了眼一旁的杜新，再看了眼自己，顿时转过身去，愤愤地跑开。
<br/>　　“哈哈哈。”铁达大叔和林飞黄源大笑一声。
<br/>　　“这小丫头，也是今天早上才招募来的，古灵精怪的，倒是有趣。”铁达大叔笑笑。
<br/>　　萧逸点点头，道，“对了铁达大叔，我还不知道我们这次的任务呢。”
<br/>　　铁达大叔立马脸色一变，低语道，“易霄兄弟，先随我们回客栈。”
<br/>　　“好。”萧逸点点头，而后目光看向跑开的苏浅沫的背影。
<br/>　　直觉告诉他，这个少女的来历不简单。
<br/>　　那长生藤武魂，也绝对不是寻常武魂。
<br/>　　......
<br/>　　客栈内，铁达大叔谨慎地关上房门，一行六人围坐在一起。
<br/>　　“林飞、黄胖子、杜新，我们合作的时间不短了，也不瞒你们。”
<br/>　　“易霄还有浅沫，虽然我今天才认识你们两个。但铁达大叔自问看人很准，信得过你们，便也告诉你们。”
<br/>　　铁达大叔看了所有人一眼，缓缓地说道。
<br/>　　见铁达大叔如此谨慎，众人脸色一正。
<br/>　　“队长，放心吧，我们的为人你还不知道嘛。”黄源认真地说到。
<br/>　　“铁达大叔直言便是。”萧逸也说道。
<br/>　　“好。”铁达大叔道，“你们可知道这破玄城，为何叫破玄城？”
<br/>　　“额，算了。”铁达大叔面露窘迫道，“我也不是太会说话，就不卖关子，直言了。”
<br/>　　“因为，这里曾经出过一个破玄境武者，而且，还是北山郡最强的破玄境。堪称地元境之下第一武者。”
<br/>　　铁达大叔说着，脸上露出了无比崇拜和向往。
<br/>　　“地元境之下第一武者？”林飞三人先是一惊，而后道，“那个狂血玄君？”
<br/>　　“不错。”铁达大叔点点头，道，“当年，曾以破玄境界，将一位地元境武者斩杀，从此声名大震，这里也被尊称为破玄城。”
<br/>　　萧逸一愣，而后也想起来。
<br/>　　狂血玄君，百多年前名震北山郡的至强武者。他活着的那个时代，整个北山郡一切武者都被他的光芒所掩盖，声名可谓盛极一时。
<br/>　　只不过，萧逸倒不知他有这么辉煌的战绩。
<br/>　　林飞忽然皱眉道，“我听说，那位狂血玄君百多年前就因为大限来临而身死了，队长你提他做什么？”
<br/>　　铁达大叔沉声道，“有人在雷鸣荒泽找到了他留下的洞府。”
<br/>　　雷鸣荒泽，位于破玄城外，是一个辽阔无边的山脉，但里面到处是沼泽瘴气，而且妖兽横行。
<br/>　　“什么？”包括萧逸在内的所有人顿时一惊。
<br/>　　“嘘。”铁达大叔做了个噤声的动作，道，“这最先是由一个在那里猎杀妖兽的猎妖师发现的，而后渐渐在我们猎妖师的圈子内传开。”
<br/>　　“现在知道这件事的人还很少，仅有一部分名气颇大的猎妖队知道。你们今天也看到了，不少猎妖队都在招募人，想必也是为此事而去的。”
<br/>　　铁达大叔脸色凝重。
<br/>　　“那位最先发现洞府的猎妖师，没敢深入，但却也在边缘找到了一本玄阶武技，还安然无恙地出来了。”
<br/>　　铁达大叔继续道，“，洞府内有狂血玄君留下的诸多宝物，我打算明日就出发，你们觉得呢？”
<br/>　　杜新连忙道，“听队长的，早早去，免得宝贝都被人拿光了。”
<br/>　　林飞和黄源皱了皱眉头，但也表示赞同。
<br/>　　萧逸思索了一下，本以为只是个难度较高的三级任务，没想到竟事关狂血玄君的洞府。
<br/>　　那等人物，就算死了，其留下的洞府也必然禁制重重，危险万分。
<br/>　　不过，所谓富贵险中求，萧逸也是个不安分的主儿，想了一下，便也答应了。
<br/>(本章完)

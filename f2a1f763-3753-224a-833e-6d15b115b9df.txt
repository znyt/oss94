　　陆雪琪可是第一次见大名鼎鼎的鬼王，仅仅是气势就让她喘不过气来。
<br/>　　刚想要起身，便被一只轻柔手掌给压了回去，压力也随之散去。
<br/>　　慕清霄并不起身，依旧端坐在桌前，品着清茶，只是抬头瞥了一眼鬼王，若无其事。
<br/>　　“鬼王客气了。”
<br/>　　鬼王也不恼怒，他知道年轻人都有傲气，更何况后者修为不弱于他，只是左侧的青龙脸色阴沉了下来。
<br/>　　鬼王来到桌前坐下，道：“不知阁下来我鬼王宗，所为何事？”
<br/>　　“好茶。”慕清霄将茶杯放在桌上，嘴角牵起一丝笑容，道：“我来鬼王宗是想借天书一阅。”
<br/>　　话语刚落，鬼王骤然色变，青龙脸色更阴沉的可怕，幽姬与碧瑶也变得紧张起来，凉亭中的气氛变得有些压抑。
<br/>　　陆雪琪倒是不明所以，她不是第一次听到天书这两字了，只是不明白天书到底是什么。
<br/>　　片刻后，鬼王脸上重新挂上一丝笑容，道：“阁下说笑了，天书乃是我鬼王宗镇派功法，怎能说借就借，而且不仅我鬼王宗，青云门也有天书，阁下何不去青云，反而来我鬼王宗呢？”
<br/>　　鬼王的言下之意就是，你不去青云门借阅天书，反而来我鬼王宗，是觉得我鬼王宗好欺负吗？
<br/>　　“实不相瞒，青云门的天书，我在半个月前就已经阅览过，这点薄面，想必鬼王不会不给吧。”
<br/>　　慕清霄简简单单一句话，就将鬼王的后话给堵死，你说去阅览青云门的天书，实不相瞒我已经阅览过来了。
<br/>　　“这......道玄真人允许你接触天书？”
<br/>　　鬼王眼中有些震惊，越发看不透面前的青年，要知道道玄的修为并不再他之下，他是用什么办法接触到诛仙剑的！
<br/>　　“此事就与鬼王无关了吧。”
<br/>　　“哼，大言不惭！”
<br/>　　就在这时，鬼王身后的青龙圣使冷哼一声，眼中杀意凌然，显然不明白，鬼王为何对这谎话连篇的青年如此客气。
<br/>　　青云门的强大，他们都亲身体验过，面前青年的年纪显然也不大，天书是圣物般的存在，他可不相信道玄真人气度有如此之大。
<br/>　　“青龙！”
<br/>　　鬼王还未说话，一旁的幽姬就低喝提醒，慕清霄有多恐怖她不知道，但绝对不会在鬼王之下，看此刻鬼王的态度就能看出来。
<br/>　　若是如此强者与鬼王宗为敌，那后果不堪设想！
<br/>　　慕清霄也不理会青龙，继续说道：“而且，我在空桑山可是帮了鬼王宗不少忙，希望鬼王让我借阅天书，我也只是借阅，并不会带走。”
<br/>　　闻言，鬼王目光看向幽姬，有些询问的意思。
<br/>　　幽姬沉思片刻，想到当初年老大被躲在暗中的强者重伤，，无情海岸上，碧瑶身上的灵气护照，道：“鬼王，当日确实多亏慕公子相救，我们才能从炼血堂余孽手中脱困，在无情海中，也是慕公子救了碧瑶的性命。”
<br/>　　“都是小事，不足挂齿，碧瑶姑娘本就无碍，我们也只是将她救上岸罢了。”
<br/>　　听到两人这么说，肯定是没错了，鬼王一时间也不知道如何是好，毕竟后者有恩与鬼王宗，他们魔教也要面子，若是恩将仇报，传出去岂不是让别人笑话。
<br/>　　见鬼王有些犹豫，慕清霄微笑道：“鬼王将天书借我一阅，算我欠鬼王宗一个人情，你看可好？”
<br/>　　鬼王眉头皱了皱，心中计算着利害关系，如此强者承若一个人情，确实难得。
<br/>　　而且，慕清霄的修为，他摸不透，后者明显能够强抢，现在却是客客气气的，若是他在敷衍，却是有些得寸进尺了。
<br/>　　“慕公子说笑了，你在万蝠古窟中救下碧瑶，又对我鬼王宗有恩，常言道，滴水之恩涌泉相报，既然如此，将天书借由慕公子一阅有何不可。”
<br/>　　慕清霄嘴角勾起一丝笑容，他的耐心也是有限的，若是如此，鬼王都不肯借天书的话，那他就只有强抢了，现在看来，鬼王也是明事理之人。
<br/>　　青龙脸色依旧有些难看，可是看鬼王的态度，显然没有他说话的余地，只好站在一旁盯着慕清霄，默不作声。
<br/>　　“慕公子，不如一起用过午膳，我在带你去天书所在地，如何？”
<br/>　　“我正在研究辟谷之术，午膳就不必了。”
<br/>　　慕清霄言辞拒绝，现在距离午膳还有一个多时辰，他可不想在继续浪费时间，正事要紧。
<br/>　　鬼王嘴角抽了抽，他本想与慕清霄拉近一些关系，没想到后者如此厚颜无耻的拒绝，还辟谷之术，你特么一个元婴期修士，辟谷个屁啊！
<br/>　　“好吧，那我这就领慕公子去阅览天书，请随我来。”慕清霄不去，鬼王也不敢强求。
<br/>　　片刻后，在鬼王的带领下，穿过凉亭拱桥，来到后山的一座秘洞前。
<br/>　　“慕公子，这是我鬼王宗藏丹书古卷之地，非我鬼王宗高层不得入内，其中机关幻境无数，请跟随我来。”
<br/>　　说着，鬼王手掌在旁边石壁上轻轻按下，“喀喀”几声，天衣无缝的石门就缓缓打开。
<br/>　　走进秘洞后，慕清霄在四周环视一圈，确实有不少阵法与机关，修为不够的人进来，绝对是有来无回。
<br/>　　秘洞不是很大，却极为宽敞，秘洞尽头有供着两尊巨大的雕刻，居然与万蝠古窟中的雕刻完全一致，一佛一魔。
<br/>　　在慕清霄古怪的目光中，鬼王神色郑重，来到雕刻前，拿起桌上的香烛，用怀中的火石点起火，伴随着青烟徐徐，一脸严肃的跪拜了下去。
<br/>　　“幽明圣母，天煞明王，圣教鬼王宗万人往诚心拜见，圣教遭厄，衰微已久，无数教众，披肝沥胆，为兴圣教，前赴后继，唯愿圣母明王，垂怜苍生，赐我福祉，再兴圣教，度化众生，共登长生不死极乐欢喜境。”
<br/>　　鬼王祭拜，慕清霄的目光则落在两座雕刻中央的石碑上，上面龙飞凤舞的刻着十个大字：天地不仁，以万物为刍狗！
<br/>　　慕清霄心神放在石碑上，脑海中&#39;轰&#39;的一声，金色光芒莫入他的脑海中，道：“加上我手中的两卷天书，总共是第三卷了。”

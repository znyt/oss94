　　“叶大哥，我们现在怎么办？”
<br/>　　祝小白脸色惨白，望向叶尘问道。
<br/>　　叶尘的脸色同样也不太好看，
<br/>　　如果早知道这里竟然是一处绝地，还不如在外面与那些人大战一场，他也未必会输。
<br/>　　现在，对方如果仗着周围的阵法禁制之力，他恐怕也未必能讨得什么好处。
<br/>　　更何况，他还要分心去照顾祝小白和苏蔓，想要全身而退，就更加困难了。
<br/>　　“要不我们先下去躲一躲吧？”
<br/>　　祝小白提议道。
<br/>　　叶尘眉头紧皱，脸上露出一抹凝重之色，
<br/>　　“这下面只怕更加凶险！”
<br/>　　他可以清晰的感应到，眼前的洞穴之中，有着无数毒虫毒兽，甚至还有着一道极为恐怖的气息，连他都没有百分之百的把握可以对付。
<br/>　　就在叶尘正纠结犹豫之时，药王宗的众高手，也已经从后面追赶了过来。
<br/>　　“桀桀！你就是叶狂仙吧？没想到啊，你竟然可以这么快就找上门来，看来本宗主还真是小瞧你了！”
<br/>　　说话的，是一名威严的中年男子，赫然正是药王宗的宗主云天青！
<br/>　　而在云天青的身后，还有着四名气息不弱于他的老者，则是药王宗的四大长老。
<br/>　　“小子！你还真是个痴情种啊，竟然为了一个女人，跑到这里来送死！”
<br/>　　其中一名长老开口道。
<br/>　　继而，
<br/>　　又有一名长老，指着一旁的祝小白，惊呼道：
<br/>　　“你们快看！那小子是不是火神宗，那个叫祝小白的余孽？”
<br/>　　“嘿！还真是踏破铁鞋无觅处，得来全不费工夫！没想到这小子居然也会出现在这里！”
<br/>　　“正好将他们一并拿下！”
<br/>　　……
<br/>　　几人叫嚣着，直接便朝下方的三人扑了过来。
<br/>　　而此时，
<br/>　　叶尘怀里的苏蔓，竟然开始撕扯自己的衣服，而且还在叶尘怀里不停的扭动、摩擦着，显然是毒力开始发作了。
<br/>　　叶尘无奈之下，只好伸手在苏蔓后脑勺上一拍，先暂时将她打晕了过去。
<br/>　　而与此同时，
<br/>　　四大长老中的一位，已经冲杀到跟前。
<br/>　　“小子！乖乖束手就擒！”
<br/>　　叶尘猛然抬头，眼眸中闪过一抹浓浓的杀意，
<br/>　　“既然你们找死，那我今日便成全你们！”
<br/>　　原本叶尘还打算，先将苏蔓和祝小白安顿下来，然后再找药王宗好好算账，现在看来，人算不如天算，唯有一战了！
<br/>　　“死！”
<br/>　　暴怒之下的叶尘，没有丝毫留手，当即祭出紫琼剑，狠狠一剑斩了出去！
<br/>　　轰！
<br/>　　一道长达超过千米的紫色剑芒，在空中划过，狠狠斩落了下来。
<br/>　　“什么！”
<br/>　　云天青和其余三大长老，顿时全都为之色变，显然都没有想到，眼前这个少年，一剑之威，竟然强大到如此地步！
<br/>　　“三长老小心！”
<br/>　　眼看如此强悍的一剑斩落下来，仅凭三长老一人，肯定难以匹敌，四人当即纷纷出手援助。
<br/>　　轰隆隆！
<br/>　　伴随着天空一阵巨响，
<br/>　　五人合力之下，总算接下了这一剑，不过五人却也被逼的连连倒退数步，方才站稳身形。
<br/>　　“好强！”
<br/>　　“看来传言不虚！就凭刚才那一剑，此子的实力，在那天榜之上，也能排进前五！”
<br/>　　“大家小心，不可轻敌！”
<br/>　　……
<br/>　　经过这么一交手，五人顿时收起了小觑之心，纷纷打起精神，开始对叶尘展开围攻之势。
<br/>　　而与此同时，
<br/>　　叶尘能够清晰的感受到，药王宗的其他高手，也纷纷朝他们的方向赶了过来。
<br/>　　望了一眼怀里的苏蔓，还有旁边的祝小白，叶尘的脸上，也终于露出一抹凝重之色。
<br/>　　他现在的情况十分不妙，苏蔓还等着他解毒，祝小白的生死，他也不能不顾，偏偏又被困在了这个绝地之中，形势可谓是大大不利。
<br/>　　“用五毒大阵！”
<br/>　　随着药王宗宗主云天青一声暴喝，五人分别占据五个方位，将叶尘围在中央，同时在空中一阵虚点。
<br/>　　轰隆隆！
<br/>　　整个山谷之中，顿时毒气四起，瞬间将上方的天空完全遮挡。
<br/>　　“小子！我不得不承认，你的实力很强，可惜这是我药王宗，今天你非死不可！桀桀！”
<br/>　　云天青狂笑一声，当即朝手腕上的乾坤镯猛然一拍！
<br/>　　轰！
<br/>　　自他的乾坤镯之中，竟然窜出了一头巨蛇，而且浑身毒气缭绕，一看就知道是剧毒之物。
<br/>　　与此同时，其他四大长老也纷纷出手，
<br/>　　轰轰轰！
<br/>　　四大长老竟然也如云天青那般，分别召唤出了四头庞大的毒物，分别是蜈蚣、蝎子、蜥蜴和蟾蜍。
<br/>　　叶尘顿时眉头一皱，
<br/>　　“又是炼尸之术！”
<br/>　　之前在逍遥山庄，叶尘和那位夏侯长老交手的时候，就曾经见识过这门邪术。
<br/>　　虽然这种旁门左道，叶尘并不放在眼里，可是眼前五人实力，远远超过之前的夏侯长老，而且五人联手，借助阵法之力，便是叶尘也也不能等闲视之。
<br/>　　轰轰轰！
<br/>　　五头巨大的毒物，分别从五个方位，朝叶尘围攻而来。
<br/>　　“哼！”
<br/>　　叶尘冷哼一声，一手搂着苏蔓，同时闪电般砍出五剑！
<br/>　　铿铿铿！
<br/>　　以叶尘如今的实力，便是元婴境后期的高手，也不敢硬接他的紫琼剑一击。
<br/>　　然而，
<br/>　　眼前的这五具妖兽尸体，是经过云天青等人，经过无数次淬炼的法宝，不但剧毒无比，而且根本没有任何痛觉。
<br/>　　纵然叶尘全力之下，一时间也难以将其毁掉。
<br/>　　轰轰轰！
<br/>　　一转眼的功夫，
<br/>　　双方已经交手了数十招，那五具妖尸在五大高手的操控之下，配合的亲密无间。
<br/>　　叶尘虽然依旧可以占据上风，可是要分心护着苏蔓和祝小白，想要取胜也是极为不易。
<br/>　　而云天青似乎也看出了一点，竟然开始向祝小白发起猛攻！
<br/>　　忽然，
<br/>　　“啊！”
<br/>　　祝小白不小心被一道毒气扫中，直接发出一声惨叫，竟然晕倒在地。
<br/>　　叶尘顿时大急，当即便要上前搭救，
<br/>　　而这时，云天青瞅准机会，突然向叶尘猛攻！
<br/>　　叶尘心头微微一叹，只好舍弃祝小白，带着苏蔓连连倒退了十多米。
<br/>　　“桀桀！”
<br/>　　云天青见逼退了叶尘，顿时发出一道邪笑，同时伸手虚抓，登时便将祝小白一把擒住，
<br/>　　“叶狂仙，我虽然不知道，你和这火神宗的小孽种是什么关系，不过你如果不想他死的话，就给我乖乖投降吧！”

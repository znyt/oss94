　　第317章
<br/>　　程大雷将樊梨花引到书房，气氛有些不对，其他人都退了出去。房间内就只剩下程大雷和樊梨花二人，于是气氛更加不对。
<br/>　　程大雷坐在椅子上，如坐针毡，会不会自己一句话说不对，樊梨花便站起来给自己一顿胖揍。
<br/>　　以她的脾气，不是做不出来呐。
<br/>　　银眸此刻端着茶进来，稍稍打破了尴尬局面，可以她的麻木，也察觉到了房间里的气氛不对，只是稍稍立一立，就退下了。
<br/>　　没有给程大雷把剑留下。
<br/>　　今日樊梨花裹着白貂毛领，绒毛拥着脸蛋，欺霜压雪的白。
<br/>　　“这个……那个……好久不见哈。”程大雷深吸一口气，今天可不能乱说话，不然有性命之忧。
<br/>　　樊梨花冷哼一声，将头侧到一边，眼里是满满的怨意。
<br/>　　“哈，我还正打算去看你呢，只是一直忙，还没抽出时间呢，你倒是来了。”
<br/>　　“忙什么？”樊梨花终于开口。
<br/>　　程大雷就简单把最近做的事情与樊梨花说了，只是将忙碌程度乘以二。程大雷发现，樊梨花还是很好糊弄呢，当然，好听些是单纯，难听些是单细胞。程大雷这么说，她就信了，眼中的怨意稍减。
<br/>　　程大雷松口气，这也不困难嘛，于是，他便镇定下来，忽想起一件事，自己敲了敲脑袋。
<br/>　　“你看我差点把正事忘了，我有个小物件，正准备送给你呢。”
<br/>　　樊梨花视线追随着程大雷，见他从炕上的柜子里拿出一个玉白手镯，递到樊梨花面前。
<br/>　　“给我的？”樊梨花目光变得柔和起来。
<br/>　　“可不是，喜欢么？”程大雷喜欢收集这些小物件，匕首，玉镯，耳坠等等，倒也可以凭此判断各地的审美。不过今日，正好应应急。
<br/>　　“来，我给你戴上。”
<br/>　　见樊梨花没有反对，程大雷便牵起她的手腕，入手只感觉冰肌玉骨，心中忍不住一荡。
<br/>　　樊梨花微微低下头，两颊飞起红霞，垂涎欲滴。程大雷暗暗叹息，倒真是个惊心动魄的美人，天予不取，怕是会折损阳寿吧。
<br/>　　这时樊梨花微微抬头，以目光打量着程大雷，程大雷一张丑脸，此刻看着也顺眼许多。二人目光相对，屋子里仿佛飘起桃花。
<br/>　　“大当家，大当家……你看谁回来了……”
<br/>　　徐神机的声音响起，一手把门帘撩开，杵在门口。看到屋内这一幕，他一张老脸顿时楞住了。
<br/>　　程大雷立刻将樊梨花的手松开，樊梨花更是尴尬得无地自容。程大雷觉得眼前已经够尴尬了，可看到徐神机身后立着的三人，尴尬程度立刻上升到最高级。
<br/>　　身后立着三个女子，风尘仆仆，打眼往房间里望着，正是苏樱、柳芷和苏樱的丫鬟小桃。
<br/>　　说巧不巧，秦蛮去接苏樱，一去月余，恰在这个时候，把苏樱接了回来。
<br/>　　苏樱本是一腔思念，见到这一幕，脸上表情也怔了怔。孤男寡女，同处一室，在这个沾衣裸袖便为失节的年代，很难让人相信程大雷是清白的。
<br/>　　更何况，程大雷也不是那么清白。
<br/>　　程大雷清了清嗓子，装出道貌岸然的模样，想着怎么把眼前的事情遮掩过去。可他还没想好说辞，苏樱已经先开口。
<br/>　　“你快出来，还有人来了。”
<br/>　　程大雷大步离开书房，见秦蛮和关鱼等人也是刚刚回来，有两个担架，上面抬着两个重伤的人。
<br/>　　程大雷走近一看，顿时吓了一跳。担架上重伤昏迷的二人，不是旁个，正是林冲和林少羽。
<br/>　　他们怎么会在这里，却又如何伤成如此模样。
<br/>　　“刘悲，刘悲！”程大雷大叫，此刻刚才的尴尬也顾不得了。
<br/>　　二人被抬到房间里，立刻有人给二人清理伤口，刘悲亲自给二人检查。二人身上的惨状，令程大雷胆战心惊。
<br/>　　从蛤蟆寨最初五个人走到现在，程大雷对于刚开始的几人还是有很深感情的。现在林少羽和林冲奄奄一息，程大雷生怕他们二人撑不过去这一关。
<br/>　　把秦蛮叫到身边，问他怎么逢着了二人。
<br/>　　秦蛮脸色也不好看，低声冲程大雷道：“少羽……好像入了正义教。”
<br/>　　程大雷一听，顿时咬牙切齿，口中大骂：“这……糊涂啊！”
<br/>　　当初长安一别，林少羽言说要和林冲退出江湖，择一山清水秀之地终老。程大雷虽然难舍，但在这样一个乱世，能如此了解一生，未必不是好事。
<br/>　　可林少羽终究闲不下来，却反而加入了正义教。程大雷现在虽然已不对正义教格外关心，但陆陆续续的消息传来，他也知道最近发生的事情。
<br/>　　李星的扬州兵攻打正义教老巢，正义教人数虽多，却只是一群乌合之众，短时间内就被击破，教众四散而逃。
<br/>　　林冲和林少羽也是正义教的教徒，路上被官兵追杀时，恰好碰上秦蛮一伙。幸运的是，秦蛮将他救回到这里，不幸的是，林冲和林少羽都受了不清的伤。
<br/>　　程大雷松口气，如果能治好他们身上的伤，二人就留在蛤蟆寨，大家还是一场兄弟，岂不快哉。关键是，林少羽能否撑过去。
<br/>　　二人全身衣物被剥尽，用湿布沾着热水清理伤口，再敷上疗伤的草药。刘悲又给二人开了两剂内服的方子，以现在的医疗水平，能做的也就仅仅是这些了。
<br/>　　程大雷一直守在二人身边，看二人身上伤痕累累，有些伤口已经重度感染，全身滚烫，以二人的水平，可能根本撑不过去。
<br/>　　不能这样……程大雷暗暗想，他急忙吩咐赵子龙、高飞豹把浩甲城的西门春长抓过来，城内还有什么名医，都一块抓过来给二人治伤。
<br/>　　仅仅靠此，或许也没办法救活二人，依照这个时代的医疗水平，像二人这样程度的伤，只能听天由命。
<br/>　　程大雷不想听天由命，而他或许也不用听天由命。
<br/>　　程大雷把自己关进办公室，深吸一口气，打开了系统界面。
<br/>　　“系统……”
<br/>　　“听到请回答，听到请回答……”
<br/>　　“无所不能的系统之神……”
<br/>　　“山贼之心……”
<br/>　　良久，良久，系统界面上浮现一行字。
<br/>　　你好……

        &lt;h3&gt;第三卷第七十八章  张函碰壁&lt;/h3&gt;
<br/>
<br/>        张函听江俞轩说陈子昂要去甸城，况且那天江俞轩给他打电话说陈子昂的耗材需要单位试用，他也帮着联系了一个朋友，价格也是按照陈子昂给别人的价格走。
<br/>
<br/>        他和陈子昂还有没有将来，他已经不知道，但是他还有俩个可爱的孩子，曾经他年少的梦想是那么的高尚，现在他的梦想是要他的孩子能安全的成长。
<br/>
<br/>        他到西城，是调查一些事情，这些事情都是机密的，他还要见闫欢。
<br/>
<br/>        阴差阳错的他们知晓了陈子昂没有考上大学的情况，当然这个不是他们调查出来的，是有人无意识的说出来的。
<br/>
<br/>        闫欢是怎么到的甸城，他也想知道，他还想知道陈子寒的情况，因为闫欢见过陈子寒，陈子寒为什么会在甸城呆那么久的时间？为什么会成为谷老大的儿子？
<br/>
<br/>        张函在绝密档案室里呆了好几天。
<br/>
<br/>        “函哥，你来看。”张倩楠喊他。
<br/>
<br/>        张函从卷宗里抬起头，当他看到陈子寒三个子时，他激动的手都抖起来了，这个档案是压在最底层，他们已经找了好久了。
<br/>
<br/>        “倩楠，放进原位置，这个事情谁也不能说。”
<br/>
<br/>        “是！”
<br/>
<br/>        “明天，我要去见一个人。”张函对张倩楠说。
<br/>
<br/>        “好的，我们这样调查好吗？”
<br/>
<br/>        “倩楠，我有私心没错，但是我更是为了我们身上的这身衣服。”
<br/>
<br/>        陈子寒在甸城一个人孤身奋战这么久，他一定很孤独吧？
<br/>
<br/>        如果陈子昂再去甸城的话是不是会给陈子寒增加困难？或者是陈子寒碰到了困难？需要帮助？
<br/>
<br/>        闫欢对甸城还算比较熟悉，他决定是找闫欢了解一下情况。
<br/>
<br/>        第二天，张函给陈子悦打了个电话
<br/>
<br/>        “子悦姐。”陈子悦比陈子昂大，原本他也曾经叫过姐，现在有了俩个孩子，无论如何也是应该叫姐的。
<br/>
<br/>        “张总。”陈子悦客气的回应着。
<br/>
<br/>        这么些年他们都不曾联系过。
<br/>
<br/>        “闫欢在吗？我想见一下闫欢。”张函说
<br/>
<br/>        “哦，你没有闫欢的电话啊？”
<br/>
<br/>        “我应该是没有她的电话。”张函记得他没有闫欢的电话。
<br/>
<br/>        “行，我把电话给你吧，你和他联系。”
<br/>
<br/>        “不用了，你告诉她，下午2点在西城mm咖啡馆见。”张函并不打算要闫欢的电话。
<br/>
<br/>        “好！”陈子悦忙忙碌碌的，他们现在光西城的那个商场都已经是西城数一数二的大企业了，里面吃喝玩乐一应俱全。人们从吃饭到采购全部都是一条龙的服务，各种特色小吃摊位都是应接不暇的，到中午了必须的等座。
<br/>
<br/>        她陈子悦已经是西省知名的企业家了。
<br/>
<br/>        闫欢说要去甸城，但是陈子昂不动，她也不动，一直等着和陈子昂一起走，因了陈子昂手头上的事情一个接一个，陈子昂也就一而再再而三的推迟去甸城的计划。
<br/>
<br/>        闫欢和陈子悦是西省的一对姐妹花，商圈里无人不知无人不晓。她们凭着一腔热血和吃苦耐劳的精神硬是创出来了一片天地，她们没有依靠，她们没有背景，她们只是普普通通的人，在创业的路上，俩个女人同心协力，携手前进，无论多苦，无论多累。
<br/>
<br/>        如今，她们站在行业的顶端叱咤风云，笑傲江湖，无论是以前还是以后，人们的衣食住行是不会改变的，只要有人消费，她们的流水就会源源不断，陈子悦成立了一个妇女儿童基金会，从每天的收入里都会拿出一部分钱来帮助一些妇女儿童，还给家乡也成立了几个贫困生帮扶站，鼓励和支持那些家里贫困的孩子上学。
<br/>
<br/>        陈子悦转述了张函的话，对闫欢说：“张函见你有什么事情你知道吗？”
<br/>
<br/>        “谁知道呢，这么多年都没有联系了，只能是下午见面说。”闫欢隐隐约约能知道张函找她是什么事情，莫非陈子昂要去西城了？
<br/>
<br/>        依着张函的能力，不可能不知道陈子昂见了谷强，而她也想了解谷强这几年过得怎么样。
<br/>
<br/>        “莫不是想和你旧情复燃？”陈子悦开玩笑。
<br/>
<br/>        “有可能哦，我要小，三上位。”闫欢作了一个鬼脸。
<br/>
<br/>        “这几年都没有联系你了，突然联系你还真的感觉怪怪的。”陈子悦一脸沉思状。
<br/>
<br/>        “下午就知道了，我下午回来给你汇报战况，看我能不能顺利的拿下。”闫欢笑着说。
<br/>
<br/>        “算了吧，你想都别想，那个人除非是他想拿下你，你才有机会。”
<br/>
<br/>        “这么了解的？”
<br/>
<br/>        “你赶紧滚吧，要不然人家等急了哦，我们这里还有上好的白茶，给张函拿一些去。”
<br/>
<br/>        “这是不是姨姐看妹夫越看越欢喜？”
<br/>
<br/>        “切，那都是过去式了，现在是看你的薄面。”陈子悦一支签字笔丢给闫欢。
<br/>
<br/>        她现在是无闫不欢，以后这个蹄子嫁人了该怎么办？真后悔妈妈没有再多生一个哥哥或者弟弟，将她收纳到陈家。
<br/>
<br/>        闫欢也是乐呵呵的去见张函了。
<br/>
<br/>        当她推开咖啡馆小间的门的时候，她呆住了，印象中的张函是瘦弱而苍白的，眼前的人是张函吗？
<br/>
<br/>        “闫欢，你来了。”张函指着对面的沙发。
<br/>
<br/>        “你怎么越来与年轻，愈来愈帅气了啊？”闫欢无法置信的看着张函，相差太大了吧！
<br/>
<br/>        “不好吗？”
<br/>
<br/>        “好啊，当然好啊。”看来你家的母老虎对你很好，过得这么滋润的。
<br/>
<br/>        “你也是越来越美丽了啊。”张函夸了一下闫欢。
<br/>
<br/>        “谢谢。”
<br/>
<br/>        俩个人沉默了一会：
<br/>
<br/>        “喝点什么？卡布奇诺、拿铁亦或者的摩卡？”
<br/>
<br/>        “我喝点冰糖柚子茶。算了，红茶。”闫欢本来想点冰糖柚子茶，但是冰糖柚子茶也算是一种功夫茶，她没有必要和张函久坐。
<br/>
<br/>        “好，来两杯红茶。”张函按了呼叫器，礼貌的对服务生说。
<br/>
<br/>        “我找你想了解一点情况。”张函开门见山。
<br/>
<br/>        “什么事情？”闫欢心里一沉，果然是找她了解情况。
<br/>
<br/>        “谷强的事情。”张函看着闫欢。
<br/>
<br/>        “我不知道，我只是见过他。”闫欢是不会再告诉张函任何情况的。
<br/>
<br/>        “你之前不是这样说的。”
<br/>
<br/>        “之前是之前，现在是现在，我已经很多年都没有见过他了，你来找我了解情况？”闫欢说的是实话。
<br/>
<br/>        “我觉得你知道的应该告诉我。”
<br/>
<br/>        “我凭什么要告诉你？凭你是张家大少？凭你是陆家的女婿？张总，这几年顺风顺水，和陆太太恩爱有加吧？”闫欢语气不善。
<br/>
<br/>        “你好像对我有意见？”张函觉得闫欢不应该对他充满敌意。
<br/>
<br/>        “此一时彼一时，张总以前可是口口声声说自己是个情圣。”
<br/>
<br/>        “难道我现在就不是一个情圣了？”
<br/>
<br/>        “噗嗤。”闫欢的一口茶喷到桌子上了。
<br/>
<br/>        张函靠在沙发上，好整以暇的看着她。
<br/>
<br/>        “对不起，我这人真的没有办法和你这个大情圣交流，告辞。”闫欢站起来就走。
<br/>
<br/>        “闫欢，我对陈子昂的心从来都没有变过。”张函对走到门口闫欢说。
<br/>
<br/>        “你变或者不变，你的心都在你身上，和我无关。”闫欢折回来看着张函：
<br/>
<br/>        “谷强，曾经是我的爱人，谁要是想动他，我拼了这条命也会让大家不安宁。”
<br/>
<br/>        “闫欢，你变了。”
<br/>
<br/>        “人都是会在改变的，你也一样。茶钱我会付的，谢谢你曾经给我温暖。”闫欢头也不回的走了。
<br/>
<br/>        张函，每个人心里都有要守护的人，如果要选择，我会选择谷强，不，是陈子寒！
<br/>
<br/>        张函没有想到他会在闫欢这里碰壁，他更没有想到的是闫欢以为他和陆玉的关系很好，才抵触他的。
<br/>
<br/>        看着闫欢离去，张函又陷入了沉思，看来，他是不是也要去甸城一趟了？

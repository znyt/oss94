　　“太强大了！”
<br/>　　“这位血公子，不愧是巨门星域第一天才啊！”
<br/>　　“那小子必死无疑了！”
<br/>　　……
<br/>　　根本没有人认为，一个只有区区金丹境的小子，能够在血无影攻击下活命。
<br/>　　“叶尘哥哥……”
<br/>　　小丫头灵溪早已经泪流满面，不敢抬头去看。
<br/>　　然而，
<br/>　　过了片刻，
<br/>　　“什么！”
<br/>　　“那小子竟然没死？”
<br/>　　“这怎么可能！”
<br/>　　……
<br/>　　伴随着众人惊呼声，紧跟着，一道灵溪无比熟悉的声音，陡然响起，
<br/>　　“你就这点能耐吗？简直蝼蚁一般！”
<br/>　　一直埋着头没敢看的灵溪，顿时娇躯一震，继而美眸发亮，
<br/>　　“这是……叶尘哥哥的声音？叶尘哥哥没死？”
<br/>　　灵溪慌忙抬头一看，
<br/>　　只见，那道瘦削的少年身影，依旧无比淡然的站在那里，甚至都没有挪动分毫！
<br/>　　“耶！叶尘哥哥好厉害！”
<br/>　　灵溪顿时兴奋都要跳起来了。
<br/>　　而一旁的灵韵，则拼命的摇了摇头，满脸的难以置信，
<br/>　　“他的实力……竟然这么强！这怎么可能？”
<br/>　　她原本以为，叶尘只是一个名不见经传的小人物，怎么也没有想到，竟然可以挡得住血无影的强大一击。
<br/>　　不但灵韵懵了，黑岩城城主以及众高手也全都懵了。
<br/>　　柳云飞也不由得眉头紧皱，眼眸中尽是骇然之色。
<br/>　　他自问，方才血无影的那一击，自己也可以抵挡，可是绝对不可能像叶尘这般轻松。
<br/>　　更重要的是，他从头至尾，竟然都没有看清楚，叶尘是如何出手的！
<br/>　　至于柳云飞身后的那名于师弟，则不由得狠狠咽了一口口水，想起之前自己挑衅叶尘时的情景，再看到眼前的这一幕，心头不由得一阵后怕。
<br/>　　就连那位血公子，同样是一脸的惊愕，他刚刚虽然没有动用全力，可是对方更加从容，甚至连脚步都没有挪动分毫！
<br/>　　“你明明只有金丹境，怎么可能有如此功力……我不信！！”
<br/>　　血公子大吼了一声，双手猛然合十，
<br/>　　“血狱神碑！”
<br/>　　轰！
<br/>　　血公子的身上，陡然有一道猩红色的光芒闪烁，
<br/>　　下一刻，
<br/>　　那血公子的头顶之上，竟然浮现出一座高达数百米的巨碑！
<br/>　　那巨碑之上，还刻着一些古老而神秘的符文，并且周身上下，散发着血腥、邪恶的气息，似乎有无数怨灵，让人忍不住为之作呕、恐惧。
<br/>　　轰！
<br/>　　看到这一幕，周围顿时再次炸开了锅，并且所有人纷纷倒退，
<br/>　　“这难道就是……血月宗的至宝血狱神碑！”
<br/>　　“传言此法宝是一件准仙器，乃是血月宗的开山祖师血月老祖，在一处绝地中所得，据说是用极阴寒铁所锻造，炼化了上百万生灵的魂魄而来，乃是有名的上古邪物！”
<br/>　　“没想到啊，血月宗竟然已经把此物，也传给血公子！”
<br/>　　“难怪这血公子，在巨门星域肆无忌惮，以他的功力，再加上此等法宝，恐怕就算是元婴巅峰境的强者，也未必能占得了便宜！”
<br/>　　……
<br/>　　听到众人的议论声，原本已经重新燃起希望的灵韵、灵溪姐妹，顿时再次陷入绝望之中。
<br/>　　有血狱神碑在手的血无影，恐怕除非是化神天君出手，才有可能将其击败了。
<br/>　　纵然叶尘的实力，大大出乎他们的意料之外，可他毕竟只有金丹境，而且还如此年轻，有怎会是血公子的对手？
<br/>　　“叶尘哥哥，你快跑！不要管我们了！快跑！”
<br/>　　灵溪反应过来之后，当即冲着叶尘大呼起来。
<br/>　　叶尘神色淡然，冲着小丫头微微一笑，
<br/>　　“丫头，放心吧！区区一只蝼蚁，还伤不了你叶尘哥哥！”
<br/>　　叶尘这话一出口，众人顿时再次哗然，
<br/>　　这家伙，莫非是真的疯了不成？
<br/>　　血公子连血狱神碑都使出来了，他居然还敢说出这种话来，是谁给的他这么大的自信？
<br/>　　血无影听到叶尘这话，更是气得一佛出世，二佛升天，两眼红的都快和兔子有一拼了，
<br/>　　“不知死活的小子！本公子今日，不但要活剥了你，还要将你元神抽取出来，放入我的神碑之中，让万鬼撕咬！”
<br/>　　轰！
<br/>　　暴怒之下的血无影，双手猛然一落，
<br/>　　轰隆隆！
<br/>　　那巨大的血色巨碑，犹如一座大山，朝叶尘狠狠砸落下来！
<br/>　　刺啦啦！
<br/>　　巨碑在空中所过之处，连空间似乎都承载不住，硬生生被撕开了一道裂缝，让人心悸。
<br/>　　“这才是血公子真正的实力吗？果然强大的可怕啊！”
<br/>　　众人再次纷纷倒退，个个满脸骇然之色。
<br/>　　“叶尘哥哥！”
<br/>　　灵溪急的眼泪哗哗直流，如果不是灵韵拉着她，恐怕她早就不顾一切的冲上去了。
<br/>　　“唉！他其实也很厉害了，可惜遇到了血无影，注定……什么！”
<br/>　　灵韵叹息着，不过话才只说到一半，突然美眸一瞪，仿佛看到了什么不可思议的事情，忍不住惊呼了起来。
<br/>　　只见，
<br/>　　面对血无影那撼动天地的一击，叶尘左腿微微后撤一步，右手握拳、后拉，然后缓缓推了出去。
<br/>　　这一拳，看上去轻飘飘的，毫无力道，就像是婴儿才刚刚学会挥拳一般，似乎也毫无任何奇特之处。
<br/>　　然而，
<br/>　　就是这么弱不禁风的一拳砸出，
<br/>　　轰隆隆！
<br/>　　彭！！
<br/>　　那原本以碾压姿态，砸落在叶尘头顶的巨大神碑，竟然轰然破碎！
<br/>　　血狱神碑的本体，更是直接飞了出去。
<br/>　　而与此同时，血狱神碑的主人，血公子猛然感觉到一股强大的反噬之力！
<br/>　　继而整个人，顿时犹如断了线的风筝一般，直接倒飞了出去，并且口中狂喷了一口鲜血。

<br/>　　血无影人在空中，顿时发出了一道不甘的怒吼声。

<br/>　　可是如今，他居然被一个只有金丹境的小子，赤手空拳打败了？

<br/>　　“蝼蚁终究就是蝼蚁，就算法宝再好，也逆不了天！”
<br/>　　就在血无影栽落在地上，还没有从惨败的巨大打击中回过神来，一道淡漠的声音，陡然在他耳边响起。
<br/>　　“逃！”
<br/>　　血无影浑身一哆嗦，下意识的想要从地上爬起，然后赶紧逃命。
<br/>　　然而这时，他又猛然感觉到胸口一沉，一只脚掌落在了他的胸膛之上，将狠狠踩回地下！

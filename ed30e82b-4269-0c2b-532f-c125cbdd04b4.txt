　　他来到拍卖行，本就是想要买一些凝练身躯之物了，不过现在楚风眠却是改变了计划。
<br/>　　只要是完成圣祭，得到的好处，比起楚风眠得到一百枚像是龙血圣王丹那样的丹药还要强的多。
<br/>　　楚风眠的身躯，想要凝练到极限，并不容易，像是龙血圣王丹这样的丹药，他至少都需要上千枚以上才行。
<br/>　　从天商会中得到的那些奇物，也不过是杯水车薪。
<br/>　　不过现在得到了这枚涅槃龙蛋，却是给了楚风眠一条捷径。
<br/>　　只要是楚风眠布置出圣祭之术来，在将这圣祭龙蛋之中的灵力抽出来的话，足以将他的身躯，凝炼到极致，省去了无数麻烦。
<br/>　　“虚空海中最大的拍卖会，可不要让我失望。”
<br/>　　楚风眠心中暗自想到。
<br/>　　一行人，在空中飞行这，身边空间凝丈为寸，几乎不到一个时辰的时间，便是横渡混元大陆，来到了混元大陆的中心。
<br/>　　“这就是混元大陆中的第一大势力，通天阁？”
<br/>　　楚风眠一眼望去，就在前方，却是有着一座巨大无比的高楼，足足是有着千米之高，万米之宽，巨大无比。
<br/>　　一眼望去，便是极为显眼。
<br/>　　在这高楼中，有着无数商铺，其中无数武者，密密麻麻，从中穿行，做着生意。
<br/>　　单单就是这通天阁，就要比起秦皇国度中的九鼎古城要繁华无数倍。
<br/>　　九鼎古城，跟着通天阁相比，都只是蝼蚁，这才是真正的交易重地。
<br/>　　“我们有着拍卖会的资格，直接前去吧，这通天阁中，还有着一些店铺，楚公子想要逛逛的话，可以等拍卖会结束在来看，毕竟拍卖会要紧。”
<br/>　　李长老笑着开口道。
<br/>　　一行人，便是直接想着让通天阁最低最为顶端飞了上去。
<br/>　　落入到了通天阁最高的一层之中去。
<br/>　　“请出示身份资格。”
<br/>　　几尊武者，看到楚风眠一行人过来，便是极为客气的开口道。
<br/>　　能够来参加这拍卖会的人，都是虚空海中一些有名气的人物了，不然的话，连拍卖会资格都得不到。
<br/>　　要知道单单这资格的价值，就足以抵得上一块王品灵石了，许多武者一身财富都买不起一个资格。
<br/>　　李长老从怀中，随意拿出一个玉符，递了过去。
<br/>　　在这玉符上，便是刻画有着通天二字。
<br/>　　“我来带两个朋友进去，不碍事吧。”
<br/>　　一尊武者接过玉符，一股灵力注入其中，上面便是显现出来了李长老的身份信息。
<br/>　　“原来是天商会的李长老。”
<br/>　　几尊武者恭敬的行礼道。
<br/>　　“以李长老的身份，当然没问题，天字第十一号包厢，请进。”
<br/>　　一尊武者，便是带着楚风眠一行人，直接走入到了一座包厢之中去。
<br/>　　这包厢隐秘无比，上面布置这无数禁制，比起天商会的包厢，还要隐秘的多，显现出这通天阁的实力来。
<br/>　　这样的包厢，一共足足有着一百个，其中位于最为顶端的二十个，便是天子号。
<br/>　　至于下方八十个，则是地字号。
<br/>　　至于玄字号，黄字号的，都是下方的座位，不过就算是座位，也都是宽敞无比的。
<br/>　　极为舒适。
<br/>　　“哦？这里包厢中的水果，居然都是玄级的灵药？”
<br/>　　楚风眠坐在其中的一座王座上，看着旁边摆放的水果。
<br/>　　这水果，是一种名为紫青果的灵药，对神力境之下的武者，都这可以固本精元的作用。
<br/>　　虽然价值不高，但是在这包厢中，居然被当做水果，倒是极为奢侈的行为了。
<br/>　　“通天阁的拍卖会上，天字号包厢中的人，都是虚空海中有头有脸的人物了，通天阁肯定会好好招待的。”
<br/>　　李长老笑了笑道。
<br/>　　“不过也就只有天字号包厢，才会如此奢侈，下面的，则就远远不如了。”
<br/>　　“这紫青果，虽然品级不高，但是吃起来倒是极为甘甜爽口，在虚空海倒是极为畅销。”
<br/>　　李长老一边说着，便是拿起了其中一枚紫青果，吃了起来。
<br/>　　楚风眠在一旁，也是同样，拿起了一枚紫青果，咬下入口，便是一股甘甜味道，充斥在楚风眠的口中。
<br/>　　这种甜味，令人都是神清气爽，还有着一股股精粹灵力，融入到了楚风眠的口中。
<br/>　　这一股灵力，对楚风眠这个级别的武者来说，倒是没有任何的意义了，楚风眠吐息一次的灵力，都是这紫青果的上百倍。
<br/>　　不过对御风境武者来说，倒是有着不小的作用了。
<br/>　　“不错。”
<br/>　　楚风眠点了点头，脸上都是露出几分满意。
<br/>　　这种紫青果，的确是极为爽口。
<br/>　　“楚公子喜欢，不妨买一些，放入到空戒之中去储存起来，反正紫青果极为便宜，以楚公子的身家，根本不需在意。”
<br/>　　李长老笑了笑道。
<br/>　　“这次的拍卖会上，居然是会有着一件极道仙兵的残片出世，不知道楚公子感不感兴趣？”
<br/>　　“极道仙兵的残片？”
<br/>　　听到李长老的话，楚风眠脸色微微有些变化。
<br/>　　一听到这极道仙兵的残片，楚风眠便是想到了人皇罐。
<br/>　　极道仙兵，在九域中本就是稀少无比，可以说真正在远古古籍中记载的，都不足两位数。
<br/>　　尤其是极道仙兵，强大无比，要被硬生生打碎，哪里那么容易。
<br/>　　楚风眠脸色平静，但是心中灵识，却是悄然一动，一下子蔓延出去。
<br/>　　他的灵识，比起生死境高阶的灵识都要强大的多，悄无声息的离开了包厢中，向着通天阁内部散去。
<br/>　　就在这通天阁内部，一座石台上，拜访这无数宝物，每一件都是价值极高的。
<br/>　　这些，应该都是这次拍卖会的拍品。
<br/>　　而就在这无数宝物之中，其中一件的气息，却是让楚风眠无比熟悉。
<br/>　　在其中一座玉盒中，正是放着一枚残破的瓦片。
<br/>　　“人皇罐碎片！真是的人皇罐碎片！”
<br/>　　楚风眠心中都是震惊无比，没想到居然又是让他碰到了一块人皇罐碎片。
<br/>　　这人皇罐碎片，破碎飘散在九域中的各个角落，能够碰到一个都是极难的一件事。
<br/>　　之前从虚空五霸的手中得到一块，已经是楚风眠极大的运气了，没想到这拍卖会上，居然还有着一块。

　　“拜见老师，不知老师唤弟子前来，所谓何事？”
<br/>　　玄都一身棕色道袍，一头青丝束起，头戴金色道冠，二十来岁的模样。一见到老子，便恭敬的行了一礼，拜道。
<br/>　　“嗯。”太上看着玄都，满意的点了点头，他这个弟子是他当年在人族传教之时所收的。当年玄都逢山拜山，遇水拜水，只为求仙问道，他一看见玄都，便觉得对自己的胃口，于是便将其收为弟子，作为传人。
<br/>　　如今玄都也的确没有让他失望，一身修为达到大罗巅峰，算得上是三教中的顶尖人物了。与他差不多的也就只有那阐教的广成子、燃灯，截教的多宝道人，无当圣母。
<br/>　　“如今人族圣皇悟道到了关键的时刻，缺少一件至宝，为师此次叫你前来，便是让你前去助那人族圣皇一臂之力。”
<br/>　　“至宝？为何至宝？”玄都不解道。
<br/>　　“河图、洛书。”
<br/>　　“河图、洛书？那不是妖族以前的先天灵宝吗，听说在三族大战之中，这两件至宝被妖师鲲鹏最后给带走了。”
<br/>　　玄都还是比较了解当年的历史的，毕竟，当年他也是人族，当时也正是他拜入太上门下不久。
<br/>　　“不错，正是那两件灵宝。”太上点头道。
<br/>　　“那徒儿此次便是要想那鲲鹏去借两件至宝吗？”玄都问道。
<br/>　　太上看了一眼玄都，取出一座三十三层的金色宝塔，玄黄之气丝丝垂下，充满着无尽奥妙，正是天地玄黄玲珑宝塔。
<br/>　　“你拿着为师的这座宝塔去北冥宫一趟，想必你知道该怎么做吧。”说着，太上将宝塔扔给玄都。
<br/>　　玄都面色淡然，眼中目光一闪，说道：“弟子知晓。”
<br/>　　“嗯，去吧。”
<br/>　　“弟子告退！”玄都恭敬的行了一礼，转身出了太清宫，接着便拿着宝塔向着北冥方向飞去。
<br/>　　北冥地处北方，这里并没有什么其他的势力，只有一方势力，那便是鲲鹏所创立的北冥宫。当年剩下的一些妖族，被圣人驱赶到北方大地，也有一些投靠了鲲鹏，如今的北冥宫，在洪荒中也是顶尖的实力，就是因为这里有鲲鹏这一位准圣后期的强者。
<br/>　　这一日，玄都只身来到北冥地界，觐见妖师鲲鹏。玄都乃是圣人弟子，一言一行都代表着圣人，鲲鹏自然是不敢怠慢，命人将其引入北冥宫。
<br/>　　“不知玄都法师来我北冥宫，所谓何事？若是鲲鹏能有所帮助，定然不会推辞。”鲲鹏此时还不知道玄都的目的，但本着圣人不可得罪的态度，鲲鹏将自己的态度尽量放低。
<br/>　　“是吗？”
<br/>　　玄都诡异一笑，说道：“那好，贫道正好有一件事情需要鲲鹏前辈帮忙，而且此事非前辈不可。”
<br/>　　“什么事？”鲲鹏一看玄都的模样，便知道不会是什么好事，心中顿时有一种不祥的预感。
<br/>　　“听说当年妖族的河图、洛书两件灵宝在前辈手中，玄都此次便是前来欲向前辈借来一用。”玄都笑了笑，开口说道。
<br/>　　鲲鹏一听此话，心中顿时就咯噔一声，脸色渐渐变得阴沉下来了，他鲲鹏本身就没有什么好的灵宝，全身上下如今最好的灵宝便是那河图、洛书了，玄都口中虽说是借，但谁都知道，这是有借无还，内在意思分明就是在抢。
<br/>　　“若是我不同意呢？”鲲鹏此时早已经没有之前那种心态了，脸色变得不太好看，沉声道。
<br/>　　“这可由不得前辈了，这是圣人的决定，前辈还是好好配合一下晚辈，否则晚辈可不好交差啊。”玄都面色也低沉下来，开口说道，隐隐有威胁之意。
<br/>　　“就凭你？不是吾看不起你，实在是你太弱了。想要从吾此处拿走河图与洛书两件极品先天灵宝，还不够！”鲲鹏冷笑一声道。
<br/>　　“哦？是吗？”
<br/>　　玄都笑了一声，毫不在意鲲鹏的态度，接着，他将老师给他的天地玄黄玲珑宝塔取了出来，悬到空中。
<br/>　　“鲲鹏前辈，你确定不借？”玄都笑吟吟的看着鲲鹏。
<br/>　　一看到玄黄塔，鲲鹏的脸色更加不好看了，因为他认出了那是太上圣人的宝塔，其内蕴含着圣人的攻击。不过，转念一想，鲲鹏心道，不就是相当于圣人的一尊分身吗，凭我准圣后期的实力，就算打不过，逃跑也应该可以吧，毕竟人家万劫魔尊还没有成圣，便能够战圣，自己虽然不及他，但对付一尊圣人分身应该也可以吧。
<br/>　　心中如此想到，鲲鹏不由得内心一定，取出河图与洛书两件法宝，法力催动，瞬间变得巨大，日月星辰，太古妖文，山河空间，瞬间出现，风云突变，周边空间被禁锢起来，河图、洛书毫光大涨，变成一道道封印，向着玄都飞去。
<br/>　　“找死！”
<br/>　　玄都见此，只听一声冷哼，他立即催动天地玄黄玲珑宝塔。
<br/>　　“嗡！”
<br/>　　法力刚一注入，宝塔就嗡嗡作响，紧接着它瞬间变得巨大，遮天蔽日，轰破周边空间禁锢，金光闪闪，宛如太阳，无尽玄黄之气犹如瀑布一般冲击下来，形成绝对防御姿态，万法不侵。
<br/>　　一股强大的圣人威压毫无征兆的落了下来，将鲲鹏压落在地。
<br/>　　接着，三十三层宝塔瞬间向着鲲鹏镇压下来，犹如三十三层天，恐怖无比的威压，直接将周边的空间震碎，一股躲无可躲、逃无可逃的恐怖禁锢之力，向着鲲鹏镇压而来。
<br/>　　“吼！可恶！怎会如此恐怖？！”
<br/>　　鲲鹏面色大骇，眼中闪过惊恐之色，仅仅一座宝塔，便让他感受到如此恐怖的威压，若是圣人出手，那还了得，到时自己岂不是不能接下一击。
<br/>　　感受着那股恐怖无比的威压，鲲鹏毫不怀疑，它能够将自己镇压。眼见危机越来越近，鲲鹏一声大吼，他将自己全部法力道则，灌注在一起，形成全力一击，向着那玄黄密布的宝塔冲击而去。
<br/>　　“轰！”
<br/>　　狠狠的一击，鲲鹏瞬间便被宝塔恐怖的威压轰出数万丈之远，周遭空间寸寸碎裂，化为空间风暴，地水火风涌动，暴乱无比。
<br/>　　再向鲲鹏看去，只见鲲鹏的一只手臂直接被粉碎，他如今的模样极为狼狈，一身血迹，满头乱发披散，眼中闪烁着阴毒狠辣之光，但更多的却是惊骇，以及深深的忌惮。
<br/>　　他怎么也想不到，圣人仅仅凭借一件灵宝一击，便将自己弄得半残。这一刻，他对圣人的强大，更是有了一个直观的了解。
<br/>　　深深的看了一眼虚空之中的玄都，鲲鹏眼中闪过一丝怨毒与不甘，之后毫不犹豫地转身向着远处飞去，若是鲲鹏真心要跑，玄都也拦不住。
<br/>　　看着逃跑的鲲鹏，玄都轻笑一声，挥手将虚空之中被玄黄玲珑塔禁锢的河图与洛书收走，不再多想其他，而是转身向着人族部落飞去。
<br/>　　毕竟，他此次的目的便是，拿到河图与洛书两件灵宝，交给人族圣皇悟道。如今宝贝已经到手了，可人族还没有去。
<br/>　　说来，自己好像也是人族啊，好久没有回去人族看看了，趁着此次机会，便回部落一趟，看看人族的情况。
<br/>　　如此想着，玄都的速度却一点也不慢，一道流光，划破天际，转瞬即逝。

　　被发现了！
<br/>　　和尚暗道一声，急忙转过身来，双臂交叉着护住头部。
<br/>　　他感受到一股劲风，那个发现他的人，从身后对他发动了攻击。
<br/>　　“砰！”
<br/>　　一道模糊的白影闪过，和尚手中涌来巨大的力道，瞬间整个人向后倒飞出去。
<br/>　　太快了！
<br/>　　面对这样强大的敌人，他根本没有半点还手的余力。
<br/>　　“哼，蝼蚁……你可知道什么叫做电子产品？”
<br/>　　一击得手，敌人却并未追击，趁着这段时间，和尚也看清楚了袭击者的样子。
<br/>　　一身白色的戏服，上面绣着一些青花，虽然是在战斗，背后的两柄短刀却没有抽出，和人类大概相似的长相，即使涂了一层白粉依旧这挡不住脸上纵横交错的尸纹。
<br/>　　如果不是身处这种环境下，估计和尚都会认为，这是一个刚刚参加完演出的戏子。
<br/>　　这……不是人类！
<br/>　　“不肯说么……那这样，就不好办了呢。”
<br/>　　戏子的声音有些阴柔，嘴上虽然这样说，但动作却没有丝毫停顿，一串残影闪过，戏子突然出现在了和尚面前。
<br/>　　“那就只能杀了你哦。”
<br/>　　咧了咧嘴，戏子不知何时抽出了双刀，径直割向和尚的脖颈。
<br/>　　“锵！”
<br/>　　眼看和尚即将被斩首的时候，一串佛珠猛然亮起了金光，险之又险的架在了刀上。
<br/>　　“咦~这只老鼠……居然还有同伴！”
<br/>　　一击未中，戏子毫不贪心，迅速的将双刀抽回，警戒着周围的情况。
<br/>　　那个潜藏在暗处的敌人，他竟然完全没有察觉，如果不是对方出手，甚至他现在都不知道。
<br/>　　也就是说，要么暗处的那个敌人善于隐藏，要么，就是实力超出他太多，不管哪种，对他来说都不是一件好事。
<br/>　　“这里是鬼棍，发现三名幸存者，队长，你那边情况怎么样？”
<br/>　　就在场面僵持的时候，和尚腰间的对讲机中，传来鬼棍的声音，稍一犹豫，他取下对讲机来。
<br/>　　“我这边碰到麻烦了，有一个敌人，十分强大，刚才如果不是有高人相助，恐怕我就要交代到这里了。
<br/>　　说罢，和尚双手抱拳，微微行了一稽，道：“不知是哪位朋友相助，小僧在这里谢过了。”
<br/>　　闻言，戏子双目也是微微一凝，竖起了耳朵来。
<br/>　　他也在等暗处的那个敌人出来。
<br/>　　“你脖子上的那串佛珠，是哪里来的？”
<br/>　　缥缈的声音响起，但刘哔却没现出身影，借助着灵力，即便是发出了声音，也都无法让人辨认出他的位置。
<br/>　　“这个……”和尚的表情微微一愣，扯下脖子上的佛珠，有些疑惑：“这是小僧下山的时候，师父交给我的，是很常见的佛器啊，先生的意思……莫非这串佛珠有什么玄机？”
<br/>　　似乎是个很重要的信息，刘哔心中微微一动，现出身来。
<br/>　　“你师父在哪？长什么样子，有什么特征，都和我说一下。”
<br/>　　看着突然出现在场中的刘哔，和尚以及戏子都有些吃惊，他们一直在警戒着周围，但对方是如何出现的，他们却一点都没能察觉到。
<br/>　　但刘哔根本没有在意他们是如何想得，只是继续开口，解释道：“如果我没猜错，你的师父，可能是我从未谋面的同门师兄，我一直在找他。”
<br/>　　闻言，和尚的脸色有些古怪，师兄……他根本不相信这个解释，这年龄相差，实在太大了吧。
<br/>　　虽然也有一些专门用来驻颜的功法，但佛门之人，可是从来不修那个，毕竟，那种东西和他们所推崇的，终归有些冲突。
<br/>　　再说，既然是同门师兄，从未谋面又是什么鬼……
<br/>　　不过，他虽然好奇，却也没问，有些事，不需要那么较真。
<br/>　　这只是无伤大雅的事情罢了，现在最重要的，是如何借助自己这个莫名出现的师叔，从这个地方逃离出去。
<br/>　　那个白色戏服的家伙，根本不是他可以招惹的。
<br/>　　“师父早就下山游历去了，小僧也不知道现在在哪，至于长相特征，好像没什么出奇的地方，如果非要说一个，可能就是驼背了。”
<br/>　　和尚眼中闪过一抹沉思，虽然不想透露太多消息，但对于刘哔的问话，还是稍稍回答了几句。
<br/>　　驼背……
<br/>　　那应该不是了。
<br/>　　想起保安小哥笔直的腰杆，刘哔皱起了眉头。
<br/>　　线索到这里，又断了。
<br/>　　他还是没能搞明白，那个袭击他们的家伙，到底是因为什么。
<br/>　　“那个……师叔，你看我们现在……”
<br/>　　看着陷入沉思的刘哔，和尚内心有些着急，小心翼翼的开口问道。
<br/>　　“师叔？”被打断思考，刘哔眼睛一瞥：“谁是你师叔？”
<br/>　　闻言，和尚的脸色顿时变的相当精彩，刘哔的言下之意，明显就是不想带他一起离开。
<br/>　　“额……您不是说我师父和您是同门是兄弟吗？”
<br/>　　有些不死心，和尚追问了一句，却收到刘哔扔过来的一个白眼。
<br/>　　“没错啊，但那和你有什么关系？”
<br/>　　看着和尚几欲吐血的表情，刘哔打了个哈欠，失去了调笑的兴趣：“我还想起有事要做，你们继续，打扰了。”
<br/>　　说罢，不等两人在说什么，刘哔身影一闪，直接从原地消失。
<br/>　　这一番变故，不仅仅是和尚，就连戏子也稍微有些发懵，一直等到刘哔离开，都没反应过来到底发生了什么。
<br/>　　“既然如此……”
<br/>　　戏子喃喃了一句，试探性的出手，却留下了一份力道，以免那人反悔，回头杀他一个措手不及。
<br/>　　双刀被和尚险之又险的躲开，戏子此时也确认了下来，那人是真的离开了。
<br/>　　随后，便再也没有半点犹豫，全力出手，十几回合之中，将和尚的头颅割下。
<br/>　　“队长，怎么不回话啊，你在哪里啊，需不需要我过去支援……”
<br/>　　对讲机中依旧传来鬼棍的声音，但和尚此时的状态，明显是给不出来任何回复了。
<br/>　　“这件事情，感觉要给大人禀报一声啊，那个家伙到底是什么人啊。”嘀咕了一声，戏子摆弄着手中的对讲机：“不过还好，大人交给我的任务倒也完成了，这个东西，应该算是电子产品吧。”
<br/>　　“不过……这东西，我记得生前也有一只啊。”戏子一边说着，一边转身向远处走去。
<br/>　　只是提着和尚头颅的身影，多少看起来都有些让人感觉诡异。

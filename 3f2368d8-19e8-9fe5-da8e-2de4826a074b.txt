　　“力哥！”
<br/>　　“力哥！”
<br/>　　“力哥！”
<br/>　　看到李霄力，地下室里负责给那些“皮箱”灌毒的马仔，立刻让开一条路，躬身低头向李霄力问好。
<br/>　　李霄力走到那个姑娘面前，蹲下身，看着眼前的姑娘露出一个微笑，问道：“小姑娘今年几岁了？”
<br/>　　表面上，李霄力是在笑，可是李霄力挂在脸上的笑落在旁边那些马仔的眼里，却是让他们不寒而栗。威虎山有座山雕“三笑杀人”，杭山市有李霄力“一笑要人命”！
<br/>　　“十、十九岁。”姑娘颤抖着声音回答道。
<br/>　　“十九岁，那应该还是个学生。”李霄力继续问道：“你叫什么名字？”
<br/>　　“刘、刘小翠。”
<br/>　　刘小翠哭着向李霄力哀求，“我不要吞毒，我不想死！我可以还钱，我愿意在这里当小姐，我什么都愿意做，请你不要让我吞毒！”
<br/>　　当小姐用自己身体来还钱也比把那些用安全套包裹好的毒球吞下去要好，人体运毒不仅要冒着毒品在体内破裂丧命的危险，还要顶着被警察发现坐牢的风险；跟自己年轻的生命相比，出卖自己的身体实在是太不值一提了。
<br/>　　旁边那些马仔听到刘小翠的话，在心中暗暗嘲笑她的天真，在他们老大眼中，刘小翠欠下的钱远不及她作为“皮箱”的价值高。
<br/>　　作为“皮箱”被送到境外以后，是连货带箱子一起卖的！
<br/>　　“我呢也不是一个不讲道理的人，只要你能把欠我们的钱还上，我可以让你离开这里。”李霄力站起身，对身边马仔问道：“这姑娘欠了多少钱？”
<br/>　　“本金加利息，一共是五万九千块钱。”马仔回答道。
<br/>　　“我哪有欠这么多钱！？”刘小翠歇斯底里的冲着那个马仔喊道：“我就借了你们五千块钱！”
<br/>　　“五千块是你借的本金，剩下的五万四千块是你的利息。”马仔面无表情的回答道。
<br/>　　“哪有这么高的利息！？”刘小翠喊叫道。
<br/>　　“我们这里的利息一直都是这么高。”马仔把一张借条放到刘小翠面前说道：“白纸黑字写得清清楚楚，你自己在上面签了字的。”
<br/>　　刘小翠在杭山市某卫校读书，来自小地方的刘小翠来到杭山市后，立刻就被这里的繁华所吸引，周围同学使用的苹果手机、名牌皮包让刘小翠羡慕不已的同时，有了攀比之心。刘小翠平时也会到这个酒吧来玩，在人的介绍下，刘小翠找到了酒吧经理，借了五千块钱用来买苹果手机，借钱的时候，介绍人的不断催促和“保证”让刘小翠没有很仔细的去看借条上的条款，就匆匆签了自己的名字；可即便当时刘小翠看了借条条款，也发现不了藏在其中的陷阱，既然是打定了注意要让你入坑，又怎么可能让刘小翠轻易发现并且心生警觉呢？
<br/>　　所以，从刘小翠找到那个介绍人，并且被带入这家酒吧的时候，她就已经成了李霄力手中的猎物，不可能再逃脱出去。
<br/>　　刘小翠一屁股坐在地上，这一刻她绝望了，彻底绝望了。就算她愿意出卖身体，就算她躺在床上不停的接客，她也还不起这笔钱，因为利息每天都在涨，而她赚钱的速度肯定追不上利息增长的速度。
<br/>　　“既然还不出钱，那就只能服从我的安排。”
<br/>　　李霄力早已敛去了脸上的笑容，他的眼神简直比冰块还要冰冷，刘小翠仅仅只是被他看一眼就忍不住打颤。
<br/>　　“把东西灌进去！”
<br/>　　李霄力对着身边的马仔点了点手指。
<br/>　　“你们别过来！”
<br/>　　刘小翠这个时候突然站起身来，退后一步，嘶吼道：“你们再过来，我就咬舌自尽！反正都是要死，我宁死也不会让你们得逞！”
<br/>　　“天真！”李霄力眼神一凝，上前一步，摊手抓住刘小翠的脖子，冷声说道：“到了这里一切都要听我的，你是死还是活，也要听我的！”
<br/>　　旁边的三儿拿来一块布塞进刘小翠嘴里，让她没办法咬舌自尽，然后用胶布捆住了她的双手，让她双手没办法挣扎。
<br/>　　李霄力从盘子里拿起一个淡粉色的毒球，在手里掂了掂，对刘小翠说道：“要把这东西塞进你身体里，不一定要你吞下去，你身上那么多洞，总能把这东西‘吞’下去。”
<br/>　　周围的马仔闻言，顿时发出****的笑声。
<br/>　　李霄力身高比刘小翠要高出一个头，力量也要比刘小翠大，他一只手抓着刘小翠被胶布捆住的双手，另一只手抓住刘小翠的裤腰，用力一扯，连同她的内裤一起扒了下来。李霄力用膝盖别开刘小翠紧紧屏住的双腿，正准备动手的时候，突然对身边的马仔问道：“她还是雏儿吗？”
<br/>　　三儿冷笑一声，对李霄力回答道：“力哥，你别开玩笑了，来这里玩的怎么可能还是雏儿。她第一次来的时候，我就看到她跟男人在厕所里玩得很嗨，然后从那个男人手里拿了几百块钱，要不然我们也不能找上她啊。”
<br/>　　“那真是太可惜了。”李霄力拿起桌子上的毒球在刘小翠面前晃了晃，说道：“如果你还是个雏儿，你还能晚点受苦，但谁让你自己不珍惜那层膜呢？”
<br/>　　李霄力把手里的毒球，塞进了刘小翠的下体，然后松开手把刘小翠仍在地上，对身边的马仔问道：“看懂怎么做了吗？”
<br/>　　“力哥，这里有好几十个呢，她那里塞得下吗？”一个马仔问道。
<br/>　　李霄力抬脚踩在刘小翠小腹上，对那个马仔说道：“你妈能从这里把你生出来，你说塞得下吗？”
<br/>　　“是力哥，我知道怎么做了。”马仔忙不迭的回答道。
<br/>　　“船我已经联系好了，今天晚上就把她跟其他‘皮箱’一起送上船，送去T国。”李霄力吩咐道。
<br/>　　“力哥，有件事我想应该告诉你。”三儿走到李霄力身边，说道：“我们这么做，影响到了T国那边的市场，T国那边普提查对我们很有意见，想要跟您谈谈。”
<br/>　　“谈？”李霄力不屑的冷哼一声，说道：“谈个屁！他有什么资格跟我谈！”

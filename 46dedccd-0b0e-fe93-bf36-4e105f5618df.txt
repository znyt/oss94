        &lt;h3&gt;第三卷第三百九十三章  她走了&lt;/h3&gt;
<br/>
<br/>        郝景文边开车边给寒伯安打电话。
<br/>
<br/>        “大哥，陈子昂带着孩子去欧洲了。他告诉我，他从欧洲直接去甸城。”
<br/>
<br/>        “什么时候走的？”
<br/>
<br/>        “我才把她送到机场，这会儿在等飞机。”
<br/>
<br/>        “那好吧，这边的公司我们都紧盯着点儿，千万不要出现任何差错。”
<br/>
<br/>        “大哥。你不知道陈子昂将头发剪了，穿了一身男装，我差一点儿都没有认出来。”
<br/>
<br/>        “男装，她想干什么？”
<br/>
<br/>        “他跟我说话时，自我介绍是陈子寒，还说他是陈思宇的男朋友，我差一点儿都想动手揍他了。”
<br/>
<br/>        “陈子寒？”
<br/>
<br/>        “是啊！他说他是陈子昂的哥哥陈子寒，从海外归来的，我简直，害，就相信了。”
<br/>
<br/>        “哈哈哈。”寒伯安笑着，心里却在思考着陈子昂的举动。
<br/>
<br/>        “好了，大哥，不说了，我去看看陈思宇，这家伙是个花痴，我要看紧点，指不定哪天见到比我帅的就被人拐跑了。”
<br/>
<br/>        “你赶紧趁早娶回家放在家里吧，这样你就不用担心喽。”
<br/>
<br/>        “嗯，你说得对。我得和她商量商量什么时间结婚。那大哥再见！”
<br/>
<br/>        郝景文挂了寒伯安的电话，转念又想给江俞轩打电话，陈子昂撩我，那我也撩撩江俞轩，不亏！
<br/>
<br/>        江俞轩从陈家回来之后就开始忙起来，西岭和高新的项目招标公告已经发出来了，他们得准备招标的事情，忙了一天还没有和陈子昂联系，想着忙完直接去陈家。
<br/>
<br/>        江俞轩正在和江俞琪研究标书，这个标有百分之九十采纳了他们的方案，就是在主街上稍微做了一下修改。
<br/>
<br/>        “姐，这高新的项目，如果我们拿下来，我们在西城的位置是无人撼动的，这个地区将会是西城面向国际招商引资的门户，后期的开发力度很大，也会得到政策更好的支持，你来看。”江俞轩走到地图前，指着地图上低矮的楼宇对江俞琪和手下的人侃侃而谈：
<br/>
<br/>        “这个地区本来就是国家级高新区之一，已经被科技部确定为要建成世界一流科技园区的六个试点园区之一，这一段时间都是在扩建中，从这里到这里都会实行改造，我们现在要做的只是一个中心，也就是说这个地方是这个区域的门户，我们要实现从无到有、从小到大、从弱到强的一系列重大跨越。
<br/>
<br/>        我们虽然只是承建部门，但是我们担负的责任不仅仅只是一个建造商，而是一个城市的奠基者，就像是脚下的地基，你的地基打不好，你盖多少层楼都是豆腐渣工程，你的地基打夯实了，能够承重，你想盖多少层就能盖多少层，同理，这个门户符合先进的社会发展潮流，跟上时代的步伐，你后期就一直朝前发展。
<br/>
<br/>        再打个比喻，你本身长得很好看，虽然穿得不怎么好，但是气质和底子在，你还是一个活脱脱的美人，如果长得不好看，穿再多名牌在身上也无济于事，除非后期整容，但是整容是要花费很大的力气的，资金和时间都是一笔很大的浪费，我们的这个门户要做一个自带气质的美人。”
<br/>
<br/>        当江俞轩说到美人的时候，脑子里一晃而过陈子昂的容貌：“陈美人不知道今天怎么样了？”
<br/>
<br/>        大家静静的等着江俞轩继续讲话，江俞轩却在发呆。
<br/>
<br/>        “俞轩。”江俞琪看着弟弟不说话，喊了一声江俞轩。
<br/>
<br/>        “嗯，嗯，所以这一次的标我们要做得百分之百的完美，我不容许我们有一点点失误之处，做标书的时候我会和大家一起。”江俞轩被江俞琪拉回了思绪。
<br/>
<br/>        “俞轩，大家来看看我们的这份全景地图，从我们现在的这个位置纵深下去就离西岭的项目不远了，你们看看西岭的项目，其实，我也想考虑西岭的项目，我们江氏集团是有实力拿下这个两个项目的。”江俞琪指着地图对大家说。
<br/>
<br/>        “西岭的项目我不赞成，我们全力拿下高新的项目就行了。”江俞轩眉头一皱，反对江俞琪的提议。
<br/>
<br/>        “为什么？”其实是下面有好几个部门的经理要求一起拿下这两个项目，江俞琪只是没有办法解释，所以才让江俞轩来说。
<br/>
<br/>        “西岭的项目目前看起来是不错，很赚钱，但是他们没有考虑到后期环保的问题，你们看这些别墅都是建在西岭山脚下，表面看起来很美，后期的社会成本没有算进去，这将是一个鸡肋，你们自己可以想一想。”江俞轩用激光笔点着他们面前的电子地图。
<br/>
<br/>        “那么别的公司也会考虑这个问题，他们都可以做，为什么我们就不可以做？”一个部门经理问。
<br/>
<br/>        “他们是他们，我们是我们，我们不能阻止一些事情的发生，但是我们不能同流合污，这个工程百害而无一利，有利的只是一些少数的有钱人，试问一下，哪些人有钱去这种地方买别墅？”江俞轩落地有声，整个办公室陷入一片沉寂。
<br/>
<br/>        “所以，我们专注一件事情，从今天起，大家齐心协力将高新的事情做好，有没有信心拿下来？”
<br/>
<br/>        “一个项目我们当然有。”大家笑着说。
<br/>
<br/>        “其实江总说的很对，我们有实力拿下两个项目，只是我们做项目也要做有益于公众的项目，这样我们的发展才能够长远。”
<br/>
<br/>        江俞琪在旁边微笑着看着自己的弟弟，她发现这几年弟弟真的是成长了很多，无论是做人还是做事都已经能够独当一面了。
<br/>
<br/>        其实江俞轩早就可以独当一面了，只是在他们眼里江俞轩永远都是一个长不大的孩子而已。
<br/>
<br/>        电话铃声打断了江俞轩的话，江俞轩看了一眼手机，对大家说：“我接个电话。”
<br/>
<br/>        走出会议室，江俞轩才接通电话：“文文！”
<br/>
<br/>        “我明天到西城。”郝景文在电话那头说。
<br/>
<br/>        “要我去接你吗？”
<br/>
<br/>        “不用，从哲接我，你猜我今天见到谁了？”
<br/>
<br/>        “你见到谁了？”
<br/>
<br/>        “陈子昂的六哥陈子寒，你见过陈子寒吗？”郝景文和江俞轩开了个玩笑。
<br/>
<br/>        “谷强？你是说谷强到了北京？”江俞轩从那一次信息会上见了谷强之后就再也没有见过谷强，但是他猜测陈子昂的六哥就是谷强，而且陈子昂也多次在他面前提起过。
<br/>
<br/>        “谷强？”郝景文反问一句。
<br/>
<br/>        “不是的，文文，你是怎么见到陈子寒的？”江俞轩急切的想知道郝景文是怎么认识陈子寒的。
<br/>
<br/>        如果有人说是陈子寒，他要知道是不是谷强，还有陈子昂现在还在西城，他是不是得陪陈子昂回一趟北京？
<br/>
<br/>        “你家陈子昂男装骗我说她是陈子寒！”
<br/>
<br/>        “陈子昂不是还在西城吗？怎么会男装骗你？”
<br/>
<br/>        “咦，你被抛弃了？陈子昂这会已经在飞往欧洲的飞机上了。”
<br/>
<br/>        江俞轩没有说话，他知道文文不会骗他。

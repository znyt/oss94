
<br/>　　<br/>　　    三楼一间房门外，齐泉和丁武领着四个人站在楼道口旁边。<br/>　　<br/>　　    “叫门，就说是楼下的住户，漏水了”齐泉跟身后的一人说道，然后让其他人全都贴着墙站好，他又低声跟丁武说道：“一会进去后你就找那个吴洋，人大概二十几岁短发，眼眶有点凹陷，嘴唇挺薄的，看着比较精神，应该很好认的”<br/>　　<br/>　　    “知道了，泉哥”丁武点头说道，手摸向了别在腰间的仿六四。<br/>　　<br/>　　    “咚，咚，咚”齐泉吩咐完，一个人来到门前敲了几下房门，几声之后屋内有人沉声问道：“谁啊？”<br/>　　<br/>　　    “不好意思，我是楼下的，你们卫生间是不是漏水了？全都渗到我家屋里去了，地板都给泡了”<br/>　　<br/>　　    屋内的人随即说道：“不能吧，我家都没太用卫生间，怎么可能漏水呢？”<br/>　　<br/>　　    外面的人急促的说道：“不是你家的话那就是楼上了，但是你得让我看看吧？看完你家要是没事，我在去楼上了”<br/>　　<br/>　　    屋内短时间内静默了下来，片刻后房门忽然被拉开了一条缝隙，从外面那人的视线望过去，屋子里面没开灯漆黑一片，什么都看不见。<br/>　　<br/>　　    “嘎吱”门外的推开房门走了进去，视线里一阵模糊，但紧接着就有一道寒光闪过，进去的人猛然感觉自己的脖子上一凉，他下意识的伸手摸了下，就发现脖子上出现一道口子，鲜血顿时“噗”的一声就喷溅了出来。<br/>　　<br/>　　    “呃，呃……”这人惊恐的捂着脖子，张嘴想要喊叫几声，但是喉结都被割断了，嗓子里只能发出呜咽的动静。<br/>　　<br/>　　    “噗咚”没到五秒钟，进去那人身体就砸在了地板上，痛苦的抽搐了几下。<br/>　　<br/>　　    外面的丁武和齐泉等人一看情况不对，连忙从楼梯上冲了上来，几人刚进屋就看见地上已经流了一大滩的鲜血，先进来的那人明显已经死透了，齐泉脸色突的一变，但反应极快的说道：“他们发现了，去两个在外面堵着”<br/>　　<br/>　　    齐泉刚说完话，从客厅那边就突然就飞过来一把椅子直接照着他脑袋就砸了过来，齐泉后面的丁武伸手一把将他拉到了身后给躲开了。<br/>　　<br/>　　    但于此同时，客厅那边的落地窗忽然传来“哗啦，哗啦”两声脆响，还有摇曳着的窗帘，两个人影直接从三楼往地面跳了下来。<br/>　　<br/>　　    齐泉和丁武还有一个人连忙跑了过去，就看见窗户被砸碎了，地上两个人影刚刚落地，其中一个抬起脑袋，阴阴的朝着楼上的人笑了笑，那笑容相当阴霾和冷淡了。<br/>　　<br/>　　    对方笑了两声之后，随即撇了撇嘴，大踏步的就要离开了。<br/>　　<br/>　　    齐泉说道：“快，出去追他们！”<br/>　　<br/>　　    没想到的是，丁武居然抱着脑袋直接就从三楼跳了下去，人“噗通”一声砸在了地上滚了几圈之后，他咬牙就站了起来，朝着那两人冲了过去。<br/>　　<br/>　　    齐泉看的一愣，他身后的同伴吸了口气说道：“真猛！”<br/>　　<br/>　　    “别他么废话了，赶紧下去啊……”齐泉连忙催促了两声，并不是所有人都有胆量从三楼往下跳的，这个高度摔死人可能有点难，但折你一两条腿却挺简单的，运气背点的话摔个半身不遂也不多。<br/>　　<br/>　　    齐泉边往楼下跑，边拿出手机给老马打了过去：“马爷人被惊了，没动上手，我们死了一个，对方两人往外面跑了，没走正门”<br/>　　<br/>　　    老马一听当即就皱眉了，一手拿着电话一手发动车子然后挂挡起步，跟齐泉说道：“你们继续追，我从后面堵过去”<br/>　　<br/>　　    “嗡”车子发动后就蹿了出去，绕着小区的正门朝着另外一侧赶去。<br/>　　<br/>　　    再说丁武这边，那两人跳下去之后没走正门，他们似乎对小区里面的环境非常熟悉，直接绕道了楼后面朝着另外一个出口跑了过去，丁武跳下来后紧追不舍，远远的就看见了两道迅速跑动的的人影。<br/>　　<br/>　　    刚刚突然被一刀抹了脖子的同伴，丝毫没有让丁武产生出一点后退的念头，他满脑子里现在就一个想法，追上对方，不行就开枪直接干了他们。<br/>　　<br/>　　    在这种念头的鼓动下，丁武甩开了两腿追的非常紧，眼看着跟对方的距离越拉越近了，“哗啦”丁武跑着跑着就从身后把仿六四给掏了出来，一拉枪栓后抬起左手枪口怼着矿泉水瓶子，就要开枪了。<br/>　　<br/>　　    忽然间，那两人前方出现了一道栅栏，跑在后面的吴洋低声说道：“你先翻过去，然后赶紧走，我拦下后面这个”<br/>　　<br/>　　    “噗，噗”丁武咬牙连续扣动两次扳机，沉闷的枪声响起后，子弹完全没有任何准头的打歪了，一点杀伤力都没有造成。<br/>　　<br/>　　    “踏踏踏，踏踏踏”跑到前面的那人，快走几步后，一脚踩在栅栏上伸手搭了上去，然后翻身快速的就跃身而上。<br/>　　<br/>　　    吴洋这时脚步一刹，身体在急速冲击中就顿住了，并且非常违反人体物理学的脚下一蹬地面，人居然只是稍微停顿了一下，就反方向的朝着丁武这边蹿了过来。<br/>　　<br/>　　    丁武根本没有料到对方在往前跑动的时候还能停住身子再冲过来，就有那么将近一秒钟的时间里，他是完全没有反应过来的。<br/>　　<br/>　　    吴洋的速度太快，几乎是闪过一道残影就冲到了丁武面前，他突然伸出左手抓住了丁武的衣领子，然后往自己这边一带，右手抬起蓄力的时候反手握着一把刀就奔着丁武的脖子抹了过去。<br/>　　<br/>　　    生死之间，丁武总算是回过神来了，他仓惶的往后仰了下脑袋，躲过了奔着脖子来的这一刀。但吴洋忽然手臂朝下压了下，中途直接变换方位，刀刃顺着丁武脖子下面的胸膛就划了过去。<br/>　　<br/>　　    “噗嗤”一刀下来，丁武的胸前就被割开了，一股冰凉的感觉从脚底蔓延到了头顶，随即吴洋抿着嘴唇再次反手握刀，左右连续划开，就看见丁武的胸膛冒出了一串血花。<br/>　　<br/>　　    “噗嗤，噗嗤……”刀子划破皮肉的声音不时传来，吴洋右手接连在丁武的身上隔了不知道多少刀之后，他抬起腿一脚就踹在了他的胸膛上。<br/>　　<br/>　　    “噗通”丁武睁着眼睛往后退了几步，踉跄着倒在了地上。

　　第77章
<br/>　　杜茂口中呼出的是酒气，酒似乎变成汗水从脸上渗出来，站在那里双手握拳，身子微微摇晃。
<br/>　　“来！”
<br/>　　口中爆喝，头顶像是响过雷霆，众人耳边嗡嗡作响，蛤蟆寨的人一时间竟怯懦不敢近前。
<br/>　　“来呀！”
<br/>　　关键时刻，林少羽挺身而出，身子在空中一跃，双脚踏向杜茂，鸳鸯连环脚的架门，杜茂胸前中了两脚，身子踉跄后退。口中发出兽吼，身子迎着林少羽撞过去，宛若蛮牛一样直接将林少羽撞飞在地。
<br/>　　林少羽一个鲤鱼打挺站起来，摸着发疼的肩膀，看向杜茂的眼神流漏出一丝畏惧。
<br/>　　不知是谁高喊了一声，众人一拥而上。人群中有人搬起那桌子，猛地砸在杜茂脑袋上。
<br/>　　杜茂身子晃了晃，摸着发疼的脑门，一把夺过那桌子，抡着将那人掀开。
<br/>　　好一头醉虎！
<br/>　　蛤蟆寨十几个人近不了身，只见他用四方桌做武器，身子摇摇晃晃，周围被他掀翻一片。地上哎呦惨叫不止，却无人能占他一招的便宜。
<br/>　　“来呀！”
<br/>　　又是一声爆喝，乔装成驼子的程大雷暗暗称赞，这等本事，怕是武松再世也就这样！
<br/>　　地上的人勉强站起，眼神看向杜茂时又是钦佩又是畏惧，却已再无人敢向前。
<br/>　　乔装成卖酒老汉的黄三元看向程大雷，那眼神仿佛再说：大当家，现在该怎么办！
<br/>　　“不用慌！”程大雷大叫一声：“让我来解决他。”
<br/>　　“你就是程大雷！”杜茂晃了晃脑袋：“怎么这么丑，快过来让我打上一拳。”
<br/>　　有还没来得及逃走的人，躲在墙角，从手指缝里露出眼睛看这两人捉对。
<br/>　　如今青牛山方圆，名声最响的就是这二人。一个打虎英雄，俨然落叶城的地下贼首，一个蛤蟆寨大当家，懂五雷法，一招击败陆哼……如今这场双雄会，上山虎碰着下山虎，水中龙碰上云中龙，究竟谁弱谁强，大家拭目以待。
<br/>　　程大雷将手中刀递给黄三元，黄三元暗赞：大当家果然英雄，要空手战杜茂，不占武器之便。
<br/>　　杜茂也将手中的桌子丢开，冲程大雷伸出手，眼中战意澎湃。
<br/>　　正在所有人都擦亮眼睛时，只见程大雷微微弯腰，手中拍着四六八拍。
<br/>　　“宝贝对不起，不是不疼你，
<br/>　　真的不愿意，又让你哭泣；”
<br/>　　咦……
<br/>　　一片寂静，空中恍惚有乌鸦飞过。大家都懵了：这究竟什么跟什么……
<br/>　　杜茂也懵了，睁大眼睛看着程大雷，满脸的问号。
<br/>　　这时候程大雷又换了曲调：“宝贝，睡吧，宝贝睡吧……”
<br/>　　杜茂拧起眉头，嘴里想把程大雷十八代祖宗都骂上。终究一股酒意往上涌，冲得他七荤八素，眼中迷茫茫一片，初见程大雷是一个，再看是两个，然后二变四，四变八……
<br/>　　他终究是喝了太多酒，杜茂虽然是一号猛人，可再猛的人也一样，喝多了也吐，开车也撞树。刚才仗着一口气撑着悍勇，可只要稍稍停歇，酒意便涌上来，然后该吐得吐，该撞树撞树。
<br/>　　脑子越来越糊涂，对程大雷的恨意也越来越重，突然大吼一声，挺身冲程大雷冲过来。
<br/>　　“小子，去死吧！”
<br/>　　身似虎，拳如龙，这杜茂曾经三拳打死老虎，这一拳如果放在程大雷身上，估计他没有老虎身子骨硬。
<br/>　　程大雷赶紧站直身子，身体绷成一道线。
<br/>　　然后没打着。
<br/>　　酒醉之后，杜茂的力气是有的，但这精准度就差了些。
<br/>　　这一拳是打空了，程大雷已站在杜茂身后，巴掌往他后脑勺重重一拍，口绽一字。
<br/>　　“倒！”
<br/>　　杜茂脑袋里七荤八素，如同开了酱料铺一样，身子晃了晃，两眼一黑，噗通倒地。
<br/>　　一般人都睁大眼睛：这样也行！
<br/>　　黄三元顿了顿，然后突然喊道：“大当家威武，一招便击败杜茂！”
<br/>　　其他人不明白怎么会事，眼前看到的也确实是程大雷一巴掌便把这打虎英雄拍倒了。见黄三元喊得这么亢奋，于是也跟着喊。
<br/>　　“好说，好说。”程大雷挥挥手，忽然道：“快把他绑起来。”
<br/>　　杜茂一倒，其他醉汉便不值一提，有的早已醉晕过去，没晕的再补一棍便是。早有准备好浸水的麻绳，将杜茂绑成一个球，往那酒桶中一塞，上面压上盖子。
<br/>　　众人化整为零的离开，在湖底捞汇合，将那醉死过去的杜茂塞进棺材里，程大雷依旧化妆成驼子模样，再又塞给守军一袋铜板后，顺利混出城。
<br/>　　杜茂真不知昏睡过去多久，迷迷糊糊醒来，就感觉浑身酸疼。他睁开眼睛，发现自己被绑在一间地牢中，周围光线昏暗，墙壁上立着火把。
<br/>　　“你们是什么人，快放开我，等爷爷离开这里，把你们刀刀杀尽。”
<br/>　　“呦，怎么兄弟，喝断片啦？”
<br/>　　一个特别讨厌的声音响起，杜茂看到一个年轻人慢悠悠踱到牢房外，火把的光打在他身上，露出一张干净的脸。
<br/>　　那些混乱的记忆重新苏醒，杜茂咬紧牙关，口中迸出三个字：“程大雷！”
<br/>　　“杜老大，幸会幸会，咱们这算是正式见面了。”
<br/>　　杜茂使劲挣扎，挣不脱身上的绳索，况且他刚刚醒转，身上也无多少力气。
<br/>　　“混蛋，王八蛋，你这卑鄙小人。”杜茂破口大骂：“有本事放开我，让我和你大战三百回合。”
<br/>　　程大雷递给他一个看白痴般的眼神：“说什么笑话呢，我是领导者，不逞匹夫之勇，我们……”程大雷用右手指节敲了敲太阳穴，补充道：“我们灵魂强大。”
<br/>　　杜茂以愤怒的眼神盯着程大雷，过半晌，终究是平静下来。
<br/>　　“这次是我走窄了，陷在程当家手中，要杀便杀，我绝无二话。”
<br/>　　“我杀你干嘛？”程大雷半蹲在牢房前，道：“现在，不如我们聊聊招安的事？”
<br/>　　“招安？”
<br/>　　“当然也可以理解成落草，我现在诚心诚意的问你一句……你愿意在蛤蟆寨入伙么？”
<br/>　　杜茂愣了会，接着放声大笑，笑声中满满的嘲讽。
<br/>　　“别这样笑，挺丑的。”程大雷心平气和道：“其实，我曾经也是一个有理想的人……”
<br/>　　
<br/>(本章完)

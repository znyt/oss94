　　八月的能力早已被狼牙记载，告知了所有受训的成员，但他归属于猎者联盟，他的默契搭档江凌天与狼牙的关系可是匪浅，不仅是黑芒战队队长江凌若的亲弟弟，更是楚涵的头号粉丝一心想进狼牙。
<br/>　　有着这层关系，所有人都下意识将八月视为了狼牙编外人员，敌军一方也极少会考虑到是否会出现这一特殊的能力。
<br/>　　而今日之所以困了第一小队这么久，也就是没有料到这一点，同样的在追击之时前方的背影模糊不清，根本分辨不出是什么身份，更无法察觉是否出手的是八月本人。
<br/>　　“会是八月？”追击中一人疑惑问道：“猎者联盟的任务，竟然还参与到了这次大战之中？”
<br/>　　“有关战役猎者联盟不会有队伍参与，但个人任务就说不准了，八月如果接了什么任务……”
<br/>　　“最终还是钱的事。”
<br/>　　徐锋没有出声，在查明真·相之前谁也无法判断，但前方那个不断逃脱他们追击范围的身影，的确不像丧尸！
<br/>　　而干扰五感这种太过于贴近强化者的稀有能力，总不可能是出现在超级丧尸身上吧？
<br/>　　所以对方是人的可能性极大！
<br/>　　一路的追击躲避让第一小队不知不觉离开了原定区域，朝着未经狼牙侦察队探索过的地方而去，黑夜之中辨别不清反向，到处都是残木与枯草，怎么看都似曾相识。
<br/>　　第一小队很快察觉到异常，他们放慢了速度，不禁思量起来。
<br/>　　“距离大部队越来越远，是不是故意引开？”一名队员开口。
<br/>　　徐锋心中郁结之气更深，总觉得这背后有什么人在戏耍他们，最可怕的是不知道对方是什么目的，处于忌惮而没有死拼？
<br/>　　“对方不可能不知道我们有所察觉，他很清楚我们的身份以及战力水准。”另一名队员开口道。
<br/>　　“如此一看，更像是八月本人了，对狼牙了解颇深。”
<br/>　　“如果是江凌天向江凌若队长打听什么？”
<br/>　　“那不会，江凌若这人……她能说个啥啊？”
<br/>　　想到这里众人也连连摇头觉得自己等人想太多，江凌若的性子跟狼牙黑芒非常契合，都是不说话就像死了一样的喜好隐藏自己，存在感相当低，也异常的安静。
<br/>　　“这是一个局。”徐锋眯起眼睛：“追与不追都是局。”
<br/>　　四名队员都看向徐锋，等待他的下令，事到如今总需要一个决断。
<br/>　　徐锋回首看了眼身后的一片漆黑，定了定神道：“追！”
<br/>　　刷刷刷！
<br/>　　五个身影毫不犹豫的提速，快速朝前追去。
<br/>　　——————
<br/>　　在第一小队与大部队脱离越来越远的时候，江凌若也在那处腐血沼泽地带遇到了极其难缠的敌人。
<br/>　　到处都是木桩的地方严重阻碍了她的移动和方向辨别，时不时从远处飞来的暗器射·入木桩之内，留下深深的痕迹，她被困在这里举步难行，无论朝哪个方向突破都会被忽然爆发的暗器潮给挡了回来。
<br/>　　而沼泽之下的那些尸骨更诡异的让人看不明白，铺了整整一片深埋在沼泽内的丧尸就是个乱葬岗，尸骨一点点下沉在泥地里，不仅让人看着惊悚，更心头压抑。
<br/>　　可到底是谁干的？第五小队？
<br/>　　只有人类才会对丧尸群如此进攻一举歼灭，可值得注意的是此处显然利用了腐血沼泽的行走困难，以及周围这一切的陈设来刻意困住江凌若，所以根本不可能是自家狼牙战团的人作为。
<br/>　　难道在这埋骨之地还有其他的人类队伍？
<br/>　　江凌若无法离开那处范围，也就无法突围去暗器射来的方向寻迹，再这么被动下去早晚会筋疲力尽，更别说她刚战斗完毕斩杀了一只丧尸小首领！
<br/>　　各处飞来的暗器还在继续，江凌若却已经不打算浪费时间，手中银色镰刀一划，直接将周身的好几个木桩割断。
<br/>　　哗哗哗！
<br/>　　连续的断裂倒下，方向整齐一划排着形成好几道，更有好几根聚在一起形成木排。
<br/>　　江凌若二话不说踩着倒下的木桩，飞快的选定一个方向掠去，身影‘刷’的一下离开那处小型沼泽带，一下子来到了边缘。
<br/>　　此时‘咻’的一声，一个身影快速的窜上树梢，紧接着就像是猿猴一样跳上了远处的另一棵树。
<br/>　　江凌若来不及追上，只能定眼捕捉那道身影，却只能看到一个穿着整齐的背影。
<br/>　　皱起眉，江凌若察觉到了一丝异样，对方并不像是要跟她死拼，反而是想引她去什么地方……
<br/>　　她眼神一动立即上前追去，就要看看是人是鬼！
<br/>　　同为狼牙三战队之一的队长，同为世上最强的新人类，江凌若和徐锋的选择一样。
<br/>　　——————
<br/>　　这时候的第五小队依旧在带领着大批丧尸群绕圈狂奔，一夜过去天已经蒙蒙亮，身后那些长相恐怖的丧尸一个个露出相貌，但总比黑暗中看不见的恐怖来的让人心定。
<br/>　　这时候忽的一个人从旁边插入第五小队之中，与他们并排奔跑中开口：“天亮了，那头已经完成，前方五百米处有接应，时间一到就回头开杀。”
<br/>　　四个早已跑的满头大汗的第五小队成员纷纷点头，跑了一路没怎么杀丧尸，早已让他们憋的不行，这会儿哪怕再累也要杀几只才行。
<br/>　　就这样一步两步，五百米的距离一霎而过，一个大圈外围无数枪口对准着这批丧尸。
<br/>　　是射击队！
<br/>　　“杀！”
<br/>　　猛然的回头，四名第五小队成员拔刀而起，直冲着眼前扑面而来的丧尸就狠狠劈了过去！
<br/>　　碰碰碰！
<br/>　　不断响起的枪声刺耳，在这天刚亮的时候惊起人一身鸡皮疙瘩，同时更有好几支小队从各个草丛中跳出来，挥刀而上就冲入了丧尸群。
<br/>　　绕了一大圈跑了一夜，爽不爽？
<br/>　　现在就给你们来一次狠的！
<br/>　　第五小队的特殊任务完毕，杀够了一波丧尸之后快速转移到后方休息，很快一个微型的根据地呈现在他们的面前，射击队百人，精英成员百人，还穿·插着诸多三战队的成员，侦查队自然随处可见，甚至还有参谋部的人在此。
<br/>　　异常的齐全，直接是一场小型战役的规模。
<br/>　　并且最让他们惊讶的是，竟然是神隐战队的队长路冰泽亲自带队！

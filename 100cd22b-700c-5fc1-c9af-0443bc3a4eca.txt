<br/>?    陈明和王蒙回到他们在客栈的房间，王蒙在外面看到布置的暗哨点点头，关门进了房间，王蒙摊开姜王陵的地图，陈明坐在地图之前看向姜王陵的位置，一条大河，几乎是将姜王陵包围，只露出了一个葫芦口一般的位置。
<br/>    
<br/>        王蒙介绍道：“这姜王陵地处密林之中，隐秘之极，也是杀生殿堂隐藏自己的手段，加上此地有着一条大河环绕，只有一个入口，只要守住了这个入口，便是如同一个铁桶一般，易守难攻，以我们如今的兵力，想要攻进去，无非是痴人说梦。兵书有云，十则围之，想要攻下如此易守难攻之地，至少要十倍的兵力才有机会攻进去，对方有着十万兵力，而我们，却是只有三万的兵力。”
<br/>    
<br/>        陈明看着地图之上的姜王陵的位置，“还真是一个好地方。”
<br/>    
<br/>        这一条大河简直是天然的屏障，阻挡了所有的进攻之路，渡河？别开玩笑了，渡河的过程危险之极，还是以少打多，对方一轮火箭，便可以点燃所有船只，几乎是可以做到没有任何战损将我方大军全军覆灭。
<br/>    
<br/>        密林，火，大河。
<br/>    
<br/>        陈明似乎是想到了什么，看向那个大河露出的葫芦口，问道：“这一条葫芦口有多长？”
<br/>    
<br/>        王蒙道：“根据我们的情报，这唯一的葫芦口有三十里之地，而且丛林密布，极难行军，杀生殿堂盘踞此地已久，熟悉地势，占尽了地利。”
<br/>    
<br/>        三十里之地，如果是这样，倒是有点意思了。
<br/>    
<br/>        陈明伸手在地图上的葫芦口点了点，指挥大战他确实不会，但是三国演义还是看过的，陆逊便是靠着火烧连营灭了刘备百万大军，“要以少胜多，不仅仅是要解除这条大河作为屏障的作用，还要让这条大河成为他们的取死之道，传我命令，命这葫芦口附近扮做商队的将军下令购买木材，然后让其他军士带领民夫前往这一带伐木，记住伐木要在葫芦口这一条线上，砍伐一条三丈宽的距离，不过不要连接起来，各自留出来一小段距离，为了不让杀生殿堂的人看出来我们的意图，派遣其他的军士在大河附近其他位置砍树，就让他们当做一次普通的购买木材。”
<br/>    
<br/>        王蒙心说这不是在打仗吗？怎么砍木头去了，别说杀生殿堂弄不清世子殿下的主意，我都不明白世子殿下到底是什么意思。
<br/>    
<br/>        不过不明白归不明白，王蒙还是按照陈明的话去做了，算了，就当世子殿下瞎胡闹吧。
<br/>    
<br/>        又是三天过去了，伐木大队进山伐木，如此大的动静，杀生殿堂怎么可能发现不了。
<br/>    
<br/>        姜王陵墓室。
<br/>    
<br/>        名叫姜王陵，自然是一位王爷的陵墓，不过杀生殿堂占据了这里之后，更是在姜王陵附近建造了很多木制营寨。
<br/>    
<br/>        杀生殿堂的殿主此刻正在听着手下的汇报，他戴着一个青铜面具，遮蔽了他的脸，身上披着一身漆黑长袍。
<br/>    
<br/>        “启禀殿主，我等十万大军还有三日便可全部汇聚在姜王陵。”
<br/>    
<br/>        殿主从自己的殿主大位上站起来，抚掌一笑道：“好，附近的飞虎军可有什么动向？”
<br/>    
<br/>        手下的斥候禀报道：“并没有什么动向，据说是等待着摄政王世子前来，否则现在大将军和摄政王对峙的局面，有任何异动都会受到大将军和摄政王的怀疑。”
<br/>    
<br/>        殿主露出一抹笑容，“哈哈哈，好，我等在这崇山峻岭之中厉马秣兵，就是为了这一刻，就让陈狱虎和陈王斗个两败俱伤，到时候父皇在内策应，我带领大军从外杀入，必定要擒拿摄政王全家，全家凌迟才能泄我心头大恨！”
<br/>    
<br/>        斥候继续道：“对了，附近最近有些商人在收购木材，所以有人在姜王陵附近伐木。”
<br/>    
<br/>        殿主笑了笑道：“陈王和陈狱虎争锋，这里的飞虎军等到陈明一到，必定会动身前往都城，到时候少不了要木材安营扎寨，建造攻城器械，这些商人自然是闻着腥味就到了。”
<br/>    
<br/>        殿主的笑容忽然凝聚在脸上，他能够掌控杀生殿堂，不是没有自己的判断的，道：“等等，你将他们砍伐木材的位置在地图上标出来。”
<br/>    
<br/>        斥候听命，在地图上点出一个个红点，如同是群星闪耀，星罗棋布，不过大多都是在大道之旁。
<br/>    
<br/>        殿主看了一眼，杂乱无章，根本没有任何规律，而且，砍木头有什么用，难道还能用木头堆死自己的十万大军？葫芦口更是一马平川，根本没有任何值得埋伏的地方。殿主挥挥手，道：“看来是我多心了，命令下去，所有人，继续潜伏，不要让这些人发现了，另外，不要派人去监视这些民夫了，省的被发现了。”
<br/>    
<br/>        斥候拱手道：“遵命！”

<br/>        摄政王府。
<br/>    
<br/>        老王爷身披一身黑袍走进王府大厅，坐在主位上的陈狱虎走下来，拱手道：“父王！”
<br/>    
<br/>        老王爷的神色有些愠怒，喝道：“你不知道我们现在在做戏吗？到底是什么事，居然要让我过来！”
<br/>    
<br/>        陈狱虎道：“父王，根据密信传来的消息，在姜王陵之中，杀生殿堂汇聚了足足十万兵力！”
<br/>    
<br/>        老王爷伸手捋了捋胡须，道：“看来燕国皇室这些年暗地里的动作不小，怪不得不惜让潜伏在我们麾下的人不计死伤的去死，也要我们打起来，可是这一点，似乎没必要惊动我吧？”
<br/>    
<br/>        陈狱虎道：“按照王蒙传来的密信上的意思，明儿似乎是想要继续进攻杀生殿堂。”
<br/>    
<br/>        老王爷神色一变，喝道：“你说什么？”
<br/>    
<br/>        陈狱虎道：“明儿似乎是准备继续打下去。”
<br/>    
<br/>        老王爷分析了一番，“按照明儿传来的消息，他只是暗中汇聚了三万飞虎军以及三十虎怅卫，这姜王陵本就是易守难攻之地，以三万对十万，而且对方还熟悉地形，这不是找死吗？”
<br/>    
<br/>        “速速传我的命令，命明儿撤军，暂时不对杀生殿堂动手了，三万大军，三十虎怅卫死就死了，但是明儿绝对不能出事！既然燕国皇室潜藏了十万大军，那这事便不能着急了，需要从长计议，这件事，已经不是明儿控制的军队能够处理的了。”
<br/>    
<br/>        陈狱虎拱手道：“我这就命人传命！”

<br/>
<br/>        忙了一整年，对于今年的收获郝景文觉得特别的有成就感，看着自己的事业蒸蒸日上，心里充满了自豪。
<br/>
<br/>        还有一点，他可以不用吃狗粮了，他可以撒狗粮。
<br/>
<br/>        物流系统在稳步前进，租赁事业目前运营看起来很是可观，应该说是更上一层楼，等陈子昂他们的系统研发成功，项目再拓展化，他们的事业就会发生质的变化，到时候会是一个新的起点，新的事业，会带动一个行业的改变。
<br/>
<br/>        他们跨界和陈子昂合作，已经初见成效，看来跨界也是很好的融合，这个世界上没有你做不到，只有你想不到，如何实现惠存和共赢，是他们这一代人需要解决的问题，站在时代的前沿，用发展的眼光来看待问题，不固步自封，一切皆有可能。
<br/>
<br/>        他，寒伯安、江俞轩、陈子昂他们只是苍穹浩瀚星辰里的一颗小行星，社会的发展时光的纽带将他们聚集在一起，利用相互吸引的作用让他们聚团却又各自散发着温热，带动着他们弱小的团体在随着时代不停的变动，不停的前进。
<br/>
<br/>        墙面上每一处点亮的红色地图，象征着他那颗跳动欢悦的心，在祖国广袤的大地上欢快的歌唱着。
<br/>
<br/>        年会结束之后，大家都散了，他回到自己的公司，静静的看着墙上的地图，办公室那一株碧绿青翠的幸福树散发着绿油油的蓬勃向上的旺盛的生命力。
<br/>
<br/>        江山如此多娇，神州如此妖娆，看改革开放的新浪潮，推动着社会前进的步伐向前跑，人生之路每一个被点亮的地图都是意义非凡的。
<br/>
<br/>        苏州和扬州，就是继杭州之后明年将要开通的城市。
<br/>
<br/>        “春水初生，春林初盛，春风十里，不如你。”郝景文想到了扬州，也想到了陈思宇，春风十里，不如你，待扬州飞絮十里红妆，此生为你点眉轻妆，携手徜徉。
<br/>
<br/>        拿起车钥匙，出门。

<br/>        ........

<br/>        陈思宇站起来升了个懒腰，对风盛夏说：“风总，我这里已经工作已经做完了，你那里还有什么事情需要我来完成的吗？”
<br/>
<br/>        “没有了，我这里也已经完成了，你先走。”
<br/>
<br/>        “那好，我先走了，风总，大家明年见。新年快乐！”陈思宇笑着和大家挥手道别。
<br/>
<br/>        陈思宇走出大门，寒冷的冬天刹那间温暖如春，郝景文斜依在宾利车身上，摇起的大衣裙摆在风中轻轻的摆动帅气的脸上鼻子有些发红。
<br/>
<br/>        陈思宇走过去捏了捏郝景文的鼻子：“来多久啦？不知道坐在车里等吗？”
<br/>
<br/>        “我就想这样等着你。”郝景文抓住陈思宇的手，她的小手有些凉。
<br/>
<br/>        “傻瓜，不嫌冷吗？“陈思宇眼里都是郝景文阳光帅气的脸庞。
<br/>
<br/>        “你在的地方都是温暖如春。”郝景文接过陈思宇手中的手袋，打开车门，看着陈思宇坐好，并细心的为陈思宇系好安全带。
<br/>
<br/>        郝景文：“先吃饭，然后回家？”
<br/>
<br/>        陈思宇：“就吃饭然后回家？”
<br/>
<br/>        郝景文：“那你还想要干什么呢？”
<br/>
<br/>        陈思宇：“好吧！”
<br/>
<br/>        郝景文：“回家的机票我已经定好了，先到杭州，然后开车去你家。”
<br/>
<br/>        陈思宇：“你也要去吗？”如果郝景文能够去她家，是不是他们的关系就更明确了？

<br/>        家里还在催婚，让她回去相亲，每次回家，三大姑八大姨的都会逼着她问：“谈男朋友了吗？什么时候结婚了？你看别人家的谁谁谁都已经有了小孩了。”

<br/>        “女孩子啊，年龄大了就嫁不出去了，赶紧的找个人嫁了啊。”
<br/>
<br/>        “要求别那么高啊，差不多的就得了。”等等此类的话让她不胜其烦，要是郝景文能和她一起去就好了。

<br/>        郝景文微微一笑，专注的看着车：“我就不去了，明天送你到机场，回去带我像叔叔问好。”
<br/>
<br/>        陈思宇听得心里一寒，嘴上却说：“好的，那谢谢你了，等会吃完饭我就，回家明天不用送我去机场的，你也得要早点回去的。”

<br/>        “机场还是要去送了，都是女朋友了，不去送机成何体统。”

<br/>        陈思宇：“你又不去，买什么？”
<br/>
<br/>        郝景文：“我不去，你得回去啊，莫非你想和我一起回我家？”

<br/>        陈思宇心里有些不爽看着窗外来来去去的车辆不说话。

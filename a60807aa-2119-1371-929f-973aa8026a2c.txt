<br/>				不多时，一袭红衣的凤九从外面走来，身后还跟着几位长老。众人看着他们几人进来，目光自然而然的落在那一袭红衣的绝美女子身上。<br/>　　<br/>　　这就是那红衣女子了？竟是生得这样的美。<br/>　　<br/>　　正想着，就见这红衣女子居然是坐在了厅中主位上，看到她坐下的位置，厅中众人眸光微动。<br/>　　<br/>　　几位长老站凤九的左右，看着厅中众人，道：“今天让诸位过来，是有一事要宣布的。”<br/>　　<br/>　　长老说着，目光从众人的身上掠过，而后落在洪家父子身上，道：“城主退位让贤，我们商量过后，决定推选出一位来当继任城主一位，而这个人，就是洪家家主。”<br/>　　<br/>　　这话一出，众人不由震惊的睁大了眼睛，目光看向那说话的长老，以及那神色如初，没有震惊之色的洪家父子。<br/>　　<br/>　　“让洪家主继任城主一位？”<br/>　　<br/>　　“不错。”长老点了点头：“以后……”话还没说完，就让其中一人打断了。<br/>　　<br/>　　“这到底是什么人挑选出来的？我想知道，城中各个世家的人都那么优秀，洪家主又何德何能可以继任城主一位？我希望有一个让我们心服口服的说法。”<br/>　　<br/>　　长老看了那名家主一眼，而后将目光看向凤九，继而道：“这不是跟你们商量，而是告诉你们我们的决定，我想，我们城主府里的人要推举什么人当城主，这应该与阁下无关吧！”<br/>　　<br/>　　听着这话，那位家主皱眉头一皱，沉声道：“如何无关？比他优秀的人大有人在，为何偏偏是他？还有，这红衣女子又是什么人？你们选洪家主来继承这城主一位，莫不是因为这红衣女子？”<br/>　　<br/>　　他的目光凌厉的落在凤九的身上，凌厉如刀的眼光带着审查与探究。这女子究竟有何本事？竟能让他们听从她的话？<br/>　　<br/>　　“呵呵，这话就说错了，我们城主府打算选谁出来，莫不是还要你们同意不成？”那大长老沉着脸色冷笑着：“刚才我就说了，今天请诸位过来，只是告诉你们一声，而不是听你们的意见。”<br/>　　<br/>　　听着这话，那位家主只觉得一口气憋在心里，虽不服，但，若是城主府和洪家的势力联合在一起，却是不得不让他们忌惮。<br/>　　<br/>　　比起那位家主，其他的几位却是将注意力放在那红衣女子身上，见她静坐着喝着茶，也没去理会那位家主说的话，不由的暗暗打量着她。<br/>　　<br/>　　这女子一身实力深不可测，气度尊华不凡，这样一个人，怎么会突然插手这城里的事？真让人想不懂。<br/>　　<br/>　　只是，这事只怕除了城主府的人和洪家的人，别人也不清楚了。<br/>　　<br/>　　“呵呵，在这里坐了这么久，也还不知这位姑娘怎么称呼？”其中一位家主说着，看着那在喝茶的凤九。<br/>　　<br/>　　听到这话，长老和洪家主皆看了凤九一眼，就听见凤九抬眸朝那位家主看去，唇角微勾，道：“鬼医。”<br/>　　<br/>　　鬼医？<br/>　　<br/>　　厅中众人微怔，这名称陌生得几乎未曾听闻，可偏偏又让他们觉得，这鬼医应该是个风云人物。<br/>　　<br/>　　：。：<br/>　　<br/>

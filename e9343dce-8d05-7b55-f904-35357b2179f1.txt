料后，还真的是惊讶了。<br/>　　<br/>　　虽然自己有些怀疑陈默的能力，但是医疗评测和医疗检查不会撒谎。另外医院的李院长也是说了，对于纪老的病，其实他们也没有什么好的办法，只能通过常规治疗手段还缓解。<br/>　　<br/>　　没有想到的是，却得到纪老经过检查后，肝脏能够再次生长，对于他们医生来说，绝对是重大的发现，是可以纳入医疗研究的。<br/>　　<br/>　　但是由于何老等人的管控，李院长也是无可奈何。对于能提升医院医疗手段和治疗手段的陈默，他早就垂涎欲滴，盼望已久的人，但是连去讨论的资格都没有，实在是有些郁闷。<br/>　　<br/>　　还在他还是能够理解一二的。再说了，不理解也没有办法，很多时候很多情况也不是他一个小小的医院院长能够阻挡的。<br/>　　<br/>　　另外，对于陈默的关注，也让他收集了一些信息。他将这些信息也提供给胡月，让他看看。对于胡老的病况，医院现在已经可以说没有什么手段了。<br/>　　<br/>　　胡老也是冠心病，但是由于还有其他一些疾病，包括一次中风病史，两次心梗病史，高血压、高血糖、高血脂，胃病和肝硬化等疾病，可以说要是不是医院现在还在想办法稳住他的身体，各种药物能用的全部再用，说不定也就去了。<br/>　　<br/>　　也是因为如此，胡老已经不适合通过手术治疗病症，也只能通过药物来缓解。也就看那天，身体一旦出现点小小的毛病，病人就会死亡。<br/>　　<br/>　　李院长其实也非常心累，对于这种病人，真的已经有些束手无策了。要是早些年，能够发现病症，还有可能治疗，但是现在，也就只能通过药物维系生命。<br/>　　<br/>　　胡月拿到李院长给他的资料，不仅有吴老的病历和检查报告，还有纪老的。另外还包括陈默上次救助车祸人员，所使用的急救手段，可以说比较全面了。<br/>　　<br/>　　这也是李院长花了不少的心思，才收集来的。对于上次急救的一些病人，也是通过电视和一些人的转述等等，才收集到的，非常的不容易。<br/>　　<br/>　　而且纪老和吴老的病历和检查报告，都是保密资料，能够拿出让胡月看，其实也是经过何老等人的同意的。不然一个小小的医院院长，就算是甲级医院的院长，也没有卵用。<br/>　　<br/>　　这些资料，胡月是真的没有想到，这个年轻人有这么大的能耐，竟然能够通过中医的手段治疗这些病症，真的是人不可貌相啊！<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>

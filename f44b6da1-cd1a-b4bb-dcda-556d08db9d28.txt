　　第80章
<br/>　　高飞豹和于虬然失魂落魄的离开蛤蟆寨，二人所想如何，程大雷不太关心。
<br/>　　他们刚走，程大雷就回到屋子。洗手净面，焚香沐浴，口中默念：脱非入欧，玄能改命。
<br/>　　自从擒住杜茂后，程大雷就一直在获得恐惧值，如今终于突破两万大关，可以在点将台召唤两次。
<br/>　　他打开系统界面，集中注意力在点将台的图像上。
<br/>　　嘟，目前拥有恐惧值20000，是否选择召唤？
<br/>　　嘟，初级点将台，每召唤一次消耗恐惧值10000，随即获得糟糕到优秀品质1 。
<br/>　　嘟，每十万恐惧值可连续召唤11次，必然获得优秀品质1 。
<br/>　　程大雷深吸一口气，口中默念：“抽……”
<br/>　　程大雷没有抽下去，注意到最后一条系统提示时愣住，一口气没上来，自己差点抽过去。
<br/>　　没想到，系统还带这么玩的，像无良的游戏商一样。
<br/>　　程大雷心里琢磨，如今点将台只是初级，优秀品质的爆率自然也很低，这其实也可以理解。等日后点将台升级，自然可以抽到高等级品质，不过所消耗的恐惧值必然也更高。
<br/>　　所以，这两万恐惧值抽下去，大概率是普通品质，也有可能直接是糟糕品质。这样，无非是给山寨加两个小喽啰。
<br/>　　而稳妥来说，程大雷应该忍耐忍耐再忍耐，等攒够十万恐惧值后，来个十一连抽。
<br/>　　这样一来可以省下一万恐惧值，二来也必定能抽到一优秀品质。如林少羽，孙九那样，哪怕他是个优秀裁缝呢，也能提升山寨实力。
<br/>　　可是好不容易攒够两万恐惧值，这样忍住不抽，其实挺折磨人的。
<br/>　　唉……
<br/>　　程大雷叹口气，将系统界面关闭。稳妥些，稳妥些，成大事者要能撑得住气，不争一时，要怎一世，要稳妥……
<br/>　　程大雷在房间内来回踱步，嘴里嘀嘀咕咕不知念着什么。忽然一咬牙，去他大爷的吧，我什么时候是个稳妥的人。
<br/>　　他再次进入系统界面，对着点将台默念：召唤，召唤……
<br/>　　点将台上篝火亮起，火焰汇聚在一处，在火焰中心似乎有什么等待出现。
<br/>　　好吧，还带过场动画的。
<br/>　　程大雷注视着这一幕，心里琢磨着，自己先用一万恐惧值过过瘾，留着一万恐惧值不用，慢慢积攒够十万恐惧值后，来个十连抽。
<br/>　　毕竟也攒了这么久，一直忍着不用，的确挺折磨人的。花上一万恐惧值，也并不过分嘛。
<br/>　　最后，从火焰中走出一人，随之，系统提示到了。
<br/>　　嘟，恭喜获得糟糕酿酒师一名。
<br/>　　姓名：张肥（名不见经传的糟糕酿酒师）
<br/>　　年龄：18
<br/>　　技能：酿酒
<br/>　　隐藏属性：无
<br/>　　程大雷看着人物属性，有些无语，若说这酿酒师，山寨是用得到的，他现在在琢磨蒸馏酒，山寨没有这样的人才，都是自己琢磨，走了许多弯路。可是这个糟糕等级，实在是，实在是……惨不忍睹呐。
<br/>　　那么，这究竟是说明自己是非酋还是欧皇呢……
<br/>　　还有一万恐惧值，要不再试试？
<br/>　　嗳，刚才都说了要攒下去的，怎么能这么不坚定。
<br/>　　嗨，反正只剩下一万了，要攒也很快的，干脆试一试，万一最后就能抽到优秀品质呢。
<br/>　　程大雷脑海中仿佛有两个小人打架，然后……他觉得后面那小人说得对。
<br/>　　“召唤。”
<br/>　　点将台再次亮起篝火，火光涌动，随着嘟地一声，一个小人走了出来。
<br/>　　嘟，恭喜获得普通武将一名。
<br/>　　姓名：关鱼（名不见经传的普通武将）
<br/>　　年龄：19
<br/>　　技能：练兵
<br/>　　隐藏属性：无
<br/>　　好吧，这下关鱼和张肥都出来了，你们大哥姓刘的呢，怎么不一块儿来。不过，很明显这次结果要好很多，已经达到普通品质了。
<br/>　　咦，果然把期望值放低后，幸福感提升很多。
<br/>　　其实练兵这个技能还是蛮有用的，如今山寨训练的事情是由秦蛮和林少羽负责。二人的本事都不弱，但练兵和能打是两回事。程大雷偶尔也上去帮帮忙，不过说实在的，他也不怎么擅长。
<br/>　　只希望这个将要到来的关鱼，不会让自己太失望吧。
<br/>　　程大雷蹲在山寨门口，画着小圈圈等待着。一直等到日沉西山，才看到四道人影沿着山路走来。
<br/>　　为首的人是孙九，孙九后面还跟着三个人。一个矮胖身材，黑渣渣胡子，这是张肥；一个身材清瘦，脸上无须，可脸像重枣般红，这是关鱼；还有一个……
<br/>　　对了，怎么是三个，不应该是两个嘛。程大雷把注意力集中在对方身上，读取着他的信息。
<br/>　　姓名：刘悲（名不见经传的优秀医者）
<br/>　　年龄：17
<br/>　　技能：草药学
<br/>　　隐藏属性：无
<br/>　　程大雷一激灵，险些吓得坐到在地，妈的，刘关张真的出现了。可这样的品质的刘关张，还真的是有辱英雄啊。
<br/>　　孙九快步上山，冲程大雷道：“大当家，这三人是我在路上的，他们慕名而来，要在山寨入伙。”
<br/>　　程大雷点点头，注意力还在刘悲身上，关张二人的到来，他并不意外，可刘悲怎么到的？程大雷心中忽然一动，莫非是山寨大旗的招募技能发动了。
<br/>　　他赶紧去大旗处看了看，见大旗的技能信息果然变成灰色。上次随着蛤蟆寨升级成二级山寨，山寨大旗的技能也获得提升，冷却CD缩短了。程大雷心里盘算着，距离上次廖甲廖乙投奔，已经过去月余，看来目前等级的大旗，是一月发动一次。
<br/>　　今天算是赶到一块儿去了。
<br/>　　山寨校场，刘关张在程大雷面前站齐，程大雷心里有些打鼓。
<br/>　　“你们认识……”
<br/>　　“我们自幼在桃园结拜，这是我二弟关鱼，三弟张肥。”刘悲回答。
<br/>　　程大雷心脏咚咚跳动，还真是缩水版的刘关张呐。
<br/>　　三人模样都很年轻，程大雷也想不明白，怎么年纪最小的刘悲却成了大哥。不过，三人站在一起，气质也是各有特点。
<br/>　　老大刘悲，十七岁，一副看谁都像欠他钱又要不回来的悲苦。
<br/>　　老二关鱼，十九岁，一副看谁都不配和我说话的傲慢。
<br/>　　老三张肥，十八岁，一副看谁都欠揍的愤怒。
<br/>　　盯着三人，程大雷忽然发现一件奇怪的事。单独看，每人的隐藏属性都是『无』，可当三人站在一起时，隐藏属性那一行却同时出现四个字：桃园结义。
<br/>　　
<br/>(本章完)

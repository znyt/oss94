　　然而，
<br/>　　就在这个时候，
<br/>　　叶尘的头顶上空，突然出现了一道巨掌，稳稳地挡住了孙长老的攻击。
<br/>　　轰隆隆！
<br/>　　周围空间，一阵剧烈晃动过后。
<br/>　　那孙长老顿时倒退了三四步，望着叶尘背后，突然出现的一道身影，脸色无比难看，
<br/>　　“风老四！你敢多管闲事？”
<br/>　　原来，
<br/>　　在关键时刻，是凤族的那位合道仙君风老四，突然出手，为叶尘挡住了这一击。
<br/>　　叶尘原本已经抬起的手臂，便又缓缓放落了下来。
<br/>　　虽然以他如今的实力，虽然并不惧这孙长老，不过他还不想过早暴露自己的底牌，既然有人出手相帮，他也乐得接受。
<br/>　　风老四双手叉腰，冷哼一声，道：
<br/>　　“老头子我就是喜欢多管闲事，怎么？你咬我啊？”
<br/>　　凤族在十三大顶级势力之中，也是排名比较靠前的存在，虽然和八大仙门还是有些差距，但也并不惧怕。
<br/>　　更何况，风老四的修为，是要稳压那孙长老一筹的，自然更加不惧。
<br/>　　反倒是孙长老，对这个风老四，颇为忌惮。
<br/>　　“风老四！你讲不讲道理？他出手杀我擎苍宗的门人，难道老夫还不能出手教训他一下了？”
<br/>　　“我去你的狗屁道理！”
<br/>　　风老四闻言，直接破口大骂，
<br/>　　“明明是他们二人打赌在先，你们擎苍宗的那小子，自己把命输给了叶小友，叶小友既然赢了，取走他的性命，又有什么不妥？”
<br/>　　“这……”
<br/>　　孙长老顿时一滞，憋的满脸通红，道：
<br/>　　“所谓赌命，不过就是玩闹而已，哪能真的取人性命？难道我们擎苍宗一位天才的性命，就如此不值钱吗？”
<br/>　　风老四双手叉腰，雪白的胡子吹起老高，
<br/>　　“我呸！你们擎苍宗的废物，自己赌命输了，死了也就死了，又能怪得了谁？我看你分明就是输不起！亏你都这么大把年纪了，居然还好意思对一个小辈动手，你也不嫌丢人！”
<br/>　　论吵架，孙长老显然不是风老四的对手，被他一通抢白，说的老脸涨红，愣是找不到话反驳。
<br/>　　就在这时，
<br/>　　又有一人站了出来，
<br/>　　“孙长老，风老这话虽然说的不好听，却也说的在理！小一辈的恩怨，就应该让小一辈来解决，我们作为长辈，出手对付一名小辈的话，就有失身份了！”
<br/>　　众人循声望去，这说话之人，赫然正是元始门门主元道一。
<br/>　　孙长老见连元道一这个东道主，都替叶尘说话，脸色顿时越发的难看。
<br/>　　而周围众人之中，本就对孙长老的行为颇为不爽，眼见风老四和元道一都站了出来，顿时也纷纷起哄起来。
<br/>　　孙长老眼看有风老四在，今日他已经奈何不了叶尘，若是再继续纠缠下去，只会辱没擎苍宗的名声，于是冷哼一声，转身返回擎苍宗的阵营之中，不再言语。
<br/>　　绝无敌深深的望了叶尘一眼，也转身走了回去。
<br/>　　叶尘转过身，冲着风老四拱手行了一礼，道：
<br/>　　“多谢风老仗义执言！一连救了小子两次，小子感激不尽！”
<br/>　　虽然叶尘并不认为，孙长老能够奈何的了自己，但是风老四两次出手相助，也让叶尘发自内心的感激。
<br/>　　尤其是之前那一次，叶尘当时还没有表现出任何价值，风老四便能够路见不平，足以说明他的为人，让人敬佩！
<br/>　　风老四摆了摆手，哈哈笑道：
<br/>　　“举手之劳而已！叶小友不必在意！说起来，我与叶小友倒是颇为投缘，不如你入我门下，如何？老夫一定倾囊相授！”
<br/>　　哗！
<br/>　　风老四这话一出口，众人顿时一片哗然。
<br/>　　尤其是擎苍宗阵营里的孙长老，更是嘴角微微一抽，脸色比吃了死苍蝇还要难看。
<br/>　　现在，明眼人都能看出来，叶尘这个绝世妖孽，和擎苍宗之间的梁子，已经彻底结下了！
<br/>　　那些势力稍弱的，即便有些招揽叶尘，此时也得好好思量思量了。
<br/>　　毕竟，一旦接收了叶尘，就等于是和擎苍宗作对，说不定会引来灭顶之灾。
<br/>　　但是，凤族，绝对是可以抗衡擎苍宗的存在，而且两大势力的关系，也一向不怎么好。
<br/>　　倘若真的让凤族得到了叶尘，凤族必定如虎添翼，对于擎苍宗来说，将会是一个天大的坏消息！
<br/>　　在众人看来，如果他们和叶尘易地而处的话，根本不用考虑，肯定立即答应下来。
<br/>　　只要有了凤族作为靠山，就再也不必担心擎苍宗的报复，等到日后修为大进，再慢慢算今日之账，岂不是美哉？
<br/>　　然而，
<br/>　　叶尘几乎连考虑都没有考虑，直接干脆的摇了摇头，
<br/>　　“承蒙风老厚爱！小子愧不敢当！”
<br/>　　哗！
<br/>　　众人再次一片哗然，
<br/>　　拒绝了！
<br/>　　他居然拒绝加入了凤族！
<br/>　　他是脑袋被门给夹了吗？
<br/>　　又或者，他已经有了中意的门派？
<br/>　　风老四也明显一怔，显然没有想到，叶尘居然会拒绝他，继而眉头微微一皱，望了一眼落霞宗的方向，缓缓道：
<br/>　　“落霞宗虽然也位列十三大势力，可是恐怕未必可以庇佑你呀！”
<br/>　　很显然，
<br/>　　在风老四看来，叶尘拒绝自己，多半是因为曦月的缘故，叶尘应该是想加入落霞宗！
<br/>　　琼碧落听到风老四这话，先是一阵激动，继而又暗暗摇头叹息，
<br/>　　风老四这话，虽然不好听，却也是实话！
<br/>　　以当下的局面来看，叶尘得罪了擎苍宗，若是真的加入落霞宗，他们落霞宗，还真庇护不了叶尘！
<br/>　　而且反而会招来灾难！
<br/>　　这一刻，
<br/>　　琼碧落真的好恨，恨自己的宗门太弱，同时也恨叶尘太会招惹是非，不该去得罪擎苍宗。
<br/>　　“曦月，你去劝劝他吧，我们落霞宗不适合他，还是让他加入凤族比较好！”
<br/>　　琼碧落叹了一口气，冲着曦月开口道，而且故意将这话，让众人都听得清楚。
<br/>　　因为只有这样，才能撇清和叶尘的关系，不至于引来擎苍宗的迁怒。
<br/>　　既然已经注定得不到叶尘这个妖孽，那还不如撇的干净一些！
<br/>　　叶尘自然明白琼碧落此举的目的，顿时冷冷一笑，道：
<br/>　　“琼宗主不要自作多情！小子不加入凤族，同样也不会加入落霞宗，因为我压根就不准备加入任何宗门势力！”

        “你对这个也有研究啊？”夏凌焰一边操作着电脑，修改自己的设计底稿，一边跟童筱颖闲聊了起来。
<br/>
<br/>        “我跟我外婆学过中草药，自然懂得一些花花草草用途，小时候，外婆还会自制花露水，给我用于驱蚊止痒。”童筱颖一本正经地说。
<br/>
<br/>        “你外婆年轻的时候一定是个大美女！”夏凌焰赞许道。
<br/>
<br/>        童筱颖咧嘴一笑：“那当然！”
<br/>
<br/>        夏凌焰笑呵呵地接着打趣道：“话说回来，你说的那个女秘书跟你的二婚老公，十有八九有一腿！”
<br/>
<br/>        “为什么？”童筱颖却不以为然。
<br/>
<br/>        “不然，你二婚老公身上怎么会有那个女秘书身上的气味。”
<br/>
<br/>        “这种纯露，只有喷在身上才会留有微淡且很不明显的香气。隔着空气，是相互染不到的。你以为是香水啊？”
<br/>
<br/>        “那你想说明什么？”
<br/>
<br/>        是啊！
<br/>
<br/>        她想说明什么呢？
<br/>
<br/>        她很想说明傅夜沉和那个女秘书落以柔之间有暧昧！
<br/>
<br/>        但是，就光凭这点，完全不成立啊！
<br/>
<br/>        万一是那女秘书落以柔自己做了许多薄荷花露，自己用不完，就免费送了傅夜沉一瓶用来提神的也说不定。
<br/>
<br/>        童筱颖一手托腮，一手拿着胶水在模型纸上涂胶，漫不经心地回答道：“想找理由离婚啊！”
<br/>
<br/>        “噗嗤”一声，夏凌焰忍俊不禁起来：“你这才结婚，就想着离婚。小筱颖，你该不会是冲着人家的钱去的，离婚了，好平分人家的财产吧！”
<br/>
<br/>        “我现在是缺钱，但也不至于这么缺德吧！”童筱颖忍不住对夏凌焰翻了个白眼。
<br/>
<br/>        “你知道你中指上这枚粉钻戒指多少钱吗？”
<br/>
<br/>        “多少钱？”
<br/>
<br/>        “以这粉钻的大小来看，起码二十万以上。”
<br/>
<br/>        “这么贵？”童筱颖惊诧。
<br/>
<br/>        夏凌焰呵呵地笑了笑后，一本正经地当起了婚姻劝导师：“筱颖啊！一个肯为你花钱，又不嫌弃你是‘二婚’的男人，你应该好好珍惜才对！”
<br/>
<br/>        “可我不爱他，他也不爱我，而且，他娶我，只是因为我长得像他的……初恋情人吧！”童筱颖喃喃自语地说，心想着，苏漫雪应该算是他傅夜沉的初恋情人吧！
<br/>
<br/>        “你跟他结婚了，还管过去的那些破桃花事做什么？有一种爱，叫日久生情。更何况，你两还是夫妻。处久了，自然就来感情了。”
<br/>
<br/>        “那撇开这些不说，最主要的问题是，他是我前夫的大哥，嫡亲的大哥啊！你说，我心里能不疙瘩吗？”童筱颖一脸挫败地叹着气。
<br/>
<br/>        夏凌焰却笑道：“前夫的大哥怎么呢？法律上规定了，女方离婚后不许嫁给前夫的哥哥了吗？”
<br/>
<br/>        “……”童筱颖顿时被夏凌焰说得有些语塞。
<br/>
<br/>        夏凌焰接着说道：“你现在就别想着什么离婚的事情了，倒不如先试着处处看。何苦总是把自己关在笼子里呢？”
<br/>
<br/>        “算了！不说我的事情了，还是先帮你赶工吧！”童筱颖苦笑，接着埋头忙活着自己手里的事情。
<br/>
<br/>        其实，一开始，她真的没想到自己会突然被大哥“二婚”。
<br/>
<br/>        她想得最多的是，一个人在云城西镇深山区里外婆那个家渡过漫漫余生。
<br/>
<br/>        随着时间的推移，童筱颖觉得，自己总有一天会忘掉傅昼景，不再爱她心里的那个龙之卫。
<br/>
<br/>        因为，她觉得时间是治愈情伤的最好疗药。
<br/>
<br/>        夏凌焰知道童筱颖有自己的想法，于是很识趣地闭上了嘴，不再跟她谈她的私事。
<br/>
<br/>        两人一起都还未吃晚饭，于是让酒店服务员送了晚餐过来，两人一边吃一边开始交流工作上的事情。
<br/>
<br/>        夏凌焰不得不说，在工作上，童筱颖是他的最佳搭档。
<br/>
<br/>        她总会在他设计灵感遇到瓶颈的时候，给他提出一些非常宝贵又有用的想法。
<br/>
<br/>        上次他设计的古建筑，提交给业主方后，对方非常满意，其实当中也有她童筱颖的功劳。
<br/>
<br/>        因为忙于工作，两人都忘了时间。
<br/>
<br/>        直到门铃响起，童筱颖主动起身去了玄关处。
<br/>
<br/>        她习惯性地先看猫眼，在看到门外的来访者后，心口蓦然隐隐作痛，脸色也跟着惨白了几分。
<br/>
<br/>        童筱颖连忙跑回夏凌焰身旁，让他起身收拾东西。
<br/>
<br/>        “是谁啊？你怎么跟见了鬼似的！”夏凌焰碎碎念道。
<br/>
<br/>        童筱颖一边将会议桌上的电脑、模型等等全部往卧室里挪，一边支支吾吾地说道：“上次我替你当了一回幌子，这次你得替我当一回幌子！”
<br/>
<br/>        “你要我当你什么幌子啊？到底是谁来呢？把你吓成这样子！”夏凌焰疑惑不解，而那边的门铃依旧在响个不停。

　　叶少阳完全不懂，只有跟着梧桐。
<br/>　　梧桐仔细观察后，用不确定的口吻说道：“这些……好像是苯教的东西……”
<br/>　　苯教！
<br/>　　叶少阳皱起眉头，跟他一起打量眼前这座坛城上的佛像，一边看一边转过去，这才看到坛城背后的真相：在坛城最高处的凹槽里面，居然镶嵌着一副棺材，不过与黄金打造的坛城不同，棺材看上去是银质的，比正常的要大一点，平坦地躺在凹槽里，严丝合缝，一看就知道是连着坛城一起打造的，而不是后来加上去的。
<br/>　　坛城上放着棺材？
<br/>　　怎么看都有点不对劲。
<br/>　　“我上去看看。”叶少阳说着，踩上了坛城的第一层，霎时间觉得冰冷刺骨，就好像大热天的突然走进了空调间里的这种感觉。而且这股阴冷的气息还有种把人往前拉扯的感觉。
<br/>　　叶少阳当场吓了一跳。最重要的额是，手上的三色莲花灯也灭了。
<br/>　　三色莲花灯，只有在检测到极其强大的邪气时，因为负荷不了才会熄灭……
<br/>　　叶少阳急忙跳下坛城，阴冷的感觉霎时间就消失了。
<br/>　　“怎么了？”梧桐好奇地问道。
<br/>　　叶少阳没作声，把手朝坛城塔基的范围里伸过去，果然，只要是约过了塔基，手臂立刻感觉到冰冷刺骨的寒气，就好像有一道无形的分界线，将坛城内外区分成两个世界。
<br/>　　梧桐感受到叶少阳的异样，也把手伸过去，顿时也体会到了叶少阳一样的感受，惊的说不出话。
<br/>　　如果是普通人经历这一幕，肯定会惊得下巴掉下来，叶少阳和梧桐都是法师，虽然吃惊，但是立刻就想明白了怎么回事。
<br/>　　“这个坛城，是一个不寻常的阵法！”叶少阳望着坛城上的棺材说道，“很可能跟这棺材有关！”
<br/>　　梧桐没有表态，弯腰下去，借着四壁发光的石头，在坛城的第一层塔基上寻找着什么，过了一会，她拉着叶少阳低头看，指着某处说道：“你来看看这个。”
<br/>　　叶少阳凑过去，看到了梧桐指的那个东西，是坛城塔基上散落的一些长长的东西，拼凑成了奇怪的图案，这些长东西看上去白生生的，叶少阳凑近看，顿时一惊，道：“我怎么看着像……人的手指？”
<br/>　　梧桐不说话，盯着那东西仔细看。叶少阳在旁边近距离地打量她的侧脸，简直……太像了。尤其是这种专注研究某种东西时的神情，跟芮冷玉几乎一模一样。
<br/>　　“这是人骨，人的指骨。”梧桐说完，等了半天没听到回音，一转头发现叶少阳盯着自己的脸庞出神，脸色微微一红，随即沉下脸来说道：“你在干什么？”
<br/>　　“我……”叶少阳吞吞吐吐地说道：“你长得很像我一个朋友。”
<br/>　　“有多像？”梧桐心不在焉的问道。
<br/>　　“这个……我要说几乎一模一样，你会不信的吧。”叶少阳试探说道。
<br/>　　梧桐有些轻蔑地笑了笑，似乎把这当成了一种泡妞的手段。“叶少阳，你现在说这个，是不是有些唐突了？”
<br/>　　“额，好吧，咱们说正事，你刚说什么来着？”
<br/>　　梧桐有点无语，说道：“我说，这些都是人的手指骨，被镶嵌在坛城上的。”
<br/>　　叶少阳集中注意，定睛看去，发现根本就不止一块，整个坛城的四周有好几处这样的地方，还真是人的手骨，被搭建成不同的形状，有的十字形，有的交叉，少则三五根，多的有十几根，这些手骨看上去都是白生生的，有点泛灰，叶少阳想着，如果是活生生的手指放在这里，慢慢腐烂风化成这样的话，那变成这样，那说明绝对有些年头了。
<br/>　　这么多的手指，摆在这里，看上去十分得触目惊心。
<br/>　　“为什么会有这么多的手指，摆成这样是什么意思？”叶少阳对这些“非主流”的邪术之类完全不懂，既然这玩意跟佛国沾点边，也只有去问梧桐了。
<br/>　　梧桐沉吟半晌，从一个凹陷处捡起了几根骨头，拿在手中端详起来，过了一会说道：“叶少阳，你看看这些手指，有什么共同之处？”
<br/>　　叶少阳好奇地接过来一个，仔细研究片刻，摇了摇头，这些骨头都完全钙化了，一抹上面都掉灰，实在没什么发现。
<br/>　　梧桐道：“这些手指，都是中指！而且都是男人的中指。”
<br/>　　叶少阳一惊，再看这些手指，说道：“何以见得？”
<br/>　　“男人的手指，就算再细，也比女人的手指要粗，而且人的五根手指是不一样的，人的手指有长短粗细，但是从比例上来说，中指都是最粗的，而且两个指节中间的长度都是不一的，这个比例，你看看自己的手指就知道。”
<br/>　　叶少阳摊开自己的双手，看了下坞根手指的区别，的确如他所说，再看这些手指，好像确实不管粗细长短，指节的长度比例都是固定的……
<br/>　　这么多，全都是人的中指，叶少阳绕着坛城走了一圈，至少有十几堆，平均下来，至少有上百根手指，叶少阳隐约想到什么，吸了口气道：“这些不会都是不同的人的手指吧？”
<br/>　　“极有可能……我想到这是什么东西了。”梧桐望着这些手指，神情有些呆滞，摇了摇头，说道：“没想到真的有人用这种残忍的阵法……”
<br/>　　叶少阳听得好奇，忙问她这些都是什么鬼东西。
<br/>　　“这些，全都是从不同的人手上割下来的中指……”梧桐看了他一眼，道，“你是法师，你应该知道，中指是灵犀指，是人身上阳气最重的部位。”
<br/>　　灵犀指的说法啊，叶少阳当然知道，人身上阳气最重的部位有三处：一个是中指，一个是舌尖，还有一处是眼珠子。
<br/>　　所以普通人遇到普通的鬼或灵异事件，咬破中指或者舌尖，甩一股血出去，一般都能逢凶化吉。一般人被夺之后，法师也是用筷子夹中指，把鬼魂拉出来，遇鬼的时候，实在凶险，可以用红线缠绕中指，一般的鬼魂就没有办法把人的魂魄拉出去。

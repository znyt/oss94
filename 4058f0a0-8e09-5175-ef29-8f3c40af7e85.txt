　　防弹突击车中，罗昊、天蛇、李毅峰等人因为汽车的失控被甩得七荤八素，晕头转向。
<br/>　　嘭！嘭！嘭！
<br/>　　在跟桥面的剧烈摩擦中，防弹突击车的三个轮胎发生了爆胎，防弹突击车一头撞在桥中央的隔离栏上，汽车引擎盖都是被撞得翘起。
<br/>　　此时，圣安地列斯大桥已经是被完全升起，天蛇、罗昊他们被彻底阻隔下来。
<br/>　　突击车中，维尔纳特得知从前方传来的消息，苍白又显得狼狈的脸上露出一丝冰冷的微笑，对着驾驶突击车的force-recon队员命令道：“加速！追上去！”
<br/>　　“妈的！”
<br/>　　天蛇在方向盘上用力拍打了一下，骂道：“真他娘的不给面子，差一点，就差一点！”
<br/>　　“准备防御！”
<br/>　　罗昊拿着M4A1战术步枪跳下防弹突击车，做好了原地进行防御的打算。原先的撤离计划已经无法再继续进行，只能从正面进行防御，再寻找机会突围。
<br/>　　“支奴干”运输机上，齐凡推开车门走下汽车，看着飞机外已经被完全升起无法通行的圣安地列斯大桥，脸色变得有些难看，通过无线电跟梅庆伟说道：“宿鸟，掉头！”
<br/>　　“收到！”
<br/>　　梅庆伟驾驶着“支奴干”运输机掉头返回。
<br/>　　不放弃！不抛弃！
<br/>　　不论在什么时候罗昊都不会抛弃身边的人，现在这个时候，齐凡和梅庆伟他们自然也不会放弃罗昊、天蛇他们独自离开。更何况，齐凡还有预备的应急预案！这次营救行动，齐凡可以说是做了充足的准备！
<br/>　　齐凡眼中寒芒闪烁，走到机舱一侧，把一块墨绿色的防水布掀开，露出防水布下的东西，三具AT4火箭弹发射筒，齐凡拿起一具AT4反坦克火箭筒，扳起发射筒上的光学瞄准镜，按住耳麦对天蛇说道：“天蛇，准备执行B方案！”
<br/>　　“收到！执行B方案！”
<br/>　　天蛇从拿出四个钢索固定支架交给罗昊、李毅峰等人，让他们安装在防弹突击车的四个角上。
<br/>　　“原来你们早就有准备，看来我是白担心了！”罗昊看了眼自己手中的钢索固定支架，勾了勾嘴角说道。
<br/>　　“虽然冰刃准备了两个撤离地点，可依然会发生意外状况，所以在从比洛多那边‘借’来这架‘支奴干’的时候，冰刃早就有所准备，设计好了B方案。”天蛇对罗昊解释道：“如果在两个撤离地点撤离都失败，那么就使用B方案，在‘支奴干’的重火力掩护下，使用钢索将两辆突击车转移到安全的位置。”
<br/>　　“重火力？你们还有重火力？”李毅峰诧异地看着天蛇。
<br/>　　“那是必须的。”天蛇抬手指了指头顶上的“支奴干”运输机，说道：“他们来了！”
<br/>　　天蛇话音刚落，一枚火箭弹便是从“支奴干”运输机上飞了下来，在大桥上那些force-recon队员惊恐的表情中，击中了一辆突击车。
<br/>　　轰——！
<br/>　　突击车瞬间化作一团火球，被炸上数米高的位置，在半空中翻了个滚，然后重重的砸落在地上，成了一堆废墟。
<br/>　　突击车上，张子航握着车载机枪，子弹像不要钱一样的朝着大桥上的force-recon队员扫去，大桥上堵着很多汽车，但此时这些汽车中的人已经全都跑完了。
<br/>　　铿铿铿铿铿……
<br/>　　子弹落在那些停在桥面的汽车上，发出“铿铿铿”的金属闷响声。有几辆汽车的引擎盖被子弹击穿，被子弹击中的汽车引擎立刻就蹿起火光来，引擎着火，发生爆炸。
<br/>　　轰！轰！轰！
<br/>　　火球和爆炸后的黑烟冲天而起，将那些原本想要逼近防弹突击车的force-recon队员又给逼了回去。
<br/>　　而此时梅庆伟驾驶的“支奴干”运输机已经是飞抵了天蛇、罗昊他们所在的防弹突击车上空，四根钢索垂下，地面上的天蛇、罗昊、李毅峰等人把钢索跟安装在突击车上的钢索固定支架相连。
<br/>　　“支奴干”运输机上，玄龟和司徒空又是对着大桥上的force-recon队员发射了两枚AT4火箭弹，炸掉了对方两辆突击车，摧毁掉两辆突击车上.50口径的勃朗宁重机枪，拔除掉对方的重火力点。
<br/>　　.50口径的重机枪即便不是使用穿甲燃烧弹这种类别的弹药，对运输机还是有足够大的威胁。
<br/>　　“支奴干”运输机吊着引擎冒出滚滚浓烟的防弹突击车快速脱离战场，掉头离开。
<br/>　　大桥上，force-recon队员还在用自动步枪对着天空中的“支奴干”运输机进行射击，但是对运输机所能造成的威胁已经可以忽略不计。
<br/>　　布谷趴在机舱金属地板上，身前架着M200狙击步枪瞄准了大桥上一名扛起火箭筒的force-recon队员。
<br/>　　即便慕容飞这把M200狙击步枪上没有安装辅助系统，但是这对于布谷来说所造成的影响微乎其微，即便是没有辅助系统，布谷照样能打出精准的狙击。
<br/>　　布谷和那名force-recon队员几乎是在同时扣下了扳机，可是M200狙击步枪中所装填的竞赛级别子弹，在速度上占了优势。
<br/>　　M200狙击步枪专用的狙击步枪子弹抢先一步击中了那名force-recon队员的身体，导致对方手中火箭筒的准星发生了偏差，火箭弹歪歪扭扭的飞出发射器，击中了被升起的大桥桥面。
<br/>　　轰——！
<br/>　　橙黄色的火焰中，大量扎着钢筋的水泥碎块砸落下来，在这如天女散花一般的水泥碎块洗礼下，停在大桥上的那些汽车却是遭了秧，破损严重。
<br/>　　被砸掉后视镜和车窗玻璃都是好的，一辆全新的宝马轿车被一大块钢筋混凝土砸得车顶全都凹陷了下去。
<br/>　　右手腕上绷带已经被血浸透染红，绷带上隐隐有着血渗出的维尔纳特看着消失在自己视野中的“支奴干”运输机用拳头狠狠地砸在身边的汽车上。
<br/>　　他最后这all-in一般的赌博还是输了！齐凡这边营救行动准备之充分超乎了他的想象！
<br/>　　“司令部那边让我们立刻返回！”
<br/>　　这时，一名force-recon队员小跑着来到维尔纳特面前，向他传达司令部让他们全部撤离的最新命令。
<br/>　　“返回？”维尔纳特皱眉道：“司令部要放弃P病毒吗？”

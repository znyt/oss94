        连星儿听着明司寒的这话，却不作声了。
<br/>
<br/>        “星儿，你又在听我说话吗？”明司寒见连星儿发愣，下意识地抬起手来，在她的眼前晃了晃。
<br/>
<br/>        连星儿回过神来后，干笑道：“恭喜你啊！”
<br/>
<br/>        但是她的这句“恭喜”，却让明司寒感觉出来，她似乎并不替他感到高兴。
<br/>
<br/>        上午的课程上完后，明司寒送连星儿回宿舍的途中，有些按捺不住地问：“星儿，你是不是有心事？”
<br/>
<br/>        此刻，连星儿怀中还抱着课本，也因为他的这句问话，双手不经意间地将课本握紧了。
<br/>
<br/>        良久后，连星儿才郁郁寡欢地说道：“我不喜欢异地恋。”
<br/>
<br/>        “噗嗤”一声，明司寒爽朗地笑了笑：“傻丫头，你跟我一起去就行了啊！”
<br/>
<br/>        “那你就不能为了我留下来吗？”连星儿反问。
<br/>
<br/>        顷刻间，明司寒脸上的笑容渐敛。
<br/>
<br/>        连星儿接着说出了自己的真实想法：“M国离家里太远，我不想离开。我也不喜欢异地恋。如果你要去M国读HF大学，那我们分手好了。”
<br/>
<br/>        撂下这话后，连星儿加快了脚下的步伐，黯然神伤地往前冲。
<br/>
<br/>        明司寒有些头疼地皱起了眉头。
<br/>
<br/>        其实，进HF大学读计算机系一直是他的梦想。当年，他因为贺思思的哥哥贺泽川失去了一次机会。这一次，他不想再失去了。
<br/>
<br/>        只是，如果他去了HF大学读书，面临的却是要跟星儿分手。
<br/>
<br/>        “连星儿这丫头的公主病又犯了？”裴萱妍得知此事后，忍不住在明司寒的面前调侃。
<br/>
<br/>        明司寒看着不知何时向自己走过来的裴萱妍，很是无语：“你怎么来了？”
<br/>
<br/>        “我不来的话，就看不到你跟连星儿的分手大戏了！”裴萱妍打趣道。
<br/>
<br/>        明司寒没好气地说：“我跟她有婚约在身，你觉得有可能会分手吗？”
<br/>
<br/>        “明家小少爷，你是真的不了解连星儿。我告诉你，那丫头从小到大，就是被他们一家子团宠着。一身的坏毛病，是个男人都受不了。”裴萱妍忍不住诋毁起连星儿来。
<br/>
<br/>        其实她自幼就听过她连星儿的大名，毕竟人家生得比她好，要什么有什么。
<br/>
<br/>        “我觉得星儿很好，不需要你在我这儿说三道四。”明司寒冷冷地说道。
<br/>
<br/>        裴萱妍瘪了瘪嘴，只好转开了话题：“我们上头说，要你帮他查点东西。”
<br/>
<br/>        “查什么？”明司寒微微皱起了眉头。
<br/>
<br/>        总觉得自己是被他们组织给缠上了，阴魂不散。
<br/>
<br/>        裴萱妍从自己身上那黑色塑身裤裤兜里掏出一个手机，向明司寒递了过去。
<br/>
<br/>        “手机被删除的所有来往聊天记录。”裴萱妍接着说道。
<br/>
<br/>        明司寒接过手机，看了看后，嗤笑了一声：“你家头儿是被戴绿帽子了吗？”
<br/>
<br/>        “这不是我家头儿的私事，但我也不能具体跟你说是什么事情。反正，你赶紧查，然后拷贝出来。这台手机是偷出来的，随时都有可能被找回去。”裴萱妍提醒道。
<br/>
<br/>        明司寒睐了裴萱妍一眼，问道：“你是打算监督我操作？”
<br/>
<br/>        “当然！”裴萱妍挑了挑眉。
<br/>
<br/>        明司寒却将手机递给裴萱妍：“我虽然答应了帮你的组织做事，但是我也得捞点好处，是不是？”
<br/>
<br/>        “你想要什么好处？”裴萱妍反问。
<br/>
<br/>        明司寒会心一笑：“你也看到了，星儿在跟我闹分手，你好像比我还了解星儿的脾气性格，不然，你去替我牵个线，让星儿不要跟我闹。”
<br/>
<br/>        “喂！”裴萱妍对明司寒简直大跌眼镜，“这是你和她的私事！”
<br/>
<br/>        “对，我在拿我的私事跟你做交换条件。很简单，你什么时候说服星儿不跟我闹分手了。我就什么时候，替你的头儿恢复这台手机里的被删除的信息。”
<br/>
<br/>        “哼！你爱干不干！”裴萱妍恼火地拿回手机，准备转身离去时，又犹豫了。
<br/>
<br/>        组织里的黑客都无法破解这台手机，不然头儿也不会要她拿着手机来找他明司寒。
<br/>
<br/>        裴萱妍忽然间的自己就是个发泄球，这两头踢来踢去。
<br/>
<br/>        “好！”裴萱妍又将手机重新塞回明司寒的手中，“我去帮你把你的小公主追回来。”
<br/>
<br/>        说完，裴萱妍就径直往女生宿舍里走去。
<br/>
<br/>        明司寒看着这台手机，意味不明地扬了扬嘴角。
<br/>
<br/>        另一边，裴萱妍直接冲到连星儿的宿舍里，拉着连星儿的手，软硬皆施地说道：“连大小姐，我求求你不要跟明家小少爷闹分手啊！你们要是分手，我就没命了！”
<br/>
<br/>        “？！”连星儿被裴萱妍说得一脸懵，“喂，你是不是发烧了？”

　　萧逸落下后，随便开辟了一个山洞。
<br/>　　布下禁制屏障，就此开始盘膝坐下。
<br/>　　闭关，自然是需要的，不然这大半月时间，该如何渡过？
<br/>　　历练的话，这些许时间，也做不了什么。
<br/>　　干脆便是闭关，稳固下修为，顺便看看能否有别的提升。
<br/>　　而在这之前，萧逸还得先恢复了伤势。
<br/>　　手中取出几粒丹药，吞服下后，配合着体内的修罗之力，快速恢复着伤势。
<br/>　　他的伤势，只有两个。
<br/>　　一个是九寒、叶殇、林星河三人围攻他时，叶殇那一掌所伤。
<br/>　　这一掌，还算不了什么。
<br/>　　但另一个伤势，林星河那道护身剑气，加上剑帝本源，直接穿透他的胸膛，让他重伤，故更为麻烦些。
<br/>　　剑帝本源，乃是剑帝留下的本源之力。
<br/>　　对剑道武者，还有剑气之类的增幅，极其可怕。
<br/>　　林星河那一道剑气，乃是斩星剑气，本就强悍。
<br/>　　再加一丝剑帝本源增幅，自然这道剑气变得相当厉害。
<br/>　　往日，一般的伤势，萧逸直接会无视掉，边历练行走，边自己慢慢恢复。
<br/>　　可这一道剑气的伤势，他起码需要稳下来全力治疗数个时辰，方能痊愈。
<br/>　　“剑帝本源，确实厉害。”萧逸眯了眯眼。
<br/>　　“不过，我还不惧之。”
<br/>　　“这一次，暂且留你一命；待古帝之墓一行后，我不会再让你有机会活命。”
<br/>　　萧逸捏了捏拳头，眼眸冰冷到极点。
<br/>　　莫忧，他必杀之。
<br/>　　既触了他的底线，他自不会这么轻易罢休。
<br/>　　今日那一记怒炎指，原本可以取她性命。
<br/>　　但最后，萧逸还是收手了。
<br/>　　原因，在于霍老院长的那道危险的眼光。
<br/>　　对，眼光。
<br/>　　自萧逸进入天藏学宫前，他就知道，他的唯一阻拦，只有霍老院长。
<br/>　　所以他一开始便有了准备。
<br/>　　最后那一瞬，他瞬间爆发兽腿的速度。
<br/>　　原本，他以为足以让霍老院长反应不过来了。
<br/>　　但待得那一指真正落下，萧逸却明显发现，霍老院长的目光，比他更快。
<br/>　　这位一代传奇的手段，比他想象的更加惊人。
<br/>　　那一道目光，甚至可以瞬间止下萧逸指尖的所有火焰。
<br/>　　可是，霍老院长没有，仅仅是看了他一眼，露出危险的意味。
<br/>　　萧逸知道，那是警告的意味。
<br/>　　意思亦不言而喻，你易霄，可以伤莫忧，但，绝不可以取她性命。
<br/>　　若萧逸那时的一指不急急收回，而是誓要取莫忧性命的话，霍老院长绝对能拦下来。
<br/>　　而霍老院长，明明能反应过来，却眼睁睁看着萧逸接近莫忧，打出那一指，原因也很简单。
<br/>　　萧逸早便注意到，霍老院长的那句话‘数年前，你能从迷雾险地捞人，确实了不起’。
<br/>　　霍老院长，那时又不着痕迹地瞥了眼高空。
<br/>　　显然，霍老院长算是微微帮他一把。
<br/>　　若是那句话，由别的人说出，萧逸自不会多想。
<br/>　　可那句话，由睿智过人的霍老院长说出，萧逸自然便晓得了。
<br/>　　霍老院长的意思，便是还他当年救霍琳琅出迷雾险地的一个人情。
<br/>　　所以，萧逸打出那一指，霍老院长未拦。
<br/>　　但，若萧逸想得寸进尺，誓要取莫忧性命，霍老院长，不会眼睁睁看着。
<br/>　　事实上，以‘易霄’的身份出现在霍老院长面前，萧逸还是头一遭。
<br/>　　但，他以自己萧逸的真实身份在天藏学宫可是数次了，自然早便知晓霍琳琅和霍老院长的关系。
<br/>　　自然的，他能想明白这一切。
<br/>　　萧逸很清楚，若他那最后一指，非要强行为之，后果，只可能是霍老院长的出手，而导致一切白费。
<br/>　　故，他急急收手。
<br/>　　随后的洛前辈落下，还有三位总殿主现身，也导致了他不得不无奈离去，宣示着这场大闹彻底落幕。
<br/>　　这等强者现身，将代表着，战场，将不再属于他萧逸。
<br/>　　而是属于五位传奇之间。
<br/>　　即便他继续大闹下去，不惜大战，其结果就算真的能杀掉莫忧，那也等同是假手他人。
<br/>　　萧逸并不希望如此，故他只能就此离去。
<br/>　　但，这不代表着他毫无收获。
<br/>　　那最后一指，他虽急急收回，未能取莫忧性命，但最后手指一瞬，他却让怒炎指化作了禁制。
<br/>　　怒炎指，本就是代表了他控火一道的最强手段。
<br/>　　简单的一指，却是他体内所有火焰的爆发，那禁制，由他所掌控的所有火焰武道力量而构建。
<br/>　　八千多条完整火焰武道，其中蕴含六种世间强悍火焰，还有上次在冰尊殿得到的那些一流火焰。
<br/>　　另还有金焱圣火等稀有火焰。
<br/>　　八千多条完整火焰武道，无规律地构建成这一火焰禁制。
<br/>　　莫忧身上这一禁制不除，她将每日都承受八千火焰各自焚烧一次之苦。
<br/>　　那等折磨，更甚置身火海熔炉。
<br/>　　那等折磨，足以让人生不如死。
<br/>　　即便有些残忍，但萧逸丝毫不会为此皱眉半分。
<br/>　　莫忧这女子，萧逸接触数次，自是知晓此人绝非善类。
<br/>　　当年在百院之争上，便要用破魂针取青麟性命，今日又以剑帝本源和剑气偷袭于他，欲取他性命。
<br/>　　再加上她竟借八殿之名，借刀杀人之事。
<br/>　　此人如此恶毒，绝非善类。
<br/>　　而另外一点，若要除去她身上这火焰禁制的话，萧逸相信，洛前辈绝对能做得到。
<br/>　　但，绝非短时间内能做到。
<br/>　　洛前辈虽深不可测，身上掌控的完整武道，浩如烟海，手段绝对极强。
<br/>　　但若是别的，萧逸还不敢说自信十足；可说到控火本事，他对自己还是有足够自信的。
<br/>　　那八千火焰武道，单单是不同的组合，就能打出不同的禁制。
<br/>　　那般无规律组合，在彻底堪破之前，根本不可能破去。
<br/>　　萧逸也绝不相信，洛前辈能掌控世间强悍火焰，还有金焱圣火等稀有火焰的完整武道。
<br/>　　所以，洛前辈休想轻易破除。
<br/>　　以莫悠的性格，必会不惜一切代价请洛前辈出手救治。
<br/>　　而洛前辈若要救治，那便休想分出身前往古帝之墓。
<br/>　　上次剑帝碑外的遭遇，已然给萧逸敲醒了一记警钟。
<br/>　　这些个强者，若非要用强的，他着实无可奈何。
<br/>　　他和无数老家伙打过交道。
<br/>　　但，这些老家伙即便实力强，也并不显得多可怕。
<br/>　　可怕的是，洛前辈这种已然实力滔天，又已然不要脸的老家伙。
<br/>　　上次，剑帝碑前，特意带了破晓钟来，一下就把他的剑帝本源装走了，根本无可奈何。
<br/>　　这一次，洛前辈没机会了。

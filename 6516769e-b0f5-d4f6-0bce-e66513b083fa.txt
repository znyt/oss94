　　叶尘听到这话，也不由得微微有些惊愕，
<br/>　　这祝小白的天赋，已经算极为不错，他那位大哥的天赋，竟然比他还有高出许多，应该的确不是一个简单人物。
<br/>　　当然，纵然是修真界中的绝顶天才，也无法和重活一世的狂帝相提并论，所以叶尘微微惊愕过后，便也不觉得有什么了。
<br/>　　“那你哥哥他现在人在哪里？”
<br/>　　祝小白顿时小脸一垮，道：
<br/>　　“我也不知道！三年前，他为了寻求突破，便独自一人离开了火神宗，我找了他很久都没有找到，有人说他离开了蓬莱仙岛，前往世俗界了，所以我才会去太初城，可惜我没有通行令牌，无法进入世俗界……”
<br/>　　太初城，便是之前两人所在的那座城池，是蓬莱仙界进入世俗界的入口。
<br/>　　叶尘沉吟了片刻，
<br/>　　“你若真的想去世俗界找你哥哥，等我解决了药王宗，就带你一起离开！”
<br/>　　祝小白顿时大喜，继而又满脸担忧之色，
<br/>　　“恩公，那药王宗虽然在五大宗门中，实力是最弱的，可是毕竟也是五大宗门之一，药王宗的太上长老云千秋，据说一身修为，早已经达到元婴巅峰境界，而且药王宗擅长炼尸制毒，手段极其狠辣，仅凭你一个人的力量，只怕未必会是他们的对手啊！”
<br/>　　叶尘哈哈一笑，没有言语。
<br/>　　祝小白见叶尘根本不听自己的劝说，也是无可奈何，心中暗想，
<br/>　　‘以他的年纪，就算是和我大哥一样的天才，修为恐怕也有限，恐怕连药王谷周围的血雾毒瘴都闯不过去，等他吃了苦头之后，我再劝他也不迟！’
<br/>　　……
<br/>　　接下来，
<br/>　　祝小白指路，叶尘御剑飞行，大约用了小半天的功夫，足足跨越了数千里，终于进入了药王宗的势力范围。
<br/>　　“恩公你看！前面那一团血雾包裹着的，便是药王宗的老巢药王谷了！”
<br/>　　“传说这药王谷中，原本就有百里瘴气，后来又被药王谷的人，以各种毒药加以培养，形成了如今的血雾毒瘴，普通人只要吸上一口气毒气，便会全身溃烂而死！”
<br/>　　说到这里，祝小白的脸上，露出一抹浓浓的忌惮之色，接着又叹息道：
<br/>　　“如果我大哥在就好了，我们火神宗的御火心经，是药王宗毒功的克星，也正因为如此，药王宗一直垂涎我们宗门的这门神功，所以才一直想方设法，要灭掉我们火神宗……”
<br/>　　叶尘闻言，也不由得点了点头，
<br/>　　“药王宗的高手，修炼的大多是木属性功法，火克木，用火功对付毒瘴，的确可以事半功倍！”
<br/>　　说完这话的同时，叶尘拉着祝小白，身体一晃，已经朝那血雾毒瘴俯冲了下去。
<br/>　　“不要！”
<br/>　　祝小白顿时大急，可是他根本来不及，也没有能力去阻止，只好全力一掌拍出，
<br/>　　“烈焰掌！”
<br/>　　轰！
<br/>　　祝小白这一掌拍出，掌劲如同烈火，瞬间便将前方的血雾毒瘴，拍散了一大片。
<br/>　　可惜，
<br/>　　两人此时进入毒瘴之中，周围的方圆百里的毒瘴，何其浓郁，很快便又再次将两人包围其中。
<br/>　　轰轰轰！
<br/>　　祝小白无奈之下，只好接连挥掌，拼命阻挡周围的毒瘴。
<br/>　　而一旁的叶尘，却犹如没事人一般，继续缓步前行。
<br/>　　“不行了！恩公，我的烈焰掌还没有修炼到家，我要撑不住了，咱们还是先退出去另外想办法吧？”
<br/>　　祝小白坚持了一会，便已经满头大汗，忍不住大声疾呼道。
<br/>　　叶尘微微一笑，
<br/>　　“你小小年纪，火功就如此了得，也算是很难得了！”
<br/>　　说完这话，叶尘心念一动，
<br/>　　轰！
<br/>　　一团赤色火焰，自叶尘的体内发出，自动成了一个能量罩，将两人包裹其中，四周的毒瘴，顿时便再也无法靠近两人分毫。
<br/>　　祝小白看到这一幕，顿时两眼瞪得老大，又惊又喜，
<br/>　　“恩公，没想到你的火功也如此厉害！你这是什么火焰？似乎和我们火神宗的火焰，都完全不同，让人不自觉的就感觉到心里发慌！”
<br/>　　叶尘双手负在身后，一边缓步前进，一边开口道：
<br/>　　“我这是三昧真火！”
<br/>　　“什么！”
<br/>　　祝小白闻言，顿时两眼再次一瞪，脸上露出浓浓的惊喜之色，
<br/>　　“这便是传说中，号称可以焚尽万物的三昧真火？这种上古神火，不是早就已经失传了吗？”
<br/>　　“我也只是在我们火神宗的古籍中，看到过一些相关信息！”
<br/>　　“据说此火，必须以修炼者的精、气、神三道本源元气为引子，而且修炼方法极为苛刻，我们火神宗一脉，也只有上古时期的祝融大神，精通此火……”
<br/>　　这祝小白虽然年纪不大，但见识倒也十分不凡，竟然说的头头是道。
<br/>　　叶尘对他顿时更加喜爱，不由得笑道：
<br/>　　“区区三昧真火罢了，也没你说的那么厉害，你若是想学，回头我教你！”
<br/>　　祝小白顿时愣住了，
<br/>　　“恩公，你不是在和我开玩笑吧？你真的愿意把三昧真火传给我？”
<br/>　　叶尘哈哈一笑，
<br/>　　“我叶狂仙一言九鼎，只要你愿意拜我为师，区区三昧真火又算什么？将来我可以将我毕生所学，都传授给你！”
<br/>　　这个祝小白天赋极佳，而且人品也很不错，比叶尘之前收到的那个徒弟，都要强上不少，因而便又动了收徒的念头。
<br/>　　然而，让叶尘没有想到的是，祝小白思量了片刻，居然摇了摇头，
<br/>　　“不行！恩公，我暂时还不能答应你，我要先找到我大哥，经过他的同意的才行！”
<br/>　　叶尘也顿时一愣，不由得笑着摇了摇头，
<br/>　　“好吧，随你！”
<br/>　　……
<br/>　　就在两人正缓缓穿过血雾毒瘴之时，
<br/>　　与此同时，药王宗之中，
<br/>　　“宗主大人，这次行动，我们竟然损失两位长老，再加上之前的夏侯长老，这笔账我可要好好和那小子算一算！”
<br/>　　说话的是一名须发尽白的老者。
<br/>　　而此时，
<br/>　　一旁的上座之上，正端坐着一名颇为威严的中年男子，
<br/>　　“公孙长老放心！我们虽然损失不小，不过总算是把那小子的姐姐抓了过来，根据之前调查的情报，只要这个女人在我们的手上，那家伙肯定会自投罗网，只要他敢踏入蓬莱仙界，就算有天大的本事，也必死无疑！”
<br/>(本章完)

　　基地运行一切进入正轨，苏重开始研究第二代抑制剂。
<br/>　　有初代抑制剂为基础，苏重只用两天就研究出第二代。然后立刻给自己注射。
<br/>　　再次感受到一次灼痛和昏沉后，苏重得到了久违的轻松。
<br/>　　“先生，根据扫描数据显示，您的体态特征已恢复正常人水平。只要不断注射二代抑制剂，从外表上看，您就是一个正常人类。”红后满脸欣喜。
<br/>　　“根据计算，您制造出解毒血清的概率已经提高到53%。”
<br/>　　“看来，我们还能继续合作下去啊。”苏重轻笑，他脸色已经透出红润，眼睛不再充斥血丝。
<br/>　　“就目前来看，是的。病毒真的有可能在您手中得到清……”
<br/>　　轰！
<br/>　　一声巨响，整个基地猛烈震荡。头顶电灯瞬间熄灭，各种仪器设备也突然没了声息。
<br/>　　走廊里亮起应急灯，红后虚影也消失在眼前。
<br/>　　“红后！”苏重眉头紧皱：“红后，基地出了什么问题！”
<br/>　　三分钟后，一阵嗡嗡声传来。整个基地灯光在闪烁中逐渐亮起，各种仪器设备也开始运作，很快恢复正常。
<br/>　　红后投影也在此时出现，看着苏重犹豫道：“先生，您的家消失了。”
<br/>　　“发生了什么？”苏重有种不好的预感，他想到了某些情节。
<br/>　　“保护伞公司在浣熊市投射了一颗小当量核炸弹，整个浣熊市已经从地球上消失。”红后小心翼翼的打量苏重脸色。
<br/>　　她本以为苏重会愤怒，可却意外发现苏重竟然非常平静。
<br/>　　“您难道不生气吗？”红后不解。在他理解中，家园被毁，不应该愤怒吗。
<br/>　　“愤怒，不。我还活着，为什么要愤怒。”苏重淡淡道。
<br/>　　如果是身体原主人可能愤怒，苏重这个外来者还真没什么感觉。
<br/>　　“那对于保护伞公司用核炸弹，清洗整个浣熊市，您难道就不感到震惊吗？”红后无法理解。
<br/>　　苏重轻笑：“你如果知道我经历过什么，就会知道我为什么不感觉震惊了。”世界毁灭都见过，苏重怎么还会为一个核洗地震惊。
<br/>　　他每次破界穿越，都相当与给世界打开一条无法弥补的裂缝。时间流逝，裂缝会越来越大，最终都逃不脱湮灭消失。
<br/>　　“虽然在意料之中，但总要做些反应才是。”苏重深吸一口气：“你说我要是真的把生化危机解除掉，那些幕后之人会不会气的跳脚？”
<br/>　　红后瞬间沉默。
<br/>　　生化危机并不是意外，是保护伞幕后黑手的灭世计划导致。但这件事情外人根本不知道，苏重却表现的了然于心。
<br/>　　红后不理解苏重从何处获得情报，但这不影响她对此事的赞同：“先生，如果您真的成功，您将成为救世主。”
<br/>　　“可别叫我救世主，我感觉这是在骂人。”从来都是他破灭世界，还从没被人叫过救世主。
<br/>　　“先生，爱丽丝回来了。还带回了一些幸存者。”红后投影出基地入口影像。
<br/>　　“轨道车已损坏，入口被掩埋。暂时来说，您已被困在地下。”红后介绍着爆炸过后基地损失。
<br/>　　“电站遭到毁灭性打击，只能启用地热发电设备。该设备技术尚不成熟，功率不足，只能支撑半个基地用电。”
<br/>　　“那就是当降低基地监控设备，一些用不到的房间和仪器，也都停止供电。我们一共也没多少人，开启半个基地，够用了。”
<br/>　　“好的先生。”
<br/>　　“博士，浣熊市一紧毁了，他们根本是我带回来的幸存者。”爱丽丝带着二十几个人找到苏重道。
<br/>　　自从苏重你已经成功研究出抑制剂后，几人便开始称呼苏重为博士。
<br/>　　保护伞那么大公司，都不能遏制病毒扩散，而苏重短短三天就能研制出病毒抑制剂，苏重的能力让几人敬佩。
<br/>　　打量这二十多个人，两个穿着战斗服的雇佣兵比较显眼，还有几个穿着警察制服，应该是浣熊市警察系统的幸存者，剩下的男男女女应该是浣熊市普通市民。
<br/>　　吸引苏重目光的是躲在爱丽丝背后的一个小姑娘，对方脸色苍白，眼眶发黑，甚至能看到皮肤下的血管。一看就不太健康。
<br/>　　察觉到苏重注视，小女孩缩回探出的小脑袋，直接躲在爱丽丝背后不出来。
<br/>　　“雷恩，马特，给他们安排住宿的地方。他们来的正是时候，浣熊市被毁，种植园必须扩大。这下我们有足够的人手了。”苏重直接吩咐。
<br/>　　几个人刚从炸弹洗地中幸存下来，脸上还残留着慌乱，也没有心思抱怨，乖顺的跟着雷恩和马特离开。
<br/>　　苏重也不怕他们造反，有红后在，基地一切情况都在监控中。钢钉能杀丧尸，也能杀人，他从不是个心软的人。
<br/>　　“博士，我有事情要和您谈一谈。”爱丽丝单独留下来，满脸严肃的对苏重道。
<br/>　　“是为了你身后的小女孩？”
<br/>　　“您知道？”爱丽丝诧异。
<br/>　　“以我对生化病毒的了解，你以为我会看不出，她也也感染了病毒了吗？”苏重淡淡道。
<br/>　　这会儿苏重也想起来，这个小女孩的父亲好像也是保护伞公司的成员。对方利用病毒给他女儿治病续命。小女和体内的病毒数量，可一点儿都不比苏重体内少。
<br/>　　“那您能救他吗？”爱丽丝满脸期待。
<br/>　　“他和我的情况不一样，和你的情况也不一样，这需要时间。不过有二代抑制剂，能保证她短时间内没问题。”苏重沉思片刻道。
<br/>　　“您知道我的情况？”爱丽惊讶。她以为自己苏重不知道呢。
<br/>　　苏重笑了笑：“我能看出来，似乎并不奇怪吧。”
<br/>　　爱丽丝苦笑：“以您在生化病毒上的造诣，确实能看出来。”
<br/>　　她还以为自己藏得很好呢。就怕苏重把她切片研究了，就和保护伞公司对待实验体一样。没想到苏重早就知道，而且一直没动。
<br/>　　“我可以提供给你血液，配合您的研究。您能治疗好她吗？”爱丽丝看着苏重曼联期待。
<br/>　　“我会尽量。”苏重挑眉。没想到爱丽丝这么看中这个小女孩。
<br/>　　不过苏重也没撒谎，之前他忙着研究病毒抑制剂，一直没来得及探究生化病毒的其他方向。
<br/>　　如今二代抑制剂成功，他有了更多时间。而且还有爱丽丝这个完美融合的标本。距离研发出彻底清除病毒的血清又前进了一大步。
<br/>　　“我会先治好她的病，然后在开始生化病毒的研究。你可以放心了。”苏重对爱丽丝承诺到。
<br/>(本章完)

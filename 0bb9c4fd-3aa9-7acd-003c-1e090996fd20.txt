　　昆仑山，玉虚宫中。
<br/>　　在共工与玄天大战到不周山附近之时，元始便一直在关注着两人。他看到玄天击败共工氏，看到共工陷入迷障之中，甚至看到共工拼尽全力，燃烧自己去撞断不周山。
<br/>　　元始的表情一直都是平平淡淡，毫无波动，但就在共工撞上不周山之时，他的脸上突然露出了无比吃惊的表情。他感受到了一丝不同的波动，一丝非比寻常的波动在不周山上出现，甚至在共工身上出现。
<br/>　　这丝波动非常细微，一般人直接发现不了，不说一般人，就算是一般圣人都发现不了。但，元始当时的神情是极为专注的，他还是发现了那一丝轻微的波动。就是因为他发现了，所以他的脸上才会露出令人吃惊的表情。
<br/>　　那是天道之力！
<br/>　　天道之力为何会在此刻出现？而且还出现在不周山与共工身上？不周山不是盘古脊梁所化吗？天道为何想要它断裂？它有何目的？
<br/>　　这一刻，元始好像发现了一个惊天的大秘密。他抬头看了看三十三天之外，那里，正是紫霄宫方向，因为，他觉得，这一切会与道祖鸿钧有关。
<br/>　　“会是你吗？”玄天在心中暗暗道。
<br/>　　是日，共工与武祖玄天争锋，大败之，怒而触不周之山，折天柱，绝地维。
<br/>　　不周山，乃是开天神尊盘古脊梁所化，本身就极为坚硬，比之大多数神铁、仙金、道材都要硬上三分，再加上有着盘古神尊遗留的威压，实乃天地之柱，天地初开之时，就算是准圣境强者也无法靠近，那时，别说是一个祖巫共工了，就算是堪比圣人境界的巫神也休想撞断天柱。
<br/>　　可惜，随着无尽岁月的流逝，盘古大神的威压在渐渐消散，到了今时今日，几乎消失殆尽，没有了盘古威压守护，不周山的档次降了不止一级，再加上有可能是天道之力的暗中相助，即便是不周山天柱再坚固，又如何能够抵挡住一个祖巫的拼命撞击？
<br/>　　“不好！”
<br/>　　眼见半截不周天柱向自己倾砸下来，即便是玄天，亦不敢有半分大意，要知道，这可是不周山啊，盘古大神脊梁所化，即便如今再怎么不如之前，玄天也不敢拿自己的身体去与它相撞，开玩笑，那不是在逞强，那是找死的行为，就算是圣人，也不敢如此。
<br/>　　眼见那半截天柱就要砸到自己，玄天来不及多想，拼尽全身法力，双手用力向着眼前的虚空一撕。顿时，玄天眼前空间就如同一张纸一般，被他撕出一道巨大的裂缝，裂缝之中，传出一阵阵巨大的撕扯之力，还有空间乱流风暴的气息。他来不及多想，连忙跳了进去，之后，空间缓缓封闭，恢复原样。
<br/>　　“轰隆隆！”
<br/>　　半截天柱砸落在地，爆发出惊天动地般的爆炸声响，一股股气浪如同波涛一般汹涌澎湃，空间在这一刻变得爆裂起来，虚空止不住的在震荡，乾坤倒转，风云失色，狂风乱舞，日月倾倒，星光黯淡。
<br/>　　此刻的空间，就如同一道平静的湖面上，落入一块巨石，湖面变得不再平静，一圈圈波纹向四周扩散开来，不周山附近的生灵，全部在此刻忍不住的爆裂开来，只是因为他们身体之内的气场与外部空间的气场相差巨大，所造成的。
<br/>　　大片大片的地面爆裂开来，高空之上，一朵巨大的云雾蘑菇缓缓升起，让洪荒许多地方的生灵都能看见。
<br/>　　天庭之上，三十三天在剧烈摇荡，不少空间在破碎，地面炸裂开，最下方的三天，甚至都全部倒塌被毁，驻扎在里面的妖族全部死亡。南天门倒塌，凌霄宝殿一样在剧烈摇晃震动，若不是有不少妖族大神通者纷纷出手，才将其慢慢稳定下来。不然，天庭的损失甚至比这还要更大。
<br/>　　“共工！可恶！”
<br/>　　天庭之内，传来帝俊的大声咆哮声，轰动天地。可见，帝俊此刻是如何的愤怒。本帝在家养伤，招谁惹谁啦？就这样无故遭殃了，简直是躺着也能中枪啊，真是欲哭无泪。
<br/>　　天庭之上的天河倒塌，九天之上，出现一个黑色的巨大空洞裂缝，滚滚天水从裂缝之中流淌出来，化为惊天瀑布，化作无尽洪流冲击下来，流入到洪荒大地之中，不知道死了多少生灵。而公共祖巫，也早已随着不周山倒下，坠入无尽洪流，淹没在九天弱水的洪流之中，消失不见，死活不知。
<br/>　　“可恶！”
<br/>　　另一面空间外，突然出现一双白皙的手，只见那人双手向外，用力一撕，那面空间瞬间便被撕开一道大口子，玄天从其中走了出来，他此刻满脸愤怒，看着倒塌的不周山。
<br/>　　玄天放眼望去，只见那支撑天地的不周山已经自半截之处断开，周山不复，天柱折断，大地震动，天缺西北，地陷东南。
<br/>　　原本不周山所在的天空之上，露出一个恐怖的大洞，滚滚不休的九天弱水源源不断的奔腾而下，数不尽的洪荒生灵被淹没其中，甚至天庭之上的妖族，有的都顺着那个巨大的窟窿被冲击而下。
<br/>　　那天河之水最是歹毒，为弱水，能侵蚀元神，融化筋骨。
<br/>　　弱水之上，鸿毛不浮，飞鸟难过，除了龙族或者擅长水系法则的大神通者，其他任何人，任你神通再广大，法力再高深，都无法在弱水中存活。
<br/>　　此刻天上通了一个大窟窿，降下无尽弱水，若是一个处理不好，人间界就有可能化为一片汪洋，准圣以下者，很难成活。
<br/>　　弱水一出，神魔辟易！
<br/>　　弱水一直在天庭天河之中，就连妖族都不敢去碰，只是偶尔取一些用来炼器，如今日这般，天河之坝倒塌，弱水纵横，天庭不知要死多少妖族，更糟糕的是，它冲击到了人间，无数生灵就会因此殒命。
<br/>　　这得是多大一笔因果啊！谁能还得清？共工？还是妖族？
<br/>　　此刻不周山断裂，支撑天地的巨柱消失，一个硕大的窟窿出现在原本不周山所在的天宇之上，无穷无尽的九天弱水从九天之上倾泻而下，在洪荒大陆上纵横肆虐，所过之处生灵不存，一片狼藉。
<br/>　　巨大的动静，惊天动地，正在大战之中的人、巫两族都纷纷停止了下来，看着九天之上的巨大窟窿，他们顿时便知道那是共工的杰作，纷纷愤怒不已。
<br/>　　但，他们也知道，既然事情已经发生了，再如何愤怒也无济于事了，此刻要做的是如何保护族人？

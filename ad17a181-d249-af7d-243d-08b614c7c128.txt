　　　　在鲁初雪经验丰富的指挥中，很快大量的物资都囤积在了一架架直升机上，高少辉也已经将和上那拿来的大东西搬上了天照3000，同时卢宏盛也速度极快的将那柄短刀淬炼完毕，配了个同样坚固的刀鞘。
<br/>　　当高少辉拿到手的时候，激动的双手都在颤抖，当他将这柄短刀拔·出的时候，一抹明黄色的微光在其刀刃的表面若影若现。
<br/>　　卢宏盛看着这柄短刀，也有些心情怪异：“这千月石好像很特殊啊？”
<br/>　　“当然。”高少辉笑着将短刀重新入鞘，道：“它可以要了我的命。”
<br/>　　不等卢宏盛继续发问，高少辉就已经离开，而那些直升机队也因为速度原因早已先走一步。
<br/>　　此时远在南署基地的迷宫战区内，守在基地中心处的新兵们望着前不久送来的第一批物资，用最快的速度重新归纳分配，在这个基地的所有战区内，有着一批人专门送食物和水，他们游走在基地各处，一边躲避着丧尸群的攻击，一边给前方战斗在一线的同伴们提供源源不断的供应。
<br/>　　这些人虽然没有正面战斗，但在团队中的贡献也不少。
<br/>　　此时更多的直升机飞来，将一批又一批的物资卸下，将这个被尸海包围的基地救活。
<br/>　　“多亏了空军一次性解决飞行丧尸。”一名后勤人员感叹出声。
<br/>　　“是啊，要不然这些物资都送不进来，我们也早晚会被困死在这里。”
<br/>　　前方第一战区附近的指挥高台处，楚涵已经从高墙上退下进入指挥间，旁边的参谋部成员依然在紧盯迷宫战区，高少辉此时也恰好驾驶着天照3000落下。
<br/>　　“东西拿来了？”楚涵并没有回头，光是凭借着细微的空气流动，便得出高少辉已经走近的结论。
<br/>　　高少辉也早已习惯这一点，哪怕他走起路来根本没有脚步声，当即开口道：“真是没想到啊楚涵，你竟然把白忧整个私人基地的东西都偷了出来，我靠我说狼牙基地怎么一下子发展这么迅猛，原来发了一笔横财啊！”
<br/>　　楚涵挑了挑眉：“什么叫偷？我那是抢。”
<br/>　　高少辉愣了愣，总觉得这句话怎么听怎么熟悉，一时没功夫多想的他将和上给的大家伙放在地上：“就这玩意儿，你们打个仗还打算开广播？”
<br/>　　毕竟这东西实在长得像老式音响！
<br/>　　楚涵惊讶的看了高少辉一眼：“你可以啊，这都猜得到！”
<br/>　　高少辉傻眼：“不是吧？你真打算放广播？我靠你专门让我去一趟段氏基地，就为了拿这？有啥破用处啊！”
<br/>　　“新发明，你懂个屁。”楚涵翻着白眼，忽的又眼神一顿：“你这短刀是出自卢宏盛之手吧？给钱了吗？”
<br/>　　高少辉原本就本打算瞒着楚涵短刀的事，但此时听到对方一上来就问这个，让他直接气红了眼：“我靠！我是用了锻造部的原材料，但淬炼石可是我自己拿出来的，那玩意儿可是可遇不可求，我敢说连白家都没有，全世界就只有我手中有这么一小块，全用在这柄短刀上了，你竟然还问我要钱！”
<br/>　　楚涵哪管什么石头，直接道：“刀在你手上，你不给钱这是什么道理？”
<br/>　　高少辉急的直跳脚：“我要短刀，还不是为了跟你站在同一战线啊！”
<br/>　　这就让楚涵听不懂了，他偏了偏脑袋：“我什么战线？”
<br/>　　“对付白家啊。”高少辉想也不想的回答。
<br/>　　楚涵神色变了变，再次看向高少辉腰间的短刀：“有什么特别之处？”
<br/>　　高少辉咧开嘴，眼神一瞥旁边没有看向这里的何峰：“秘密。”
<br/>　　楚涵眯了眯眼睛没有继续问下去，转而冲着蒋天庆问道：“战况如何？”
<br/>　　“迷宫战区一切顺利，目前为止没有出现伤亡。”蒋天庆立即将最新情况说出：“第一战区全是狼牙老兵在战斗，丧尸群几乎一进来就死·光了，哪怕是高阶丧尸也一个没放走，高墙上的三战队状态照旧，但有着第一战区狼牙老兵的把守，他们的节奏也慢了下来，上下两批队伍在合作，射击队已经一个多消失没开枪了。”
<br/>　　当楚涵开始紧急调整，狼牙战团总计一万人不到的老兵全部进入第一战区后，那些原本由新兵负责的丧尸群就被杀的毫无招架之力。
<br/>　　毕竟狼牙战团的每一名老兵，都是从最基础的丧尸战开始成长的，而在他们手中死去的丧尸数量早已不可计数，杀丧尸的熟练程度也早就炉火纯青。
<br/>　　情况似乎顺利的过分，可在指挥间内的所有人除了高少辉都很清楚，越是看似顺利的情况，越代表着异变随时可能发生。
<br/>　　果然就在第二批来自段氏基地的直升机队放下物资离开，后勤人员刚将最新的一批物资分配到全战场的各个队伍手中时，基地外时不时响起，已经让所有人都习惯的响亮声音，不知什么时候起已经很久没有再出现。
<br/>　　“准备。”这时候楚涵的声音忽然在扬声器内响起，让基地内所有人都听得到：“高墙新武器能量耗光，开始正面对抗尸潮！”
<br/>　　防守队在高墙内不断朝外发出的强大威力新武器，终于在两天后失去了效果，这两天内的尸潮战大多数丧尸都是由这些新武器解决，依靠的更是科技取胜。
<br/>　　可当新武器的能量耗光，尸潮的数量还剩下一半的时候，剩下的才是这场战斗最为艰难的时刻！
<br/>　　冲入第一道防线内的丧尸群数量开始随之成倍增多，到后来不仅一批接着一批的涌·入，还开始呈现源源不断的拥挤状况。
<br/>　　每一个通道都被丧尸群堵的严严实实，就像是一群老鼠从外冲进来，虽然没有秩序但却如同潮水般凶猛迅速。
<br/>　　在第一战区的全体狼牙战团老兵们，也渐渐开始难以招架，当越来越多的老兵身上出现丧尸抓伤后，第一道防线终于被尸潮攻破，开始蔓延进入基地之内，而后流入有着八万新兵把守的迷宫战区。
<br/>　　高墙上的三战队情况更是一下子步入危机状态，没有新武器威胁的高阶丧尸一只又一只的爬上高墙，从外面看基地这一圈的高墙外壁上密密麻麻的都是体型巨大的丧尸，就像是成群结队的壁虎一般牢牢钉在墙面上！

        &lt;h3&gt;第四卷第三章  相见2&lt;/h3&gt;
<br/>
<br/>        谷强听到闫欢提到了陈子悦和陈子昂，他心里一紧，他没有回答谷正娟的问话，而是紧紧的盯着闫欢问：“是一个人吗？”
<br/>
<br/>        “我不知道是不是一个人，陈子悦和我一直是运营者，而陈子昂是股东，她还有别的生意在做。”
<br/>
<br/>        谷正娟很好奇，因为他多次听到陈子昂的名字，谷强在信息化项目上会经常打电话和陈子昂沟通，而且陈子昂有安排人过来，但是一直没有见过陈子昂本人，因为谷强说过他们很像，谷正娟一直想知道陈子昂是什么模样。
<br/>
<br/>        “那陈子昂是做什么生意？”秦少卿替谷正娟问出来了。
<br/>
<br/>        “信息化项目，很厉害的一个女人！”闫欢笑着说。
<br/>
<br/>        “哇，强子，那应该是一个人了，你们这样一说，我好想见一见陈子昂是一个什么样的人，不会你真是她失散的哥哥吧？哇，想一想你有那么厉害的家庭，我都要对你刮目相看了。”谷正娟在桌子底下踢了踢谷强。
<br/>
<br/>        “唉，闫欢，陈子昂会来甸城吗？”谷正娟问闫欢。
<br/>
<br/>        “我们投资的项目她是同意了的，而且听她说甸城也有信息化项目，她已经安排人过来了，至于她自己过不过来不一定，毕竟这边的项目对她来说只是小事情，也不重要。”闫欢淡淡的说。
<br/>
<br/>        谷正娟听了闫欢的话，心里很震惊，他们谷家在甸城是有多少年了，她的父亲谷老大和秦寿都愿意让谷强来做信息化项目，而闫欢竟然说甸城的项目对陈子昂来说并不重要，那陈子昂是有多厉害？
<br/>
<br/>        不过想一想也真是的，一个商场他们都耗资十几个亿，可想而知生意做得肯定不错。
<br/>
<br/>        谷正娟心里开始盘算了，他们的房子不收房费，要作为入股的形式来做这个事情，入股肯定是比收房租要划算很多。
<br/>
<br/>        谷强低下头，喝了一口汤，他感受不到汤是什么味道，心里只是觉得又感动又苦涩，他的妹妹这是带着她的身家财产都过来了。
<br/>
<br/>        那样闫欢和陈子昂是不是提前预先沟通好了？罢了，原本他还怕闫欢知道他的身份，会对他不利，如今看来她们都已经商量好了。
<br/>
<br/>        谷强不知道闫欢从来都没有给任何人说过他的事情，他也不知道陈子昂也没有给任何人说过他的事情，只是这俩个女人都是奔着同一个目的而来。
<br/>
<br/>        秦少卿也是知道陈子昂的，他从陈子昂提前安排过来帮忙做方案的员工嘴里经常听到他们公司的事情，也对陈子昂充满了好奇。
<br/>
<br/>        他记得有一次他提起信息化项目的时候，祁峥和陆玉在旁边，祁峥说：“其实要说信息上的事情，我真的是佩服陈子昂，短短的几年已经站在行业顶端，一般人很难做到这个份上，无论是财力还是软硬件实力都在我们之上。”
<br/>
<br/>        “在怎么厉害的企业都得依靠国家和有关部门的支持，没有支持你再有本事也发挥不出来。”陆玉冷冷的说。
<br/>
<br/>        “你说得对。”祁峥微微一笑。
<br/>
<br/>        “所以，别说某个人厉害，而是我们的社会发展快，国力强大，与她个人有多大的关系？”陆玉嗤笑一下接着说：“有些人不过是仗着运气好了点，如果真的不让她发展，就像是碾死一只蚂蚁一样容易。”
<br/>
<br/>        “话不可这样说，如果个人没有能力，你想扶持也扶持不起来，这些都是相辅相成的，我们现在的社会是需要这样有担当有责任的人作为中流砥柱来促进社会的发展，作为我们来说，我们要摒弃个人好恶，进行纠正引导他们向正规的地方发展，而不是不让他们发展。”秦少卿反驳陆玉的话。
<br/>
<br/>        秦少卿从回忆中回过神，看着闫欢笑着说：“我也曾经听过你们口中的陈子昂，按照你这样的说法，我也很想见见这个人，感谢她来为我们甸城的发展做贡献。”
<br/>
<br/>        “嗨，你们见到我就等于见到她了，我和她还真的是很像，闫女士，你觉得我们像不像？”谷强笑着问闫欢。
<br/>
<br/>        “是有点像，刚刚看到你的时候，我都吃了一惊，你和他的哥哥像。”
<br/>
<br/>        “你还见到了她哥哥？”
<br/>
<br/>        “是啊，她好几个哥哥呢。嗨，怎么都不吃饭了？来来来，干杯！”闫欢成功的将大家的注意力提起来，也成功的将谷正娟的胃口吊起来了。
<br/>
<br/>        “来来来，欢迎闫欢来甸城投资，希望我们能够合作成功！”谷正娟和闫欢碰了一下杯。
<br/>
<br/>        谷强这几年一直跑着信息化项目，他将祁峥召集到了甸城，这样祁峥可以带领人做调研、做方案，陈子昂也安排了公司的几个技术骨干一起让祁峥带领着，他们将甸城整个地区都进行了预测，这个项目快折腾了三年。
<br/>
<br/>        而谷老大的那个事业因为谷强在做信息化，进度缓慢。
<br/>
<br/>        他将老三沧岚雪安排出去开发别的市场，先从正轨的市场做起，名义上他们是正轨的生意人，加上谷老大的仁善，加上谷强的运作，谷家也是商界一股清流。
<br/>
<br/>        提起谷家，大家也是无人不知无人不晓，可是大家很少见到谷老大和他的儿子谷强，这俩个人基本不在大众露面，人们并不知道这俩个人的模样，这更增加了谷家的神秘性和好感度，你看，人家谷家做了好事情，连面都不露。
<br/>
<br/>        谷强也只是在极少的圈子里活动，包括在甸城，除非有必要他才出面。
<br/>
<br/>        沧岚雪出去了会给谷老大汇报工作，谷老大也会和谷强商量，谷强的意思先打好基础，站稳脚跟，所以这几年谷家出去的钱财很多，做的事情基本收益很少，但是赚取了很好的名声。
<br/>
<br/>        他们的灰色事业也在进行，只是少量进行，由老三老四在小打小闹，只要不出大的问题，保证收益就行。
<br/>
<br/>        闫欢和陈子昂的到来，虽然说还可以拖上一段时间，但终究不是办法，如果他们在甸城投资，那就更脱不开身，谷强要想好怎么做对他们才更有利。
<br/>
<br/>        “闫欢，投资有风险，还得谨慎，你们这投下去的不是小数目，好了有收益，不好，怕是连本都拿不回来。”谷强看着闫欢。
<br/>
<br/>        “这个我们都考虑好了，我们是抱着必胜的信念来的，要做就做到最好！我们没有想到会失败！”
<br/>
<br/>        “如果失败了呢？”
<br/>
<br/>        “除非你......你们不支持我们！”
<br/>
<br/>        “怎么会不支持呢，你是来拉动消费，搞活经济，是来投资的，我们当然是举手欢迎。”秦少卿很开心，他都没有想到闫芈的姐姐这么厉害。

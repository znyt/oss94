　　枪声便是号令，那一声震耳的枪声响过之后，无数的大唐禁军从密林中跳了出来，左手刀右手枪，对眼前一切能动的生物展开了屠杀。
<br/>　　是的，就是屠杀！在唐人的眼中，那些无法沟通的土著与猴子根本没什么区别。
<br/>　　而且在眼下这种环境里面，你不杀他，他就杀你，根本没有什么怜悯可言，如果不想死就只能放手去杀。
<br/>　　可以连发，而且装弹迅速的左轮手枪在这次的屠杀中起到了很大的作用，打的那些‘猴子’一点反抗之力都没有，往往是刚一冒头，就被不知从什么地方射来的子弹爆掉了脑袋。
<br/>　　还有一点就是岛上的土著都十分迷信，看着唐军手中火光一闪，接着一声巨响过后便有人死去，大部分的土著瞬间就没了战斗的意志，丢掉手中的一切就开始逃窜。
<br/>　　这些敌人都是天兵，是神仙，是不可力敌的存在，这便是土著们心中对大唐禁军唯一的印象。
<br/>　　先知和族长以及那些族老们全都懵了，他们没想到唐军会来的这么快，而且自己族中的抵抗竟然对他们形同虚设，根本起不到任何作用，只刚一接触便已经溃散。
<br/>　　只有刚刚的那个少族长，可能是因为神仙草吸的多了，竟然不顾一切的抄起一张放在角落里的木弓，嚎叫着冲了出去，开弓放箭动作流畅的一逼，只是那箭飞出去之后直接插在了一个土著的胸口，然后就看那家伙哼都没哼一声就栽了下去。
<br/>　　不过，少族长的攻击也就只有这些了，还没等他第二枝箭搭上，一颗不知从什么地方飞来的子弹已经撬开了他的头颅。瞬间，白的脑浆，红的鲜血爆的到处都是。
<br/>　　外面杀声震天，部落中唯一像样点的房间里面却安静的可怕，少族长死了，不过却没有人看上一眼，族长还有好几个孩子，死的只不过是大一些的，他还不缺继承人，完全没有必要在这个时候出去报仇。
<br/>　　……
<br/>　　乒乒乓乓的枪声很快就稀疏了起来，片刻之后外面彻底安静了。
<br/>　　而就在部落的先知打算出去看看的时候，一个高大的人影从外面走了进来，一身漆黑的护甲，脸上涂满了绿色的树汁，手中提着一把让无数人战栗的大唐横刀。
<br/>　　“你们有没有会说大唐官话的？”那人影进来之后，根本没有任何废话，甚至都没有看清房间中有几个人，便冷声问道。
<br/>　　“我，我会一些！”先知到底是先知，结结巴巴的回答道，虽然并不标准，但至少可以交流。
<br/>　　“会说人话就好，跟老子出来！”高大人影点了点头，转身便走。
<br/>　　而那先知则是看了一眼族长，又看了看几个族中长老，这才叹了口气跟着那人影走了出去。
<br/>　　到了外面，看到了外面的景像之后，先知立刻倒吸了一口冷气。
<br/>　　只见处眼所见的到处都是族人的尸体，鲜血几乎流成小溪，一些还没有死透的族人正在努力挣扎着，但最后都逃不过那锋利的横刀。
<br/>　　一个两千人的部落，几乎在眨眼间被屠戮一空，活着的只有区区两、三百人，而且还是个个带伤，只不过这些伤并不影响行动。
<br/>　　而反观大唐一方，这次行动也有不少禁军搭上了性命。
<br/>　　因为那些土著的武器竟然全都带着剧毒，只要被他们的武器伤到，基本上就没救了，剧毒会随着血液飞快的流便全身，便受伤的人失去意识，随后慢慢停止呼吸。
<br/>　　这些该死的土著他们没有炼铁的技术，所有的武器都是一些木质或骨制的，为了提高这些武器的杀伤力，他们想到了用毒。
<br/>　　当然，这并不是对唐军准备的，而是为了他们日常打猎准备的，唐军只是适逢其会而已。
<br/>　　“老子死了三十二个兄弟，你说，该怎么办？”就要先知为眼前看到的一切所震惊时，那高大的汉子已经把刀架到了他的脖子上，同时冷声问道。
<br/>　　先知本以为那刀会划过自己的喉咙，不过等了片刻发现那汉子并没有真的要杀他，这才长长出了口气，用颤抖的声音结结巴巴的说道：“尊敬的大唐将军，族人愚昧，并不知道天兵的厉害，千百年来他们根本就没有见到过外人……”。
<br/>　　“放屁，没见过外人，你的大唐官话是跟谁学的！”那汉子显然是觉得先知在骗他，手里的刀不由紧了紧。
<br/>　　“我，我是部落里的先知，我的官话是跟着我的父亲学的，而我的父亲则是跟他的父亲学的。”先知只觉得脖项处一阵刺痛，知道那是刀锋划破皮肤才有的感觉，连忙急声说道。
<br/>　　脖子上的刀松了松，先知的耳中传来那汉子的声音：“哼，先知，看来老子还真抓了一个大人物。”
<br/>　　“不敢，不敢！”听着那汉子语气中的嘲讽，那先知连忙摇头。
<br/>　　不过那汉子显然并不想听所谓先知的解释，想都没想便说道：“为什么昨天要攻击船队。”
<br/>　　“这都是误会，一切都是误会，都是因为族人没有见过外面的一切，所以见到船队才会自发的发起攻击，小人也是刚刚才收到消息，正想着带人去道歉，没想到将军就来了。”先知用并不怎么熟悉的汉话结结巴巴的说着，一边说还一边观察着那高大汉子的脸色。
<br/>　　先知的话里有前后矛盾的成份，首先他说自己的官话是跟他爹学的本身就有问题，至于其他的也还有更多。
<br/>　　不过高大的汉子似乎并不想在这个问题上过于纠结，只是冷冷的看着他，半晌才开口说道：“刚刚跟你在一起的想必也都不是普通人吧？把他们叫出来，跟老子走。”
<br/>　　“是，是，请将军稍等！”先知连声答应着，飞快的跑回了屋子里面，时间不大便带着其他人从里面走了出来，向高大汉子介绍道：“尊敬的将军，这位就是我们的族长，这几位……”
<br/>　　高大汉子撇撇嘴打断那先知的话：“老子才不管谁特么是族长，现在跟老子去见一个人，路上你们最好都老实一点，否则别怪老子翻脸无情！”
<br/>(本章完)

　　很快双方的第三局对决到来，这一场ka战队并没敢在将盲僧这个英雄放出来。

<br/>　　很快双方便是选人完毕，这一场两边都是选择了比较靠后期的阵容，不过就算是如此，最终叶帝他们还是取得了胜利。
<br/>　　到了后期，他们的团战能力再一次展现了出来，根本没有一丝差错，无论是adc的输出位置，还是坦克的保护都是非常的完美。
<br/>　　“让我们恭喜荣耀白队率先晋级到四强之中，虽然我们想到了结果，但是没想到比赛会结束的这么快，3:0，真的是太干脆利落了。”米粒激动的大声喊道。
<br/>　　“接下来就看ST和eP战队究竟谁能够获胜了，获胜的将会在半决赛和大魔王他们对决，不过我这心里总有一种预感，也许大魔王他们会和ST战队再一次相遇，两支战队自从世界赛开始后，已经是相爱相杀很多次了。”林雅楠跟着开口说道。
<br/>　　场下此刻无数的国内观众已经是纵情的欢呼了起来，看到白队展现出了这么强的统治力，他们每个人都有一种与有荣焉的感觉。
<br/>　　“66666666666666，太强势了，这三场比赛打的真心爽，没有任何的失误。”
<br/>　　“无敌，真的是太无敌了，夜雪的盲僧还是那么的厉害，真的是服气了。”

<br/>　　“接下来就看半决赛了，我估计肯定还是白队打ST，eP战队虽然强，但是他们也就是下路强而已，不过ST战队这边可是整体都很强啊，而且他们的下路也是世界有名的，emp和aco两人想要在coco和newlife身上取得优势，还真不太容易。“
<br/>　　“加油加油，现在的比赛真的是越来越精彩了，我现在已经是迫不及待的希望他们快点对决了。”
<br/>　　直播间里的众多观众此刻也都是激动的在那里议论着，显然对于他们来说荣耀白队这么快获胜，一个个也都是非常的开心。
<br/>　　场上的叶帝几人此刻也是在比赛室内开心的拥抱着、欢呼着，虽然比赛结果并没有出乎意料，但是这种胜利的喜悦还是让人非常的开心。
<br/>　　反观另一个比赛室内，此刻KA战队的气氛倒是变的有些低落了，这一场比赛输了，就代表着他们要告别世界赛了，他们努力一年为之奋斗的目标，就要到此结束了。
<br/>　　甚至这个时候beng已经是忍不住流下了眼泪，对于一个职业选手来说，世界冠军永远是他们追逐的目标，而这一次又一次失去冠军的争夺机会，对于beng来说无疑是一个巨大的打击。
<br/>　　他已经正在在这个赛场上很多年了，如今他也算的上是一个老将了，现在就是连他自己都不知道还能不能坚持到明年了，今年是他最后的希望，只是可惜在这个时代他遇到了这么多天才的人物。
<br/>　　过了一会后，叶帝等人来到了ka战队的比赛室中，这个时候beng一行人也是已经调整好了情绪，双方没有多说什么，只是紧紧的拥抱了一下。
<br/>　　大家都是职业选手，自然是能够体会到对方的那种感情，失败对于每一个选手来说，都是很难接受的。
<br/>　　接受过采访过后，叶帝等人便是向着基地返去，接下来的几天分别是其他几支战队的对决，而在这些对决过后，仅仅只是休息2天的时间，他们就将会迎来半决赛的对弈，可以说时间很紧，由不得他们有丝毫的松懈。
<br/>　　已经打到了这个时候，他们每天都需要充足的训练时间来保持自己的手感，如果一天不打游戏的话，很可能他们在接触游戏的时候就会感觉到一丝陌生。
<br/>　　回到基地自然是免不了又一番庆祝，不过大家都是以饮料来替代酒。
<br/>　　本身酒对于一个职业选手来说就是一个非常大的杀手，在现在这么紧张的时候，他们自然是更加的不敢喝了。
<br/>　　“明天的对决不出意外的话，应该还是ST战队获胜，毕竟他们的整体实力摆在那里呢，半决赛又是你们两支战队对决，到时候你们可要努力了，千万别进不了决赛。”蒋念开口说道。
<br/>　　“你们还是担心下自己吧，我们都已经晋级半决赛了，你们现在连比赛都还没打呢。”香蕉开口说道。
<br/>　　“我们的实力，你们还不清楚么，对战og战队根本一点压力都没有好么。”飞刀自信满满的说着。
<br/>　　“确实，你们下半区的实力确实是有点弱了，你们晋级半决赛问题应该不大，我现在比较关注的是雷霆战队和tm战队的对决，如果雷霆战队能够战胜tm，那对于咱们华夏赛区来说，就是一个很大的惊喜了。”叶帝点头应道。

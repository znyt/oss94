　　苏莫在苍穹神宫中待了足有十日，又为少量的人融合了战魂，最后才和众人依依道别。
<br/>　　尤其是夕儿，非常的不舍，抓住苏莫的手，几乎不愿意松开。
<br/>　　临走之前，苏莫做了详细的安排，夕儿、战乱等人离开之后，苍穹神宫由帝释、海明江和东海龙王三人共同执掌。
<br/>　　以后可以招收弟子，但只招收天才弟子，择优而选。
<br/>　　苍穹神宫不允许大规模的扩张，免得引起其它势力的注意，低调发展。
<br/>　　苏莫给苍穹神宫留下了海量的资源，他身上只留下了四十万下品灵晶，其它所有的物品，全部留给了苍穹神宫。
<br/>　　包括灵晶，灵石、宝器、武学等等的一切，庞大到恐怖的财富，他丝毫不吝啬的全部留给了苍穹神宫。
<br/>　　玉镯空间中的灵石矿脉，同样也留给了苍穹神宫，埋在了神宫府邸的大地之下，让他们自行开采。
<br/>　　一切安排妥当之后，苏莫才离开途山镇，而夕儿、司空炎等人也同一时间离开了途山镇，去追寻更广阔的天地。
<br/>　　她们会离开古周星，甚至可能会离开古灵星河，她们也不知道此行的目的地。
<br/>　　但有个共同的目标，那就是加入某一方超级大势力。
<br/>　　苏莫离开之后，就向神武学府的方向飞去，他一直催动着‘隐身战魂’，整个人隐藏不出，非常的谨慎。
<br/>　　此行去神武学府，他必须慎之又慎，苗头稍有不对立刻遁走。
<br/>　　但是，他能肯定，现在的古周星上一定有不少巫族强者，他想逃走的话很难，必须要从星空逃离。
<br/>　　就算是从星空逃离，估计都很危险，而且他的隐身战魂，好像不一定能瞒的过武尊境强者。
<br/>　　其实，苏莫目前最好的办法，是隐藏在古周星上，隐匿不错，才是最安全的办法。
<br/>　　但是，随着他在古周星的消息传回巫族，巫族必定有大量的强者来到古周星，甚至是大祭司亲临。
<br/>　　古周星虽然大，他也很难逃出大祭司的搜寻。
<br/>　　所以，他才会返回神武学府，短时间之内，神武学府还是比较安全的地方。
<br/>　　他现在被排进了真圣榜第一，天赋足够厉害，想必神武学府不会不管他。
<br/>　　苏莫一路疾行，没有丝毫的停顿，数日之后便临近了神武学府，那硕大的陨落星辰，出现在了他的视线中。
<br/>　　他立刻减缓了速度，仔细的查看四方，不过并没有任何发现，没有发现巫人王，也没有发现任何巫族之人。
<br/>　　或许对方隐藏了起来，不可能在此光明正大的堵截他。
<br/>　　苏莫松了口气，他依旧没有现身，隐藏着身形，神不知鬼不觉的进入了神武学府中。
<br/>　　进入学府之后，他立刻便回到了内府火元峰，回到了自己的洞府之中。
<br/>　　一切都是平静无比，没有任何异常，苏莫身形显现了出来，去旁边两个洞府转了一圈，发现聂乾和韩盈都不在，洞府中都落了一层厚厚的灰尘。
<br/>　　“奇怪！”
<br/>　　苏莫微微皱眉，他临走之时，告诫两人不要轻易离开火元峰，免得遭到太史世家派系的报复。
<br/>　　没想到两人居然还是离开了火元峰，不知道去了哪里？
<br/>　　叹了口气，苏莫盘坐在自己的洞府内，低头沉思自己接下来的行动，完全没有了修炼的心情。
<br/>　　巫族是庞然大物，就算仅是天冥星支脉，也是强大无比，根本不是他目前能抗衡的存在。
<br/>　　他需要寻找靠山，寻找能庇护他的靠山，而火元圣师就是当前最好的人选。
<br/>　　只不过，先不说对方能不能抗衡巫族，对方愿不愿意为了他而抗衡巫族还是两说呢！
<br/>　　唰！
<br/>　　就在苏莫思索之间，一道白色倩影落在了他的洞府门前，是木璃导师。
<br/>　　“导师让你去见他！”木璃目光望向苏莫，淡淡的说道。
<br/>　　“嗯！”苏莫点了点头，立刻长身而起，他正想去找火元圣师呢，没想到对方居然就要见他了！
<br/>　　不过，他感觉今天木璃导师看他的眼神怪怪的，仿佛看怪物一般。
<br/>　　“木璃导师，你经常待在火元峰吗？”苏莫笑着问道，他想和对方聊聊，探探火元圣师找他得目的。
<br/>　　“走吧！”
<br/>　　木璃导师却是根本没有多聊的想法，身形一闪，便破空离去。
<br/>　　苏莫顿时无语，随即摇了摇头，立刻离开了洞府，飞到了火元峰的上方，火元圣师居住的宫殿门前。
<br/>　　宫殿大门敞开，木璃导师并不在此，只有火元圣师一人，闭目盘坐在宫殿中央的蒲团上，仿佛是一位迟暮的老人。
<br/>　　苏莫深吸口气，大步的走进了宫殿中，来到了对方的身前。
<br/>　　“见过前辈！”苏莫抱拳向对方一拜。
<br/>　　火元圣师闻言，缓缓的睁开了眼眸，深邃苍桑的目光落在了苏莫的身上。
<br/>　　“苏莫啊！听说最近外面出了个真圣榜，你名列榜首？”火元圣师轻声问道，面色平静，仿佛是一名慈祥的老爷爷。
<br/>　　“呃……！”
<br/>　　苏莫闻言一怔，随即尴尬的点了点头：“这个……的确是榜首！”
<br/>　　他心中了然，现在大荣城的拍卖会早就结束了，他是真圣榜第一的消息，估计已经在古周星无人不知无人不晓了。
<br/>　　而且要不了多久，便能在整个古灵星河传开。
<br/>　　“听说你和巫族之人有仇，一名巫族天才要杀你？”火元圣师又问道。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，略一沉吟，又道：“我和巫族天冥星一脉有大仇，天冥星一脉的巫族人都想要杀我！”
<br/>　　他没有隐瞒此事，如实告诉了对方，想看看对方愿不愿意给予他庇护。
<br/>　　“哦？什么大仇？”火元圣师苍老的眉头微微皱起，沉声问道。
<br/>　　“我杀了巫族近千名年轻子弟，还灭了几个天冥星上的巫族小镇，杀了一百多万巫族族人！”苏莫语出惊人。
<br/>　　火元圣师闻言沉默不语，眸中一片深邃，让苏莫看不出对方的半点想法。
<br/>　　“前辈，此仇可有办法化解？”苏莫试探性的问道，当然，他并不是真的要和巫族化解仇恨，而是暂时度过这次危机而已。
<br/>　　“你认为此种仇恨，还有办法化解？”火元圣师平静的反问道。
<br/>　　“好像不太可能！”苏莫摇了摇头，这是血海深仇，他也知道很难化解。

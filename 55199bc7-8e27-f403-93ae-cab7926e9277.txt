　　“不过…”萧逸皱了皱眉。
<br/>　　“既然同为圣皇境，却特地要划分出武道大能和绝世强者，划分得如此清晰。”
<br/>　　“原因，只有一个。”
<br/>　　萧逸皱眉看着云渊长老二人，“这之间的实力差距，异常庞大。”
<br/>　　“聪明。”云渊长老点了点头。
<br/>　　讳执事笑笑，“我早便说过了，与聪明人说话，确实是比较省心而且是比较愉悦的事。”
<br/>　　“可我还是有些不明白。”萧逸直言问道。
<br/>　　“不明白什么？”云渊长老问道。
<br/>　　萧逸理了理脑海思绪，道，“掌控9100条完整武道，便是圣皇境巅峰。”
<br/>　　“掌控9501道，便是武道大能。”
<br/>　　“掌控9990道，乃是绝世强者。”
<br/>　　“这三者之间，不过相差了数百条武道，怎可能会有庞大的实力差距？”
<br/>　　会特别在圣皇境巅峰后，划分出武道大能与绝世强者两个称呼。
<br/>　　证明着三者间的实力差距，异常庞大，有着极其明显的区别。
<br/>　　可是，要知道，无极境之后的武者，其实力来源，来源于天地武道；其差距，亦在于掌控天地武道的多少。
<br/>　　圣王境时，每差距一重，便相差100条完整武道。
<br/>　　便是这100条完整武道，让得有了圣王境之中，相差一重，等同秒杀，这样的说法。
<br/>　　而到了圣皇境，掌控千条武道，踏入圣皇境；随后，每掌控千条，修为升一重。
<br/>　　也就是说，圣皇境中，每差距一重修为，掌控的完整武道便相差1000条，这样的差距无疑更大。
<br/>　　与圣皇境一重，掌控千条武道；到了圣皇境二重，便是掌控两千条完整武道了；圣皇境三重，三千条完整武道…
<br/>　　而到了圣皇境九重后，按照划分，便是掌控九千条完整武道。
<br/>　　可在这之后，圣皇境巅峰、武道大能、绝世强者，三者间，不过是相差几百条武道罢了。
<br/>　　云渊长老闻言，明白萧逸的意思，笑笑，问道，“你应该与圣皇境巅峰，以及武道大能层次的武者，都交过手吧。”
<br/>　　“也该知道他们间的差距有多巨大。”
<br/>　　“当然。”萧逸点了点头。
<br/>　　圣皇境巅峰与武道大能层次的武者，他都交手过。
<br/>　　如天冥府的那位大长老，掌控九千三百条完整武道。
<br/>　　虽未到达武道大能层次，但显然是圣皇境巅峰中的佼佼者。
<br/>　　而武道大能层次，就如北隐宗大长老，掌控九千五百条以上的完整武道。
<br/>　　萧逸与这二者交手过，甚至可以确定，若是这二人交手的话，北隐宗大长老，几乎可以秒杀天冥府大长老。
<br/>　　这就是武道大能与圣皇境巅峰的差距。
<br/>　　但，二者真正的武道差距，不过是两百条完整武道左右罢了。‘
<br/>　　区区两百条完整武道，在圣皇境这个动辄数千条完整武道的层次中，根本算不得什么。
<br/>　　但另一方面，萧逸确信这些划分绝不会有问题。
<br/>　　炎龙大陆的修炼体系，在上古时期便已发展到巅峰，几近完美。
<br/>　　更别说千万年后的今日了。
<br/>　　萧逸皱着眉。
<br/>　　讳执事笑笑，道，“我原以为，你现今这般实力了，不应该会有这些问题或疑惑。”
<br/>　　“哦？”萧逸疑惑地看着讳执事。
<br/>　　云渊长老笑道，“你竟然拿武道的多少来衡量实力差距？小子，你可不是那些傻瓜菜鸟。”
<br/>　　“武道的单纯叠加和融合间的区别，你不是早就知道了吗？”
<br/>　　“叠加和融合的区别？”萧逸闻言，霎时眼前一亮。
<br/>　　“想起来了吗？”云渊长老笑道，“对比一条条完整武道杂乱地储存于小世界，与一条条完整武道契合地融合，两者的实力差距，自然是极大的。”
<br/>　　萧逸面露恍然之色，“对，武者在圣境之后，早便在走一条只属于自己的武道了。”
<br/>　　讳执事接话道，“武道大能，为何是武道大能？便是因为在武道这一方面，他们已经是大能。”
<br/>　　“九千五百条以上的完整武道，数量何其多，他们所掌控的武道知识，又何其丰厚？”
<br/>　　“这等人物，你可以想象，即便九千五百条完整武道，他们无法悉数融合；但他们本身所走的武道，该完善到何等地步了。”
<br/>　　“至于绝世强者，那则更加恐怖了。”
<br/>　　“9990条完整武道，你可想而知，他们已经在自己的武道上，走出极远的距离了。”
<br/>　　“故他们能有绝世称号，放眼天下，几无敌手。”
<br/>　　“我懂了。”萧逸恍然，点了点头。
<br/>　　看似只相距几百条完整武道。
<br/>　　但实则，他们三者间所完善的自身武道，根本不在一个层次。
<br/>　　到达那个地步，属于自己完善的武道，已层次极高；每差一条，每融合一条，可能都是天差地别的实力差距。
<br/>　　拿他萧逸自己来说。
<br/>　　他是圣王境九重修为，掌控的完整武道，达到九百条以上。
<br/>　　可这其中，真正属于剑道的，不过是一半左右。
<br/>　　剩下的一半，是属于火焰一类的。
<br/>　　而那一半剑道武道，便是他自己完善着的剑道，亦是他的剑道实力。
<br/>　　可想而知，绝世强者那9990条完整武道中，存在了多少条剑道武者，又融合了多少。
<br/>　　拿剑修来说，绝世强者层次的剑修，其自身完善的剑道已经相当惊人；剑道实力之强，自是不言而喻。
<br/>　　其它一道的武者，同样如此。
<br/>　　“十八府的府主，便是绝世强者吧。”萧逸问道。
<br/>　　“不错。”云渊长老点点头，“十八府府主，现在剑域之内的气宗器宗两宗宗主，还有林家主等人，都是绝世强者。”
<br/>　　“都是？”萧逸皱了皱眉，“可我感觉，林家主的气息，远超过两宗宗主。”
<br/>　　“那是自然。”云渊长老回答道，“林家主，可是掌控了9998条完整武道的绝世剑修。”
<br/>　　“而两宗宗主，充其量在9995条左右徘徊。”
<br/>　　“绝世强者之后，每差一条完整武道，实力之差距，远远超过绝世强者与武道大能的差距。”
<br/>　　“那是更加庞大的差距。”
<br/>　　“仅差一条，便差距如此大？”萧逸愈加皱眉，“我没猜错的话，达到万条武道后，武者将会踏入新的境界。”
<br/>　　“对，但那不是你现在该想的事。”讳执事陡然脸色一板。
<br/>　　萧逸未在意，继续问道，“绝世强者后，距离下一个境界，不过剩10条完整武道罢了。”
<br/>　　“为何不尽快突破？参悟区区10条完整武道，应当不是难事。”
<br/>　　拿萧逸自己来说，一月时间就能参悟过百条完整武道。
<br/>　　两人莫名一笑，“对，参悟是不难，区区10条完整武道罢了。”
<br/>　　“但，你知道自己差了哪10条武道吗？”
<br/>　　“什么？”萧逸闻言，霎时脸色大变。

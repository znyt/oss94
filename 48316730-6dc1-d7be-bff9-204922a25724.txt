　　锵…锵…锵…

<br/>　　寒霜剑与裂天剑，也不断在做着碰撞。

<br/>　　“你怎么可能有如此战力。”
<br/>　　“我不信。”
<br/>　　白墨寒满脸不可置信之色，渐渐地，露出了些许疯狂。
<br/>　　他连武魂虚影都凝聚出来了，已然实力全开。
<br/>　　却仍旧被萧逸压着来打。
<br/>　　这对拥有着不败神话的他来说，是不能接受的事。
<br/>　　但，事实摆在眼前。
<br/>　　萧逸的每一剑劈出，都让他如遭重击，体内一阵气血翻滚。
<br/>　　另一边。
<br/>　　寒冰屏障内，七位巅峰强者，脸上写满了急切。
<br/>　　“怎么回事儿，萧逸怎么可能忽然变强了？”宗主急切地说着。
<br/>　　同时，手中之剑，打向寒冰屏障的攻击更为急速。
<br/>　　他只想快些破去寒冰屏障，出去对付萧逸。
<br/>　　一旁的段云长老看着远处的战斗，脸色有些凝重。
<br/>　　“不是萧逸变强了，是寒霜剑变强了。”
<br/>　　“什么？”宗主皱起了眉头。
<br/>　　段云沉声道，“寒霜剑，已经无限接近于上品元器。”
<br/>　　“萧逸在将体内元力，倒灌入寒霜剑中。”
<br/>　　“以此暂时增幅寒霜剑的力量。”
<br/>　　“倒灌入寒霜剑？”宗主脸色一惊，道，“这不可能。”
<br/>　　“中品元器，内头蕴含的力量，何其庞大。”
<br/>　　“若想让中品元器增幅到这个层次，所需耗费的元力，绝对是个天文数字。”
<br/>　　一旁的大长老沉声道，“不，段云长老说得没错。”
<br/>　　“我曾经见过萧逸凝聚的真气漩涡，足足七百丈。”
<br/>　　“我猜测，萧逸当时应该还是藏拙了。”
<br/>　　“他的真正气泉，应该在七百丈以上，八百丈左右。”
<br/>　　“八百丈气泉？”另外六位巅峰强者，瞪大了眼睛。
<br/>　　“白长老也不过五百丈气泉吧。”
<br/>　　“这岂不是比白长老的气泉大了一半都不止？”
<br/>　　段云点了点头，道，“如此庞大的气泉，难怪能倒灌力量进寒霜剑。”
<br/>　　“只是，以萧逸现今修为，坚持不了多久的。”
<br/>　　“他现在，体内的元力必定在以一种恐怖的速度消耗着。”
<br/>　　“顶多能维持这样的战斗十分钟左右。”
<br/>　　“不过，为免有意外，我们还是得尽快脱困。”
<br/>　　此时，原本厚实的寒冰屏障，已被生生刮去了一层又一层。
<br/>　　薄了大半以上。
<br/>　　要不了多久，七人就能脱困而出。
<br/>　　另一边，远处。
<br/>　　萧逸仍在一剑剑轰着白墨寒。
<br/>　　白墨寒嘴中，不断溢着鲜血，伤势也在渐渐变重。
<br/>　　萧逸见状，狞笑一声，“若没了裂天剑，我杀你如杀鸡。”
<br/>　　萧逸不仅增幅了寒霜剑的力量。
<br/>　　同时，也早就使用了修罗变，增幅了肉体力量。
<br/>　　20滴修罗之力，已一瞬间消耗。
<br/>　　当然，肉体力量现今只有地元九重。
<br/>　　对这个层次的战斗帮助不大。
<br/>　　但，多少是有一定的增幅。
<br/>　　时间，渐渐过去，数分钟后。
<br/>　　白墨寒的伤势，愈加严重。
<br/>　　他的挥剑速度，反应速度，也在渐渐变慢，变弱。
<br/>　　一分钟后。
<br/>　　白墨寒再次猛地吐出一口腥血。
<br/>　　锵的一声。
<br/>　　手中裂天剑，脱身而出。
<br/>　　萧逸狞笑一声，“你败了，也该死了。”
<br/>　　“你…你…你不能杀我。”白墨寒脸上，终于露出了恐慌。
<br/>　　萧逸没有再废话，一剑刺出，直取白墨寒咽喉。
<br/>　　但，恰在此时。
<br/>　　远处，一道恐怖的火焰，忽然打来。
<br/>　　火焰的速度，快到极致。
<br/>　　同时，另一个方向，也是一道恐怖火焰打来。
<br/>　　嘭的一声。
<br/>　　第一道火焰，轰开了萧逸的剑。
<br/>　　第二道火焰，直直打在萧逸身上，将萧逸轰飞。
<br/>　　萧逸并没有受太重的伤，只是最终微微溢出一丝鲜血。
<br/>　　刚才的两道火焰，绝对达到了巅峰战力的层次。
<br/>　　又忽然打出，猝不及防下，才让萧逸吃了些亏。
<br/>　　此时，萧逸的目光，直视远处，目光有些凝重。
<br/>　　那里，数支强悍的武者队伍，正飞速而来。
<br/>　　下一秒，已来到白墨寒身边，并将白墨寒牢牢保护起来。
<br/>　　“天炎卫！”萧逸咬了咬牙。
<br/>　　天炎卫，国主麾下，最强武者力量。
<br/>　　人数极少，只有寥寥数十人。
<br/>　　但，里头成员，清一色天元三重以上。
<br/>　　领头的四位队长，更皆是天元九重，位列巅峰强者层次。
<br/>　　这是国主手中的最强利剑。
<br/>　　“天炎卫，你们来得正好。”白墨寒脸上的恐慌，早已褪去。
<br/>　　取而代之的，是毒辣和狰狞。
<br/>　　“给我宰了那小子。”
<br/>　　这支队伍中，领头的四人，点了点头。
<br/>　　“有我等在，白长老不必担心。”
<br/>　　说罢，领头的四人，大手一挥，“上，拿下萧逸这逆贼。”
<br/>　　数十天炎卫，瞬间出手。
<br/>　　萧逸皱了皱眉头。
<br/>　　领头的这四人，毫无疑问就是天炎卫的四位队长。
<br/>　　修为达天元九重，乃是巅峰强者。
<br/>　　不过，大概也只在木青云那个层次。
<br/>　　他还不惧。
<br/>　　“拦我者，死。”萧逸冷喝一声，仗剑而上。
<br/>　　一群天元三重罢了，外加四个实力和木青云差不多的巅峰强者罢了，他还不放在眼中。
<br/>　　但，下一秒。
<br/>　　这四支队伍，却玄妙地跃动了起来。
<br/>　　同一时间，领头的四个队长，忽然气势飙升，实力大增。

<br/>　　四位队长，正源源不断地吸收着手下数十队员的元力，为己所用。
<br/>　　这绝不是一加一等于二那么简单。
<br/>　　阵法存在的意义，乃是让一加一，等于三，乃至四。

<br/>　　嘭的一声爆响。

<br/>　　“最后更强行困住我们，再与白长老鏖战。”
<br/>　　“如此连番大战，消耗过大，他早已无法保持最强战力。”
<br/>　　“落败，只是迟早的事。”

<br/>　　宗主、血谷主和血成河，以及木青云，联手而上。

<br/>　　四人联手，再次直取萧逸而去。
<br/>　　而四位天炎卫队长，则停下了攻击，并非要放过萧逸。

<br/>　　“此次，有劳天炎卫相助了。”

<br/>　　一道恐怖剑气，仿佛划破了空间的限界，直直打来。

<br/>　　那原本淹没萧逸的火海，瞬间在剑气下熄灭。

<br/>　　“终于肯出现了吗？”
<br/>　　另一边，段云等人，脸色大变，“大自在剑道？”

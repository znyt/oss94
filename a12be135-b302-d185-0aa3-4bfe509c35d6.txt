<br/>				<br/>　　谢玉堂看着他走回来，不由动了动嘴唇，歉意的的道：“对不起，都是我们不好，要不是我们，你也不用去帮他们烤肉。”<br/>　　<br/>　　他垂低下头，这一路上他隐隐猜到凤九的实力在他之上，只是不知到底有多厉害，但他知道，若是平时凤九根本不会答应帮他们烤肉的，都是因为他们才低的头。<br/>　　<br/>　　听到这话，凤九微怔了一下，继而笑道：“这没什么，反正闲着也是闲着。”她笑了笑，来到火堆边坐下，对他道：“你身上的伤得换下药，来，我帮你看看。”<br/>　　<br/>　　“哦。”谢玉堂没有再多说什么，只是听话的将外衣脱了下来，露出了里面绑着的那些伤口。<br/>　　<br/>　　“怎么办？伤药只剩下一瓶了。”谢诗思小脸一皱，有些担心药用完了伤口却还没好。<br/>　　<br/>　　“我这里有，你那瓶先收起来吧！”凤九说着，解开谢玉堂伤口处的带子后，查看了下伤口，再从空间里取出一瓶帮他敷上。<br/>　　<br/>　　当看到那处较为严重的伤口因汗水和天气的炎热而溃烂时，她眉心微皱：“伤口有些溃烂，得重新处理下。”<br/>　　<br/>　　“没事，随便包些药上去就行了。”他也知道这里没有条件，只能简单包扎了。<br/>　　<br/>　　凤九瞥了他一眼，继而取出小刀来，道：“我帮你将烂掉的肉剔除。”说话间，她将小刀在火焰上烧了一下。<br/>　　<br/>　　而听到他的话的谢家兄妹却是脸色微变：“剔除？你、你怎么行？”他又不懂医，拿把刀在剔他的肉？他怎么敢？<br/>　　<br/>　　看着凤九手中那把尖锐而锋利的小刀，谢玉堂的脸色白了白，额头的汗水也渗了出来：“不、不用了，我看还是回去再处理吧！应该没什么……”话没说完，就被打断了。<br/>　　<br/>　　“等到回去你这条胳膊也就废了。”凤九淡淡的说着，瞥了他一眼：“莫不是，你怕疼？不敢让我剔？”<br/>　　<br/>　　谢玉堂咽了咽口水：“这根本不是怕不怕疼的问题好吧？你都不懂……”<br/>　　<br/>　　他的声音一顿，整个人屏起了呼吸僵硬着身体，因为那拿着小刀的人根本不待他将话说完，就已经拿着小刀在他那伤口上划着，看着他心头一紧，整个人都紧绷着，一句话也说不出来。<br/>　　<br/>　　“放松点，不用紧张，熟能生巧，我多拿你练练也就是了。”凤九闲闲的说着，一边剔除着他伤口上溃烂的肉。<br/>　　<br/>　　谢玉堂瞪着一双眼睛盯着他。什么叫熟能生巧多拿他练练？这人怎么这般没良心呢？身体紧绷着，一双眼睛却不敢往伤口上移，只盯着眼前的人看着。<br/>　　<br/>　　凤九在剔除好他伤口上溃烂的肉后，又清理了一下，这才重新抹上她的药后再将伤口包扎了起来。<br/>　　<br/>　　“行了。”她到一旁清洗了下手，这才走回来，道：“我就先去睡会了，有事再喊我。”声音一落，便跃到树上去休息。<br/>　　<br/>　　谢玉堂正看着重新包扎好的伤口，感觉伤口上了药后好像没那么疼了，反而有一种凉凉的感觉，不由的朝树上的凤九看了一眼。<br/>

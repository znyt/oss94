<br/>        干了这件事，估摸着应该是众多大势力都咽不下这口气，目测要追杀自己一万年。

<br/>        是时候先把后路规划一番了。

<br/>        正在这时，小胖子在陈明的耳边轻声道：“师父，你到时候跑路了，我怎么办啊？”

<br/>        “对了，你这一行反正跋涉数颗星辰，为师交给你一个历练任务，你现在也凝聚了十文道的道种，对于算卦有些了解，你这一路过来，想办法骗取一些为富不仁的凡人十万两黄金，分给穷苦百姓，明白了吗？”
<br/>    
<br/>        小胖子这才明白了过来，原来燕山，是一个这样一身正气的宗门！“师父，弟子记住了！”
<br/>    
<br/>        打开一个牢门，小胖子也是一脸委屈的住了进去。

<br/>        陈明走进进去，挥手，这里的景象在外界变成一片黑色，在这里却是变成了灯火通明，陈明看向周围，笑道：“我知道你们有方式可以通知你们背后的无面者，不过，通知他们的时候，记得告诉他们一句话，无面者信条：作为一个骄傲的无面者，从来不屑于借刀杀人！”
<br/>    
<br/>        陈明的手中，出现无面者的面具，周围的无冕者议会的修行者都是吃了一惊，这个星龙太子，居然是一个无面者！
<br/>    
<br/>        怪不得戴着一张滑稽面具！
<br/>    
<br/>        陈明继续道：“内斗是内斗，对外是对外，对外来说，
<br/>    
<br/>        这一次，是我们无冕者议会胜了，对内来说，你们这一次内斗败了，我已经公开了自己的身份，妖族星龙太子，欢迎诸位背后的无面者前来杀我！”
<br/>    
<br/>        众多无冕者议会的修行者都是知道，陈明这就是在挑衅！
<br/>    
<br/>        大胆的公开自己的身份，而陈明想要的，就是他们背后的无面者不会透露这里发生的事。

<br/>        每一个无面者，都是高傲之极的存在，自然不会借刀杀人，内斗，借刀杀人是耻辱。

<br/>        无面者之中流传着一句话，那就是一个无面者，只会死在另外一个无面者的手中。

<br/>        就是这么一个奇怪的组织，却是靠着阴谋和诡谲，对抗着其他仙国，实力之雄厚，无敌于星空。

<br/>

<br/>        既然诈不出来，那就算了吧。

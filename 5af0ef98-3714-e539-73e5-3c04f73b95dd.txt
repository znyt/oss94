<br/>　　陆勇当机立断立即组织人退到特等区。特等区那里还有个高墙还可以踞守。
<br/>　　也就在一阵狂轰而去，这个钢门终于不堪重负，直接给轰出个窟窿。
<br/>　　一大群被困在基地外面的丧尸刹那间就好像是那开了闸的水库一样，全部朝其中涌了进去。
<br/>　　在平民区内，还有不少宪兵队员在战斗，他们就听到一阵震天动地的爆响，然后就是数不尽的怪物和丧尸犹如潮水一样的涌入。
<br/>　　“不好！门破了！！”
<br/>　　几个宪兵队面色煞白，他们难以想象，这个花费了无数钢材才制作出来的巨大钢门居然破了，这个钢门，可以抵抗十二级大地震，可以抵抗大爆炸，是不可能被攻破的存在，可是如今，他却破了，这就代表曙光基地已经失守了。
<br/>　　他们这群宪兵已经失去了任何的情报，他们的队长都死了，所以他们根本不知道陆勇已经放弃外墙退守特等区了，他们依然还在死战。
<br/>　　不过很快他们就被丧尸群给吞没了，在大门口这里有几百个丧尸被活生生的压成了番茄酱，大门就这么大，几万个丧尸要一起冲进来其中还有怪物，一下子几百个丧尸全部被挤爆了。

<br/>　　林峰的车队已经抵达了曙光基地，唇亡齿寒，所以这一战，也和林峰息息相关，不得马虎。
<br/>　　林峰跳下车，几百个队员也都跳下车，在基地外，一地的尸体，一地的坑坑洼洼，那都是被炮弹轰出来的。
<br/>　　“这么惨烈？”
<br/>　　林峰也是震惊了，他是第一次看到这种人间地狱的样子。
<br/>　　现场还有最起码几千只丧尸在那里啃着尸体，这些丧尸看到林峰这群活人，然后全都是长开血盆大嘴朝林峰撕咬而来，有的丧尸嘴巴里还有一些肉块没有吞下去，血糊糊的，甚至还有一些肠子挂在嘴边，就好像是过年吃的腊肠一样，无比的狰狞可怖。
<br/>　　不过林峰已经不再是之前的那个被人堵在厕所里打的那个高中生了。
<br/>　　他早就已经习惯了这些样子，虽然有些恶心，但手上动作可不慢，直接数刀斩杀而去，他没有去对付那些普通的丧尸，而是把所有的注意力全部放在了几个五阶以上的丧尸，这些丧尸爆发力强还有一定的智慧，能跑能跳，是个不小的威胁，其余五阶以下的丧尸，凭借队员们精良的武器弹药足够造成客观的威胁。
<br/>　　哒哒哒。
<br/>　　顿时枪声再度响起，杀戮盛宴也再次开席。

<br/>　　那个黑袍人也口鼻之中鲜血狂喷。
<br/>　　因为异能消耗了他太多的精神力。
<br/>　　若不是有这些鲜血在刺激他的神经，他已经死了。

<br/>　　他挣扎的站起身，擦了擦嘴角的血迹，抱起了死去的同僚，回去复命。

<br/>　　“怎么可能，不是说这基地的门非常厚的吗，就是那几百吨炸药都炸不开，怎么会被攻破。”

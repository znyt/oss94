<br/>    凌仙张开双臂，如同在拥抱整个天地，他长吸一口气，然后道：“试试看吧，这世间的仙人，到底能耐我何！”
<br/>    
<br/>        凌仙踏步，朝着远方飞去，他的体内，道宫开始流转，无尽的灵气，朝着他飞去，如同形成一个巨大的漩涡！
<br/>    
<br/>        两个小店之中，万八千忽然感受到了周围的灵气忽然一阵暴动，他记得这种感觉，他笑了笑，然后道：“二师兄，来了。”
<br/>    
<br/>        陈灵儿看了一眼，现在是白天，她就算赶过去，也没有什么用处，这灵气的暴动只有二师兄才有这个本事。二师兄是来了神山吗？“有我的丹药，他应该能够撑到晚上了。而且，三师兄可不会这么容易让二师兄死掉。”
<br/>    
<br/>        万八千摇了摇手中的铃铛，亡命很快走了进来，亡命拱手道：“主公，有何吩咐？”
<br/>    
<br/>        万八千道：“带着你的人，去找一个被所有人追杀的人，然后，救下他。”
<br/>    
<br/>        亡命有些疑惑，问道：“这世上有着能够被所有地仙追杀的人？”
<br/>    
<br/>        万八千道：“那是自然，我的二师兄，他有着这个资格！”
<br/>    
<br/>        亡命瞬间明白了过来，原来是这位的二师兄！
<br/>    
<br/>        他的四师姐是鬼医，那么二师兄，肯定更加厉害。
<br/>    
<br/>        听万八千这么一说，亡命反而觉得没有任何奇怪的地方了。
<br/>    
<br/>        万八千继续道：“记住了，不能暴露我们的关系，而且，非死不救。”
<br/>    
<br/>        亡命拱手道：“属下，明白了！”
<br/>    
<br/>        万八千挥手道：“去吧！”
<br/>    
<br/>        亡命从院子里走了出来，来到小店的背后一座客栈之中，淡淡道：“走了，有任务了，在我的手下，必定要完成主公的一切任务！”
<br/>    
<br/>        黑压压的地仙从客栈之中涌出，惊起一地的风沙。
<br/>    
<br/>        万八千看向远方，“二师兄啊，就是年轻气盛。你叫我一声师弟，我叫你一声师兄，身为同门，自当同生共死！”
<br/>    
<br/>        凌仙飞空而过，不过片刻便能看得到远方有着一个地仙朝着他飞来，那地仙道：“这种气
<br/>    
<br/>        息，你修行了不该修行的功法！”
<br/>    
<br/>        凌仙看向这个地仙，“嗯，月宫的人。”
<br/>    
<br/>        月宫地仙道：“既然你修行了不该修行的东西，就应该躲藏在无人知道的地方，你此行，唯有一死！”
<br/>    
<br/>        凌仙笑道：“只怕是你要死在我的前面了。”
<br/>    
<br/>        月宫地仙大笑道：“大言不惭，你不过超脱，也敢在地仙面前妄自称尊？”
<br/>    
<br/>        凌仙抖了抖肩膀，“看来，你并不知道我修行的功法和术法为什么被称作是禁忌。”
<br/>    
<br/>        通天桥前，月宫的银色马车逐渐飞了过来，里面的，是月宫太子，他们不得不来，周围的灵气都在暴动，这种感觉，只能是有人修行了禁忌的功法！
<br/>    
<br/>        那种功法和术法，是用来杀仙的！
<br/>    
<br/>        他们修仙，而对方杀仙，这便是纷争。
<br/>    
<br/>        月宫太子从马车之中走出，在前面，金镶玉，星龙仙子，九太子，即便是虚空太子，都是来到了此处。
<br/>    
<br/>        不过虚空太子依旧是保持着神秘，他并没有离开马车。
<br/>    
<br/>        月宫太子道：“有人修行了禁忌。”
<br/>    
<br/>        金镶玉道：“这个时候，突然有人修行了禁忌功法，而且还肆无忌惮地施展了出来，必定有诈！”
<br/>    
<br/>        九太子却是看向在通天桥前，有着一个戴着斗笠，怀中抱刀的家伙正在睡觉，“这家伙哪儿来的？”
<br/>    
<br/>        烟生笑道：“都什么时候了，九太子还在琢磨这些。对方修行的术法，乃是搏仙术，这搏仙术需要仙人的鲜血才能修行到巅峰，此等修行界的恶徒，绝对不能放过！”
<br/>    
<br/>        从虚空太子的马车之中，传出虚无缥缈的声音，“这里是无字劫中，并且在这个时候，突然有修行了禁忌功法的人施展了出来，有问题。他难道就不怕死吗？不，这世间的一切都怕死。那么，他到底是为了什么？”
<br/>    
<br/>        金镶玉道：“若是我们全部去追杀他，那么就会让渡劫者有了可乘之机！这是渡劫者的手段，他们想要我们离开此地，去追杀这个修行了禁忌功法的人！”
<br/>    
<br/>        月宫太子道：“但是，这件事，我们不能不管！”
<br/>    
<br/>        众人开始争论了起来，谁也不想因为追杀这个修行了禁忌功法的人，而放过了渡劫者。
<br/>    
<br/>        在通天桥前打瞌睡的抱刀人忽然醒了，他起身，走到几人之前，淡淡道：“杀他而已，何须你们，我去便够了。”
<br/>    
<br/>        九太子看向这个抱刀人，问道：“你是谁，你有这个实力？”
<br/>    
<br/>        抱刀人呵呵一笑道：“若不是这一次，乃是无字劫，我们无冕者议会可不会出手。你们几个是法修，修行《道经》的修行者对你们克制更大。”
<br/>    
<br/>        “我是苦修，虽说也有克制，但是并没有你们法修克制的那么大。”
<br/>    
<br/>        抱刀人离开了此地，直到这时，虚空太子才传出了声音，“这个家伙，应该就是无冕者议会之中的刀枪剑戟四道之中的刀道执刀人首席了。他是所有执刀人之中，同阶刀道第一人！而且，根据可靠的消息，他应该就是烈刀仙王的转世，一位真正的轮回仙。”
<br/>    
<br/>        无面者议会有着四大首席，便是这刀枪剑戟四大首席，而这四位的天赋与实力，并不下于他们。
<br/>    
<br/>        而且，执刀人的战斗，更为血腥，他们是真正从厮杀之中起来的，他们从来不比试，他们只杀人，刀道第一人，说明同阶之中，所有挑战他的人，都死了。
<br/>    
<br/>        而他，更是一位轮回仙，无冕者议会全是轮回仙，这是不可能的，但是四大仙国知道的是，无冕者议会，真的拥有很多的轮回仙，这也是为什么无冕者议会能够力抗四大仙国的原因。
<br/>    
<br/>        凌仙看向身前的月宫地仙，叹道：“到底是哪个王八蛋，说的我修行搏仙术需要仙人的鲜血，简直就是鬼扯！”
<br/>    
<br/>        月宫地仙却是没有管那么多，“哼，一个超脱而已，也敢出来送死，我这就送你上路！”
<br/>    
<br/>        凌仙道：“看来，你是真的不知道我修行的功法，为啥是禁忌功法啊，让你见识见识吧。”
<br/>    
<br/>        月宫地仙朝着凌仙飞来，凌仙呵呵一笑，开启，天地之间为之一振，灵气瞬间狂暴，形成一个巨大的漩涡围绕着凌仙。

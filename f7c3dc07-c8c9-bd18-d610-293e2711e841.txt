　　嗖嗖嗖！！
<br/>　　犀利的破空声响彻天际，三道人影急速而来，浩大磅礴的气息铺天盖地，毫不掩饰。
<br/>　　这是三名青年，个个修为强大，气势不凡。
<br/>　　而其中一人不是别人，正是几日前辈苏莫击败的天虚宗真传弟子武仁，至于另外两人，气息比武仁还要强大。
<br/>　　魔渊附近的所有人，见到这急速而来的三人，均是面色微变，知道接下来又有大事要发生了，苏莫要面临灾难了。
<br/>　　唰唰唰！
<br/>　　三名青年转瞬即至，瞬息间便来到了魔渊，他们身形不停，直接向苏莫飞了过去，而后瞬间便将苏莫包围了起来。
<br/>　　站在苏莫身边的那名御剑谷紫衣青年见此，急忙闪声离开，远远的和苏莫拉开了距离。
<br/>　　“苏莫，这回看你还怎么抢？“武仁居高临下的俯视着苏莫，眸中冷笑连连，这一次他定不饶苏莫。
<br/>　　苏莫眼眸微眯，抬头打量着眼前的三人。
<br/>　　这三人除了武仁之外，另一人是一名高高瘦瘦的黑袍青年，气息比武仁更强三分。
<br/>　　至于另外一人，苏莫也认识，而且非常的熟悉，正是天虚宗妖孽天才扬天成。
<br/>　　而此刻的扬天成，已经不再是武帝境九重的境界，而是武尊境一重的修为境界。
<br/>　　“苏莫，好久不见！”扬天成嘴角噙着若有若无的笑意，眸中尽是戏虐之色。
<br/>　　上次在太阴族的会武，扬天成丢进了脸面，对于苏莫更是恨之入骨。
<br/>　　他前几日突破到武尊境之后，便一直想找苏莫报仇，没想到心想事成了。
<br/>　　他刚刚出关，便在宗门内听闻武仁在魔渊被苏莫击败的事情。
<br/>　　所以，他毫不犹豫，便和武仁一起，联合另一名真传弟子禄冲，一起赶了过来。
<br/>　　“的确是好久不见！”苏莫淡淡的点了点头，心中却是一沉，这三人来着不善，他怕是有麻烦了。
<br/>　　“苏莫，上次你用阴险的手段，害我当众出丑，这笔账今日该算一算了吧？”扬天成冷声说道。
<br/>　　“你要怎么算？”苏莫皱眉问道。
<br/>　　扬天成闻言，略一沉吟，傲然道：“我也不难为你，当众磕头道歉，我便既往不咎！”
<br/>　　扬天成的神态颇为傲慢，仿佛他非常大度一般，与在太阴族会武之时，那种温文尔雅的姿态迥然不同。
<br/>　　“磕头道歉？”苏莫闻言，眼眸深处闪过一道冷光。
<br/>　　“不错，你磕头道歉，我便饶了你！”扬天成面色郑重的点头，但心中却是暗想，他要狠狠的折辱苏莫一番。
<br/>　　就算苏莫磕头道歉了，他也不会轻饶。
<br/>　　“还有我！”
<br/>　　这时，一边的武仁开口了，冷声道：“苏莫，只要你将这段时间以来猎杀的魔物骨刺，全部交给我，方能赔罪！”
<br/>　　“呵呵！”
<br/>　　苏莫笑了起来，怒急而笑，随即他长身而起，目光环视三人，轻笑道：“你们一个让我磕头道歉，一个让我献上骨刺赔罪，难道我是罪人吗？”
<br/>　　苏莫对于武仁和扬天成非常不屑，这两个家伙没有屁的本事，这欺人的本事倒是不小。
<br/>　　“难道你不是？”扬天成和武仁齐齐开口反问，在两人看来，苏莫就是罪人。
<br/>　　对于扬天成而言，苏莫在会武之中，故意让他出洋相，此番所谓罪不可赦。
<br/>　　而对于武仁而言，苏莫将他以及二十余名天虚宗弟子打伤，抢夺了魔渊的控制权，同样是罪大恶极。
<br/>　　“呵呵！”
<br/>　　苏莫再次轻笑一声，随即脸色冰冷了下来，目光扫视着武仁和扬天成，说道：“你们两个垃圾玩意，没有让我赔礼道歉的资格！”
<br/>　　“你说什么？”
<br/>　　“找死！”
<br/>　　武仁和扬天成两人，闻言顿时暴怒，眸中爆射出冰冷的寒光，他们没有想到，到了这个时候，苏莫居然还敢如此的猖狂。
<br/>　　“苏莫，天堂有路你不走，地狱无门你自来投！”
<br/>　　“是你自己找死，别怪我们了！”
<br/>　　武仁和扬天成纷纷厉喝一声，身上气势冲天而起，就欲直接出手。
<br/>　　所有在场之人，无不心惊肉跳，这苏莫不是一般的狂啊！
<br/>　　面对此种危局，居然还敢辱骂武仁和扬天成，这已经不是大胆了，而是在自寻死路啊！
<br/>　　扬天成乃是天虚宗的妖孽，现在突破到了武尊之境，实力绝对不会比武仁差。
<br/>　　而除了两人之外，还有一名黑袍青年。
<br/>　　那名黑袍青年，名为禄冲，同为天虚宗的真传弟子，而且名头比武仁还要响亮，实力也更为强大。
<br/>　　三人联手的话，苏莫定然会九死一生。
<br/>　　吕罡脸色有些苍白，他已经远远的飞开了，免得扬天成等人找他的麻烦。
<br/>　　至于苏莫的安危，他也顾不上了，不过在他看来，苏莫虽然不可能是扬天成三人的对手，但以苏莫的实力，成功逃命问题不大。
<br/>　　就在扬天成和武仁准备动手之时，蓦地，苏莫率先动手了。
<br/>　　只见他的周身道道剑光猛然浮现，而后直接向扬天成三人暴刺而去。
<br/>　　苏莫不得不率先出手，因为他处在扬天成三人的包围圈之中，这是非常危险的，他必须冲出包围圈。
<br/>　　唰唰唰！！
<br/>　　扬天成三人，没想到苏莫二话不说，就出手了，急忙身形闪烁，躲避灵剑的攻击。
<br/>　　而此时，苏莫则身形冲天而起，瞬间冲出了三人的包围圈，飞到了九霄之上。
<br/>　　“苏莫拿命来吧！”
<br/>　　扬天成见此，立刻在半空中盘膝而坐，身前出现了一柄古朴的木琴。
<br/>　　铮！
<br/>　　屈指一弹，一声嘹亮的音符响彻而起，而后化为了一柄犀利的剑光，直冲九天，向苏莫袭杀而去。
<br/>　　“死吧！”
<br/>　　武仁也是毫不含糊，大弓在手，瞬间取出了三支庚金破虚箭，三间齐发，洞穿虚无，发出刺耳的厉啸声，直击苏莫。
<br/>　　至于那名黑袍青年禄冲，则是没有出手，而是身形如电，直接向苏莫追了过去。
<br/>　　“你们简直是在找死！”飞上九天的苏莫，身形立刻停了下来，口中一声暴喝，眸中爆射出犀利的杀机。
<br/>　　扬天成和武仁两人，已经不是简单的要找回场子那么简单了，而是要置他于死地。
<br/>　　既然如此，那就别怪他不客气了！
<br/>　　这三人想要杀他，就要有被杀的觉悟。

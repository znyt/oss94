　　啸天狼族的人大怒，这苏莫下手居然如此之狠，接下来墨狼已经无法继续参加会武了。
<br/>　　不过，他们也无法说什么，毕竟这种会武，只要不伤人性命，都不算违规。
<br/>　　“幽狼，下一轮你应该知道怎么做了吧？”
<br/>　　啸天狼族之中，领头的森狼半圣对身边满身银色毛发的幽狼说道。
<br/>　　“族叔放心，我会打败苏莫，让他败的很惨！”幽狼沉声说道，他必须要打败苏莫，一是替墨狼出口气，二是为啸天狼族争回脸面。
<br/>　　毕竟，族中顶尖天才，被苏莫瞬间击败，这实在是让他们啸天狼族面上无光。
<br/>　　“嗯！”森狼半圣点了点头。
<br/>　　此时，苏莫和墨狼的战斗早已结束，战台上的比斗，已经继续开始了。
<br/>　　不过，整个武斗场依旧嘈杂不已，大量的人依旧在讨论苏莫，讨论他的实力，讨论他的底细。
<br/>　　实在是苏莫的表现，太过惊人了，让所有人都是感觉不可思议。
<br/>　　不过，苏莫虽然厉害，好像只是靠着诡异的能力，真正的修为实力，还有待考究。
<br/>　　看台上。
<br/>　　苏莫静静的观战，他都能感受到有无数的目光，在他的身上打量。
<br/>　　有太阴族之人，其它六大势力之人，还有道宗的弟子。
<br/>　　吕罡、林皇、唐古等一众道宗弟子，面色复杂，道宗之前丢尽了脸面，没想到居然是靠一名武帝境八重的武者，来夺回颜面。
<br/>　　苏莫端坐不动，耳边充斥着冯光絮絮叨叨的声音。
<br/>　　“苏莫，你适才是如何击败墨狼的？那紫色雾气是战魂吧？那是什么战魂？居然如此诡异！”冯光眸光闪亮，满脸的好奇之色，口中吐沫星子飞舞，噼里啪啦的问出一个个问题。
<br/>　　“冯长老，你的问题这么多，让我如何回答？”苏莫道。
<br/>　　“嘿嘿，那你告诉我，你那紫色雾气是什么？”冯光长老笑道，他是真的好奇了，同时心中暗暗猜测，难怪真贤圣长老和苏莫关系不一般，原来苏莫是妖孽，看来真贤长老是看重苏莫的天赋。
<br/>　　“这是梦魔战魂！”苏莫轻声说道，并没有什么隐瞒。
<br/>　　“梦魔战魂？”
<br/>　　不仅是冯光听到了，周围的道宗长老和弟子们也听到了，均是心中暗思，这是什么战魂？难道与梦有关？
<br/>　　“梦魔战魂有什么能力？”冯光睁着大眼睛，如同好奇宝宝一般，继续问道。
<br/>　　“冯长老，等会武结束之后再说吧！”苏莫摇了摇头，不想与对方细谈此事。
<br/>　　“那好，我们回去再谈！”冯光无奈的点了点头。
<br/>　　此时，战台上由于妖孽天才登场了，乃是御剑谷的妖孽卓剑豪。
<br/>　　卓剑豪背上背负着一柄黑色大剑，其身穿锦袍，身形挺拔，星眉剑目，相貌颇为英俊，就算比之扬天成，都不差多少，但气质和扬天成完全不同，其整个给人一种无形的锋芒。
<br/>　　他挑战的对手，是一名闪灵族的天才，名为灵晔，虽然实力不如灵枫，但也是闪灵族之中能排进前三的天才。
<br/>　　此人瘦小无比，身高不足四尺，仿若是一名十岁的孩童一般。
<br/>　　战台上。
<br/>　　卓剑豪背负着长剑，与灵晔相对而立，淡然无比。
<br/>　　“卓剑豪，让我领教一番你的本命剑阵吧！”灵晔面色凝重的说道，虽然他实力很强，又有瞬移的能力，但也丝毫不敢小看对方。
<br/>　　“你没有资格见识我的剑阵！”卓剑豪淡淡的说道。
<br/>　　“是吗？那谁才有资格？”灵晔闻言双眸微眯下来，对方还真是狂妄，他连见识的资格都没有吗？
<br/>　　“除了灵枫，炎澜天、扬天成、幽狼和雷云皓五人，七大势力之中，没有人有资格让我动用剑阵！”卓剑豪的眸中，闪烁着无与伦比的自信之光。
<br/>　　这是妖孽天才的自信，一种无敌的自信。
<br/>　　无数观战之人暗暗点头，卓剑豪的实力毋庸置疑，七大势力之中，的确只有另外五位妖孽能与之争锋了。
<br/>　　至于苏莫，的确很强，手段也很诡异，但真实的实力估计还是不能和卓剑豪等妖孽抗衡。
<br/>　　毕竟，苏莫才武帝境八重的修为。
<br/>　　“自大！”灵晔冷笑一声，蓦地，他的身形突兀的消失不见，再出现之时，已经出现在了卓剑豪的身后。
<br/>　　咻！
<br/>　　一道寒光闪现，灵晔的手中握着一杆短枪，短枪洞穿长空，枪尖瞬间便临近了卓剑豪的后背。
<br/>　　“好快！”苏莫神情一凝，这个灵晔的瞬移速度，以及出手的速度，比其他的闪灵族之人快多了。
<br/>　　就在短枪要洞穿卓剑豪的后背之时，只见卓剑豪身形微微一扭，瞬间便避开了短枪。
<br/>　　下一刻。
<br/>　　锵！
<br/>　　一声剑鸣厉啸长空，一道夺目的剑光冲天而起，是卓剑豪出剑了。
<br/>　　其出剑的速度，快到让人无法看清，只能看到寒光陡然一闪，空间瞬间裂开，夺目的寒光直击灵晔。
<br/>　　“不好！”灵晔顿时面色一变，他立刻再次施展了瞬移，刷一声，身形瞬间出现了百里之外。
<br/>　　躲过了卓剑豪一剑，灵晔低头看向肩膀，只见肩膀上出现了一道浅浅的剑痕，流出一丝腥红。
<br/>　　他，还是被尖锋扫中了。
<br/>　　回想到对方刚才的一剑，他面上露出心悸之色，快，好快的剑！而且威力绝伦。
<br/>　　“好恐怖的剑啊！”
<br/>　　“太快了！”
<br/>　　“根本就看不清是如何出剑的！”
<br/>　　“不愧是剑道奇才，妖孽天才！”
<br/>　　众人亦是心惊不已，这卓剑豪终于开始展现真正的实力了吗？而且这还是普通的一剑，不知道其剑阵的威力，能达到什么地步！
<br/>　　苏莫也很是惊讶，相比于对方，他在剑道上的造诣，实在太低了。
<br/>　　剑，对于他来说，只是一种武器，单纯的武器，但对于剑道高手来说，却是一种修炼道统。
<br/>　　此人在剑道上的造诣，绝对不凡，他也很好奇，对方的本命剑阵，有多厉害，等会倒是可以切磋一番。
<br/>　　战台上，卓剑豪一剑斩退灵晔之后，身上有锋锐的剑威升腾而起。
<br/>　　“瞬移，对我无用！”
<br/>　　卓剑豪的目光无比凌厉，淡淡的对灵晔说了一句，然后出手了，他手臂一抖，长剑凌空一刺。

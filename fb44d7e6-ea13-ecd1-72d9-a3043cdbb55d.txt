<br/>  老道士至从上次山神事件之后，王康就发现，他是个人才。<br/>　　<br/>　　  怎么说呢？<br/>　　<br/>　　  这个人喜欢研究一些奇怪超脱常理的事情，例如用醋在白纸上写画，然后用火烤后，就能显现。<br/>　　<br/>　　  还有热油锅探物，将手伸进热油里取出锅底物品，手拿出来毫发无损。<br/>　　<br/>　　  外人看来很神奇，这个人有法术，有异能。<br/>　　<br/>　　  其实根本不是，原理也很简单，是利用了不同物体，有着不同的密点和沸点。<br/>　　<br/>　　  “油锅取物”中的油锅，其实并非装满了油，而是装了油和醋的混合物。<br/>　　<br/>　　  不过，因为醋的密度大，油的密度小，所以两者混合时，醋沉到下面，油浮在上面，看起来就像是装了一锅油，实际上，只有上面一层浮着油。<br/>　　<br/>　　  然后给油锅加热。醋的沸点比油低，看着锅里沸腾，其实温度并不高。<br/>　　<br/>　　  这时探手取物，根本不会烫伤。<br/>　　<br/>　　  如此种种，还有几例，都是老道士自己研究出来的。<br/>　　<br/>　　  这个就有点厉害了。<br/>　　<br/>　　  不过他研究出这些，都是用来坑蒙拐骗，但不得不承认，也是有着厉害之处。<br/>　　<br/>　　  因此，王康就将此事情交给了他。<br/>　　<br/>　　  院外，有着士卒站岗巡逻，见得王康到来，开门相迎。<br/>　　<br/>　　  进了院子，明显发现不同，虽是民居但经过特殊改造，将两处连通，里面是很大的。<br/>　　<br/>　　  院子里两旁，种满了移植过来的各种花卉，玫瑰，牡丹，兰花，百合，一串红，芍药花等多种。<br/>　　<br/>　　  其里花香飘散，芳香四溢。<br/>　　<br/>　　  院内还有专门雇佣的妇女在浇花侍弄，见得王康走进，都是起身行礼。<br/>　　<br/>　　  “康少爷，”<br/>　　<br/>　　  “您来了，”<br/>　　<br/>　　  她们都知道，这个年轻的少爷才是现在新奉县的主人。<br/>　　<br/>　　  “老白呢？”王康问道。<br/>　　<br/>　　  “白主管在后院，用不用去通报。”<br/>　　<br/>　　  “不用了，你们干活吧！”王康说了一句，便向后院走去。<br/>　　<br/>　　  后院是连着几座屋子，王康进了其中一间，老道士正在一个器皿面前忙碌着，就连王康进来，都没察觉。<br/>　　<br/>　　  或许是长久习惯，老道士虽然不在做着道士的活，但还穿着道袍。<br/>　　<br/>　　  “老白。”王康叫了一句。<br/>　　<br/>　　  闻言，老道转头惊道：“哎，康少爷来了。”<br/>　　<br/>　　  “研究的怎么样了？”王康问着走到他身前。<br/>　　<br/>　　  “已经做出几种了，我正在研究看能不能找到其他的香精油。”<br/>　　<br/>　　  老道说着又道：“您发明的这个蒸馏器真的是太好用了，很方便。”<br/>　　<br/>　　  “哈哈，那是自然，做出来的有成品吗？给我看看。”王康问道。<br/>　　<br/>　　  “有的，您稍等。”<br/>　　<br/>　　  老道应道，不一会从里屋拿出来一个小木盒，他将木盒打开，里面放置着三个小瓶子。<br/>　　<br/>　　  瓶子晶莹剔透，里面分别盛放着几种液体，分别是粉色，红色，淡蓝色。<br/>　　<br/>　　  “康少爷，这个就是您要做的香水，”老道说着道：“我已经试过了，效果极好。”<br/>　　<br/>　　  王康拿起红色的小瓶，拔开小木塞轻闻了下，一股玫瑰的清香散发出来。<br/>　　<br/>　　  他的眼中也有激动之意，在他的指导下，香水终于问世。<br/>　　<br/>　　  同样是针对女人的设计发明，王康始终相信，女人的钱是最好赚的。<br/>　　<br/>　　  其实人们自古以来就有熏香的习惯，如香粉，香包，焚香，香汤浴等，古代也是有香水的，叫做蔷薇水，又叫古剌水。<br/>　　<br/>　　  一种由蔷薇花蒸馏而成的香水，五代时期由伊朗，阿拉伯等地传入我国。<br/>　　<br/>　　  到了宋代人们也掌握了蒸馏香水的技术。<br/>　　<br/>　　  但在现在这个时代是没有，因为现在都没有蒸馏技术。<br/>　　<br/>　　  不过王康早就做出来了，那还是他曾酿酒的时候使用，通过蒸馏酿出了高度白酒。<br/>　　<br/>　　  基本的器皿有了，其实制作香水就很简单了。<br/>　　<br/>　　  香水，是用一定比例的植物香精油加上一部分的酒精和纯净水混合制作而成。<br/>　　<br/>　　  先向瓶子中倒入大概百分之六十左右的纯净水，然后滴入百分之十的酒精，剩下的就可以滴入香精油了，这是标准配比。<br/>　　<br/>　　  唯一的难点在于香精油。<br/>　　<br/>　　  那香精油怎么来呢？这就要用到蒸馏了。<br/>　　<br/>　　  采集花期最好时的花朵，直接或稍微阴干后，放入蒸馏器内蒸馏产生精油液体。这个是制作植物精油最原始简单的方法。<br/>　　<br/>　　  接下来就是混合配比了，然后冷却存放，最后过滤提纯，这样香水就成了。<br/>　　<br/>　　  而放置香水的透明小瓶呢，就是由玻璃制成。<br/>　　<br/>　　  玻璃，也是王康近期搞出来的发明，在这个时代，所用最多接近玻璃的是琉璃。<br/>　　<br/>　　  琉璃相当的贵，是完全的奢侈品。<br/>　　<br/>　　  当时王康装修天上人间，大部分的钱都投在了这个上面。<br/>　　<br/>　　  因此，在条件成熟后，他便主持发明了玻璃。<br/>　　<br/>　　  玻璃其实在古代也有，在北宋时期，出现了商品玻璃，并开始成为工业材料，到了清初，为适应研制望远镜的需要，制出了光学玻璃。<br/>　　<br/>　　  不过，我国古代的玻璃，一直是较为简单的低温铅钡玻璃。<br/>　　<br/>　　  因为化学成分和烧成温度的原因，遇到热水就炸，因而根本无法用来做生活器皿。<br/>　　<br/>　　  不过在这个时期，这样的玻璃也没有，才会造成琉璃的高昂价格。<br/>　　<br/>　　  制作玻璃需要的材料有，石英砂，石灰石，长石，纯碱！<br/>　　<br/>　　  石英砂是石英石经过破碎加工而成的颗粒状，这个在新奉县就有。<br/>　　<br/>　　  石灰石，长石更是多见。<br/>　　<br/>　　  而纯碱呢，古代早就使用。<br/>　　<br/>　　  其实日常中所见之物，究其制作，原材料都是很长见得。<br/>　　<br/>　　  然后就是配料了，<br/>　　<br/>　　  将各种原料按照合适的比例混合。<br/>　　<br/>　　  然后怎么做呢？这一点是跟冶铁比较相似的，溶炼。<br/>　　<br/>　　  将配制好的原料经过高温加热，形成均匀无气泡的玻璃液，然后是将熔制好的玻璃液转变成具有固定形状的固体制品。<br/>　　<br/>　　  最后让它逐渐冷却退火，最终成型。<br/>　　<br/>　　  当然说起来是很简单，实际制作过程中，经历了不少的试验。<br/>　　<br/>　　  像现在也只能做一些小的瓶罐，大面积尺寸的玻璃还没成功，因为缺乏合适的模具。<br/>　　<br/>　　  不过这个问题，王辅臣他们已经在研究了，相信不日就会成功，因为王康建新城，建造现代房屋，<br/>　　<br/>　　  就准备用上玻璃。<br/>　　<br/>　　  而且，玻璃更能为他带来巨额的财富，再加上香水的研制成功，绝对会掀起一场新的风暴……<br/>　　<br/>　　  【作者题外话】：第四更，求票，感谢，(是不是学到了，一写这个就很慢，为了更加合理，)<br/><br/><br/>

　　各大世家如何管理自己的承包的地皮李承乾并不十分关心，如果他们连这点能力都没有的话，根本就不足以成为什么世家。
<br/>　　之所以对他们提出三个建议，是因为李承乾需要利用这些世家的资源来拉到大唐的经济，否则单靠他自己一个人瞎折腾，就是累死他也不可能把整个大唐的经济带动起来。
<br/>　　所以他不惜放弃一些自己的利益来满足这些家伙的要求，同时也在政策之类的问题上对世家放开了许多，好让他们在那些已经承包的土地上拿出更多的力量。
<br/>　　总之一场‘见面会’，把所有的事情搞定之后已经是天色微明，拖着疲惫的身体，李承乾回到自己的‘兰若寺’，一头扎进温暖的被窝睡死了过去。
<br/>　　至于说上朝……谁爱去谁去吧，老子现在还是个病人，老头子已经批准可以在除夕之前不用去上朝。
<br/>　　这一觉睡的，那叫一个天昏地暗、日月无光，醒了的时候已经是日上三竿，竟然睡了一天一夜还要多。
<br/>　　“我的哥哎，你可算是醒了，再不起来弟弟我都要急死了。”刚一睁开眼，人还没有回过神来，李佑那小子的声音就在炕边上响了起来。
<br/>　　“你啥时候来的？”用力揉了揉脸，李承乾打了个哈欠，不紧不慢的问道。
<br/>　　“我昨天就来了，来了三次，早，中，晚，今天双是从早上等到现在，我说哥啊，你这也太能睡了吧？到底几天没有睡过觉了？我上次好几天没睡，六个时辰也足够我睡醒的了，可你这足足睡了一天一夜还要多。”
<br/>　　被李承乾一问，等了他很久的李佑发起了牢骚，唠唠叨叨说个没完没了，像个老太太一样。
<br/>　　“停，要钱没有，要命一条！”呆呆的听着李佑发牢骚，李承乾的第一个反应就是这货想要借钱。
<br/>　　“啥钱啊？提啥钱啊，你没有我有啊，你说缺多少，我给你弄去。”李佑的反应让李承乾更加迷惑，完全不知道他在发那门子疯，甩甩头让自己更加清醒一些：“不是，你到底想干啥？说清楚点！”
<br/>　　“哥，你怎么给忘了？前段时间您不是说带我装逼带我飞么？”李佑略有失望，看着李承乾的眼神满是委屈，像一只受了伤的小狗，这在后宫小霸王身上可是难得一见的事情。
<br/>　　李承乾愕然抬头看着李佑，当时不过就是吹个牛逼而已，这小子怎么就当真了呢？
<br/>　　“哥，哥，你不是真给忘了吧？”李佑眼中满是失望。
<br/>　　“哦哦，没，怎么能忘了呢，等我收拾一下，然后就出发，我带你去看一样好玩的东西。”回过神的李承乾安慰了李佑一句，便从炕上跳下来，在仅有的两个侍女服侍下开始洗漱、更衣。
<br/>　　半个时辰之后，李承乾带着李佑还有赶来凑热闹的长乐出现在了将作监。
<br/>　　“老柳，前几天让你搞的那个东西弄出来没有？”看到柳敏的第一眼，李承乾不等他上来见礼便把他拉到一边。
<br/>　　“啥东西？您可是有一个多月没来过了。”柳老头一脸懵圈，看着李承乾的眼神像是在看一个傻子。
<br/>　　“啧”李承乾啧了一声：“就是三个多月前我让你搞的那个大风筝，搞好了没有。”
<br/>　　不得不说，李承乾脑子里的时间概念很弱，不管时间多长都会用类似‘前几天’，‘前段时间’这样的量词来衡量。
<br/>　　“哦哦，那个啊，早就搞好了。”一听三个月前，老柳一阵恍然，连连点头说道：“东西现在就在后院，殿下要看看么？”
<br/>　　“不看了，如果搞好了就装车，给本宫拉到龙首原那边去。”李承乾嫌来来回回的走浪费时间，决定先把东西拉到龙首原再说。
<br/>　　“喏！”柳敏答应着，躬身送李承乾一行离开，然后便命人把那个硕大的风筝装车。
<br/>　　一路无话，却说到了龙首原之后，一众工匠在李承乾的指点下把风筝全部组装完毕之后，一个硕大无朋的大家伙终于出现在众人眼前。
<br/>　　“我的个乖乖，哥，你这弄了个啥东西？”李佑围着那个有着木制框架，数层绸布包裹的大风筝转了好几圈，一脸迷惑的问道：“这还是风筝么？谁能放得动啊？”
<br/>　　近两丈的翼展，九尺左右的长度，三角形的巨大身躯尾部被修成燕尾形，若是拿到后世的现代，几乎所有人都会认得——滑翔翼。
<br/>　　没错，这就是李承乾新鼓捣出来的东西，一架滑翔翼，一个可以装逼可以飞的东西，放在大唐绝对是无与伦比的高大上。
<br/>　　“哥，这到底是个什么？你可别说真的是风筝，人家才不会信。”长乐同样呆呆的看着那个硕大的风筝，她不知道为什么风筝的下面为什么会有一个三角形的架子，也不知道为什么后面还有一个袋子，这些东西很明显不是风筝上应该有的东西。
<br/>　　“这叫滑翔翼，一种可以飞的东西。”万事俱备之后，李承乾摩拳擦掌的开始装备，护肘、护膝、手套、头盔各种装备都穿好之后，站到了滑翔翼的旁边。
<br/>　　什么情况？这是要干什么？长乐和李佑以及包括柳敏在内的工匠全都紧张了起来，似乎预感到有什么事情即将要发生。
<br/>　　“小夜，你们搞定了没有？”站在滑翔翼边上等了一会儿之后，李承乾对躲进一边‘照相车’中的夜魅喊了一声。
<br/>　　“好了！”片刻之后，夜魅从‘照相车’里出来，手里拿着一个并不大的玻璃片，对着李承乾晃了晃。
<br/>　　“收好了，千万收好了，若干年后这东西可值老钱了，这可是鉴证历史的东西，先万不能丢了。”李承乾对夜魅嘱咐着，不过听起来却十分怪异，让人有些摸不到头脑。
<br/>　　不过管他呢，现在几乎所有人的目光都盯在李承乾的身上，小雨姑娘和夜魅两个更是紧张的手心冒汗。
<br/>　　因为此时的李承乾已经钻到了滑翔翼的下面抓着三角形的架子提了提，感觉了一下重量之后，便顺着前面的山坡冲了下去。
<br/>　　由特殊木料制成的滑翔翼重量很轻，加上外面包着的绸布总份量也只有不到五十斤，提起来虽然有些费力，但却并不十分困难，在顺着山坡往下跑的情况下，并不是多大的问题。

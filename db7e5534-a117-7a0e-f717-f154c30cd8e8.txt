        高考结束后，童筱颖和苏漫雪同时进行了自行估分。
<br/>
<br/>        童筱颖果然如那几个家庭教师所说的一样，考上一本大学完全没问题。
<br/>
<br/>        至于苏漫雪，可能勉强能过临海大学附属学院的分数线，而且到时候还需要傅老太爷去找找关系才行。
<br/>
<br/>        如今，高考都结束了。
<br/>
<br/>        童筱颖盼着傅昼景能来接她回家，但是她的期盼还是落空了。
<br/>
<br/>        傅昼景回了简讯给她，让她自己坐高铁或者是坐飞机回滨江城来。
<br/>
<br/>        滨江城离临海城也不算太远，开车也就两个小时左右能到。
<br/>
<br/>        童筱颖收拾了行李，向傅老太爷和冷夫人告别后，打算自己回滨江城。
<br/>
<br/>        苏漫雪则因为忌惮傅夜沉，执意要留在傅家大庄园里陪冷夫人和傅老太爷，心里想着自己要傍着这两座大山，傅夜沉才不会把她怎么样。
<br/>
<br/>        也的确，只要她苏漫雪安安分分地待着傅家大庄园里，不去招惹童筱颖，傅夜沉暂时自然不会“处置”了她。
<br/>
<br/>        苏漫雪心里十分清楚，傅夜沉那颗心全在童筱颖身上。
<br/>
<br/>        如今，童筱颖成为了傅夜沉的弟妹，他不敢再对童筱颖有非分之想，但又不想惹人非议，索性，他就拿她苏漫雪来当幌子。
<br/>
<br/>        傅夜沉步步为营，全都只是为了护童筱颖周全。
<br/>
<br/>        童筱颖拖着行李箱，刚出傅家大庄园的正大门。
<br/>
<br/>        一辆霸气的黑色越野车在她面前停了下来。
<br/>
<br/>        童筱颖觉得这车很眼熟，抬眸看去的时候，只见后座的车窗落下，车内传来了傅夜沉的声音。
<br/>
<br/>        “上车。”
<br/>
<br/>        “大哥？”
<br/>
<br/>        “快点上车。”
<br/>
<br/>        “我自己……”
<br/>
<br/>        “童筱颖，你是想我压你上车吗？”
<br/>
<br/>        他不可违抗的命令，让童筱颖倒吸了口冷气。
<br/>
<br/>        此时，从驾驶座上下来一个身着黑色西装革履的男司机。
<br/>
<br/>        他小跑到童筱颖面前，替童筱颖拉开车门后，又提着童筱颖的行李箱去了后备箱。
<br/>
<br/>        童筱颖紧抿着粉唇，一脸不悦地刚坐上车，就听到身边的傅夜沉声音冷淡的开口：“是阿景让我顺道带你回滨江城的。”
<br/>
<br/>        得知是傅昼景的意思，童筱颖情不自禁地嘴角微扬。
<br/>
<br/>        傅夜沉瞥了一眼童筱颖，看着她嘴角的笑容，心里虽然开心，但也痛着。
<br/>
<br/>        她只有在提及他弟弟的时候，她才会发自内心的笑，看来她是真的爱上他的弟弟了。
<br/>
<br/>        为什么她爱的人，不是他呢？
<br/>
<br/>        哪怕先遇上，也不一定会爱上吗？
<br/>
<br/>        傅夜沉的心深沉了几分。
<br/>
<br/>        “你之前是学室内设计的？”途中，傅夜沉总是在有事没事地搭话。
<br/>
<br/>        他外表看起来，可不像是一个非常善言辞的人。
<br/>
<br/>        殊不知，他也只有在她一个人面前比较聒噪而已。
<br/>
<br/>        “嗯。”童筱颖应了声。
<br/>
<br/>        “这个暑假，你还打算出去实习吗？”傅夜沉又问，其实他算是明知故问吧！阿景掌管着傅氏集团旗下的建筑公司——地利置业，筱颖毕业后自然是在阿景手下做事。
<br/>
<br/>        童筱颖无奈地笑了笑：“哪都不想去，在家当全职太太。”
<br/>
<br/>        其实，她还得回“夜惑”去驻唱。只不过……这件事情，她不能跟大哥提，万一被昼景也知道了的话，那就糟糕了。
<br/>
<br/>        听她这么一说，傅夜沉又沉默了。
<br/>
<br/>        他忽然想起了什么，从自己的公文包里，掏出了一个精致的小方盒，递给了童筱颖。
<br/>
<br/>        童筱颖惊讶地瞥了一眼，并未接，只是好奇地问：“这是什么？”
<br/>
<br/>        “陨石项链！”傅夜沉淡漠地回答。
<br/>
<br/>        “我想，大哥最初应该是想送给大嫂的吧！因为我跟大嫂长得一模一样，让大哥认错了人，这东西也送错了人。”童筱颖落落大方地说。
<br/>
<br/>        傅夜沉顿时皱起了眉头。
<br/>
<br/>        她说他送错了人？！
<br/>
<br/>        不！
<br/>
<br/>        他没有送错！
<br/>
<br/>        这就是他送给她的。
<br/>
<br/>        但是，在她眼里，却认为是他送错了人。
<br/>
<br/>        心痛，已经不是一二两次，再痛一次，又何妨？
<br/>
<br/>        傅夜沉微微闭上眼睛，冷冷地回答道：“你不要我就把它扔了。”
<br/>
<br/>        童筱颖怔了怔，讷讷地接过礼盒，只好先将这小方盒收回手提包里，不再多话，而是静静地望着车窗外。
<br/>
<br/>        两个小时后……
<br/>
<br/>        傅夜沉将童筱颖送到了地利公寓楼下，便让司机开车扬长而去。
<br/>
<br/>        童筱颖开门回到家中后，心里顿时迷茫了。
<br/>
<br/>        以后，她又要过着这种一个人吃饭、一个人睡觉、一个人逛街、一个人做任何事情……的生活吗？
<br/>
<br/>        这样的婚姻生活，真的不是她想要的。
<br/>
<br/>        童筱颖心里一边想着，一边朝厨房走去，打开冰箱，里面除了速食和一些饮料以外，其他什么也没有。
<br/>
<br/>        难道，她不在家的这段日子，昼景都没有回家做过饭吗？
<br/>
<br/>        这不太像昼景的作风啊！
<br/>
<br/>        关上冰箱，童筱颖决定先去超市买些食材回来。
<br/>
<br/>        这边傅氏集团旗下地利置业大厦，傅夜沉突然造访傅昼景的办公室，让傅昼景赶到有些意外。
<br/>
<br/>        “大哥，你回来得真快！”傅昼景打趣地说。
<br/>
<br/>        “你让我顺道接你老婆一起回来，就不怕我把你老婆拐跑？”傅夜沉开玩笑似的反问。
<br/>
<br/>        傅昼景倒是不以为然：“大哥，你不会。”
<br/>
<br/>        更何况，大哥还有个未婚妻苏漫雪，他对自己的未婚妻那般宠爱，又怎么可能会对筱颖那丫头感兴趣。
<br/>
<br/>        傅昼景笃定，笃定到他连苏漫雪的样貌为何会整得和童筱颖一模一样的这个问题，都懒得去思考。
<br/>
<br/>        傅夜沉不经意间地苦笑了笑。
<br/>
<br/>        此时，傅昼景的助理赵斌敲门进来，见傅夜沉也在，忙颔首问候了一声：“大少爷，您好。”
<br/>
<br/>        傅夜沉微微点了下头。
<br/>
<br/>        傅氏集团，现在已经有一半的事情由傅夜沉在接手管理。
<br/>
<br/>        赵斌自然不敢怠慢傅夜沉。
<br/>
<br/>        所以，这一进门，他自然是先问候傅夜沉，再向傅昼景通报：“傅总，季总来了。”
<br/>
<br/>        “让她在贵宾室等等。”傅昼景回答道。
<br/>
<br/>        傅夜沉见状，会心一笑：“那我先走了。你忙。”
<br/>
<br/>        “大哥，慢走。”傅昼景微微颔首，然后目送傅夜沉离开。
<br/>
<br/>        傅夜沉并未想到，弟弟阿景即将见的这个“季总”，就是季思妍。
<br/>
<br/>        对于季思妍已经不在地利置业实习的事情，傅夜沉是知道的。
<br/>
<br/>        季思妍现在在父亲的支持下，另行开了一家自己的建筑公司，上次和傅昼景达成一致后，和傅昼景除了业务来往，私底下的来往也很频繁。
<br/>
<br/>        这些事情，其实是傅炎廷的特助林贵打听道后，告知了傅炎廷，而爷爷傅炎廷前几天才跟他私底下提起过。
<br/>
<br/>        季思妍和傅昼景来往频繁，只怕是为了“远东城军机部”那十个亿的项目。
<br/>
<br/>        一块味道鲜美的肥肉，自然很多人都会想去抢着吃。
<br/>
<br/>        傅夜沉却觉得，这种事情很寻常。
<br/>
<br/>        傅氏集团已经拿下了这个项目的设计权，傅夜沉跟傅昼景也交过底，他想拿这项目的施工权。
<br/>
<br/>        傅昼景只说，他尽力而为。
<br/>
<br/>        傅夜沉离开后，傅昼景去了一趟贵宾室。
<br/>
<br/>        他一进门，看到季思妍起身朝他走过来，便问道：“今天，你怎么过来了？”
<br/>
<br/>        傅昼景记得，今天好像没什么业务要跟季思妍交流。
<br/>
<br/>        季思妍拿起手机，使了使眼色，瘪嘴道：“你不接我电话，也不回我电话，我就只好以工作的名义来找你喽！”
<br/>
<br/>        说到手机，傅昼景这才想起，于是连忙从兜里掏出手机来看看，原来，手机被他放在口袋里不小心碰成了静音状态。
<br/>
<br/>        点开手机屏幕，里面还有几条短信是筱颖发过来的。
<br/>
<br/>        和以前一样，她是在问他会不会回来吃饭。
<br/>
<br/>        “晚上，去我家吃饭。”季思妍接着说道。

　　“自从两年前和神女一别，我这两年来一直期待能再次见到神女，瞻仰神女荣光，今日我有一个不情之请，不知神女能否答应在下？”扬天成挺拔而立，目视云悠悠，不卑不亢的说道。
<br/>　　“什么请求？”云悠悠疑惑的问道。
<br/>　　“待这次会武结束之后，我想与神女单独一叙，畅谈武道，不知神女是否同意？”扬天成面色真诚的说道，他眼眸中流露出浓烈的期待。
<br/>　　众人闻言惊讶，原来这个扬天成，居然也是心仪神女。
<br/>　　一瞬间，在北方看台之上，很多太阴族的内族天才，以及南方看台上，七大势力的顶尖天才，目光都凌厉了起来。
<br/>　　这些人之中，大多数都是心仪神女之人，但没有人敢明目张胆的说出来，扬天成居然如此大胆，当众要求和神女独处！
<br/>　　不少人心中冷笑了起来，心仪神女倒是很正常，但是在这种场合提出要求，神女怎么可能会同意！
<br/>　　果然，只见云悠悠柳眉微瞥，歉意一笑，婉拒道：“不好意思，待会武结束，我可能要立刻闭关，怕是没有多余的时间。”
<br/>　　扬天成闻言，也不气馁，他知道云悠悠很难同意他的要求，虽然他是天才，但追求对方的顶尖天才不在少数。
<br/>　　据他所知，在太阴族内族之中，都有很多的妖孽天才，在追求对方。
<br/>　　“神女，倘若我能得到此次会武的第一名，不知神女是否愿意给我这个机会！”扬天面上带着一丝傲然，眸中神采奕奕，显得自信无比。
<br/>　　众人闻言心中暗惊，这扬天成居然如此自信，能在会武中夺得第一名？
<br/>　　这是自大还是自信？
<br/>　　要知道，七大势力之中，除了道宗没有强大的妖孽天才之外，其它几大势力，可都是有非常强大的妖孽人物。
<br/>　　比如黑炎族的炎澜天、啸天狼族的幽狼，闪灵族的灵枫，御剑谷的卓剑豪，雷云禽族的雷云皓。
<br/>　　这几位妖孽人物，都是战力极度强大之辈，无论是天赋还是名气，都不比扬天成弱。
<br/>　　这扬天成居然如此狂言，能从这些人中脱颖而出，获得第一？
<br/>　　众人心中惊疑不定，对于接下来的会武，则是更加的期待了起来。
<br/>　　苏莫的眼眸已经眯了下来，凌厉的目光落在了扬天成的身上，锋锐冰寒，冷冽刺骨。
<br/>　　他这才刚见到青璇，居然就有情敌出现了！
<br/>　　这个家伙，居然公然的在他的面前追求青璇，这不是找虐吗？
<br/>　　这种事情，他是绝对不会让其发生的。
<br/>　　苏莫嘴角勾起一丝冷笑，得第一名吗？有他在这里，谁能得到第一名？
<br/>　　看台上，云悠悠柳眉微瘪，面上闪过犹豫之色。
<br/>　　她不想同意这个要求，但对方已经给自己加了限制，是在夺得第一名的基础上，若是她还不同意，就显得有些不近人情了。
<br/>　　毕竟，得到会武第一名，会得到她师尊的指点，若是她连和对方一叙的面子都不给，也有点说不过去。
<br/>　　“好吧！若是你能得到会武的第一名，我们可以一叙，畅谈武道！”
<br/>　　最终，云悠悠还是点了点头，同意了扬天成的请求。
<br/>　　“好，有神女这句话，这一次会武我就算舍命，也要得到第一名！”扬天成面上露出喜色，眸中精光爆射，只感觉体内充满了力量。
<br/>　　这是多么好的机会，如果能和对方单独相处，就能结下友谊，就能为日后俘获对方的芳心打下根基。
<br/>　　如果他能和神女结为道侣，以对方的身份，和纯阴之体的双修体质，能给他带来无穷无尽的好处。
<br/>　　其实，扬天成也没有绝对的信心能得到第一名，毕竟太多强大的对手了。
<br/>　　不过，他也是在逼迫自己，有压力才有动力，这一次，第一名非他莫属。
<br/>　　随即，扬天成目光望向南方看台，开始挑选对手。
<br/>　　南方看台之上，不少均是眸中战意升腾，不少爱慕云悠悠的人，都是对于扬天成产生了敌视。
<br/>　　“林皇，出来一战！”扬天成的目光落在了林皇的身上，朗声喝道。
<br/>　　对于扬天成来说，太弱的人，没有资格让他挑战，但太强的人，现在也不是挑战的时候。
<br/>　　所以，林皇算是合适的对手，击败以后，才会有足够的震慑力。
<br/>　　众人见此，不禁无语，之前炎澜天挑战唐古，现在扬天成又挑战林皇，这是欺负道宗无人啊！
<br/>　　道宗的一众长老们，脸色不禁更难看了，真是人弱被人欺啊！
<br/>　　林皇捏了捏拳头，长身而起，一步踏出，飞落在了战台之上。
<br/>　　他无法拒绝，即便没有信心，也只能倾力一战。
<br/>　　“林皇，之前唐古在炎澜天的手上没撑过两招，你在我手上，同样撑不过两招。”扬天成淡笑着说道，其言语虽然有些狂妄，但面上的笑容却给人一种和煦之感。
<br/>　　“废话少数，战吧！”
<br/>　　林皇朗喝一声，全身气息涌动，战魂释放而出，特殊体质的血脉力量，同样催动了起来。
<br/>　　林皇并不是双生战魂，他只有一种战魂，是一种巨大的蜥蜴战魂。
<br/>　　这种蜥蜴，名为赤灵狂蜥，特殊的异兽，非常强大。
<br/>　　而林皇的血脉体质，同样很强，名为紫纹灵体、排名比金刚之体都要略高。
<br/>　　紫纹灵体一催动，林皇的全身皮肤之上，顿时布满了一道道纷乱的紫色纹路。
<br/>　　这种紫纹灵体，无法增强林皇的身体力量，也无法增强他的玄力，但却能让他沟通周围无形的天地之地，战力倍增。
<br/>　　“杀！”
<br/>　　怒喝一声，林皇率先出手了，大手一张，猛然向扬天成凌空一抓。
<br/>　　伴随着林皇的招式，他背后的赤灵狂蜥战魂，同样利爪一探。
<br/>　　轰隆隆！
<br/>　　霎时之间，一只长达百丈的巨大手爪，碾碎虚空，裹挟漫天狂风，滔天气势，向扬天成狠狠的抓了过去。
<br/>　　巨爪杀机无限，气息如同爆发的火山，狂猛霸道，凶厉狠辣。
<br/>　　这一抓非常的强大，林皇没有任何的保留，直接动用了巅峰的战力。
<br/>　　更加惊奇的是，这巨爪的攻击，居然得到了周围天地之力的加持，方圆千里之内的灵气仿佛被引动了一般，飞速的汇入巨爪之中，让得巨爪威力再增。

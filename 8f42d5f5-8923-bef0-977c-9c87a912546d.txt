　　“这…这…”
<br/>　　北隐宗大长老不可置信地看着胸膛处。
<br/>　　苍老的眼眸下，余光瞥了眼。
<br/>　　身后，一抹狰狞的笑容，不知为何，竟让他浑身发冷。
<br/>　　那抹笑容，来自于萧逸。
<br/>　　刚才，紫炎游龙与元力游龙交锋不停时。
<br/>　　萧逸一个脚步爆发，瞬间便来到了北隐宗大长老身后，一拳轰之，洞穿了他的身体。
<br/>　　北隐宗大长老操控那一条条元力游龙时，需得全力为之，分心不得。
<br/>　　可萧逸操控那数以千计的紫炎游龙时，却轻而易举。
<br/>　　这时便看出那双兽手的控火能力之恐怖了。
<br/>　　如此轻易地操控数千狂猛紫炎游龙，萧逸一个心念间，便能让紫炎游龙与元力游龙全力交锋。
<br/>　　故他能轻松爆发至北隐宗大长老身后，打出这致命一拳。
<br/>　　北隐宗大长老也万万想不到，操控如此威力狂猛且数量庞大的紫炎游龙下，萧逸还能分心朝他攻击。
<br/>　　没有防备下，自也瞬间让自己陷入了必死之况。
<br/>　　“看来，今日要埋尸金光禁地之下的，是你北隐宗大长老。”萧逸冷笑一声。
<br/>　　“小子。”北隐宗大长老咬了咬牙，脸上的暴怒已然全无，取而代之的是一抹恐慌与惊骇。
<br/>　　“杀了老夫，你有何好处？”
<br/>　　“今日若就此作罢，老夫保证北隐宗与你恩怨全消，如何？”
<br/>　　北隐宗大长老虽被穿身，但一身精纯修为下，还没那么容易死。
<br/>　　他乃是武道大能，九千六百条完整武道加身下，生命力异常强悍。
<br/>　　只要萧逸现在愿意放他，他顶多是重伤的下场。
<br/>　　当然，他现在也动弹不得，更不敢动弹。
<br/>　　因为，萧逸穿身的那只手臂，只需轻松一动，就能立刻取他性命。
<br/>　　那散发着寒芒的恐怖爪子，瞬间就能捏碎他的心脏。
<br/>　　“求饶吗？”萧逸戏谑一笑。
<br/>　　“是。”北隐宗大长老毫不犹豫地吐出一个字。
<br/>　　对比死亡，他更愿意吐出这个字。
<br/>　　他是成名多年的强者了，活了数百岁月，甚至更久。
<br/>　　人越老，越怕死，看来果真不假。
<br/>　　“此次作罢，你有何要求，老夫尽皆满足。”北隐宗大长老的语气，异常恭顺。
<br/>　　“若我说不呢？”萧逸狞笑着。
<br/>　　“不？”北隐宗大长老双眼一眯。
<br/>　　“老夫提醒你，我来金光险地，是有要事要办。”
<br/>　　“我若身死，北隐宗必定震怒，全力追查。”
<br/>　　“你想接受整个北隐宗的怒火？”
<br/>　　他愿意暂时放下尊严求饶，但不代表他会无休止地求饶下去。
<br/>　　他相信，放眼中域，没人愿意承受整个北隐宗的怒火。
<br/>　　“你之前不是说了？”萧逸冷笑一声。
<br/>　　“我若将你埋尸金光禁地这深不可测之地，谁能知晓？”
<br/>　　“至于你所谓的条件，好处…呵。”
<br/>　　“对于你这样的武道大能来说，自信无比，一身宝贝都放在身上乾坤戒中吧。”
<br/>　　“我杀了你，便能得之，放你作甚？”
<br/>　　“你…”北隐宗大长老霎时大怒，却也知晓萧逸之话属实。
<br/>　　而最重要的是，他知道，今日萧逸绝不会饶他性命。
<br/>　　一股气势，悄然在他身上凝聚。
<br/>　　“小贼，老夫要死，你也需得来陪葬。”
<br/>　　愤怒的话语落下，气势陡然激增。
<br/>　　“想自爆，拉我一起死？”萧逸冷笑一声。
<br/>　　穿过北隐宗大长老的狰狞拳头，忽然重重一捏。
<br/>　　北隐宗大长老的心脏，以及体内自成空间的小世界，瞬间捏碎。
<br/>　　无数精纯元力，霎时从北隐宗大长老身上溢散。
<br/>　　而北隐宗大长老原本打算自爆而凝聚的气势，也瞬间消散。
<br/>　　“我的修为…我…”
<br/>　　北隐宗大长老脸色大变，但话语，却未能继续说下去。
<br/>　　下一秒，北隐宗大长老已然生机散尽，就此身死。
<br/>　　萧逸收回了拳头。
<br/>　　嘭…嘭…嘭…
<br/>　　一条条紫炎游龙，霎时从远方疾速而来。
<br/>　　萧逸身影一闪，回到了地面。
<br/>　　高空中，紫炎游龙，已彻底吞噬了北隐宗大长老的尸体。
<br/>　　不过片刻，高空中爆响不断，火焰连绵。
<br/>　　待得火焰消散，里头已是一片空白。
<br/>　　萧逸根本不需要将北隐宗大长老的尸体埋尸金光禁地之下。
<br/>　　武道大能的身体，确实强悍；甚至在无数天地规则力量加持下，难以受损。
<br/>　　但紫晶灵炎焚毁万物之效，却无物可挡。
<br/>　　此时，北隐宗大长老的尸体，已被焚毁成虚无。
<br/>　　高空中，一枚乾坤戒缓缓飘落。
<br/>　　萧逸一把捉过，这就是北隐宗大长老随身携带的乾坤戒。
<br/>　　萧逸感知了一眼，脸色大喜。
<br/>　　里头，重宝无数，想必是北隐宗大长老这位武道大能的全部家当了。
<br/>　　顺带一提，在萧逸操控紫炎游龙焚毁尸体前，已眼中光芒一闪，吸了北隐宗大长老的武魂力量。
<br/>　　“呼。”萧逸轻呼出一口气，收起了乾坤戒后，淡淡一笑。
<br/>　　“原本还以为要借助雾妖的力量，没想到压根不需要。”
<br/>　　不错，在萧逸之前的打算中，是以一身实力与北隐宗大长老交手。
<br/>　　随后趁机让雾妖去斩杀。
<br/>　　上一次被北隐宗大长老追杀，他毫无办法。
<br/>　　可去了一趟风刹殿总殿后，风圣壶和雾妖大大成长后，他便已有了底牌。
<br/>　　这也是他刚才不惧北隐宗大长老的原因。
<br/>　　不过，他没想到那双兽手和兽脚如此力量惊人。
<br/>　　在其增幅下，加上自己的一身手段，竟然直接斩杀了北隐宗大长老。
<br/>　　不仅不必雾妖出手，还省了自己不少麻烦。
<br/>　　萧逸看了眼自己的双手，自笑一声。
<br/>　　他不得不再次惊疑，八龙焚火炉内封印的妖兽尸骸，到底是属于何等妖兽，竟有如此恐怖力量。
<br/>　　那般控火之能，甚至比火族血脉还要强大几分，让得他的紫晶灵炎威力大增。
<br/>　　萧逸面露喜意，忽然，似是想起了什么，思索了起来。
<br/>　　他记得，之前八龙焚火炉上，第二道封印刚被冲破，出现了这左手时，他便尝试过左手的威力。
<br/>　　对比刚才双手以及左脚尽皆覆盖的威力，显然刚才的威力更强。
<br/>　　“难道说…”萧逸皱着眉。
<br/>　　半晌，萧逸暂时稳下心头思索，看向金光禁地之内。
<br/>　　“金光禁地吗？下面到底有什么？”萧逸疑惑自语一声。

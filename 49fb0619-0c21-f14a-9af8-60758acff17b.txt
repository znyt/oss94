　　子午跟SHADOW是老朋友更是老对手，子午了解他的手段，他们之间交手过很多次，但是子午并没有见过SHADOW。
<br/>　　黑客都是保持着神秘的，子午没有见过SHADOW，SHADOW也没有见过子午，当然这是在子午去代码小镇之前。
<br/>　　不过，两个人如此近距离的见面，却是实打实的第一次。
<br/>　　智天使站在一旁，眼睛在子午和影天使身上扫了扫，嘴角露出一丝微笑，说道：“老朋友见面了，不应该拥抱一下吗？”
<br/>　　“没兴趣。”
<br/>　　子午冷哼一声，仔细打量着影天使，随后问道：“你现在的样子，是你真实的外貌吗？”
<br/>　　影天使笑了笑，回避开这个问题，用另外一种方式回答道：“是不是我真实的外貌又有什么关系呢？世人只知道这个世界上有个人叫SHADOW，至于SHADOW是男人还是女人，是帅是丑，是老是少，又有什么关系呢？”
<br/>　　“把没脸见人都能说得这么天花乱坠，果然是了不起。”子午一脸讥诮的看着影天使。
<br/>　　对于子午的嘲讽，影天使也是不以为意，说道：“没想到GHOST不仅手上的技术了得，就连说话都这么风趣。”
<br/>　　“其实我妈一直以来的愿望是让我当一个相声演员，只可惜我对这些东嘴皮子的事情不感兴趣。”
<br/>　　子午站在影天使面前，两个身高相差不大的人就这么面对面的站着，相互打量着对方，过了一会儿，子午说道：“时间差不多了，可以开始了，早点比完早点回家。”
<br/>　　影天使饶有兴趣的看着子午，说道：“你就这么有把握能赢我？”
<br/>　　“我们之间交手过这么多次，各有胜负，但是总得来说，你输多赢少。”子午淡淡的回答道。
<br/>　　“呵呵，不过这次可不一定哦。”
<br/>　　“那也要比过之后才知道。”
<br/>　　子午和影天使之间的比试还没有正式开始，但是两个人之间的火药味就已经非常浓烈了。
<br/>　　在智天使的带领下，子午和影天使走到那两间早就准备好的房间前，房间里的布置一模一样，一张桌子，一把椅子，一台电脑，还有墙壁上一个液晶屏。墙壁上的液晶屏给子午和影天使在比试过程中进行“交流”的，而这所谓的“交流”其实就是对对方进行干扰，高手之间的对决是容不得慢点疏忽的，一句垃圾话都有可能影响到一场对决的最终胜负。
<br/>　　“两台电脑的配置和性能都是一模一样的，如果你不放心的话，可以去检查。”影天使对子午说道。
<br/>　　原本影天使以为子午并不会去检查，但是令他没有想到的是，子午居然真的去检查桌子上那两台电脑了，子午双手一边在键盘上敲打着，一边说道：“不是我信不过你，我是信不过他。”
<br/>　　子午瞥了眼站在影天使旁边的智天使，说道：“玩谋略的人心都脏。”
<br/>　　智天使这个时候是有些庆幸的，本来在安排这两间房间的时候，他的确是想动点手脚，但是却被影天使给阻止了，因为他想要跟子午光明正大的分出一个胜负，为此，吴云还对影天使的高傲表示出了不屑；不过，智天使现在明白，对于房间里这个男人，站在自己身边的影天使要比自己更加了解。最了解自己的人，永远都是自己的对手，因为只有自己的对手才会不停的去研究自己，思考着打败自己的方法，不是么？
<br/>　　子午是玩电脑的高手，很快就把两个房间里的电脑都检查过了一遍，并没有什么问题；如果真有什么问题是绝对瞒不过他的。
<br/>　　“如果没问题，我们可以开始了。”
<br/>　　影天使跃跃欲试，对于今天这样的一次比试，他期待已久了。正如之前子午所说的那样，他跟子午之间的较量，输多赢少，两个人之间最后一次较量，他最后还是输了。
<br/>　　子午拉开椅子做了下来，“我就坐这儿吧。”
<br/>　　影天使点了点头，去了隔壁房间。两个人进入房间后，房间门被智天使关上，这两间房间的门是通过房间里两台电脑控制开关，无法由外部手段进行关闭，这也是保证了这次比试的公正性。
<br/>　　咔哒！
<br/>　　咔哒！
<br/>　　子午和影天使操作电脑，将对方房间的门锁上锁。
<br/>　　“你电脑上现在能看到一个五秒钟的倒计时，五秒钟后，键盘自动解锁，此后的一个小时你可以通过各种手段在我的门锁进行设置，目的只有一个，防止我从这个房间逃出去。”
<br/>　　影天使出现在墙壁的液晶屏幕中，“当然，我也会在你房间的门锁上做同样的事情，时间是一个小时，一个小时后系统会把设置程序锁定，同时，房间里的塑胶炸药的倒计时会启动，时间是十五分钟，十五分钟能从对方设置锁定的房间中逃出去的那个人能活下来。”
<br/>　　子午在进入房间的时候，就看到了放在房间四个角落里的塑胶炸药，四个角落中的塑胶炸药上都有计时器，上面的时间是十五分钟，还没有启动。
<br/>　　“GHOST，准备好了么？”影天使看着摄像头，对子午问道。
<br/>　　“在比试开始前，我有一个问题想要问你。”子午说道。
<br/>　　“你问。”影天使说道：“只要是我能回答的，我一定会回答你，我总不能让你带着遗憾离开这个世界。”
<br/>　　“我真不明白你哪来的迷之自信。”
<br/>　　子午轻轻摇头，看着液晶屏幕中的影天使问道：“你是‘暗月’的人？”
<br/>　　“这是显而易见的。”影天使回答道：“如果我不是‘暗月’的人，我又怎么可能会跟智天使在一起，我还可以告诉你，我不仅是‘暗月’的人，而且我在‘暗月’里的地位还不低，我是‘十二主天使’之一的影天使，也是‘十二主天使’中最神秘的‘主天使’。”
<br/>　　影天使的回答，显然是有些出乎子午意料的，他猜到了SHADOW跟“暗月”有关系，但是没有料到他居然会是“十二主天使”中的一个。
<br/>　　“还有一个问题。”子午看着影天使，问道：“‘暗月’到底是一个什么样的组织？”

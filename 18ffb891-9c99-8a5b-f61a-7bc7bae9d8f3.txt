　　“可是陛下，这能行么？虽然听起来像是那么一回事儿，但是这真的能行么？”褚遂良对李承乾的判断持怀疑态度，他并不认为这是一个好主意。

<br/>　　另外像薛仁贵、苏定方他们，也都在外面牧守边疆多年，也该调回来了。

<br/>　　想着一些有的没的，李承乾结束了这一天的早朝，略有些兴奋的回了自己的书房。

<br/>　　李承乾此时正在与房玄龄、李靖等人讨论各大军区长官换防的事情，一时间没有反应过来长乐到底要做什么，愣了半天才问道：“怎么了这是？干嘛生这么大气？”
<br/>　　“我怎么能不生气，这么大的事情，你只给了我十万贯，你觉得这点钱能干什么？连修场地的钱都不够！”长乐一脸的郁闷，一副你不给钱我就要哭的表情。
<br/>　　“这么大的事情只有十万贯，这钱好像是少了些。”房玄龄坐在一边，点了点头，对长乐表示支持，作为数朝元老，老房的确有说这话的资格。
<br/>　　“房伯伯说的是呢，我哥他就是抠，而且这十万贯还不单单是这次长安杯的钱，而是长安杯和华夏杯两个大型比赛全部的钱。”长乐见有人支持自己，立刻来了精神，把自己面临的困难一股脑的说了一遍。

<br/>　　“这怎么可能。”长乐一而再，再而三的听李承乾强调这件事情的可行性，不知不觉也开始怀疑自己是不是真的做错了。
<br/>　　“怎么不可能，正所谓世间万物，一通百通，前几天我在早朝上说的那些你应该已经知道了吧？一次华夏杯的利润远在五万万贯之上，可为什么到了你的手里却是赔钱呢？”李承乾无奈的说道。

<br/>　　“明白了就快点去办事，尽快把这件事情操办起来，别再拖了。”李承乾笑着对长乐摆了摆手，示意她可以离开了。

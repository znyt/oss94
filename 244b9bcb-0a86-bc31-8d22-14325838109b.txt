　　“正好赶上幻影旅团集合吗？”苏重眉头微皱。

<br/>　　要是被这些人缠上，确实有些麻烦。而且，团长库洛洛能力特殊，叫做盗贼的极艺，能够窃取他人念能力。谁知道他掌握了多少诡异能力？
<br/>　　先下手为强！
<br/>　　轰！
<br/>　　苏重身影瞬间消失在原地，眨眼间就来到新长身边。
<br/>　　信长眼睛陡然一缩，好快的速度！
<br/>　　几乎是下意识的，右手猛然一紧，长刀骤然出鞘。
<br/>　　锵！
<br/>　　拔刀斩！
<br/>　　嗤！
<br/>　　一道半月状白色剑气骤然飞射而出。
<br/>　　砰！

<br/>　　吱吱吱！
<br/>　　剑气不断切割念力网，却始终拿念力网没办法。
<br/>　　“拔刀斩吗？借助刀鞘作用，提高出刀速度。同时把念力化作半月剑气飞出，用作远程攻击。不错的能力，可惜还不够。”
<br/>　　苏重五指微微曲奇，武道念力线从手指吐出长长垂下。右手猛然一挥。
<br/>　　猫刃！
<br/>　　铛！
<br/>　　一声脆响，半月剑气瞬间被猫刃切碎，化作点点荧光消失不见？
<br/>　　“不是强化系？”信长心脏猛然一条。听窝金说，苏重力量强横，他还以为，对方和窝金一样，走的是强化身体的路子。没想到，竟然使用出了念力线。这是操控系，还是放出系？
<br/>　　不等他想明白，苏重已然欺身而进。右手朝着信长狠狠抓了过去。
<br/>　　猫刃！
<br/>　　锵锵锵！
<br/>　　顷刻间火花四溅。只是刚刚触碰，信长就心中一震。看似柔软的线条，竟然那意外的坚韧？自己斩击有多么强大，就是钢铁也能轻易斩断。竟无法砍断几根小小念线？！
<br/>　　嗤！
<br/>　　只是略微失神，一道念线突破信长长刀防御，朝着他抽了过来。
<br/>　　脸色一变，地面泥土炸开，身体猛然后跃。接连三个起落，退出数十米。斜眼一看，肩膀上无声无息出现一道口子，血液狂涌，瞬间染红衣衫。切口整齐，好似被神兵利刃击中一般。
<br/>　　“竟然那么锋利！”信长瞳孔骤然一缩。这种伤口他太熟悉了。被他用剑斩杀的人，留下的就是这样的伤口！
<br/>　　“刚才后退，只要慢上一个呼吸，自己这条胳膊就被砍断啦！”
<br/>　　“这么敏锐？”苏重眼中闪过一道讶然。念线只是刚刚进入对方身周，竟立刻被发现，及时撤退才免了断臂之苦。
<br/>　　信长的圆只能覆盖身周两米，范围并不大。却格外敏锐，就是一只蚊子奋进这范围，也会顷刻被发现砍死。
<br/>　　“可惜，能发现，如果跟不上，也是白搭！”
<br/>　　砰！
<br/>　　地面猛然炸开，苏重身影瞬间出现在信长勉强。双手举起，朝着信长狠狠挥击。
<br/>　　双猫刃！
<br/>　　双手瞬间化作一片虚影，无数线条狠狠斩向信长。
<br/>　　锵锵锵！
<br/>　　火星四射，信长几乎使劲了全身力气。身上汗水不断流出，衣服眨眼间就被汗湿。好像刚刚洗过一般！
<br/>　　不断出现的念线斩击，逼着信长精神高度集中，不敢有哪怕一丝一毫的大意！这几乎是他练剑以来，出刀最快的一次！
<br/>　　嗤嗤嗤！
<br/>　　苏重身形一晃，推开数步。信长却呆呆站在原地，身上骤然爆发出十多道血光。脸色几乎瞬间就苍白下去。
<br/>　　苏重脸色漠然，右手成爪，自右向左，斜斜挥下！五道透明念力线割破空气，发出嘶嘶鸣叫。
<br/>　　信长一脸绝望，依照念线的锋利，自己一旦被切中，必死无疑！
<br/>　　砰砰砰！
<br/>　　密密麻麻的念弹骤然从我进背后冒出，划着弧线，好似一群闻到了血腥味的食人鱼，凶猛的冲向苏重光。
<br/>　　叮叮叮！
<br/>　　蛛网猛然出现在身前，铺天盖地的念线瞬间就将苏重覆盖。
<br/>　　嗤！
<br/>　　蛛网整个被推的向后退去。苏重被裹挟后退，竟在地上拖出两道长长的痕迹？
<br/>　　“窝金、信长，你们可真狼狈啊！”侠客跳到泥坑边上，蹲在那里笑嘻嘻的看着窝金。
<br/>　　“你的刀练到狗身上去了吗？信长。”侠客双手插兜，慢悠悠走到信长旁边。
<br/>　　砰！
<br/>　　信长心神一松，整个人向后倒去，一下子就砸在了满是鲜血的地面上。张开口想说什么话反击，却眼皮沉重不一会儿就昏迷了过去。
<br/>　　刚才那短暂的交锋，太过激烈，他的精神高集中，实在是太累了。加上失血过多，看到同伴心神放松，竟然直接昏迷了过去。
<br/>　　“都来了吗？”苏重眼睛眯起，左右打量。
<br/>　　库洛洛额头十字太过显眼，苏重一眼就看认了出来。体型庞大，表情木讷的富兰克林也很好辨认。
<br/>　　看对方十指断开，露出黑洞洞中空手指，好似枪管。刚才如雨般念弹，就是对方发出来的。

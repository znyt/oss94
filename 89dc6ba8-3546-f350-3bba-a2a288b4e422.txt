　　当场摇了摇头，道:“算了，你跟我弄一些火把吧，我教你怎么做。”
<br/>　　叶少阳顺带找他要了不少法药，虽然自己包里还剩下不少，不过毕竟不是自己所在那个世界，他人生地不熟的，很多东西不知道上哪里去买。这个时候，他才意识到老郭有多重要，以往有老郭在，什么都不要操心，缺什么就要什么，老郭也总能第一时间送到，现在，什么都要靠自己了。
<br/>　　阿兰之前住的那个小屋，叶少阳已经让人收拾干净，开窗透气，之后又把阿兰抬了进去。让阿兰仍旧住在这里，一来是为了让腊梅回来还能容易找到她，二来这里是宅院里最偏僻的地方，门外还有个小院子，方便作法。
<br/>　　叶少阳让秋镇长找十几个男子过来，家丁不够，秋镇长只是一个镇长，家里没养这么多仙人，就从治安所调了几个人过来，叶少阳跟他们吩咐了一番到时候怎么做，自己在院子里布置起来。
<br/>　　秋镇长跟在后面，看着叶少阳把阴阳镜挂在门上，有点不放心地说道：“叶先生，你把这东西挂在这，腊梅来了，看见这个，不会怀疑吗？”
<br/>　　叶少阳笑道：“这个你就放心了，邪物眼里见不到法器，在触发之前，真正的法器她是看不见的。”
<br/>　　九段光的法器，一个幽魂绝对看不出问题。
<br/>　　还有一些东西，秋镇长已经让人去准备了。这会儿闲着没事，阿兰被送进房间里，在床上睡着了，秋镇长让人送了一壶茶过来，三人坐在台阶上一边喝茶一边等着。
<br/>　　“叶先生，今天晚上……不是我信任你，我是想知道，你是有把握的吧？”
<br/>　　“当然，一个幽魂而已。”叶少阳叹了口气，看着自己在院子里布置的这些，心中不由苦笑：对付一个幽魂而已，自己什么时候需要布置这些东西了？
<br/>　　上一次这么大张旗鼓地对付一只幽魂，叶少阳自己都记不住是什么时候了，估计至少是十年以前了。
<br/>　　没想到现在一切又回头了。
<br/>　　不过，反过来看，这件事也反映了人间法师的强大：
<br/>　　人间法师，因为有肉身的束缚，不能飞天遁地，不能刀枪不入，受伤了会疼，伤重了会死，与鬼妖尸灵相比，这些都是硬伤，不过，人间法师最大的优势，就是会借助外力：法器、法药，各种阵法，道家对于阴阳二气的掌控、佛家对佛光真言的运用，说白了就是从天地万物中汲取一切可以用到的力量，让自己能以羸弱之躯，抗衡比自己强大的多的各种邪物。
<br/>　　叶少阳从前也认识到这一点，但是如今，这种感觉更加的强烈，并且让他有了一种新的感悟：平衡。
<br/>　　法师与邪物之间，是一种平衡的关系，这种平衡一直维持到现在，从来没有一方能够压倒另一方，这其实也是天地大道的一种表现。其实不光是法术界和邪物，社会的各方面，正与邪、黑与白，一切能够长久维持的对立关系，归根究底都是一种平衡。
<br/>　　正是因为有这种平衡，这个世界才能够一直维持下来。
<br/>　　想到这里，叶少阳仿佛突然开悟，对于道家的阴阳辩证的关系，又多了一层理解。
<br/>　　他正胡思乱想着这些玄乎的东西，秋镇长突然叹了口气，说道：“叶先生，我有个小小的请求。如果不是到了最后一步，我想请叶先生留下腊梅的性命。”
<br/>　　叶少阳笑道：“她都已经死了，哪还有什么性命可言。”
<br/>　　“啊，我的意思是，超度之类的？我不懂这个，那个……鬼如果死了会怎么样？”
<br/>　　“人死为鬼，鬼死为精魄，你一个普通人，还是不要知道这么多了。”叶少阳道，“我白天都没有杀她，今天如果她来，我当然没理由再杀她了。”
<br/>　　秋镇长谢过，叹道：“其实腊梅也挺可怜的，她生前的经历……你也是知道了的，是个苦命人。”
<br/>　　叶少阳道：“要不是看在这一点，我也不会跟她客气的了。”
<br/>　　秋镇长道：“她的可怜，还不止于此……”
<br/>　　话刚说到这里，屋里一个老妈子走出来，告诉秋镇长，姨太太醒了，正在找他。
<br/>　　秋镇长喜出望外，朝叶少阳看去。
<br/>　　叶少阳让他自己进去。
<br/>　　秋镇长进去了好一会，然后出来告诉叶少阳，阿兰请他进去。
<br/>　　叶少阳带着翠云一起进去。
<br/>　　阿兰躺在床上，老妈子已经帮她擦洗过了身子，脸上的油彩也被洗干净了，看上去是个细皮嫩肉的美女，见到叶少阳，他让老妈子扶自己坐起来，对叶少阳道谢。刚才秋镇长在屋里把一切都告诉她了。
<br/>　　叶少阳道：“你不用客气，也别害怕，就在这里躺着就行，今晚只要她来，我就一定能收了她。”
<br/>　　阿兰说道：“叶先生，希望你抓到腊梅之后，能让我跟她说说话，我想告诉她一些真相。”
<br/>　　叶少阳点点头，也没有细问，让她继续休息，跟大伙一起出去了。
<br/>　　晚饭时候，秋镇长张罗了一桌丰盛的晚宴，请叶少阳和翠云吃饭。他的太太也出来打了个招呼，虽然看上去珠光宝气的，不过这位太太人还不错，听说了下午的事情之后，特意来跟叶少阳说了几句客气话，然后留下秋镇长作陪，自己先回房间了。
<br/>　　因为惦记着晚上的事，秋镇长没什么心情喝酒，叶少阳也不喜欢喝酒，倒是对一桌饭菜大感兴趣，他穿越过来也有几天了，平时在翠云家都是粗茶淡饭，眼前对着一桌子鸡鸭鱼肉，也是食指大动。
<br/>　　翠云平时也没吃过这么好的菜，等太太一走，跟叶少阳一起狼吞虎咽起来，也顾不上什么形象了。
<br/>　　吃完饭，叶少阳让翠云先歇着，自己跟秋镇长来到阿兰居住那个小院，一切都布置妥当了。叶少阳画了两张隐气符，跟秋镇长在花园里找了个角落躲起来。
<br/>　　叶少阳不确定腊梅今晚一定会来，如果她来，那就说明她背后一定有靠山撑着，想要前来害死阿兰，一了百了。
<br/>　　（少阳回到民国，并不是重新开始的故事，与过去会大有相关，大家别着急，真相很快就会显现）

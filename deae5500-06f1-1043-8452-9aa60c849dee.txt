　　梦魔战魂构筑的梦境，一切都是虚构的，一切都是由苏莫创造的。
<br/>　　因此，梦境中的一切，都是完全不存在的。
<br/>　　但是，巫人王却是不知道，就算知道他也无法破除，他只感觉自己身上的血液流尽了，全身提不起半点力量，虚弱到了极致。
<br/>　　“你……你只不过是用卑鄙的手段而已，有种放我出去，我们公平一战！”巫人王声音孱弱的道。
<br/>　　“公平？”
<br/>　　苏莫闻言嗤笑一声，不屑的说道：“没有什么是绝对的公平，结果代表了一切。”
<br/>　　咔嚓！
<br/>　　话音一落，寒光一闪，苏莫手起刀落，直接将巫人王的一条手臂骨骼斩了下来。
<br/>　　啊！
<br/>　　巫人王口中顿时发出凄惨的嚎叫，剧烈的疼痛深入骨髓，痛彻心扉。
<br/>　　咔嚓！
<br/>　　又是一声脆响，巫人王的一条腿骨同样为斩了下来。
<br/>　　啊！
<br/>　　巫人王再次痛苦的嚎叫，就算是他修为再高，也忍受不住这种剧烈的疼痛，整个人都抽搐了起来。
<br/>　　“巫尘，你不得好死！”巫人王怒吼，他的气息却是越来越弱。
<br/>　　不过，巫人王也是顽强，到了现在精神力依旧在拼命的挣扎，想要挣脱梦魔空间的束缚。
<br/>　　苏莫的脸色也稍稍发白，他要靠精神力和梦魔战魂的力量，极力稳固梦魔空间的稳固。
<br/>　　这对于他的精神力，也是巨大的消耗，对方挣扎的越厉害，他的消耗也越大。
<br/>　　“巫人王，你的这骨骼和血肉，你说我是直接将其毁灭，还是拿去喂狗呢？“苏莫微笑着问道。
<br/>　　“你……可恶啊！”巫人王闻听此言，顿时气的全身颤抖了起来。
<br/>　　“这血肉这么恶心，喂狗的话真是脏了狗的嘴，还是毁灭了吧！”苏莫摇了摇头，随即，手中魔力吞吐，瞬间便将那堆血肉和骨骼毁灭的一干二净。
<br/>　　随即，苏莫手中光芒一闪，出现了一根长鞭。
<br/>　　这长鞭极为惊人，其上布满了一根根锋利的倒刺，闪烁着森冷的寒光。
<br/>　　“你想干什么？”巫人王见此，顿时恐惧了起来。
<br/>　　苏莫脸色冷漠，也不言语，举起长鞭便向巫人王劈头盖脸的抽打了起来。
<br/>　　啪！啪！啪！啪！！
<br/>　　一声声清脆的响声，在幽暗的囚室中不断的响起，伴随着巫人王凄厉的惨叫。
<br/>　　啊啊啊啊！！
<br/>　　巫人王的口中哀嚎不断，但苏莫完全置若罔闻，不停的抽打。
<br/>　　时间缓缓流逝，一个时辰又一个时辰。
<br/>　　三个时辰之后，巫人王已经不成人形，全身骨骼被抽的碎裂了，内脏流了一地，模样惨不忍睹。
<br/>　　这一幕无比的血腥，无比的凄惨！
<br/>　　巫人王已经停止了惨叫，整个人已经有些迷糊了，三个时辰的折磨，他的意志和精神，几乎要崩溃了。
<br/>　　“巫人王，振作起来啊！我才抽了五千鞭子，还差五千鞭子才到一万之数呢！”苏莫大声呼喊道。
<br/>　　“巫……巫尘，你杀了……杀了我吧！”巫人王声音虚弱无比，微不可查。
<br/>　　“杀了你？哪有这么容易！”
<br/>　　苏莫脸色郑重的摇了摇头，如是说道：“我准备折磨你十天十夜，不然也太无趣了吧！”
<br/>　　言罢，苏莫继续抽打，更加的用力。
<br/>　　这梦魔空间完全虚构，连时间都是假的，他就算在梦魔空间中待上十天，外界也就是片刻的功夫而已。
<br/>　　同时，苏莫心中暗暗盘算，能不能将对方的战魂和血脉吞噬，这对于他来说并不难。
<br/>　　但是，想要瞒过外界的大祭司等巫族高层，怕是不太可能。
<br/>　　若是为了吞噬巫人王，而让她暴露出去，那就太不值了。
<br/>　　相比于吞噬对方，直接杀死对方则更容易一些。
<br/>　　……
<br/>　　外界，混天魔域之中。
<br/>　　苏莫身形不动，面色稍稍有些苍白。
<br/>　　而在他对面的巫人王，脸色灰暗无比，双眸神光涣散。
<br/>　　不仅如此，其身体仿佛已经失去了控制，慢慢飘荡了起来，随着魔气的翻涌而不断的沉浮。
<br/>　　“巫人王败了！”众人看到此情此景，心中不免震动不已。
<br/>　　同阶无敌的巫人王，居然就这么轻易的便败了，这巫尘的实力不仅强大，而且诡异！
<br/>　　“大祭司，不会出问题吧？”四祭司皱眉问道，无论是巫尘还是巫人王，都是一等一的天才，一个都不能有事。
<br/>　　“再等等吧！”大祭司说道，他修为虽然高深莫测，但也无法探知苏莫和巫人王的战斗情况。
<br/>　　不过，他能掌控两人的战斗结局，保证不会有人陨落。
<br/>　　混天魔域之中。
<br/>　　突然，苏莫眼眸中恢复了清明，眸中闪过一道冷光。
<br/>　　巫人王的意志和精神，已经完全被他击垮，所以他收回了梦魔空间。
<br/>　　唰！
<br/>　　瞥了一眼已经如尸体般的巫人王，他大手一探，直接将对方抓了过来，一把扣在了对方的脖子上，直接将对方的脖颈捏的稀烂。
<br/>　　巫人王并没有死，只不过是意志跨了，精神处于严重的絮乱状态，过段时间便能恢复。
<br/>　　“住手！”就在苏莫准备捏死对方的时候，一声轻喝突然响起。
<br/>　　一股浩瀚无比的气机顿时压在了他的身上，让得他全身剧震，身体瞬间凝固，混天魔域也随之消散。
<br/>　　唰！
<br/>　　大祭司飞了过来，出现在苏莫的身边。
<br/>　　瞥了巫人王一眼，大祭司对苏莫道：“巫尘，你已经胜了，此战就此结束吧！”
<br/>　　“大祭司，巫人王之前口口声声要杀我，难道我就这样放了他？”苏莫面带不甘之色，不过他早知道大祭司会出手阻拦。
<br/>　　在苏莫说话之间，他微微催动了吞噬战魂，将吞噬之力控制成了一道细微的丝线，从他的手臂内穿过，进入了巫人王的体内。
<br/>　　他，要吞噬对方的血脉。
<br/>　　虽然这样吞噬的有限，但能吞噬多少是多少。
<br/>　　“只是一时气话而已，巫尘你无需当真！”
<br/>　　大祭司摇了摇头，笑道：“若是你败了，老夫也不会让你有事；你们两人都是我族的天才子弟，任何一人陨落，都是我族的损失！”
<br/>　　“可是，他的修为这么高，若是事后找我麻烦怎么办？”苏莫皱眉问道，他只是在故意拖延时间。
<br/>　　他正在吞噬巫人王精血中的血脉之力，只需几个呼吸之间，就能将对方的血脉之力吞噬的一干二净。

<br/>　　巫人王的特殊体质，好像叫什么天玄武体，苏莫不知道这种体质有何厉害。
<br/>　　但对方之前战力无双，强大的有些过分，绝对不单单是战魂的力量，肯定借动用了体质的效果，不然的话，绝对不可能这么厉害。
<br/>　　所以，苏莫对于这种体质有些眼馋，现在壮着胆子偷偷的吞噬了起来。

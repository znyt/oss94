　　因为年代久远，时过迁境，加上那个时代被打下了战争的烙印，很多陈年鬼事，被遗忘在了历史的角落。
<br/>　　然而那个时代，是一个灵异事件频发的时代。
<br/>　　那些被占了道观寺庙的和尚道士，不管有没有捉鬼降妖的本事，多数都打着法师的招牌，行走江湖，总比要饭的好。
<br/>　　不过风险还是很大的，万一遇到对付不了的恶鬼厉妖，一个不小心就交代了。
<br/>　　然而，真有本事的法师，还是有的。而且当时，唯物主义在华夏还未扎根，民众对于鬼神之事，还是笃信不疑，对待法师的态度也是极为崇拜，这其中包括很多军阀权贵，为了做一场法事，不惜一掷千金的事毫不稀奇。
<br/>　　捉不完的鬼，收不完的妖，从某种意义上来说，这个混乱的时代，也是法师的黄金时代。
<br/>　　………
<br/>　　漂浮在一片火红色的汪洋大海之中，叶少阳的意识是模糊的，随着海浪上上下下，缓缓前行。
<br/>　　在遥远的天际，出现了一道道的暗影，看上去好像是一座山，隐藏在云层中间。
<br/>　　没等叶少阳仔细看个清楚，一个熟悉的声音在身后响起：“少阳！”
<br/>　　叶少阳悚然一惊，回头看去，是芮冷玉，穿着刚见面时候她穿过的那身紧身皮衣，长发飘飘，英姿飒爽，看上去像个女特工，站在海水中间，缓缓地沉了下去，只露出脑袋和一双手。
<br/>　　“少阳，救我……”芮冷玉吃力地说道，眼神中充满了希望，对叶少阳求救。
<br/>　　叶少阳立刻游了过去，就在这时，自己的左侧也传来了另一个熟悉的声音：“少阳……”
<br/>　　转头看去，是小九，跟芮冷玉一样，她也漂浮在水中，不停地挣扎着，她的眼中只有绝望，也没有向叶少阳求救，只说了一句话：“去救冷玉……”
<br/>　　叶少阳左右看看，两个人都是命悬一线，都要沉下去，但是两人之间却隔着一段距离，叶少阳冥冥中有一种强烈的感觉：只能救一个人！
<br/>　　救冷玉，还是小九？
<br/>　　叶少阳陷入了痛苦的挣扎之中。这时候，他看到一道黑色的巨浪，从远处奔袭而来，按照这个速度，最多也就十秒钟之内一定会砸过来。
<br/>　　必须要救一个！
<br/>　　这个念头在叶少阳心底浮现，想要狠下心随便去救一个，至少不能让她们两个都死，救一个……可是，救谁？
<br/>　　在这万难的抉择之中，叶少阳的内心始终在挣扎中煎熬，没有办法做出决定。他没法接受救一个就要牺牲另一个。巨浪拍击过来，将芮冷玉和小九一起淹没……
<br/>　　“不！！”
<br/>　　叶少阳大声喊叫着，猛地坐了起来，睁开了眼睛。
<br/>　　一切都消失了。没有海浪，也没有芮冷玉和小九。叶少阳开明白，自己是做了一个梦。
<br/>　　他剧烈地喘一会，左右看去，发现自己在一个完全陌生的房间里，这个房间挺破的，没灯，叶少阳黑暗中打量起来：一个书桌，一架木柜，还有自己睡的这张床，都是木头的，看上去很古老的感觉。
<br/>　　这是什么地方？自己为什么在这？
<br/>　　叶少阳一下就懵了，昏沉沉的大脑中，记忆开始恢复，他想起了空界之战，想起了最后一幕：自己用一身法力敲响了东皇钟，之后浑身虚脱，好像就昏过去了，没错，那是自己最后的记忆。可是……自己现在在哪？
<br/>　　叶少阳想来想去，觉得自己最大的可能是被人给救了，毕竟自己那些小伙伴当时都在身边，他们当然不可能放着自己不管。
<br/>　　叶少阳觉得，自己最先应该确认的，是这里究竟是哪个空间——是空界、鬼域，还是人间？
<br/>　　叶少阳撑了一下身子，想要坐起来，结果肋下撕裂一般的疼痛，让他闷哼一声倒在床上，翻来覆去半天也没缓过去。
<br/>　　在疼痛中，他想到了这是芮冷玉那几根手指留下的伤口：在空界受伤，就算投影回到人间，肉的伤口依然会存在。也正是基于这一点，叶少阳判断出这里是人间，因为这种撕裂的一跳一跳的疼痛，只有肉身才有。
<br/>　　“喂，有人吗？”叶少阳躺了半天，感觉上好了一些，忍不住大声喊了一声，他想的是不管这里是哪，小伙伴们不可能把自己这个伤员一个人丢在这，一定就在附近。
<br/>　　果然，门吱呀一声被推开了。叶少阳趴在床上，听见脚步声有两个，他疼的抬不起头，问道：“是谁？”
<br/>　　“哎呀，你表弟终于醒了，阿弥陀佛，菩萨保佑，我看他伤那么重，还以为他活不过来了呢，呸呸，醒来就好，醒来就好。”
<br/>　　一个尖嗓门的女声喋喋不休的说道，带着浓浓北方话的口音，叶少阳倒也能听懂，心中疑惑起来：表弟？什么鬼？
<br/>　　在巨大的好奇心驱使下，叶少阳强撑着翻过身来，抬头看去，先看到了一盏油灯，照着两个人的眼睛，一个是老太太，长得像叶少阳小时候在红楼梦电视剧看过的那个刘姥姥，一张嘴一口大黑牙。
<br/>　　另一个是看上去三十来岁的女子，长的倒还可以，左边眉毛上有一颗美人痣，脸上也有不少雀斑，不过整体看上去还算是美女。
<br/>　　叶少阳注意的不是她的容貌，而是这两个女子的打扮：两人都穿着淡色的对襟的那种布衣，头发往后梳，盘起来。这身打扮……叶少阳没法形容，怎么说来着，反正就是电影里经常见到的那种过去的贫苦人家的女人打扮。
<br/>　　女鬼？
<br/>　　叶少阳本能的第一反应，但是一看又不是。
<br/>　　两个女人笑吟吟地盯着他，像刘姥姥的那个老女人又开口了：“后生，你怎么样？”
<br/>　　“你们……这是……你们是谁？”叶少阳喃喃问道。
<br/>　　老女子愣了一下，朝那个略有风韵的年轻女人看去。
<br/>　　年轻女人嗔了叶少阳一眼，对老女人道：“他八成发烧烧糊涂了，又昏睡了几天，连我都不认识了。”转头对叶少阳道：“我是你表姐啊。”
<br/>　　表……姐？

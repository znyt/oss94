　　第652章
<br/>　　阿喜派出许多人手，分布帝国各地，刺探各路诸侯的情报。但帝国太大了，通讯手段又落后，阿喜虽然加力在做这件事，但许多地方还是情报的空白地带。
<br/>　　但长安城不一样，长安是帝国权力中心，斗争尤为激烈。阿喜早就派出去人手，收集朝廷的情报。于是，蛤蟆城能在第一时间收到尉迟离的死讯。
<br/>　　当然，不仅仅是蛤蟆城，帝国两帝十八王再加上一些排不上名号的势力，怕都会将眼睛安插在长安，关注这里的一举一动。
<br/>　　和珅闻讯过来，将这个任务领了，他会书一份挽联，当然是以程大雷的名义，然后让长安城的弟兄送到尉迟离的葬礼上。
<br/>　　“大当家，他们该不会以为咱们是幸灾乐祸，故意示威吧？”
<br/>　　“呃……这怎么会，咱们和将军府又没恩怨，关键是提醒相府注意些。”程大雷顿了顿道。
<br/>　　和珅点了点头，忽然感慨一句：“尉迟离这一死，怕长安就要乱了。”
<br/>　　程大雷心中也明白，尉迟离死后，长安的权力天平将会失衡，相府当然会借此扩大自己的势力，原先归附将军府的世家豪族也该做出他们的选择。而在这场乱局中，李乐天将何去何从……
<br/>　　当然，长安城就算打得稀巴烂，程大雷也不会太关心。关键是长安城的乱局，会影响到凉州的形势。
<br/>　　原先凉州是三足鼎立，谁也奈何不得谁，但现在是程大雷与宋伯康两家对峙，二人之间迟早会有一战，或早或晚。
<br/>　　程大雷希望这个时间来得稍晚一些，可以留给自己发展的机会，如非必要，程大雷是不愿意与宋伯康动手的，互相打来打去，也是白白消耗兵力粮草。
<br/>　　长安一乱，相府必然集中注意力在长安，如此，倒也给了自己发展喘息的机会。
<br/>　　据程大雷得到的消息，宋伯康已经在京州三水关前屯兵，狼子野心十分明显。
<br/>　　程大雷问和珅：“现在招募人口的事情怎样？”
<br/>　　“比原先的情况好一些，陆陆续续有人从外地迁过来，目前朔方城的人口还不到两万人。”说起这件事，和珅也有些着急，毕竟没有人口，其他什么事情都操办不起来。
<br/>　　程大雷同样没有什么太好的办法，毕竟该做的都已经做了，目前似乎只有等待。可这样不知等到什么时候，自己方才可以完成任务。
<br/>　　“朔方城修得如何，几座破房子，该不需要修这么久吧？”程大雷问道，他接了两个任务，招募人口的任务急不得，可修整城池应该用不了太长时间。
<br/>　　“这件事已快完成了，前些日子耕田播种浪费了些时间，眼下腾出人手来，马上就可以完工，也就在这一两天。”
<br/>　　在建筑精通的加持下，这件事当是快的。程大雷现在也只能等着，看完成这个任务后，会得到怎样的奖励。
<br/>　　在和珅与阿喜离开后，程大雷继续翻着案上的情报。一件件一桩桩纷乱无序，类似于百里胜闯出三水关，去向不明，三水关守将负伤之类的事情。
<br/>　　之后，桌上一份刚刚送来的情报引起程大雷的注意：关于戎族的……
<br/>　　阿喜收集各路诸侯的消息，但关于戎族是最难收集的。原因，程大雷自己也亲身体味过。戎族居无定所，互相间打来打去，也没办法打听到确定的消息。
<br/>　　距离程大雷从戎族返回蛤蟆城，已过去半年时间。这半年内，草原上又发生一些事情。
<br/>　　野原火拔地而起，占据了戎族王城，之后，便以这座城池为中心，向外扩张自己的势力。
<br/>　　戎族间你打我我打你，也不算什么新鲜事，偶尔某一天互相不打了才算奇怪。但这次野原火的扩张势力，明显和以往有些区别。
<br/>　　他这次给程大雷的感觉，是要一统戎族诸部落。
<br/>　　戎族数百个部落，有大有小，大的上万人，小的几百人。他们虽然生活方式差不太多，却有不同的信仰，分布在不同地区，想要将他们统一在一起，以前草原也不是没有人想这么干过，但几乎无人可以做到。
<br/>　　更值得人深思的是：他统一戎族之后，接下来要做什么？
<br/>　　程大雷叹了口气，手指在桌案上敲了敲。人无远虑，必有近忧，现在远虑和近忧都有了。
<br/>　　看起来，这世道不会一直这么糟糕的，而是会越来越糟糕。
<br/>　　嘟，修整城池任务完成，奖励宝藏一份。
<br/>　　程大雷还未读完系统提示，便有一人着急跑进来：“大当家，喜事，喜事呐。”
<br/>　　“什么喜事？”
<br/>　　“我们修屋的时候，挖到一个地窖，里面好像有东西。和总管让我请大当家去看看。”
<br/>　　程大雷听罢便明白，是系统奖励的宝藏找到了。不过，这份奖励当真有些水，程大雷现在最不缺的便是金银珠宝。
<br/>　　关键是即便有这些，也花不出去，如果到江南，还可以找那些世家豪族换换一些粮食兵器什么的。但程大雷如果真的到了江南，够得上这些大家族，那么他就直接动手抢了，也不需要拿金银财宝之类的换。
<br/>　　不过，既然找到这些，便去看看，或许会有什么稀罕玩意。
<br/>　　这已经是程大雷对这批宝藏唯一的期待了。
<br/>　　嘟，奖励临时技能：超级声望广播，持续时间三天。
<br/>　　是否使用？
<br/>　　咦！
<br/>　　程大雷迈出去的脚步停住，原来这次系统不仅一项奖励。超级声望广播？这究竟是什么东西？
<br/>　　超级声望广播：临时技能，持续时间三天。三天内，你的名声将传遍帝国，无人不知，无人不晓。
<br/>　　程大雷算是明白了这个技能代表什么，简单来说，也就是可以在三天内扩大自己的声望。让本来已经臭名远扬的程大雷，变得更加臭名远扬。
<br/>　　恍惚，是没多大用处。
<br/>　　系统一遍遍询问他是否使用，程大雷暂且摁了下来，先随着人去看看刚发现的那批宝藏。
<br/>　　那宝藏在一间大宅子下，也因为掘土修墙才会被发现。程大雷到的时候，周围已经挤满了人。
<br/>(本章完)

　　唰！
<br/>　　人影一闪，苏莫的身形出现在了第三层空间之中。
<br/>　　这层空间更加的辽阔，一眼望不到尽头，仿佛是一处位面小世界一般。
<br/>　　只不过，略显虚空的空间，显得不太真实。
<br/>　　而这层空间之中，巫凌的投影依旧在等待了，其伫立于高空之上，双眸紧闭。
<br/>　　嗖！
<br/>　　苏莫飞身上前，与对方相隔百里而立。
<br/>　　“能闯到第三层，你很不错！”巫凌的投影睁开了眼眸，目光凝视着苏莫，淡淡的开口道。
<br/>　　苏莫没有理会对方，一个没有多少意识的投影，没有必要理会。
<br/>　　他身上魔力涌动，魔气滚滚，整个人伫立在一片黑雾之中，仿佛化生成了魔头。
<br/>　　“你的魔功不简单，想来应该是尊级上品吧！”巫凌投影面色稍稍凝重了起来，再次开口说道。
<br/>　　“嗯？”苏莫闻言一怔，随即仔细打量了一番对面的投影，面带惊疑之色，他发现这具投影与之前的明显不同了。
<br/>　　此投影虽然依旧是武帝境三重的境界，但身上充满了灵性，拥有很高的自主意识，不再是单纯的投影了。
<br/>　　“看来你也不简单！”苏莫嘴角勾起了一丝笑容，他的魔功可是高达圣级上品，只不过修炼的火候太低，才第一层境界。
<br/>　　“想要打败我，就拿出你全部的实力吧！”巫凌投影淡淡的说了一句，身上的气势立刻爆开了。
<br/>　　一股浩大，伟岸的气息冲天而起，搅动空间，威临八荒。
<br/>　　“七成实力而已！”苏莫面色淡定，丝毫无惧，五成实力的投影都接不住他全力的一招，七成实力又能强到哪里去！
<br/>　　即便对方拥有很高的自主意识，但实力终究是根本。
<br/>　　“败！”
<br/>　　就在此时，巫凌的投影率先出手了，其身形冲天而起，居高临下，一掌镇杀而下。
<br/>　　霎时之间，一道巨大的玄力掌印，从天而降，向苏莫当头砸下。
<br/>　　这掌印如同一座宫殿般大小，栩栩如生，其上的纹路都清晰可见，澎湃的掌劲如同喷发的火山，汹涌浩荡，威不可挡。
<br/>　　“就拿你，试试混天魔域的威力！”苏莫轻笑一声，身形不退反进，直接向着巨掌迎了上去。
<br/>　　与此同时，苏莫立刻施展了浑天魔域，滚滚魔气从他身上汹涌而出，如同滔天骇浪，席卷八方。
<br/>　　无尽的魔气形成了一片领域，化为了一片漆黑的独立世界。
<br/>　　轰！
<br/>　　巨大的掌印镇压而下，直接轰进了混天魔域之中，向苏莫狠狠的袭去。
<br/>　　“灭！”苏莫身处混天魔域之中，轻喝一声，一剑逆斩而上。
<br/>　　一道惊天的黑色剑气冲天而起，裹挟无边的锋芒，直击巨大的掌印。
<br/>　　混天魔域之中，一股无形的气机，加持在苏莫的攻击之上，让得他的攻击威力倍增。
<br/>　　下一息，剑气重重的斩击在了巨大的掌印之上。
<br/>　　轰！
<br/>　　轰隆隆！
<br/>　　巨响连连，轰鸣震天，巨掌被斩碎，剑气同样崩溃。
<br/>　　苏莫身形不停，继续想巫凌的投影冲去，并且身上的魔威更加庞大，滚滚魔气浩荡，混天魔域急速扩张。
<br/>　　巫凌投影脸色微变，立刻身形急速暴退，他知道若是陷入这无比的魔气领域之中，将极为不妙。
<br/>　　不过，苏莫的速度不比他慢，再加上混天魔域的扩张速度，巫凌投影根本逃不出去。
<br/>　　几乎是瞬息之间，巫凌投影便被笼罩进了混天魔域之中。
<br/>　　混天魔域之内，仿佛自成一界，一片暗无天日，魔气滚滚的邪魔之地。
<br/>　　“杀！”苏莫厉喝一声，随即心念一动，二十八柄本命灵剑激.射而出，向巫凌投影绞杀而去。
<br/>　　现在苏莫也没有什么顾忌，首先现在是在宝塔之中，其次又是在混天魔域之中，外面的人不可能发现他动用灵剑阵。
<br/>　　而且，就算有人能发现，估计也只有大祭司和四祭司才能发现。
<br/>　　再者，这无尽星空之中，拥有灵剑阵的人，并不是只有他一人，苏莫可是听说了，本命灵剑阵在古灵星河，也是不少强大剑客喜欢的攻击手段。
<br/>　　只不过，因为精神力的原因，很多人的本命灵剑阵，威力并不是太强。
<br/>　　轰隆隆！！
<br/>　　本命灵剑阵化为浩大的剑气风暴，以毁灭之势，碾压一切。
<br/>　　这一次，本命灵剑阵上裹挟的剑气，不再是玄力，而是汹涌的魔力，魔光耀眼至极。
<br/>　　浩大的剑气风暴，好像是巨大的魔兽，张开血盆大口，欲将巫凌投影一口吞下。
<br/>　　“化灵掌！”面对本命灵剑阵的绞杀，巫凌投影怒喝一声，一边暴退，一边大手一张，一道虚幻的掌印向剑气风暴迎击而去。
<br/>　　这虚幻的掌印极为古怪，威势并不强大，却蕴含着无比玄妙的奥义。
<br/>　　嗤嗤嗤！！
<br/>　　瞬息之间，掌印击中剑气风暴，没有爆响声，更没有爆炸声，掌印直接融进了剑气风暴之中。
<br/>　　而后，掌印消失无踪，浩大的剑气风暴顿时威力大减，仿佛被融化了一般，威力十不存一。
<br/>　　随即巫凌投影再次出掌，又攻击出了一道凝实的掌印，瞬间便将剑气风暴击溃，二十八柄本命灵剑迸射八方。
<br/>　　巫凌投影，完美的破除了本命灵剑阵的攻击。
<br/>　　“的确是厉害啊！”苏莫眼眸闪亮，这个巫凌只有七成实力的投影，就有这般强大，那这一关能有多少人闯过去？
<br/>　　此人若是有十成实力，再加上战魂的力量，绝对是同阶无敌的存在了。
<br/>　　当然，苏莫也远没有动用最强的战力，他只是要试试混天魔域的威力而已。
<br/>　　浑天魔域，差不多能增加他两、三成的实力。
<br/>　　虽然只是两三成的实力，但对于他这个级别的武者而言，已经非常恐怖了。
<br/>　　实力到了他这个级别，以前的大三宝术，都已经没用了。
<br/>　　苏莫以前抽时间试验了一次，大三宝术抽取三成的精气神，连一成的实力都增加不了了。
<br/>　　“懒得和你浪费时间了，该结束了！”苏莫不想过多纠缠，准备结束战斗了。
<br/>　　下一刻，他身上剑威轰然爆发，凌厉无比的剑威冲天而起，形成威临八荒的剑势，剑之战魂、剑之武魂均是被苏莫催动了起来。
<br/>　　剑势，便是剑威强大到极致，形成的一种剑之大势，威力无匹，威慑八方。
<br/>　　耀眼的魔光闪烁，苏莫高举手中的长剑，瞬间连斩三剑。
<br/>　　三道浩大无匹的黑色剑气，裹挟汹涌的魔浪，封锁了巫烈投影的闪避空间，形成绝杀之势。
<br/>　　下一刻。
<br/>　　轰隆隆！！！
<br/>　　震天的爆响不断的响起，面对苏莫的绝杀，即便巫凌投影战斗意识再强，但受限于实力的限制，又没有战魂之助，根本挡不住苏莫的攻击。
<br/>　　一轮攻击过后，漫天魔气消散，巫凌投影直接被苏莫绞杀，化为了一片虚无。
<br/>　　呼！
<br/>　　苏莫轻呼一口气，收起所有灵剑，随即看向前方出现的楼梯，大步的走了过去。
<br/>　　三层已过，他要登上塔顶了！

　　他们三人，号称三仙，在这武胜国度，便是仙人一般的存在，高高在上的老祖宗，一方巨头。
<br/>　　如今楚风眠的话，听到他们耳朵中，简直是赤果果的羞辱。
<br/>　　其中一尊白发老者，三仙之一的风生子，更是满脸杀机，看向楚风眠怒声喝道。
<br/>　　“楚风眠，你到底要做什么？莫非是要跟我们三仙岛彻底为敌吗？你今日杀了我们三仙岛的岛主，今日必须要来给个交代才行！”
<br/>　　风生子一步上前，看了一眼楚风眠，又看了一眼下方的寒雪阁，冷哼开口道。
<br/>　　“今日的事，若是你不给个交代，无端端的敢来挑衅我们三仙岛，今日你跟这寒雪阁，都要灭亡！”
<br/>　　风生子开口，话语之中便是无尽优越，高高在上，看着楚风眠，好像是想要怎么捏死都可以。
<br/>　　他们毕竟可都是巅峰圣者，楚风眠的实力，他们也都曾经听说过。
<br/>　　但是却都丝毫不以为然，能够击败寻常圣者又如何？寻常圣者跟巅峰圣者之间也是有着无边无际的差距的。
<br/>　　今日在他看来，楚风眠便是一只随时可以捏死的蝼蚁，可是楚风眠展现出的自信，却是让他似乎察觉到有些不对。
<br/>　　不过很快风生子便是冷笑了起来。
<br/>　　再自信又如何？他不相信一个御风境的小辈，可以跟巅峰圣者来抗衡。
<br/>　　“给个交代？今日你们三仙岛的岛主来围攻寒雪阁，别说跟你们没有关系，杀了他，也是他自找的。”
<br/>　　楚风眠看着风生子，毫不在乎他言语中的威胁之意，大声说道。
<br/>　　“今日的事，他敢来围攻寒雪阁，别说跟你们三个老家伙没有关系。”
<br/>　　楚风眠瞥了一眼那风生子，冷笑开口道。
<br/>　　一个巅峰圣者，楚风眠丝毫不放在眼中。
<br/>　　死在楚风眠手中的巅峰圣者，已经不是一个两个了，多一个风生子也不多。
<br/>　　威胁楚风眠？就凭这风生子，还没有这个资格。
<br/>　　“围攻寒雪阁的事，我们的确知道不错，但是那是北邙学院的命令，难道你敢违背北邙学院的命令？”
<br/>　　风生子冷笑一声，直接给甩到了北邙学院那边去，一个大帽子扣了下来。
<br/>　　“但是你敢杀了我们三仙岛的岛主，便就是对我们三仙岛的挑衅。”
<br/>　　“今日你必须要给我们三仙岛赔礼道歉才行！”
<br/>　　这风生子的眼神中，露出了几分贪婪之色道，指着寒雪阁说道。
<br/>　　“你们寒雪阁，必须要交出五百弟子来，送给我们三仙岛作为奴隶才行！同时要交出你身上的无数奇遇，还有丰饶碑，统统交出来！这件事就算是一笔勾销了。”
<br/>　　“放肆！”
<br/>　　寒月漓在一旁，听到这风生子的话，都是勃然大怒了。
<br/>　　以她淡漠的性格，如今都被彻底的激怒了，让寒雪阁交出五百弟子来当做侍女，这件事就是对他们寒雪阁的侮辱。
<br/>　　更何况还要让楚风眠交出奇遇，交出丰饶碑来，这根本就是在威逼。
<br/>　　“哼，差点忘了，还有你，寒月漓，你也必须要自废修为，以你的命，来换寒雪阁！”
<br/>　　风生子看着寒月漓，眼神中尽是贪婪道。
<br/>　　“据说你是玄寒之体！先天体质！我们若是得到了你的先天之力，也就可以渡过生死大劫，踏入生死境，以你的命，来换寒雪阁！”
<br/>　　风生子看向楚风眠，镇定自若说道。
<br/>　　“楚风眠，你如今选吧，是乖乖送人出来，还是让我们杀了你，在毁了这寒雪阁，倒时候所有人都会是我们三仙岛的奴隶！”
<br/>　　“我选择第三个。”
<br/>　　楚风眠看着风生子，眼神中杀意浓烈，他的心中越是杀意浓烈，他的表情也就越是冷静。
<br/>　　知晓楚风眠的人，便是能够过来，这是楚风眠真的怒了。
<br/>　　楚风眠一开始没有动手，也正是给了这风生子一个机会，可如今既然这风生子自己找死，楚风眠也不会放过了他。
<br/>　　风生子冷哼一声道。
<br/>　　“第三个？小子，你只有这两个选择！”
<br/>　　楚风眠眼神中闪过一抹杀意。
<br/>　　“那就是你们三仙岛的所有人都要死！来偿还你们今日围攻寒雪阁的恩怨！”
<br/>　　听到楚风眠的话，风生子不屑一声，他大步上前，猛然出手。
<br/>　　“想要杀了我们？楚风眠看来你是失心疯了！”
<br/>　　一时间狂风围绕，一重重的狂风，凝聚在了他的手心之中，冲着楚风眠猛然一掌轰了过去。
<br/>　　这一掌便是真正的杀招。
<br/>　　同时在风生子的头顶之上，却是突然出现了一片黑暗。
<br/>　　这黑暗，遮天蔽日，仿佛是蒙蔽了天际一样。
<br/>　　“楚风眠，你想不到吧，我们三仙岛可是有着秘术，可以蒙蔽天意，生死大劫，你以为我没有办法出手？”
<br/>　　风生子大笑一声，灵力暴起，一身力量毫不掩饰的爆发了出来。
<br/>　　化为了一道道巨大风刃，向着楚风眠斩杀了过来。
<br/>　　“居然是有着如此秘术？”
<br/>　　寒月漓眉头不由一皱，这风生子是巅峰圣者的人物，应该是无法全力出手的。
<br/>　　毕竟他一旦全力出手，便是就会引来生死大劫，但是如今这蒙蔽天意的秘术，却是足以全力出手而不引来生死大劫。
<br/>　　一尊全力之下的巅峰圣者可是跟那种惧怕生死大劫的巅峰圣者截然不同。
<br/>　　以寒月漓的实力，若是风生子忌惮生死大劫，她还有着跟风生子一战的资本，可现在，风生子的实力已经是远远的超越了她。
<br/>　　一招一式之中，顿时向着楚风眠跟寒月漓二人轰杀过来，要将他们二人一齐击败。
<br/>　　“这秘术倒是不错，只是可惜，你的任何挣扎都没有任何的意义。”
<br/>　　看着风生子全力出手，楚风眠表情一如既往的平静。
<br/>　　对他来说，风生子不管是顾不顾及生死大劫，都没有任何的区别，因为在楚风眠的手中，都是蝼蚁。
<br/>　　“风生子，你的实力的确不错，但是可惜你今日必定陨落！”
<br/>　　楚风眠冷哼一声，一掌打出，化为了一团战龙之力，如同一条咆哮战龙，轰击而出。
<br/>　　一招之下，这漫天风斩，被一瞬之间，全部都被摧毁殆尽。
<br/>　　“什么？这小子居然可以抗衡风生子？”

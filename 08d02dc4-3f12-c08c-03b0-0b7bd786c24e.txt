　　第436章
<br/>　　草原上，一行三人从远处过来。
<br/>　　程大雷骑在黑牛背上，长长打了一声哈欠。徐神机、银眸骑着马跟在他身后。银眸依旧是面无表情，她一向如此，想要从她脸上看到任何波动都是奢侈。徐神机却是长吁短叹，一路啰啰嗦嗦说个不停。
<br/>　　“大当家，咱们歇一歇吃点东西吧？”
<br/>　　“唉，我这把老骨头，活不了多久喽，说不得那一天就一命呜呼，临死前，我可不想当个饿死鬼。”
<br/>　　……
<br/>　　程大雷没有搭理他，骑在牛背上的他却突然停下脚步。皱起眉头，目光狐疑的盯着四周。
<br/>　　哇，大当家，你终于肯歇一歇了么。我还觉得你特意要饿死我呢，快把干粮拿来，我要吃东西。”徐神机大叫。
<br/>　　程大雷摇摇头，视线投在一个方向上。
<br/>　　“血腥气。”
<br/>　　某些事情当真是在不知不觉间发生，程大雷在战场上出生入死多少次，对于鲜血的味道已经有种类似野兽的直觉。
<br/>　　他拨转牛头，循着血腥气往前走，徐神机叫苦连天的跟在他身后。当三人登上一个山坡时，突然被眼前出现的一幕吓得呆住。
<br/>　　这是烧过火的痕迹，冬天的草原百草枯槁，一场野火可以连绵烧起十几里的草原。但很显然，眼前这不是野火，四周有挖开的断火壕，是有人故意在此地放火。而他们在此地焚烧的是尸体。
<br/>　　几十具尸体整整齐齐摆在一起，叠成一个明显的『X』字，现在尸体已经被烧焦，火焰已经熄灭，几只秃鹫在啄食尸体上的残渣。
<br/>　　程大雷拣一根木棍，检查尸体上的伤口。发现这些尸体大部分是男人的，有老人也有孩子，青壮年居多。都是先辈杀死后摆在这里的，码得整整齐齐，像某种宗教仪式。
<br/>　　“哇！”
<br/>　　程大雷正认真思考着，只听身后的徐神机突然吐了起来。
<br/>　　“你怎么了，身体不舒服？”
<br/>　　“不是，我只是突然觉得烤肉的味道……好香……哇……”
<br/>　　“呃……”
<br/>　　程大雷抽抽鼻子，同时也开始觉得胃部有些异样。他们的确好久没吃过东西，而眼前尸体烧焦的味道……勾起了他的食欲。
<br/>　　程大雷好险是没有吐出来，赶紧带着徐神机和银眸撤了。
<br/>　　三人坐在大树下歇脚，取出干粮进食。银眸嚼着风干的肉干，用眼神示意程大雷和徐神机要不要吃一些。
<br/>　　二人立刻摆手，示意自己并不需要。
<br/>　　“大当家，会不会又是他们干的？”徐神机。
<br/>　　程大雷点点头：“看手法应该是同一拨人，只是不知道他们要干什么。”
<br/>　　一路从凉州过来，这样的事情已不是第一次遇见。草原之上，许多小部落被洗劫，女人掳走，男人被杀死。都是同样的手法，杀人之后焚尸体，摆成一个整齐的『X』字。
<br/>　　过程中，程大雷抓了一个戎族，据他口中得知。北蛮部的草原上冒起一伙人，他们烧杀劫掠，短短时间，已经有十几个部落被屠杀。曾有几个部落联合起来，要将这伙横空出世的杀贼一网打尽。结果，反而是他们被一网打尽。
<br/>　　据说，为首之人是个独眼龙，没人知道他是从哪里来的，仿佛天降杀孽给草原。
<br/>　　“独眼龙……”程大雷嘴里念叨着这三个字。
<br/>　　徐神机似乎知道程大雷在想什么，道：“大当家，不是福德勒，那人叫做野原火。”
<br/>　　“嗯。”程大雷摆摆手：“反正只要不遇见，这件事就和我们没关系，我们的目的还是去找少羽。”
<br/>　　“是呐，只是不知少羽现在身在何处。”徐神机叹了一声。
<br/>　　如今，三人已进入北蛮部的草原，处于并州边境，下一步，就是穿过草原，进入帝国。
<br/>　　正与徐神机交流的程大雷突然闭上嘴巴，微微侧头，道：“有人来了。”
<br/>　　他的声音刚刚落地，徐神机耳边就听到马蹄声，七八匹快马翻过草坡，将大树下的三人围住。
<br/>　　为首之人脸上一道长长的刀疤，盯着银眸手中的干粮，露出贪婪的表情。
<br/>　　然后，他的目光落在银眸脸上。
<br/>　　“好俊的姑娘，正好让兄弟们享用享用。”
<br/>　　一帮人哈哈笑着，有人道：“男人宰了，女人带回去，也给兄弟们解解馋，男人的肉可以炖锅好汤。”
<br/>　　“只是有个老头，肉太柴已经嚼不动了。”
<br/>　　徐神机忙点头：“是呐，是呐，我的肉太老，不好吃，你吃他们就好了。”
<br/>　　程大雷白了他一眼，目光看着这伙人，道：“帝国人？”
<br/>　　“哈，我不是帝国人，也不是戎族人，老子不服王法，不怕老天，是天王老子独一份。”刀疤脸大笑。
<br/>　　此刻银眸已经起身，双手抓住了鬼面斧。程大雷挥挥手：“记得留个活口。”
<br/>　　银眸拎着比她个头还高的斧头，拖拉着走向敌人，她的速度很慢，刀疤脸几个哈哈大笑，止不住的兴奋。
<br/>　　待走到对方马前时，刀疤脸的笑容停住，饶有兴致的打量着银眸。
<br/>　　“小姑娘，你还是和爷爷一起回去吧，爷爷不仅不杀你，还会好好待你。”
<br/>　　银眸抬起头，身子从地上蹦起，双手扬起大斧，跳着劈下。
<br/>　　一斧砍掉马头。
<br/>　　鸦雀无声，所有人都呆呆看着这一幕。这小姑娘浑身上下，像是没有四两力气，可一斧砸下来，却能割断肌肉，砍断骨头，敲掉整个马头。
<br/>　　刀疤脸狼狈从地上站起来，还没回过神，迎接他的就是一柄斧头。
<br/>　　从头到尾，被劈成两半。
<br/>　　程大雷窝在大树下，眼睛看着这一幕。即便他迈入绝世后，单纯比力气也不敢说胜过银眸，何况这帮来路不明的马匪。
<br/>　　银眸杀人与她的表情一样，都很干净，人命她眼里，似乎和一只鸡一只狗没什么区别。他能用最残忍的手段敲掉敌人的骨头，同样也不介意轻巧的割断敌人喉咙。
<br/>　　七八个马贼很快死在她的斧下，只剩下一人，早已被吓破了胆。突然大叫一声，纵马向远处起身。
<br/>　　程大雷突然起身，身体像是离弦的箭，在空中踏步，将对方从马背上拽了下来。
<br/>(本章完)

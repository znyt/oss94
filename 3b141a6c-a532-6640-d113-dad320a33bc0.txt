　　商朝都成朝歌，此刻已经被大阵封天。谁又曾想到，整个朝歌便是一座大阵，人不能入侵，甚至就连仙人也攻不进去，诸侯联盟军队只好在外将其围住。
<br/>　　“如今该如何是好？纣王便在城中，我等却无法入侵。”武王姬发焦急问道。
<br/>　　“武王莫急，我师尊自会处理，你只需稍等即可。”申公豹经历过了许多，此刻也变得淡定了许多。
<br/>　　“元始圣主竟然亲自出手，那再好不过了。”
<br/>　　姬发闻言大喜，他能夺得商朝天下，他知道不是靠自己，而是靠阐教，阐教能够让他做皇帝，也能够让其他人当大王，所以他对于阐教这等仙人，一向是极为敬畏的。
<br/>　　“嗯。”申公豹淡淡点点头。
<br/>　　此时，朝歌城中，元始的身影由虚化实，渐渐完整地出现在纣王面前。
<br/>　　“本座该说你是纣王呢，还是天外之人？”
<br/>　　元始目光平淡，打量着眼前这个所谓的纣王。在他身后，一众大臣早已消失不见，不知去了何处，留下的只有五人，分别是异界烛龙敖无天、诸葛亮、吕布、岳飞，以及一个白发女子，称号为死亡主神。
<br/>　　五人之中，元始只是随便瞟了一眼，便将目光重新投到纣王身上，相比较其他人，他对于这位纣王更感兴趣。
<br/>　　五人之中修为最高的也只是那名为“死亡主宰”的女子，是一名伪混元强者。怎么说是伪呢，那是因为此女境界虽是混元，但战力弱得一个准圣强者便能将之斩杀。元始在她身体之中感受到一股类似圣人鸿蒙紫气的存在，只是比鸿蒙紫气弱化了数百倍，想必这是其他世界的法则本源吧。
<br/>　　“你来了，此界最为神秘的元始天尊，你称呼朕为纣王即可。”纣王的目光之中依旧不见任何焦虑等情绪，只是带着一股好奇的目光打量着元始。
<br/>　　“汝好像并不怕本座，是觉得本座不会杀你，还是你有更强的底牌，能够保证汝之性命不受威胁。”元始心念一动，身影变得模糊起来，让纣王等人不能看清楚。
<br/>　　“本座猜你是不是觉得本座对你有所顾忌，便不会对你出手？”
<br/>　　“呵，随便圣人怎么想。”纣王呵呵一笑，毫不在意道。
<br/>　　“你是在拖延时间，似乎是在等待着某个人。”元始亦不着急，两人就好似多年未见的朋友，淡淡地交谈起来。
<br/>　　闻言，纣王瞳孔一缩，但紧接着便恢复了平静，好似什么都没发生过，这种变化虽小，但元始还是发现了。
<br/>　　“圣人是在说笑吗，朕有什么人可等的。”纣王目光平静道。
<br/>　　“哦？是吗？”元始淡淡一句，接着语气一转，继续道：“那么，此人又会是谁呢？”
<br/>　　元始伸手向着虚空一抓，接着，在冥界天空之上，出现一只巨大的遮天大手，向着六道轮回之处一抓，在众冥界高手惊骇的目光下，将一个毫不起眼的鬼差抓了起来，之后消失不见。
<br/>　　转眼间，那人便出现在元始以及纣王面前。纣王看到此人，面色再也不复之前那般从容，面色一变，但随即又恢复淡然，问道：“此人是谁啊？”
<br/>　　“纣王真的不认识此人吗？”
<br/>　　元始轻笑一声，停顿数息，接着道：“此人似乎并不是我洪荒之人，三年前不知何故潜入冥界，诱导僵尸一族与冥界骷髅一族大战，之后又诱发其他大战，弄得冥界不得安宁。”
<br/>　　说到此处，元始停下，看了一眼目光依旧的纣王，接着道：“之后，此人竟然觊觎起六道轮回。你说，这样的人，来冥界是为了干什么呢？”
<br/>　　“圣人都不知，朕又岂会知晓？”纣王道。
<br/>　　“是吗？”
<br/>　　元始淡淡道：“你应该知道，本座是靠因果法则成圣的，那么，为何此人身上的因果与纣王身上的一模一样呢。纣王该如何解释本座这个问题呢？”
<br/>　　哗！
<br/>　　这一刻，纣王终于不复之前那般淡定了，他面色一变，道：“朕，终究是小看了你，洪荒最为神秘的圣人。”接着，他又淡定道：
<br/>　　“不过，你就算知道又如何？”
<br/>　　“哦？看来纣王还有些手段啊。呵呵！”
<br/>　　元始轻笑一声，一掌将地上那鬼差装扮之人拍碎，淡然道：“那么，本座毁了这个人不人、鬼不鬼的东西，想必纣王也不会生气吧。”
<br/>　　“噗嗤——”
<br/>　　血族分身被毁，纣王随即便是一口逆血喷出，显然是受到了极大的重创，不仅毁了一个分身，更重要的是元神伤的重创，无法弥补。
<br/>　　既然撕破了脸皮，纣王也不准备继续装下去了，反正他有系统，随时可以返回，但就如此离开，他内心却极为不甘。所以，就算走之前，他还准备恶心元始一番。
<br/>　　纣王擦去嘴角鲜血迹，冷冷道：“元始，不要以为你是圣人，就可以为所欲为。朕知道你想知道朕的秘密，但朕就是不告诉你，你能奈朕何？”
<br/>　　“有意思，很久无人如这般对本座说话了。”元始目光依旧淡然无比，并没有因为对方的话语而感到丝毫羞怒。
<br/>　　“朕有人皇之气护身，你若是不想跌落圣坛，修为巨损，就别惹朕！”纣王冷然道。
<br/>　　“跌落圣坛？修为巨损？呵，你想多了。”
<br/>　　元始淡然一笑，随后霸气道：“谁告诉你，我是圣人了，本座就是因果之主，谁敢让本座因果缠身。”
<br/>　　“你不是圣人，那么天罚圣主又是怎么一回事？”纣王闻言一愣道，但紧接着，他好似想到了什么，惊骇道：
<br/>　　“那是你的三尸之一！因果之主，难道你是混沌魔神？”
<br/>　　纣王语气之中带着浓浓惊骇之意，一切事实都不敢置信，一个圣人竟然仅仅只是三尸之一，任谁也不会相信。
<br/>　　周围早已被元始遮蔽了，任何天机都传不出去。尽管他说得不对，但元始并没有反驳，他这样理解也不差多少。
<br/>　　“就算你是混沌魔神又如何，想要杀朕，可没那么简单！”
<br/>　　纣王冷笑一声，他一挥手，三十六条紫气真龙腾空而起，一声声龙吟直冲九霄，同时滚滚气运弥漫开来，将这片天地遮蔽，完全隔绝天道。
<br/>　　若是圣人到此，便不能够借助天道之力战斗，战斗力将大大折损。从这一步来说，纣王做得完全正确，他只是没料到，到此的元始并不是圣人。
<br/>　　“起阵！”
<br/>(本章完)

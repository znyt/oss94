　　　　高漫秋走在有些湿·润的石子路上，前方是带路的水族人，与当初他们所想的一样，水族与神秘家族有着很大区别，乍一眼看上去根本差别不到这些人到底有何特别之处，甚至路径所看到的都是一众淳朴村名。
<br/>　　但高漫秋可没有忘记，这些人可以自由在水中畅行无阻，天生自带无敌的水性，更有着神秘家族普通成员没有的敏锐。
<br/>　　高漫秋的到来引起了水族全族的好奇，无数双眼睛看着这个女人从石子路走来，就这么一步步走到圣女的屋檐下。
<br/>　　“请。”水族的领路人开口，而后避开到几米开外。
<br/>　　高漫秋望着眼前这座屋子，轻笑，而后踏步进入。
<br/>　　待高漫秋的身影没入在屋内，屋门闭合后，其后方的整个围观水族群众，这才炸开了锅。
<br/>　　“又一个非我族人进来了！”
<br/>　　“还是圣女让进的。”
<br/>　　“据说这个女人在外面请求拜访圣女？”
<br/>　　“是啊！这是个高家嫡系，那个四偏门的高家。”
<br/>　　“四偏门知道水族在此？知道圣女出世？”
<br/>　　“这个女人跟我们感觉很奇怪，有着压制感，神秘家族最强的人不是白忧么？”
<br/>　　讨论声不大，并没有传进屋子，此时在鱼柔女的屋内只有两个人，这两个女人就这么相视而坐，见面后谁也没有开口说话。
<br/>　　桌面上是一壶清茶，热气徐徐而上中映照着两女的面容带着一丝雾气，鱼柔女手腕翻转，变戏法似得将一缕热水从水壶中引出，形成一道溪流流入高漫秋面前的杯中。
<br/>　　高漫秋眼神无波，端起杯子轻抿了一口，顿时觉得神清气爽又双目清澈不少。
<br/>　　她眼神一动，抬头道：“这茶有特殊功效？”
<br/>　　鱼柔女轻笑：“银净水配干葵草。”
<br/>　　高漫秋眉宇一挑，这两种东西都是她闻所未闻。
<br/>　　鱼柔女又给高漫秋添了一杯，道：“狼牙在数次危机中不败，那么多战士长久作战而不疲，都是靠着水族出产的银净水。”
<br/>　　高漫秋听着，淡淡点头：“难怪……”
<br/>　　“你还有什么想问的？”鱼柔女淡笑着，说不清是什么神情。
<br/>　　高漫秋摇摇头：“没有什么需要打听，我只是依循尊卑，代表高家前来拜会水族。”
<br/>　　鱼柔女再次唇角勾起：“那你呢？我可是听到了你是以你自己的名义前来，要拜访我。”
<br/>　　高漫秋眼神带笑，与鱼柔女平视：“我么？我是神女，对圣女好奇而已。”
<br/>　　话落，顿时一股淡淡的争锋感在屋内蔓延，神女和圣女之间的气势较量也初现成型。
<br/>　　“这还是千年来的第一个神女。”鱼柔女收回视线，淡然道：“自你进来后我就有所感应，却没想到你的神女等级，竟然也能感应到我的存在。”
<br/>　　“高家的诅咒最轻。”高漫秋忽然说了这么一句话：“当年的事，你的传承里应对有记忆。”
<br/>　　“不错。”鱼柔女目光顿时犀利，定定的看着高漫秋：“但我却看不透，你们高家到底是真忠心，还是太厉害？”
<br/>　　“说到厉害，谁比得了肖家？”高漫秋不退不让，一字一顿：“他们先随波逐流等待至今，再以全族为血，祭奠了一个千年老妖啊！”
<br/>　　“肖梦琪还是肖七？”鱼柔女眼神深邃了起来。
<br/>　　“肖梦琪是个混血，男生女名，随着肖家旧址被找到后，他已经被完全吞噬，再也没有出来过。”高漫秋说着一顿，紧接着迟疑道：“肖七这个名字……”
<br/>　　“恩，肖家第七代传人，那个绝世神女。”鱼柔女眯起眼睛：“她不在这里，你可放心了。”
<br/>　　高漫秋松了一口气，起身慎重道：“多谢。”
<br/>　　走出·水族旧址，高漫秋早已满目坚定，一步步踏上了前方高家精英所在的道路。
<br/>　　肖七不在，甚好。
<br/>　　——————
<br/>　　刷刷刷！
<br/>　　飞奔的步伐在丛林里穿梭，一个年轻男子速度如猎豹，风一样的由一棵树的枝头跳到另一棵树的枝头。
<br/>　　这是已经完全恢复伤势的江凌轩。
<br/>　　他在步纱的住处呆了很长一段时间，根本没有系统治疗的情况下，伤势的恢复只能依靠本能，早在半个月前江凌轩就已经可以离开，但他还是等到步纱被白允儿带走，这才走出屋子独自行动。
<br/>　　猎者联盟的事已经放下，时间过去这么久猎王自有应对方案，用不着当下的江凌轩来操心。
<br/>　　只不过除了这些，他身上还有属于他自己的个人任务。
<br/>　　任务发布者：何培源。
<br/>　　任务资金：无。
<br/>　　这是一项没有报酬的任务，来自于一个老人临终前的委托。
<br/>　　“阳谷的入口并不一定是阴谷出口，还有别的办法可以进入。”
<br/>　　此时江凌轩就飞驰在茂密的植被上方，寻找着何培源所说的那个阳谷入口，这条路颇为难走，雨林环境的茂密植被让下方地面到处都是瘴气和蛇虫，除非有专业的队伍一同前行，否则很容易就死在里面，并且难以辨别方向。
<br/>　　而走在上方却需要极强的体能和耐力，脚步不能停，否则一不小心就会落下去，不知道掉进哪只狂化动物的口中。
<br/>　　最重要的是这片地带，直升机飞入总是莫名失常，不是需要迫降就是被藤蔓缠住。
<br/>　　江凌轩只能等自己的伤势完全恢复后，在自己战力处于巅峰状态时，采取第二种方法。
<br/>　　一路走来他知道自己不是第一个用这种办法前来的人，因为看到了不少人类留下的痕迹，只是那些人是谁，有没有成功进去又活着出来，江凌轩无法猜测。
<br/>　　他知道的情报很少，也是第一次脱离猎者联盟，以单独的身份前来闯荡这个危险至极的地方。
<br/>　　摸了摸藏在里衣内的一口铜镜，江凌轩不知道这古董一样的铜镜有什么作用，只清楚这玩意儿肯定不简单。
<br/>　　躺在山间这么久，江凌轩不知道前方的阴阳谷内已经有三大神秘家族的成员入驻，也不清楚自己即将卷入一场可怕的斗争里。
<br/>　　他只是非常巧合的在这个时间点，找到了阳谷入口！
<br/>　　“就是这里！”没多想，江凌轩双眼一亮，稳稳当当的踏在了那块巨石之上。
<br/>　　默数了三下，江凌轩依照着何培源所说，成功开启了阳谷入口。
<br/>　　圆形石台飞速下潜，一下子带着江凌轩降落至地底，顿时一股奇怪的温度和湿度扑面而来，带着淡淡的腐朽气息。
<br/>　　江凌轩紧了紧胸口裹着的铜镜，小心翼翼的在黑暗中摸索前进。
<br/>　　只是江凌轩不知道，在他进入阳谷内的时候，不远处的一簇低矮植被后方，楚云天将这一幕看的清清楚楚。
<br/>　　“第二批，一个人。”楚云天叨念着什么，在手中的本子上记下。
<br/>　　此时的他衣衫褴褛，还有些瘦骨如柴，蓬头垢面的样子显然已经在此处呆了很久，他是第一个从阴谷出来的人，并没有冒然行动，而是立即找了个地方隐藏自己，以路过的小生物喂食，将每一个进入阳谷的人都记下。
<br/>　　阳谷的入口规则他早已摸透，之前的白家和罗家二十人，也是踏了三下后潜入下方。
<br/>　　但是楚云天还是没有行动，他还在等。

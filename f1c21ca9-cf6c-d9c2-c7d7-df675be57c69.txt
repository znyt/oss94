　　自己还躺在厢房的床上，外头是白天（这里有极昼极夜，但没有人间大部分地方那么规律的日夜交替）。芮冷玉感觉自己全身都是麻痹的，半天才缓过来，靠在床上大口喘气。
<br/>　　“出什么事了？”
<br/>　　道风的声音传来，芮冷玉转头去看，那个“小子”还保持着背对自己的姿势，这是自己睡前特意摆的姿势，虽然不用避嫌，但是被人盯着睡觉的感觉还是有点别扭，于是每次睡觉之前，芮冷玉就强行把小人转向脸朝墙的那边。
<br/>　　芮冷玉坐起来，缓了几口气，道：“做恶梦了。”
<br/>　　想到那荒诞的梦境，于是跟道风说了一遍，道风听完，却半天不作声。
<br/>　　“这个梦我越想越怪，道风，你说这个梦会不会有什么预示？”
<br/>　　“不要多想。”道风半天只说了这句话。
<br/>　　芮冷玉还是忍不住去想，尤其是梦中，后卿对自己反复说的那句话，一直历历在目：我一直在你身边……
<br/>　　这句话什么意思？
<br/>　　芮冷玉越想心里越犯嘀咕，又跟道风去说。道风沉默半晌，突然说了句：“你对当日你获救的经过，还记得多少？”
<br/>　　芮冷玉想了一下，道：“不太记得了，我只记得当时自己走进了燃烧的尸血里，后来就昏过去了，等我醒来，我已经被带回人间了，这中间的事情……我是一点也想不起来了，难道发生了什么我不知道的事？”
<br/>　　道风没有回答，他当然不会把自己的猜测说出去。
<br/>　　“道风，你说，我现在算是什么形态，是人，还是鬼，或者僵尸什么的？”芮冷玉自己分析道，“我有肉身，在人间还是需要吃饭睡觉的，按说我是人类，但那天我明明吸收了将臣之血……我很担心我成了僵尸了。”
<br/>　　“你是人。”道风答。
<br/>　　“你怎么知道？”
<br/>　　“你的血中有人的阳气，我才吸的。我吸僵尸血做什么。”
<br/>　　芮冷玉听他这么说，也就稍稍放心了，只是总忍不住想起梦境中的经历，在心里形成了疙瘩。
<br/>　　大明朝时期的北京城。
<br/>　　比南京那边人多，城门内外，人络绎不绝，干活的、做买卖的，还有一些达官贵人，形形色色，从城门洞下往来进去。
<br/>　　叶少阳一行人在远处看着，如果不是他们心里有底，谁能想到，这些形形色色的人们，竟然全都被改写了记忆，被硬塞进这个虚假的世界？
<br/>　　每一个生灵，在这里都被迫扮演着崭新的角色，自己却不知道。
<br/>　　看着他们，叶少阳内心深处，突然涌起了一股浓浓的恐惧，跟碧清说了。
<br/>　　“这是真正的众人皆醉我独醒啊，如果我现在去跟他们说出真相，他们反倒会觉得我是神经病。”叶少阳轻轻摇了摇头，这种奇怪的感悟，他说不上来。
<br/>　　碧清也望着城门下那些行人，说道：“可是，你怎么知道你才是醒着的？没准你们也是被人灌输了记忆，活在你们那个世界呢，如果有人去说你们那个世界是假的，你们也会骂别人是神经病吧。”
<br/>　　叶少阳顿时哑然，一个人，判断现实处境的真假，无非就是记忆，可如果记忆本身就是作假了的呢？但反过来，如果不依靠记忆，那还能依靠什么呢？想了半天，还真的什么都没有。
<br/>　　这件事，真的细思恐极。
<br/>　　进城的比出城的盘查得要严格得多，尤其是他们这种赶着马车、一看就是外地来的，需要出示户籍证，这东西叶少阳也是有的，但想到自己在南京那边闹出的事，估计自己八成已经成了通缉犯，于是推说没带，然后报了个假名字。
<br/>　　守城的士兵倒也没说不让进，拉到边上，询问他们进城的目的之类的。
<br/>　　叶少阳随口敷衍，然后塞了几张银票过去，城门官见钱眼开，看他还带着两个女眷，应该不会是啥坏人，就放他们进去了。
<br/>　　叶少阳一行人在街上找了一家挺上档次的客栈，安顿下来。
<br/>　　“接下来有什么打算？”安顿下来之后，碧清问道。
<br/>　　“等小马他们来到，一起撸袖子干。”叶少阳也是打定了主意，先把皇帝拉下马再说，至于那个九星叠气阵位置所在，以他的设想，一定是在皇宫之类的地方，也不会太难找。
<br/>　　碧清道：“你连法器都没有，如果引来圣灵会的人，拿什么对付？”
<br/>　　“我能感觉到，我那些法器都在不远某个地方，明天过去看看。”
<br/>　　“就算你那些法器圣灵会的人用不上，也不会好好放着留给你的，没准有什么阴谋。”
<br/>　　叶少阳也是这么想，道：“嗯，但必须拿回来。”
<br/>　　碧清盯着他，说道：“你可别忘了答应我的。这次无论如何也要帮我把炼尸缸拿到，也不枉我跟了你这么久。”
<br/>　　叶少阳挠着头，“你这话容易让人污秽啊，什么叫跟了我这么久……”
<br/>　　碧清笑道：“不是吗，每天陪着说话，同吃同住……”
<br/>　　两人正调侃着，门外突然传来一阵急促的脚步声，把木头楼梯踩的咚咚响，下一刻，房门就被人敲响了。
<br/>　　叶少阳过去开门，立刻冲上来两个人，一左一右按住他的胳膊。
<br/>　　另一个进到屋里，拿了桌上的油灯，照着叶少阳的脸，喝问道：“可是杨绍业？”
<br/>　　叶少阳心里咯噔了一下，没等开口，边上一人呼啦一声抖开了一张纸，是一张通缉令，整个半张都是自己的画像，别说画得还挺像。
<br/>　　“都统，就是他！”一个人嚷起来。
<br/>　　几个人一起比对，那个都统点了点头，皮笑肉不笑地看着叶少阳，“您是小侯爷啊，还是个皇亲国戚，怎的赶出偷袭白虎堂、杀死特使宗师这样的事……”表示很不可思议，随即笑道：“您而今做了通缉犯，就别再摆侯爷的架子了，跟兄弟们去衙门走一趟吧。”
<br/>　　说完，冲着站在窗前的碧清努了努嘴，手下们争相出列，狞笑着朝碧清走过去，想趁机在绑人的时候占点便宜。
<br/>　　叶少阳摇了摇头。
<br/>　　那都统说道：“我说小侯爷，现在摇头也晚了。”

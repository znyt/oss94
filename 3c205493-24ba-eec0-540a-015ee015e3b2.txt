　　第618章
<br/>　　莫将难笑吟吟看着柳追风，一副智珠在握的表情。柳追风可是一股不小的势力，若能得到他的相助，自己的大事便更有把握。
<br/>　　心中雄心壮志，想着拉起一支雄兵，与各路诸侯争一争天下。
<br/>　　柳追风目光闪烁，显然已是动了心。无意间，他与周桃的目光碰撞，忽又生起不忍的念头，心中暗自思付道：我又怎能欺她。
<br/>　　他忽然仰头哈哈大笑，道：“阁下的好意我心领了，但小桃愿不愿意跟我，还要看她的心思，你我都做不了主。”
<br/>　　说着，他骑在马上向周桃伸出手，道：“你愿不愿意随我走？”
<br/>　　这话出口，程大雷不免对柳追风刮目相看，能成一派宗师，心中的境界毕竟不低。
<br/>　　倒是莫将难愣了愣，揣测不明白柳追风的心思。但他心中还是蛮自信的，双手背在身后，冲周桃道：“你若愿意随他走，是你的自由，我不会留你。”
<br/>　　程大雷目光落在周桃身上，心中还是存了看戏的心思。
<br/>　　周桃看看柳追风，又看看莫将难，莫将难目光有意无意的看着她，嘴角挂着淡淡的笑容。
<br/>　　“好，我随你走。”
<br/>　　周桃忽然向前踏出一步，手搭在柳追风手腕上，微微踮脚，便上了马背。
<br/>　　莫将难脸上的笑容僵固住，周桃的选择完全在他预料之外。
<br/>　　柳追风却是喜形于色，轻轻一拽缰绳：“好，我带你走。”
<br/>　　几个人骑在马上，都是柳追风打招呼唤来的武林同道，柳追风一声喝，这些人已打算离开。
<br/>　　“呔。”莫将难喝道：“阁下想走，是不是要与我招呼一声。”
<br/>　　大家都在绿林道上混，就算没碰过面，柳追风也听说过莫将难的名声。
<br/>　　“怎么，阁下还想留柳某人不成，别人怕你万马堂，我可不怕。”
<br/>　　莫将难呵呵冷笑：“今日我们弟兄们在此起事，你若透漏风声，引得官兵过来，岂非坏了我们的大事。”
<br/>　　“你想怎样？”莫将难的手已悄悄握住剑柄。
<br/>　　“倒是不想怎样，只是想留柳兄暂歇几日，等过了这件事，在下一定送柳兄离开。”
<br/>　　柳追风居高临下的看着莫将难，忽然哈哈大笑，只听锵啷一声，他身上佩剑出鞘。
<br/>　　“好好好，阁下想要留我，且问我这口剑答不答应。”
<br/>　　扶风一剑柳追风，号称剑能追风，他骨子里的自信，都是从这口剑来的。真要动起手来，哪怕今天场上不少绿林好手，他也丝毫不惧。
<br/>　　莫将难一挥手，数人将柳追风一行人围了上来。都是万马堂的人，其他江湖人仍旧做壁上观。
<br/>　　明知柳追风在剑道上的造诣颇深，莫将难心中也并无畏惧。他冲身边一人抱拳，道：“请韩兄出手。”
<br/>　　程大雷的目光这才落在此人身上，他五短身材，属于丢在人堆里都认不出来那种。
<br/>　　他向前跨出一步，拦在柳追风的马前，冷冷道：“请了。”
<br/>　　柳追风一看到此人，便如临大敌。程大雷不认得这五短身材的刀客，柳追风却是认得的。
<br/>　　漠北孤鹰韩醒，曾经用一柄刀平了塞外大盗一阵风。怪不得莫将难有恃无恐，原来是手下有这种高手。
<br/>　　柳追风不敢托大，从马上跃下来，也就是他刚刚落地的光景，一柄刀向其斩来。杀的就是柳追风立足未稳，这一闪而逝的机会。
<br/>　　韩醒如何来历，无人知道，他出世时就用一柄刀屠了一阵风。一阵风手下上百人，被他一夜之间杀尽。这倒也算不得什么，江湖人武艺出众，并不是做不到这一点。关键是，所有被他杀死的人，都是一刀毙命，身上并无其他伤口。
<br/>　　如此想来，就有些恐怖了。
<br/>　　出招快准狠，讲究一刀毙命，不费二刀。柳追风的确遇上了劲敌，亏得他躲避及时，才没被韩醒一刀斩了。
<br/>　　静心防御，柳追风慢慢稳住局势，手持着佩剑与韩醒打在一起。
<br/>　　这时候，他的同伴也与万马堂的手下动起手来。都是江湖上的好手，可万马堂毕竟人多，且每个人都是杀人不眨眼的恶匪，这帮人陷入围攻中，已经渐渐落在下风。
<br/>　　如果两人公平对决，柳追风也未必不是韩醒的对手，可今天的情形实在对他不利。
<br/>　　他无心恋战，不想与韩醒纠缠，向后纵跃退出战斗后，带着一行人往外冲。
<br/>　　可身周密密麻麻都是万马堂的手下，目光粗略看过去，至少也有数百人。
<br/>　　纵然每个人武艺都是不错，却没能力闯出重围。柳追风等人抱团围在一起，面前是长刀短枪围着他们。
<br/>　　莫将难道：“柳兄，这不过是一场误会，你若愿意与我们弟兄一起起事，我愿奉柳兄为我军中大帅，你我兄弟一起去天下，岂不快哉。”
<br/>　　柳追风阴森森的冷笑：“姓莫的，你忒小看我了。今日弟兄们陷在此地，拼死也要拉你几条命下来。”
<br/>　　“好。”莫将难道：“柳兄英雄气概，今日我就送柳兄归天，成全柳兄的名声。”
<br/>　　柳追风情知今日情况危急，说不得自己一条性命就要交待在此地。他目光转了一圈，忽然落在程大雷身上，眼前顿时一亮，心中有了主意。
<br/>　　“程当家，弟兄们愿意拼死救了你，你可愿意帮我们突围？”
<br/>　　程大雷立在那里，笑呵呵道：“好说，好说。”
<br/>　　柳追风没能耐突围，可眼前还有一尊大贼，只要替程大雷摆脱束缚，再给他一柄剑，柳追风不信有人能拦得住他。
<br/>　　“弟兄们，与我冲啊。”
<br/>　　柳追风拼死向程大雷冲过来，莫将难心中一惊，冷不防柳追风还有这一招。他急忙挺大刀护在程大雷身前，免得程大雷被劫走。
<br/>　　一群人乱杀乱砍，程大雷慢慢退到农神庙里，免得刀枪不长眼，伤到自己。
<br/>　　外面砍杀声响起，程大雷躲在农神庙里，目光瞅着外面的情形。
<br/>　　“大当家……”
<br/>　　突然一个声音隔着窗户轻轻唤他。程大雷一回头，看到飞天蛤蟆白元飞趴在窗户外。
<br/>　　“大当家，你没事吧？”
<br/>　　程大雷刚才就在人群中扫到白元飞，所以才能有恃无恐，他道：“暂时没什么事，你怎么来了？”
<br/>(本章完)

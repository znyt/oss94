        &lt;h3&gt;第四卷第二章  相见&lt;/h3&gt;
<br/>
<br/>        谷强和谷正娟应约到了秦少卿的家里。
<br/>
<br/>        闫欢和闫芈及保姆已经将饭菜做好的，当闫欢和闫芈略微相似的脸同时出现在谷强和谷正娟的眼前时，谷强和谷正娟都有些吃惊。
<br/>
<br/>        谷强是见过闫欢的，俩人在一起呆过一段时间，而谷正娟没有见过闫欢，不过俩个人如此像，不用说一定是姐妹两了，闫芈曾经说过闫欢是她的姐姐。
<br/>
<br/>        谷强和谷正娟都是场面上的人，心思也是一瞬间就转过来了。
<br/>
<br/>        “闫芈，你走这么久都没有想过我啊。”谷正娟笑着对闫芈说。
<br/>
<br/>        “当然是想的啊，要不然我会邀请你们来家里？”闫芈和谷正娟拥抱了一下，然后看着谷强，“要不要拥抱一下？”
<br/>
<br/>        谷强张开怀抱。
<br/>
<br/>        闫芈笑着说：“还是算了，我不敢。”
<br/>
<br/>        “来，我给你们介绍一下，这个是我的姐姐，闫欢！姐，这个是谷家兄妹，谷强、谷正娟，你要是想要在甸城呆下去，一定得靠他们，黑道白道没有他们搞不定的。”闫芈为姐姐闫欢介绍着。
<br/>
<br/>        “你好。”闫欢看着谷强，他还是一如既往地的冷冰冰，一如既往地帅气吸人。
<br/>
<br/>        “你好，欢迎来甸城。”谷强看着闫欢，他不知道闫欢来甸城的目的，她为什么要到甸城来？
<br/>
<br/>        “闫芈你和你姐姐真像啊。”谷正娟看着闫欢，笑着对闫芈说。
<br/>
<br/>        “我们俩个人相差两岁。”闫欢微笑着对谷正娟解释。
<br/>
<br/>        “来，平时难得相聚，快快快，谷强，不醉不归。”秦少卿笑着招呼谷强和谷正娟。
<br/>
<br/>        “将我喝醉了，我可就要住你家里的。”谷强微微一笑。
<br/>
<br/>        “反正今天周末，住家里有何不可？”
<br/>
<br/>        “凤姨，收拾好客房。”秦少卿吩咐保姆。
<br/>
<br/>        五个人坐下吃饭喝酒，谷强、谷正娟、闫欢都是有心思的人，但是三人因为都是在商场上摸爬滚打过来的，相互之间都隐藏的很好，尤其是谷正娟在谷强的面前更是表现得很自然，对闫欢客气和疏离，对闫芈亲热有加。
<br/>
<br/>        谷强因为和闫欢呆着的那一段时间，谷正娟装作不知，所以心里有一些震惊，她也不敢表现出来，而谷强以为谷正娟不知道，所以也就装作不认识闫欢。
<br/>
<br/>        闫欢是知道谷强的身份，她也装作不认识谷强。
<br/>
<br/>        “我邀请你们来，不光光是吃饭，还有一件事情要请你们帮忙。”入座之后酒过三巡，秦少卿也不客气，开门见山的说。
<br/>
<br/>        “什么事情，你吩咐一声不就行了。”谷强看了一眼闫欢。
<br/>
<br/>        “是这样的，我姐姐准备来甸城投资，她看好了皇后大酒店旁边的那个大楼，想租下来。”秦少卿说。
<br/>
<br/>        “投资？”谷强和谷正娟都看着闫欢，眼里有震惊。
<br/>
<br/>        “是的，我准备在甸城投资超市，你们那个大楼正合适。”闫欢又展颜一笑。
<br/>
<br/>        “我们的那个大楼已经出租出去了，给人家办公了。”谷正娟皱着眉头。
<br/>
<br/>        闫欢跑来甸城投资？
<br/>
<br/>        “这个好说，你在附近不远的那个大楼不是已经装修好了，让人搬过去就是。”秦少卿现在为了弥补闫芈，帮着闫欢说话。
<br/>
<br/>        “你说得简单，我让人搬人家就搬啊？”
<br/>
<br/>        “你们先谈好租赁期限和租赁方式，她是过来投资的，周一上班我会将她的投资方案拿出来上会，从政策上给予支持。”秦少卿并没有理会谷正娟的话而是对着谷强说。
<br/>
<br/>        “你吩咐就好，我按照你的吩咐做。”谷强点点头，然后对谷正娟说：“我们甸城是发展中城市，需要引进的项目很多，秦总负责招商引资，我们是需要配合他的。”
<br/>
<br/>        “我是担心有些人房费不到期，人家不搬。”
<br/>
<br/>        “这个好说，给予他们一定的补贴或者是少收一个月的租金。”秦少卿开口了。
<br/>
<br/>        “租金我可以出。”闫欢看了一眼当着她面讨论问题的三个人。
<br/>
<br/>        “这个不是租金的问题。我担心......”谷正娟看了一眼谷强。
<br/>
<br/>        “你们什么都不用担心，按照我吩咐的来做。”秦少卿坚定的说。
<br/>
<br/>        “呵，这个事情你和陆玉商量了吗？”谷正娟看了一眼秦少卿，言下之意是你能做主吗？
<br/>
<br/>        “我是甸城招商引资的负责人，我做不了主谁能做主？”秦少卿的语气冷下来了。
<br/>
<br/>        这个谷正娟就是一个神经病，哪壶不开提哪壶。
<br/>
<br/>        “你这样说我就放心了，明天我回去就让他们搬家！”谷正娟见秦少卿变脸了，便改变了语气。
<br/>
<br/>        谷正娟不知道闫欢知不知道她，虽然她没有直接参与之前的事情，但确是她联系了当初甸城的人。
<br/>
<br/>        陆玉是知道闫欢的，就是不知道陆玉会不会继续和闫欢对着干下去，而闫欢是不是抱着有什么目的来甸城？
<br/>
<br/>        “谢谢两位谷总和少卿，闫欢以后在甸城就仰仗几位照顾了，来，我敬大家一杯。”闫欢感激的说。
<br/>
<br/>        “你准备做什么？”谷强问闫欢。
<br/>
<br/>        “我在西城的项目总建筑面积25万平方米，商业面积20万平方米，是最大的商业单体项目。按照国际购物中心来做的，汇聚国内外600余家知名品牌，主力购物区域分：名品门店区、精品百货区、品牌连锁餐饮区、文化休闲区、中央景观区、地铁潮店区，打造以生活、休闲、娱乐为一体的具有国际综合服务水平的现代化互动体验式都市型购物中心。
<br/>
<br/>        不仅有着armani  jeans、tommy  hilfiger、guess等国际品牌，最意想不到的就是将带来最颠覆的革命性购物体验：购物中心在商场核心位置营造400多平米水景带，全球最大的室内瀑布飞流直下，以及全场绿化，60部进口电梯打造科学的交通组织，有最长的飞天扶梯直达六层餐饮中心，我想将西城的模式搬过来。”闫欢讨论是工作来时，完全是变了一个人，自信而强大。
<br/>
<br/>        “这样投资下来得多少资金？”谷正娟看着自信的闫欢，眼里是欣赏，这个女人不一般啊。
<br/>
<br/>        谷强看着闫欢，这么多年没有见，她究竟做了些什么？她完全变了一个人，那个战战兢兢的闫欢不复存在了。
<br/>
<br/>        “我们在西城光一个商场投资了将近十九个亿。”闫欢微微一笑。
<br/>
<br/>        秦少卿和谷正娟包括闫芈都吃了一惊，他们怎么都没有看出来闫欢是一个有钱人。
<br/>
<br/>        “不是你一个人投资吧？”谷强压抑着心里的震惊。
<br/>
<br/>        “我们的商场是三个人合伙开的，陈子悦、陈子昂和我。”闫欢看着谷强。
<br/>
<br/>        谷强心里一紧！
<br/>
<br/>        “陈子昂？是你口中说的那个陈子昂吗？”谷正娟问谷强。

　　　　山谷内，无数目光汇聚在一条匍匐在地紫金巨蟒身上，在天雷天火如此密集的轮换轰炸下，他还没灰飞烟灭，不得不说是一个天大的奇迹。
<br/>　　换了任何二阶精怪，恐怕就算有十条命，也都死完了。
<br/>　　不过见这条蛇萎靡的样子，看来也仅此而已了，但是区区精怪层次，能接下二十道天雷，二十道天火，已经非常的了不起。
<br/>　　嚓...
<br/>　　第二十一朵劫云花盛开了，傲然怒放，雷音滚滚，看样子准备乘胜追击，直接将这条蛇精给毁灭。
<br/>　　整个山谷光芒一闪，这依然是一道闪电雷，已迅雷不及掩耳之势，携带万钧之力，携带天威浩荡，径直划破长空，留下一道白引，狠狠朝着地面的紫金巨蟒劈下。
<br/>　　“特么的，给我适可而止”
<br/>　　只见匍匐在地的紫金巨蟒仰天咆哮，已是庞然大物的它整个蛇躯猛然暴涨，扶摇直上，威武不能屈，直接变成了五十米巍峨的惊天巨蟒，蛇口一张，朝着闪电雷一口就吞去。
<br/>　　这一吞，仿佛要将整个世间都给吞下去，山谷的天为之一暗。
<br/>　　这道闪电雷无影无踪了，好似从来不曾出现过。
<br/>　　“这...”
<br/>　　天雷被吞了？是这样吗？还有这种操作？
<br/>　　没有人看清楚，这条惊天巨蟒好像是冲着空气咬了一口，但劈下来的闪电雷就消失了。
<br/>　　无数关注这场天劫的生物都呆若木鸡，反应不过来，而身为正主的涂小安一双蛇眸亮的犹如电光。
<br/>　　他成功了，成功将那道闪电雷吞了进去，或许说不上吞，张开蛇口便行。
<br/>　　此时，他的腹中有一股残暴惊人的力量在疯狂乱窜，在破坏，在毁灭，他很痛苦，仿佛有一双拳在他的身体中乱拳乱打，带着一股麻痹感。
<br/>　　整个巍峨身躯忍不住地颤动着，明明好像失去了感知，但又痛的直达灵魂深处。
<br/>　　“给我吸收”
<br/>　　在劳资的身体中，让你天雷在强，也要成为我涂小安的养分。
<br/>　　“叮...恭喜宿主吞噬天雷，成功获取百分之10的天威之能”
<br/>　　什么？
<br/>　　自己吞噬天雷，吸收到了天威？
<br/>　　系统什么都是数据化，这百分之10说明什么，说明涂小安窃取到了一小部分的天威，也仅仅只是一小部分，还没彻底的掌握。
<br/>　　或许多吞几道，有可能彻底掌握天威之能。
<br/>　　原来只有将天雷给吞进去，才有机会得到天威之能。
<br/>　　涂小安顿时狂喜，终于如愿以偿的得到了天威，虽然仅仅只是一小部分，但是他的天劫还远远没有结束，他还有大把的机会。
<br/>　　如果涂小安彻底掌控了天威之能，那么他的元素属性能力释放，都是携带着天威，威力不可同日而语，逼格无限提升。
<br/>　　他释放出来雷电，就是天雷，他释放的火焰，就是天火。
<br/>　　谁要是跟他打，那么就是经历一场噩梦般的天劫。
<br/>　　此时，身体上的皮肉之痛，瞬间得到缓解，富贵险中求，拼了，只要渡过这场天劫，他涂小安不但成为三阶妖灵，超凡脱俗，能是拥有天威浩荡。
<br/>　　轰...
<br/>　　第二十一颗天外陨石晃动了，电光火石间的迅速砸落，一路伴随着激烈的爆炸，如星空坍塌。
<br/>　　“来的好”
<br/>　　五十米的惊天巨蟒仰天咆哮，巨大血盆大口张开，无畏无惧，朝着天外陨石一口吞了下去。
<br/>　　这个画面凶残无比，惊心动魄，简直闻所未闻。
<br/>　　所有生物撑着惊骇的目光真真切切的将这一幕烙印在眼中，这条蛇精将天火给吞下去了？
<br/>　　我的妈呀...
<br/>　　这应该说他脑洞大开，还是拼命作死？
<br/>　　天火焚烧一切，你给吞进去？
<br/>　　之前闪电雷的吞，让众人无法看清楚，但这次吞天火，是看的一清二楚了。
<br/>　　震撼，太震撼了。
<br/>　　光这种勇气都足够所有生物冲着这条蛇精竖起大拇指。
<br/>　　你牛逼！
<br/>　　吞下天火的惊天巨蟒巍峨的蛇躯就地打滚起来，仿佛承受了巨大的痛苦，蛇尾不断的在地面拍打，无数条不知向外延伸了多少里的裂缝席卷。
<br/>　　已惊天巨蟒为中心的地面，疯狂的地震着，地面大肆的裂开，飞沙走石，另外伴随着惨绝人寰的蛇类嘶吼声。
<br/>　　痛...痛死我了。
<br/>　　天火一吞，涂小安的腹中像是被万根灼热的利刀刺着，一股绞心的疼痛遍布他的全身。
<br/>　　一阵又阵的疼痛犹如江海大潮一般的疯狂涌来，一波又一波。
<br/>　　天火在焚烧，焚烧涂小安的五脏六腑，若不是他有火焰之力护体，真的要灰飞烟灭了。
<br/>　　但就算有火焰之力护体，他无比强大的蛇躯也在地面翻过来，翻过去，蛇口发出撕心裂肺的嚎叫。
<br/>　　这种痛比刚才吞下天雷的痛更加的难以忍受，涂小安犹如在地狱油锅中翻滚着，霎时，体内犹如有只魔鬼在作怪，翻江倒海，又仿佛是有一枚核导弹在里面发生了“核爆炸”似的。
<br/>　　世间无任何的痛，能与之相比。
<br/>　　涂小安只能忍，拼命忍，暗暗运转吞噬之力，吸收这道天火的能量。
<br/>　　半晌!
<br/>　　“叮...恭喜宿主吞噬天火，成功获取百分之10的天威之能”
<br/>　　又是百分之10吗。
<br/>　　很显然，涂小安想彻底的掌控天威，那么就要获取百分之百的天威之能，少一个点都不行。
<br/>　　也就是说，涂小安要继续这样去吞噬天雷天火。
<br/>　　可将天雷天火吞下去，是要承受非人的折磨，这种折磨一次就宛如死上了十回。
<br/>　　偷天之力，岂能不付出代价。
<br/>　　只是这种代价太痛苦了，颇有种一朝被蛇咬十年怕井绳。
<br/>　　涂小安沉寂在渡劫之中，完全不知道外界的震惊。
<br/>　　那些关注这场天劫的妖灵跟精怪都要疯了，这还不死，还不死，吞下了天火还没有被焚烧的灰飞烟灭？
<br/>　　仅仅是痛苦嚎叫，四下翻滚，可翻滚翻着，好像又没什么事情了。
<br/>　　妖孽，这才是变态的妖孽。
<br/>　　吞下天火都能没事，看样子还将天火给消化了，你这是什么无敌铁胃，如此恐怖如斯。
<br/>　　麻木了，一个个都有点麻木了，这条蛇精怎样才会死。
<br/>　　之前挑衅天劫就罢了，现在更是变本加厉，连天雷天火都吞，这世间还有什么是这条蛇不敢吃的。

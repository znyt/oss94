　　单单是这些功法武技，都足以让剑道门弟子的实力整体提升一大截。
<br/>　　当然为了防止不劳而获，楚风眠也跟北邙学院之前的规矩一样，指定下来了贡献点制度。
<br/>　　只有交纳足够的贡献点，才可以学习其中的功法武技。
<br/>　　因此，在仪式结束后，不少的弟子都已经是离开，想办法去赚取贡献点了。
<br/>　　整个剑道门内，都是热火朝天，人人向武。
<br/>　　如此长久以来，未来剑道门，可能超越七大宗门，成长成庞然大物一般。
<br/>　　片刻之后，人群散去，在宫殿之中，也只剩下了楚风眠跟寒月漓，洛汐，以及弑剑君，邓灵等人。
<br/>　　弑剑君，邓灵，是楚风眠特意留下的。
<br/>　　“不知门主还有什么吩咐？”
<br/>　　弑剑君看着楚风眠恭敬的说道。
<br/>　　“我打算要在剑道门之中，开辟两个新的堂口，这两个堂口分别是云羽堂，执剑堂。”
<br/>　　楚风眠开口道。
<br/>　　现在剑道门之内，只有两个堂口，还是太少了。
<br/>　　楚风眠建立剑道门，也打算完整的传承曾经剑道门的剑术。
<br/>　　曾经剑道之主建立起的剑道门，虽然已经彻底的被毁灭，但是其中的剑术，都被楚风眠所传承，如今也该继续流传下去了。
<br/>　　然后另外的云羽堂，也是楚风眠曾经答应云老的，将云羽天宫传承下去。
<br/>　　“这两个堂口的堂主之位，我还没有决定，你们有何人选？”
<br/>　　楚风眠看着弑剑君开口道。
<br/>　　他现在手下能用的人，的确不多，除了弑剑君，洛河之外，其他人的实力还太弱了。
<br/>　　担任堂主之位，也会被人不服。
<br/>　　所以楚风眠一开始，只是定下了北邙，寒雪两个堂口。
<br/>　　“云羽堂的话，云老是最好的人选。”
<br/>　　弑剑君想了一下，开口道。
<br/>　　“应该无人比起他对更加了解云羽天宫。”
<br/>　　“云老？”
<br/>　　楚风眠倒是差点忘记了，毕竟云老是器灵，行为受限。
<br/>　　但是在这剑道门之中，云老倒是足以自由活动，传道解惑，倒是没什么问题。
<br/>　　“云老你意下如何？”
<br/>　　楚风眠看着旁边开口道，不一会云老便是现身出来。
<br/>　　“全凭门主安排。”
<br/>　　云老开口的时候，嘴角露出了几分笑意，显然是同意了。
<br/>　　云老是云羽天宫的器灵，见证了云羽天宫的兴衰，一直以来都是想要重建云羽天宫，恢复当年的辉煌。
<br/>　　让他亲手去做，也是他最为欣喜的了。
<br/>　　“那好，云羽堂的堂主，就由云老担任。”
<br/>　　楚风眠点了点头，随即看向了弑剑君。
<br/>　　“那另外的执剑堂呢？”
<br/>　　“执剑堂，就由我来担任堂主吧，至于寒雪堂的堂主之位，交给洛河前辈就是。”
<br/>　　弑剑君看了一眼楚风眠，笑着开口道。
<br/>　　“我也算是半个门主的弟子，担任这执剑堂的堂主之位，也不算数给门主丢脸吧。”
<br/>　　楚风眠听到这话，犹豫了一下，还是点了点头。
<br/>　　“那好，就由你担任执剑堂的堂主，这里，是我得到，记载下来的剑术，你要将他们，发扬光大。”
<br/>　　楚风眠拿出了一枚玉符，递给了弑剑君。
<br/>　　在这里面，都是楚风眠挑选出来的剑术，许多剑术，都是天级剑术中的佼佼者。
<br/>　　弑剑君接过，打开看了一眼，就是脸色大变。
<br/>　　以他的见识，足以知晓这其中剑术的珍贵，现在楚风眠却是全部交给了他。
<br/>　　“等我离开后，剑道门，就由你们负责了。”
<br/>　　楚风眠看着下方的众人，开口道。
<br/>　　“门主你要离开？”
<br/>　　听到这话，弑剑君，邓灵等人都是大惊失色。
<br/>　　“不错，我要离开剑道门一段时间，前去归墟天，寻找宝藏。”
<br/>　　楚风眠开口道。
<br/>　　“在我离开的期间，如果遇到危险，你们就开启护山大阵，闭关不出，这是护山大阵的操控方法。”
<br/>　　楚风眠手心一动，飞出来的几枚玉符，飞向弑剑君等人。
<br/>　　“如果无事，都下去吧。”
<br/>　　交代完这一切后，楚风眠挥了挥手。
<br/>　　他能够做的也都做了，剑道门，不可能一直在楚风眠的庇护之下，楚风眠终究会离开的。
<br/>　　他给剑道门开了一个好头，但是未来剑道门如何，也不是楚风眠一人可以掌控的了。
<br/>　　“是。”
<br/>　　弑剑君等人，纷纷离开。
<br/>　　云羽天宫之中，只剩下了楚风眠跟洛汐，寒月漓三人，当然还有云老。
<br/>　　这云羽天宫是云老的本体，楚风眠在其中的一切交谈，都不可能瞒得过云老的耳朵。
<br/>　　“门主真的打算要去归墟天？”
<br/>　　云老现身，看着楚风眠，有些担忧的开口道。
<br/>　　“现在的归墟天，可不比从前。”
<br/>　　“我知道，现在归墟天内，是什么情况？”
<br/>　　归墟天的危险，楚风眠当然知道。
<br/>　　现在整个九域，都剧变变的截然不同了，处处都是危险，归墟天，可是满地都是宝藏之地。
<br/>　　不止是宝藏，最为吸引人的，还有许多古老的传承，古老的灵器，像是云羽天宫的传承，在归墟天内，只能够算是中游。
<br/>　　楚风眠后面前去归墟天的时候，曾经发现过比起云羽天宫还要古老的遗迹。
<br/>　　单单是那些遗迹之上留下的禁制阵法，就足以阻挡楚风眠，当楚风眠无法进入其中。
<br/>　　这些，才是最足以吸引无数强者前去的，为了争夺宝藏，必然会掀起一场场厮杀。
<br/>　　但楚风眠要得到机缘，再进一步，踏入帝尊境，这归墟天，他是必须要去的。
<br/>　　整个九域，也找不到任何一处地方，可以跟归墟天相比机缘。
<br/>　　“归墟天最近的事情，我不知道，只是在我离开的时候，已经是有强者，闯入了归墟天之中。”
<br/>　　云老想了一下回答道。
<br/>　　“那些强者，应该都不是九域的强者，而是来自域外，对了，我还记得有魔族的存在，有一位魔族之中的强者，也去了归墟天，单单是他一个人，就收取了数十座古老遗迹。”
<br/>　　“我也是运气好一点，才躲过一劫。”
<br/>　　“魔族？魔族强者？”
<br/>　　楚风眠眉头一皱。
<br/>　　他刚刚来到九域的时候，就碰到了魔族了，现在这归墟天内，又有魔族。
<br/>　　如此之多的魔族，居然也都来了九域。
